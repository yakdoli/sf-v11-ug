##### Using Properties Model {#using-properties-model style="tab-stops: 0pt"}

The following steps explain the positioning of the Tab using the *Properties* model.

1.   In the **Controller**, create an instance of **MobTabModel**, set the ContentPosition property with the desired value as an argument and pass the instance through **View Specific Data** to **View** as given below:**

| [        {]{style="FONT-FAMILY: 'Courier New'"}                                                                                                                     |
