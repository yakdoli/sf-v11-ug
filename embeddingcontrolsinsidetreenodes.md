##### Embedding controls inside Tree Nodes[]{style="FONT-SIZE: 9pt"} {#embedding-controls-inside-tree-nodes style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

TreeView allows any custom controls to be conveniently embedded inside it, which is done using the Template feature by placing the controls inside template tags and assigning the template id of the control to the **TemplateID** property of the node inside which the custom control should be embedded.

 

::: {align="center"}
  ------------ -----------------------------------------------------------------------------------------------------------
  Property     Description
  TemplateID   Specifies the id of the template that contains any custom ASP.NET control to be embedded inside the node.
  ------------ -----------------------------------------------------------------------------------------------------------
:::

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"}
