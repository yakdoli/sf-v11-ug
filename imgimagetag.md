### IMG - Image Tag {#img---image-tag style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

The **Image** tag is used to display an image in the HTMLUI control. The HTMLUI control supports the following attributes for the image element, which can used in developing advanced HTMLUI applications.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

[·      ]{style="FONT-FAMILY: Symbol"}**src**: Source attribute is used to access the location of the image file. The image may be located in the local folder or in the URI.

[·      ]{style="FONT-FAMILY: Symbol"}**alt**: Alternate Text tag provides a tooltip test when the mouse is moved over the image. In case the image is not rendered in the control, this text is displayed in place of the image.
