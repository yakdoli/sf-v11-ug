The following are the system requirement for the Essential Studio:

 

::: {align="center"}
  ------------------------- ------------------------------------------
  .NET Framework versions   .NET 2.0 and higher version
  Operating System          All windows client and server
  Free Space                Minimum 3 GB free space in system drive.
  Memory                    Minimum 256 MB in RAM
  ------------------------- ------------------------------------------
