## Panning {#panning style="tab-stops: 0pt"}

Map Panning can be done by dragging the map. in any direction. Panning can be disabled by setting EnablePan property as False.

Panning can be done using Pan() method and by the NavigationControls.

 

**Disable Pan**

It is possible to prevent the Map from panning using Disable Pan. EnablePan is a property which is used to prevent the Map from Pan operation if its value is set as false.
