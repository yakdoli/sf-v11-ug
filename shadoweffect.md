##### Shadow Effect {#shadow-effect style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

Menu allows you to render the structure with shadow effects, in addition to style settings. Shadow effects can be set for the entire menu structure to enhance the look and feel of the menu.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

::: {align="center"}
  ----------------- -------------------------------------------------------------------------
  Menu Property     Description
  ShadowColor       Specifies the color to use for dropdown menu shadow.
  ShadowDirection   Specifies the direction of shadow offset, in degrees.
  ShadowStrength    Specifies distance in pixels to which the shadow filter effect extends.
  ----------------- -------------------------------------------------------------------------
:::

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

The shadow effect can be applied using the **ShadowStrength**, **ShadowDirection** and **ShadowColor** properties.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"}
