#### Setting Visual Styles {#setting-visual-styles style="tab-stops: 0pt"}

The appearance of the TaskBar control is customized by using the **VisualStyle** property. The various built-in visual styles are listed below.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

::: {align="center"}
+-----------------------------------+-----------------------------------------------------------------------------------------+
| Property                          | Description                                                                             |
+-----------------------------------+-----------------------------------------------------------------------------------------+
| VisualStyle                       | Sets the visual style for the TaskBar control. The options provided are as follows.     |
