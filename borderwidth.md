##### Border Width {#border-width style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

[·      ]{style="FONT-FAMILY: Symbol"}**border-width**: Specifies the thickness for the border of the rendered table. The user can specify the border-width in units.

[·      ]{style="FONT-FAMILY: Symbol"}**border-bottom-width**: Specifies the thickness for the bottom border of the rendered table

[·      ]{style="FONT-FAMILY: Symbol"}**border-top-width**: Specifies the thickness for the top border of the rendered table

[·      ]{style="FONT-FAMILY: Symbol"}**border-left-width**: Specifies the thickness for the left border of the rendered table

[·      ]{style="FONT-FAMILY: Symbol"}**border-right-width**: Specifies the thickness for the right border of the rendered table
