### Offset {#offset style="tab-stops: 0pt"}

The **Offset** function returns a reference to a range that is offset a number of rows and columns from any given range or cell.

 

**Syntax:**

**Offset( range, rows, columns, height, width )**

 

**where,**

[·      ]{style="FONT-FAMILY: Symbol"}range is the starting range from which you want to apply the offset.

[·      ]{style="FONT-FAMILY: Symbol"}rows is the number of rows you want to apply as the offset to the range. This can be either a positive or negative number.

[·      ]{style="FONT-FAMILY: Symbol"}columns is the number of columns you want to apply as the offset to the range. This can be either a positive or a negative number.

[·      ]{style="FONT-FAMILY: Symbol"}height is the number of rows that you want the returned range to be.

[·      ]{style="FONT-FAMILY: Symbol"}width is the number of columns that you want the returned range to be.
