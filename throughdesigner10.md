##### Through Designer {#through-designer style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

Add CallbackPanel control to a page and then add controls and static text to the control.

The CallbackPanel control can be created as follows.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

1.   Create a new ASP.NET Web application. For details, see [Creating ASP.NET Web Application]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: windowtext; FONT-SIZE: 9pt; TEXT-DECORATION: none; text-underline: none"}.

12.  Drag the CallbackPanel control onto the Web Form. Refer to the [Concepts and Features]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: windowtext; FONT-SIZE: 9pt; TEXT-DECORATION: none; text-underline: none"} section to customize the various settings of the control.

13.  Add controls to the CallbackPanel. (For details, see [How to add controls into the panel]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: windowtext; FONT-SIZE: 9pt; TEXT-DECORATION: none; text-underline: none"}.)

14.  In Java Script, in response to some event, say button click, for example, call \'callback\' method with some arguments on this control\'s client side object.

15.  Listen to the CallbackPanel\'s CallbackRefresh server side event for updating and refreshing its child controls.
