#### Collapsing the Accordion Contents {#collapsing-the-accordion-contents style="tab-stops: 0pt"}

Accordion provides support to collapse all the accordion contents at the same time.

 

Properties

+-------------+-----------------------------------------------------------------+----------------------+------------------+-------------+
| Name        | Description                                                     | Type of the property | Value it accepts | Dependency  |
+-------------+-----------------------------------------------------------------+----------------------+------------------+-------------+
| Collapsible | When set, all the accordion contents can be collapsed at a time | bool                 | true/false       | NA          |
|             |                                                                 |                      |                  |             |
|             |                                                                 |                      |                  |             |
+-------------+-----------------------------------------------------------------+----------------------+------------------+-------------+

 

Using Builder

The following steps explain the method to enable collapsing the accordion contents through Builder.

1.   In **View**, create the contents of the accordion with header and div tags and invoke the accordion helper with control ID as the first argument, followed by the **Collapsible** method with the argument set to 'false'.
