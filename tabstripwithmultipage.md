##### TabStrip with MultiPage {#tabstrip-with-multipage style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

The Tabstrip control can be enhanced by integrating it with MultiPage control.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

::: {align="center"}
  ------------- --------------------------------------------------------------------------------------
  Property      Description
  MultiPageID   Specifies id of the multipage instance to be integrated in tabstrip.
  PageViewID    Specifies the id of the specific page to be integrated with the  corresponding item.
  ------------- --------------------------------------------------------------------------------------
:::

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

1.   Open an ASP.NET web application.

109.   Set the ID of the Multipage instance to the **MultiPageId** property in the tabstrip control. If you do so, the first tab will be connected to the first page, the second tab will be connected to the second page and so on. So, when the user selects the first tab, the first **PageView** instance will be displayed, and so on. Also if you want to set any page to any particular tab element, it could be done using **PageViewID** property of the tab element. Here is the sample code.
