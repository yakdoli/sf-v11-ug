#### Adding animations to the Dialog {#adding-animations-to-the-dialog style="tab-stops: 0pt"}

The Dialog supports a number of in-built animations.

Properties

+------+------------------------------------------------+------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+------------+
| Name | Description                                    | Type of property | Value it accepts                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          | Dependency |
+------+------------------------------------------------+------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+------------+
| Show | Sets the animation for the dialog open action  | enum             | [DialogAnimations]{style="COLOR: #2b91af"}.bounce, [DialogAnimations]{style="COLOR: #2b91af"}.blind, [DialogAnimations]{style="COLOR: #2b91af"}.clip, [DialogAnimations]{style="COLOR: #2b91af"}.drop, [DialogAnimations]{style="COLOR: #2b91af"}.explode, [DialogAnimations]{style="COLOR: #2b91af"}.fold, [DialogAnimations]{style="COLOR: #2b91af"}.highlight, [DialogAnimations]{style="COLOR: #2b91af"}.puff, [DialogAnimations]{style="COLOR: #2b91af"}.pulsate, [DialogAnimations]{style="COLOR: #2b91af"}.shake, [DialogAnimations]{style="COLOR: #2b91af"}.slide | NA         |
+------+------------------------------------------------+------------------+                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           +------------+
| Hide | Sets the animation for the dialog close action | enum             |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           | NA         |
+------+------------------------------------------------+------------------+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+------------+

Using Properties Model

The following steps explain the addition of animations to the Dialog using the Properties model.

1.   In the **Controller**, create an instance of **DialogModel**, define the **Show** and **Hide** and pass the instance through **view specific data** to **View** as given below.**
