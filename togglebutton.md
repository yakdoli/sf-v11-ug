### ToggleButton {#togglebutton style="tab-stops: 0pt"}

The ToggleButton control in Essential Tools for Mobile MVC  allows the user to enable or disable particular actions. The toggle button allows the user to have both text and images on the button.

The key features of the ToggleButton control are:

[·      ]{style="FONT-FAMILY: Symbol"}Support for icons

[·      ]{style="FONT-FAMILY: Symbol"}Four built-in mobile skins
