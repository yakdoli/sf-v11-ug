#### Client Side Methods {#client-side-methods style="tab-stops: 0pt"}

 

Rich text editor supports rich set of client side methods to control its behavior.

+----------------------------------------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+-----------------------------+---------------------------------------------------------------+-----------------------------+
