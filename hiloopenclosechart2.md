#### HiLo Open Close Chart[]{#p51} {#hilo-open-close-chart style="tab-stops: 0pt"}

 

HiLo Open Close chart is a special kind of chart that is normally used in stock analysis. This chart expects four Y values for every point in the series. These values should represent the High, Low, Open and Close values of the stock, in that order, for that period.

 

The following code example illustrates how to create a HiLo Open Close Chart.
