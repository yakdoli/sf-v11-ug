##### Adding Controls into the Panel {#adding-controls-into-the-panel style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

You can add any html or web controls into CallbackPanel control through designer or code.

 

You can drag from the Toolbox, any html or web controls to CallbackPanel at design time and also edit these controls. If necessary, the panel will resize itself to accommodate the control.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

![](ImagesExt/image72_244.jpg){border="0"}

Figure 163

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

You can create any controls programmatically and add them to the Controls collection of existing CallbackPanel control. Also, you can use **Tag** property of CallbackPanel to attach any object to this control\'s view state.

 

To add controls into CallbackPanel on page callback see the following code example.
