##### Visibility of Popup Parts {#visibility-of-popup-parts style="tab-stops: 0pt"}

[·      ]{style="FONT-FAMILY: Symbol"}Calendar Button visibility can be enabled by setting the IsCalendarEnabled property to true.

[·      ]{style="FONT-FAMILY: Symbol"}Clock Button visibility can be enabled by setting the IsWatchEnabled property to true.

[·      ]{style="FONT-FAMILY: Symbol"}None Button visibility can be enabled by setting the IsEmptyDateEnabled property to true.
