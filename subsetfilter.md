### Subset Filter {#subset-filter style="tab-stops: 0pt"}

Subset Filter is a control used to filter the number of records that have to appear in a row and the number of fields that have to appear in the column. This control will popup when we press the **subset filter** button placed on top of the column axis and row axis element builder.
