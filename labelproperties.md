##### Label Properties {#label-properties style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

The following table lists the properties of the Label with their descriptions.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

::: {align="center"}
  ------------ -------------------------------------------
  Property     Description
  Text         Specifies the label content.
  Foreground   Specifies the foreground of the label.
  Location     Specifies the location of the label.
  Background   Specifies the background color for label.
  ------------ -------------------------------------------
:::
