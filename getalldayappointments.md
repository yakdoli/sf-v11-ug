#### GetAllDayAppointments

GetAllDayAppointments function is used to get the AllDayAppointments from the schedule. It takes only one argument called the ResourceID. It will return the entire AllDayAppointements  collection as an object.

The following codes will display the entire AllDayAppointments subject through an alert box in the schedule.[]{#_GetAppointmentsDataInternal}
