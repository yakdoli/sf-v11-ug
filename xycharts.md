### XY Charts {#xy-charts style="tab-stops: 0pt"}

 

XY charts are used to visualize the relationship between two variables that relate to the same event. XY charts are used to graph paired data. X is usually assigned to an independent variable, the variable for which the value is controlled. The Y variable is then the depended variable, the values that follow from each value of the X set.

The types of XY charts are as follows:
