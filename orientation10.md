#### Orientation {#orientation style="tab-stops: 0pt"}

TabSplitter Items are placed horizontally or vertically by using the **Orientation** property. This is a dependency property which sets the orientation of the TabSplitter Item. It supports the following types of orientation.

 

[·      ]{style="FONT-FAMILY: Symbol"}**Horizontal**: TabSplitter Item in the TabSplitter is placed horizontally

[·      ]{style="FONT-FAMILY: Symbol"}**Vertical**: TabSplitter Item in the TabSplitter is placed vertically
