**Using AJAX to Load Grid**

Follow the below steps while using AJAX to load grid:

1.   While using Ajax post scripts will not be evaluated.  To overcome this use \$.fn.sfEvalScripts(data) function.

The following code illustrates this:

+--------------------------------------------------------------------------------------------------------------------------------+
| **[\[Script\]\                                                                                                                 |
| \                                                                                                                              |
| \                                                                                                                              |

| [type: [\'post\']{style="COLOR: maroon"},]{style="FONT-FAMILY: 'Courier New'"}                                                 |
|                                                                                                                                |
| [url: [\"Home/JQueryAjaxAction\"]{style="COLOR: maroon"},]{style="FONT-FAMILY: 'Courier New'"}                                 |
|                                                                                                                                |
| [datatype: [\'html\']{style="COLOR: maroon"},]{style="FONT-FAMILY: 'Courier New'"}                                             |
|                                                                                                                                |
| [success: [function]{style="COLOR: blue"} (data) {]{style="FONT-FAMILY: 'Courier New'"}                                        |
|                                                                                                                                |
