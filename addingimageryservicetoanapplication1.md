**About Bing Maps Key:**

Bing Maps Key is important to enable the Imagery Service. With a Bing Maps Key, users will be validated by Microsoft. Invalid keys will not be accepted. For more information about obtaining a Bing Maps Key and using Bing Maps Services, please visit

<http://www.microsoft.com/maps/product/licensing.aspx>

**Zooming and Panning**

Zoom level of the Imagery Service enabled map will be increased by 1. The map will be zoomed based on the zoom level. If the zoom level is increased, the map will be drilled in (zoomed in). If the zoom level is decreased, the map will be drilled out (zoomed out). Zoom can be done through the mouse scroll and the Navigation control of the maps.

Panning is used to navigate through the map. Panning can be done by click-and-drag on the map or through the Navigation control.

For information about navigation control please refer to Chapter 11.
