##### Background Repeat {#background-repeat style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

The **background-repeat** property repeats the background that is set for the HTML element, horizontally or vertically.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

[·      ]{style="FONT-FAMILY: Symbol"}**repeat-x**: Repeats the background image horizontally

[·      ]{style="FONT-FAMILY: Symbol"}**repeat-y**: Repeats the background image vertically

[·      ]{style="FONT-FAMILY: Symbol"}**repeat**: Repeats the background image both horizontally and vertically

[·      ]{style="FONT-FAMILY: Symbol"}**no-repeat**: Fixes the background image at the top-left corner of the element and does not repeat the image display

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"}
