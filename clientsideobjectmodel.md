##### Client-Side Object Model {#client-side-object-model style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

The client-side methods can be used to control the behavior of the AutoCompleteTextBox control that allows users to interact with it. Using the **ShowDropDown** client-side method, we can view all the autocomplete options of the AutoCompleteTextBox. The client-side method supported by the AutoCompleteTextBox is given below.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

::: {align="center"}
+-----------------------------------+-------------------------------------------------------------------+
|                                   |                                                                   |
|                                   |                                                                   |
| Client-Side Method                | Description                                                       |
+-----------------------------------+-------------------------------------------------------------------+
| ShowDropDown                      | Displays all the autocomplete options of the AutoCompleteTextBox. |
+-----------------------------------+-------------------------------------------------------------------+
:::
