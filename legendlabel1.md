#### Legend Label {#legend-label style="tab-stops: 0pt"}

[]{style="COLOR: #15428b"} 

Legend contains the legend items, which are nothing but the series representation. You can set unique labels for those series using the LegendLabel property of the particular series, so that it can be easily identified by the end user.
