## Switch Silverlight Version {#switch-silverlight-version style="tab-stops: 0pt"}

[\
]{style="COLOR: #001554"}Silverlight version switcher utility is used to switch to the respective version in that machine. If both Silverlight4 and Silverlight5 versions are installed in the machine, this utility is used to switch to the particular version.

Switch to Silverlight4

Run the **SwitchSilverlight4.bat** file ***(\<Install Location**[\>]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; FONT-SIZE: 9pt"}\\Syncfusion\\Essential Studio\\x.x.x.x\\Utilities\\SilverlightVersionSwitcher)***, it will move Silverlight4 assemblies to the **precompiled assemblies** folder and configures the toolbox to Silverlight4.

 

Switch to Silverlight5

Run the **SwitchSilverlight5.bat** file ***(\<Install Location**[\>]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; FONT-SIZE: 9pt"}\\Syncfusion\\Essential Studio\\x.x.x.x\\Utilities\\SilverlightVersionSwitcher)***, it will move Silverlight5 assemblies to the **precompiled assemblies** folder and configures the toolbox to Silverlight5.
