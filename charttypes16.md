### ChartTypes {#charttypes style="tab-stops: 0pt"}

Essential Chart includes a comprehensive set of 8 basic Chart types for all your business needs. Each one is highly and easily configurable with built-in support for creating stunning visual effects.

Chart types are specified on each Series by using the Type property. All the chart types are required to have at least one X and one Y value. Certain chart types need more than one Y value.
