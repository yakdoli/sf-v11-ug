## Tooltip {#tooltip style="tab-stops: 0pt"}

The tooltip is a common graphical user interface element. This enables you to display information about the UI operation. In windows application tooltip will be displayed on mouse hover. Mobile application does not have mouse hover event. To overcome this Essential Tools for Windows Phone provides tooltip with relative element positioning property to set tooltip for an element. Opening a tooltip can be achieved by any possible ways.

[  ]{style="COLOR: #c00000"}

IT Scenarios

This control enables you to display information about the UI elements for better understanding. You can also display tooltip for an operation.  

 

Feature Summary

Tooltip control supports the following feature:

 

[·      ]{style="FONT-FAMILY: Symbol"}Relative Target

[·      ]{style="FONT-FAMILY: Symbol"}Relative Placement

[·      ]{style="FONT-FAMILY: Symbol"}Tooltip Shapes

[o  ]{style="FONT-FAMILY: 'Courier New'"}Corner Radius

[·      ]{style="FONT-FAMILY: Symbol"}Closing modes
