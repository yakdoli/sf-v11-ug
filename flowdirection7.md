#### Flow Direction {#flow-direction style="tab-stops: 0pt"}

 

Flow Direction of the Gallery control is set using the **FlowDirection** property.

 

::: {align="center"}
+-----------------------------------+---------------------------------------------------------------------------------------+
| Property                          | Description                                                                           |
+-----------------------------------+---------------------------------------------------------------------------------------+
| FlowDirection                     | Sets the flow direction for the Gallery control. The options provided are as follows. |
|                                   |                                                                                       |
|                                   | []{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"}  |
|                                   |                                                                                       |
|                                   | [·      ]{style="FONT-FAMILY: Symbol"}LeftToRight                                     |
|                                   |                                                                                       |
|                                   | [·      ]{style="FONT-FAMILY: Symbol"}RightToLeft                                     |
+-----------------------------------+---------------------------------------------------------------------------------------+
:::
