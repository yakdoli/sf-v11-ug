#### Open on click {#open-on-click style="tab-stops: 0pt"}

Menu supports triggering the menu open and close actions on click events. By default, these actions will be triggered on mouse over events.

Properties

 

  ------------- ------------------------------------------------------------------------------- ------------------------------------------------------------------------------------------------ ------------------------------------------------------------------------------------------------------ ------------
  Name          Description                                                                     Type of property                                                                                 Value it accepts                                                                                       Dependency
  OpenOnClick   When set to true, the menu open and close actions take place on click events.   [[bool]{style="COLOR: windowtext; TEXT-DECORATION: none; text-underline: none"}]{.UGHyperlink}   [[true/false]{style="COLOR: windowtext; TEXT-DECORATION: none; text-underline: none"}]{.UGHyperlink}   NA
  ------------- ------------------------------------------------------------------------------- ------------------------------------------------------------------------------------------------ ------------------------------------------------------------------------------------------------------ ------------

 

Using Builder

The following steps guide you in enabling the menu open and close actions on click event, through builder:

1.   In **View**, create ul-li hierarchy of menu items and invoke the menu helper with the menu content ID as the first argument and enable the **OpenOnClick** method with the desired option as argument.
