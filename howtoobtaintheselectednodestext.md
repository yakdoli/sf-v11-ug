##### How to obtain the selected node\'s text? {#how-to-obtain-the-selected-nodes-text style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

The selected node\'s text can be obtained by triggering the **NodeSelected** event for which the **AutoPostBackOnNodeSelect** must be set to **True**.
