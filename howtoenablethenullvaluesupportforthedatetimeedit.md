#### How to enable the Null Value support for the DateTimeEdit {#how-to-enable-the-null-value-support-for-the-datetimeedit style="tab-stops: 0pt"}

NullValue support in the DateTimeEdit control can be enabled by setting the IsEmptyDateEnabled property to true. You can also specify the Value when you set the null to the DateTime property by using the NullValue property.  You can set the WatermarkText when the Null Value is set to the DateTime property.
