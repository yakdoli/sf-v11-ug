#### Changing the flow directions {#changing-the-flow-directions style="tab-stops: 0pt"}

The flow direction for the TaskBar is set through the **FlowDirection** property.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

::: {align="center"}
+-----------------------------------+---------------------------------------------------------------------------------------+
| Property                          | Description                                                                           |
+-----------------------------------+---------------------------------------------------------------------------------------+
| FlowDirection                     | Sets the flow direction for the TaskBar. The options provided are as follows.         |
|                                   |                                                                                       |
|                                   | []{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"}  |
|                                   |                                                                                       |
|                                   | [·      ]{style="FONT-FAMILY: Symbol"}LeftToRight                                     |
|                                   |                                                                                       |
|                                   | [·      ]{style="FONT-FAMILY: Symbol"}RightToLeft                                     |
+-----------------------------------+---------------------------------------------------------------------------------------+
:::
