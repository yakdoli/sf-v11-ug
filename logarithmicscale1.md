##### Logarithmic Scale {#logarithmic-scale style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

Essential Circular gauge for Windows Phone supports logarithmic label ticks. This can be achieved by setting the **IsLogarithmic** property to True, and specifying a valid LogBase value. The value of the **LabelTickSet** is calculated based upon the base provided in the Logbase.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

The following code snippet illustrates on how to set the logarithmic scale for a Circular gauge.
