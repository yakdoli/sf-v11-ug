##### Server- side Events {#server--side-events style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

The server-side events can be handled to execute a function on some user action. The server-side event of GenericDropDown control has been given below.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

::: {align="center"}
  ------------------- ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
  Server-Side Event   Description
  TextChanged         Specifies the function to be executed when the text in the input area of the GenericDropDown control changes. This event is handled only when the AutoPostBackOnTextChanged property is set to True.
  ------------------- ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
:::
