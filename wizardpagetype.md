##### Wizard Page Type {#wizard-page-type style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

You can set the type of the wizard page by using the **PageType** property. There are three types of wizard pages.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

[·      ]{style="FONT-FAMILY: Symbol"}Blank

[·      ]{style="FONT-FAMILY: Symbol"}Interior

[·      ]{style="FONT-FAMILY: Symbol"}Exterior

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"}
