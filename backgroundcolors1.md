### Background Colors {#background-colors style="LINE-HEIGHT: 150%; tab-stops: 0pt"}

Essential Schedule supports customizing the background colors. The following properties are used to change the colors.

::: {align="center"}
  ----------------------- ------- ------------------------------------------------------------------------------------
  Property                Type    Description
  SelectionBackground     Brush   It can be used to change the Selection Background Color of the Schedule Control.
  ShadedBackground        Brush   It can be used to change the Shaded Background Color of the Schedule Control.
  ScheduleBackground      Brush   It can be used to change the Schedule Background Color of the Schedule Control.
  HeaderBrush             Brush   It can be used to change the Header Background Color of the Schedule Control.
  StrokeLine              Brush   It can be used to change the Border Color of the Schedule Control.
  AppointmentBackground   Brush   It can be used to change the Appointment Background Color of the Schedule Control.
  ----------------------- ------- ------------------------------------------------------------------------------------
