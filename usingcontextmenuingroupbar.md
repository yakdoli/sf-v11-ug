#### Using Context menu in GroupBar {#using-context-menu-in-groupbar style="tab-stops: 0pt"}

 

To enable the context menu support in the GroupBar, set **IsEnabledContextMenu** property to ***true***. This dependency property sets the value that indicates whether the context menu is enabled.

 

Here are the benefits of using context menu in GroupBar control.

 

[·      ]{style="FONT-FAMILY: Symbol"}Cut, copy and paste the content of GroupBar

[·      ]{style="FONT-FAMILY: Symbol"}Sort the content of groupbar

[·      ]{style="FONT-FAMILY: Symbol"}Add the GroupBarItem

[·      ]{style="FONT-FAMILY: Symbol"}Rename the GroupBarItem

[·      ]{style="FONT-FAMILY: Symbol"}Delete the GroupBarItem

[·      ]{style="FONT-FAMILY: Symbol"}Move the content of GroupBar either up or down

[·      ]{style="FONT-FAMILY: Symbol"}Add the content of GroupBar item.

[·      ]{style="FONT-FAMILY: Symbol"}Rename the content of GroupBar item.

[·      ]{style="FONT-FAMILY: Symbol"}Delete the content of GroupBar item.
