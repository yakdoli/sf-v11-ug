## Switch MVC Version {#switch-mvc-version style="tab-stops: 0pt"}

In order to run the samples, MVC Version Switcher Utility is used to switch to respective MVC version installed in the machine.

MVC3 to MVC2:

1**.  ** Run the **SwitchMVC2.bat** file ***(\<Install Location**[\>]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; FONT-SIZE: 9pt"}\\Syncfusion\\Essential Studio\\x.x.x.x\\Utilities\\MVCVersionSwitcher)***, it will move MVC2 assemblies to the **precompiled assemblies** folder and configures the sample browser to MVC2.

2**.  ** Run the **Assembly Manager** from the dashboard manually to place the MVC2 assemblies in GAC.

 

MVC2 to MVC3:

1**.  ** Run the **SwitchMVC3.bat** file ***(\<Install Location**[\>]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; FONT-SIZE: 9pt"}\\Syncfusion\\Essential Studio\\x.x.x.x\\Utilities\\MVCVersionSwitcher)***, it will move MVC3 assemblies to the **precompiled assemblies** folder and configures the sample browser to MVC3.

2**.  ** Run the **Assembly Manager** from the dashboard manually to place the MVC3 assemblies in GAC.
