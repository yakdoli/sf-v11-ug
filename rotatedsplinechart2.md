#### Rotated Spline Chart {#rotated-spline-chart style="tab-stops: 0pt"}

[]{#p23} 

[]{style="COLOR: #15428b"} 

A Rotated Spline chart is similar to a Spline chart. It plots one or several series of data and joins each series by smooth, rotated spline curves instead of straight lines.

 

The following code example illustrates how to create a Rotated Spline Chart.
