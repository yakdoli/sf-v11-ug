#### Change the alignment of the text in the UpDown control {#change-the-alignment-of-the-text-in-the-updown-control style="tab-stops: 0pt"}

You can change the alignment of the text in the UpDown control by using the TextAlignment property. The following code snippets can be used to set TextAlignment for the UpDown control.
