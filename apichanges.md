## API Changes {#api-changes style="tab-stops: 0pt"}

[]{#p12} 

[]{style="COLOR: #15428b"} 

Essential Chart Silverlight has naming conversions similar to Essential Chart WPF. This naming conversion helps the Chart WPF users to easily upgrade to Chart Silverlight.

 

The Chart Silverlight namespace has been renamed as in WPF Chart.  The following the namespace has changed in Chart Silverlight.

::: {align="center"}
  ------------------------------ --------------------------
  Namespace (Old Release)        Namespace (New Release)
  Syncfusion.Silverlight.Chart   Syncfusion.Windows.Chart
  ------------------------------ --------------------------
:::
