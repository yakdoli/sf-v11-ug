#### Adding controls into Splitter {#adding-controls-into-splitter style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

You can add any html or web controls into SplitPane or SlidingPane controls through designer or code.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

You can drag any html or web controls from toolbox, to the SplitPane or SlidingPane at design time and also edit these controls.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

![](ImagesExt/image72_509.jpg){border="0"}
