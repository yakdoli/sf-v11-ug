##### Adding a Pointer Cap to the Circular Gauge {#adding-a-pointer-cap-to-the-circular-gauge style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

The **Pointer Cap** can be endlessly customized using the various in-built properties. Some of the major properties are listed below.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

[·      ]{style="FONT-FAMILY: Symbol"}**BackgroundBrush** - Sets the Background Brush for the pointer cap

[·      ]{style="FONT-FAMILY: Symbol"}**BorderBrush** - Sets the Border Brush for the pointer cap

[·      ]{style="FONT-FAMILY: Symbol"}**BorderWidth** - Sets the Border width of the pointer cap

[·      ]{style="FONT-FAMILY: Symbol"}**PointerCapRadius** - Specifies the radius for the pointer cap
