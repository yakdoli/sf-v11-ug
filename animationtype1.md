##### Animation Type {#animation-type style="tab-stops: 0pt"}

The type of animation that is generated while expanding or collapsing the TreeViewAdv is controlled by using the **AnimationType** property. This property includes the following options.

 

[·      ]{style="FONT-FAMILY: Symbol"}**Fade**---animation fades out when the TreeViewAdv is collapsed or expanded

[·      ]{style="FONT-FAMILY: Symbol"}**None**---no animation is generated

[·      ]{style="FONT-FAMILY: Symbol"}**Slide**---animation slides when the TreeViewAdv is collapsed or expanded

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

The default value of this property is set to ***Slide***.

 

Use the following code to set this property.
