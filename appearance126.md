##### [Appearance]{#Appearance} {#appearance style="tab-stops: 0pt"}

The appearance of the OlapPager can be customized using the **CssClass** property. The custom CssClass style should have the following CSS styles associated with it:

[·      ]{style="FONT-FAMILY: Symbol"}MovePrev---Defines styles for the **Move Previous** button

[·      ]{style="FONT-FAMILY: Symbol"}MoveNext---Defines styles for the **Move Next** button

[·      ]{style="FONT-FAMILY: Symbol"}MovePrevDisabled---Defines styles for the **Move Previous** button's disabled state

[·      ]{style="FONT-FAMILY: Symbol"}MoveNextDisabled---Defines styles for the **Move Next** button's disabled state

[·      ]{style="FONT-FAMILY: Symbol"}PagerPanel---Defines styles for the Pager panel

[·      ]{style="FONT-FAMILY: Symbol"}PageBox---Defines styles for the **CurrentPage** text box

[·      ]{style="FONT-FAMILY: Symbol"}Text---Defines styles for spans used in the pager
