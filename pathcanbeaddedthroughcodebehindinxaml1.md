#### Path can be added through Code Behind in XAML {#path-can-be-added-through-code-behind-in-xaml style="tab-stops: 0pt"}

When the parameters are provided in code and when you run the sample, the path will be displayed.

We can add the PathLabel in OnPoint and OnMiddlePoint.

OnPoint---The Pathlabel displayed in a Label Point mentioned.

OnMiddlePoint---The PathLabel displayed in a middle Point of the Path.
