## Linear Gauge {#linear-gauge style="tab-stops: 0pt"}

Linear gauge measures the values of scales and presents it in Horizontal or Vertical sliding or meter. Linear gauge provides more powerful functionalities such as State Indicator, Orientation, different types of scale and pointer with label ticks. Apart from this, complete customization options are available using style and template properties of Linear Gauge class. It supports different frame types, such as bolted rectangle, cropped rectangle, rectangle and rounded rectangle with inner gradient.
