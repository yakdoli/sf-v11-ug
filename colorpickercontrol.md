## Color Picker Control {#color-picker-control style="tab-stops: 0pt"}

 

The Color Picker (Palette) control provides a rich visual interface for color selection. The Color Picker for Windows Phone 7 provides PhoneAccentColors and their variations and a picker to choose required color. This provides you a wide range of color options.

 

It Scenarios

Using this control you can choose required color for doodle application or customize the background, foreground and border color at run time.
