### Merge cells {#merge-cells style="tab-stops: 0pt"}

Spreadsheet control provides support to merge two or more cells. When a group of cells is merged, the contents of the upper-left cell will be taken as the content of the merged cell, and the rest will be deleted.

 

You can merge/unmerge the group of cell using the *MergeCommand*. To merge the cell, pass the command parameter as *MergeAndCenter*. To unmerge the cells pass the command parameter as *UnMerge.*
