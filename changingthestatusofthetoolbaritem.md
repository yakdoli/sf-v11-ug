##### Changing the status of the Toolbar Item {#changing-the-status-of-the-toolbar-item style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

Changing item status

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

An item can be disabled or enabled using the **SetDisabled** method. Depending on the parameter values that are passed the status of the item will be changed accordingly.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

::: {align="center"}
+-----------------------+-----------------------+------------------------------------------------------------------------------------+
|                       |                       |                                                                                    |
|                       |                       |                                                                                    |
| Method                | Parameter             | Description                                                                        |
+-----------------------+-----------------------+------------------------------------------------------------------------------------+
| SetDisabled           | obj, bool             | Sets enable / disable item.                                                        |
|                       |                       |                                                                                    |
|                       |                       | First parameter is identifier of HTML-element, index of item array or item object. |
|                       |                       |                                                                                    |
|                       |                       | If second parameter is false then item will be disabled.                           |
+-----------------------+-----------------------+------------------------------------------------------------------------------------+
:::

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

Here, when an item is selected, the SetDiabled method will be called and the item\'s html id is passed along with the boolean value as true which will disable the item.
