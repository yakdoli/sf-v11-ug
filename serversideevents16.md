##### Server-Side Events {#server-side-events style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

The server side events can be triggered to execute a function on some user action. The various server side events are as follows.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

::: {align="center"}
  ------------------- --------------------------------------------------------------------------------------------------------------------------------
  Server-Side Event   Description
  ItemCollapse        Specifies the function to trigger when a parent item is collapsed.
  ItemCommand         Specifies the function to trigger, either when the selection of an item is canceled, or before navigating to the item\'s url .
  ItemExpand          Specifies the function to trigger when a parent item is expanded.
  ItemSelect          Specifies the function to trigger when an item is selected.
  ------------------- --------------------------------------------------------------------------------------------------------------------------------
:::
