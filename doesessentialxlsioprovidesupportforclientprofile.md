### Does Essential XlsIO provide support for Client profile? {#does-essential-xlsio-provide-support-for-client-profile style="tab-stops: 0pt"}

Yes, Essential XlsIO provides support for Client profile. In order to use Essential XlsIO in an application (which targeted to Client profile), the user should include the following assemblies

[·      ]{style="FONT-FAMILY: Symbol"}Syncfusion.Core.dll

[·      ]{style="FONT-FAMILY: Symbol"}Syncfusion.Compresssion.Base.dll

[·      ]{style="FONT-FAMILY: Symbol"}Syncfusion.XlsIO.ClientProfile.dll
