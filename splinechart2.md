#### Spline Chart {#spline-chart style="tab-stops: 0pt"}

[]{#p22} 

[]{style="COLOR: #15428b"} 

The Spline chart is similar to the Line chart and includes all the features supported by the Line chart. The only difference is that the chart points are joined with smooth curved lines.

 

The following code example illustrates how to create an Spline Chart.
