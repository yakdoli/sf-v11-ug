#### Layout Type {#layout-type style="tab-stops: 0pt"}

 

You can set the layout type for the TabControlExt control by using the TabItemLayoutType property. This dependency property sets the layout type of the Tab Item. It provides the following layout types.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

[·      ]{style="FONT-FAMILY: Symbol"}**SingleLine**--tab items are displayed in single line

[·      ]{style="FONT-FAMILY: Symbol"}**MultiLine**--tab items are shrunk and displayed in multiple lines

[·      ]{style="FONT-FAMILY: Symbol"}**MultiLineWithFillWidth**--tab items are displayed in multiple lines without being shrunk
