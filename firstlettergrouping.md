##### First Letter Grouping          {#first-letter-grouping style="tab-stops: 0pt"}

 

You can group the items using the *GroupProvider* property. To group the items based on the first letter of the item, use the *FirstLetterGroupProvider*[ ]{style="COLOR: blue"}instance created for the *GroupProvider*. The *PropertyValue* is used to specify the field name to which the first letter grouping has to be performed.
