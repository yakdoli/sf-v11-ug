#### Enable or Disable Zooming {#enable-or-disable-zooming style="tab-stops: 0pt"}

Zoom can be enabled or disabled by the EnableZoom property. If enable property is set as true then the Map can be zoomed. If EnableZoom property value is false the Map cannot be zoomed.
