#### Column Chart {#column-chart style="tab-stops: 0pt"}

[]{#p32}[]{style="COLOR: #15428b"} 

Column Charts are among the most common chart types that are being used. It uses vertical bars (called columns) to display different values of one or more items. It is similar to a bar chart except for the bars are vertical and not horizontal. Points from adjacent series are drawn as bars next to each other.

It is used for comparing the frequency, count, total or average of data in different categories. It is ideal for showing the variations in the value of an item over time.
