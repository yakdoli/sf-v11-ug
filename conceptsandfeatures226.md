###### 4.1.7.1.2.1 Enable Calendar Zooming Navigation Mode {#enable-calendar-zooming-navigation-mode style="tab-stops: 0pt"}

The schedule **CalendarNavigationMode** property's default value is **Default**. If we changed the property value to **Windows7** then the calendar can navigate to the month, year, and 10-year views when clicking on the header.
