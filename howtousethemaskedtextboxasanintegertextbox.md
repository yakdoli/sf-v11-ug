#### How to use the MaskedTextBox as an IntegerTextBox {#how-to-use-the-maskedtextbox-as-an-integertextbox style="tab-stops: 0pt"}

 

You can use the MaskedTextBox as an IntegerTextBox by using the Mask property. "0" is the Masking element that represents the digit. You can customize the number of digits by using the Mask property.
