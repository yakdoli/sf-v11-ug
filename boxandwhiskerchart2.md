#### Box and Whisker Chart {#box-and-whisker-chart style="tab-stops: 0pt"}

[]{#p56}[]{style="COLOR: #15428b"} 

John Tukey published an efficient method for displaying a five-number data summary. The graph is called a Box and Whisker plot (also known as BoxPlot) and summarizes the following statistical measures.

[]{style="COLOR: #15428b"} 

[·      ]{style="FONT-FAMILY: Symbol"}Median

[·      ]{style="FONT-FAMILY: Symbol"}Upper and Lower Quartiles (75 percentile to 25 percentile)

[·      ]{style="FONT-FAMILY: Symbol"}Minimum and Maximum Data Values

[]{style="COLOR: #15428b"} 

Essential Chart Silverlight supports the Box and Whisker Chart Outlier and Extreme Points Calculation features. You can enable or disable the outlier feature by using the DefaultOutlierVisible attached property of ChartBoxandWhiskerType class. The Box and Whisker chart requires minimum 5 Y point values.

 

The following code example illustrates how to create a Box and Whisker Chart.
