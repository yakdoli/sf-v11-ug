### How to Add a Sort Icon (Up / Down Arrow) in a GridControl\'s Column Header {#how-to-add-a-sort-icon-up-down-arrow-in-a-gridcontrols-column-header style="tab-stops: 0pt"}

 

**Introduction**

 

Add the **GridSortColumnHeaderCellModel** to the GridControl\'s **CellModels** collection to include the **SortColumn** HeaderCell. Then assign this as the [CellType]{style="COLOR: black"} and set the **tag** property to either ListSortDirection.Ascending or ListSortDirection.Descending to show the up / down arrow mark.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"}
