### Column Formatting {#column-formatting style="tab-stops: 0pt"}

Essential Grid for ASP.NET MVC uses the following methods for formatting the grid cells.

[·      ]{style="FONT-FAMILY: Symbol"}**Format()**---Used to set the format for the column.

[·      ]{style="FONT-FAMILY: Symbol"}**CssClass()**---Used to set the CssClass for the column.
