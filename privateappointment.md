##### Private Appointment {#private-appointment style="tab-stops: 0pt"}

You can set the appointment to private or not use the **IsPrivate** Property. It is a True / False Property. The default value is False. You can also set the private option in the appointment window. You can change it in appointment window by checking / unchecking the private button.
