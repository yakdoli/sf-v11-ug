#### How to Prevent Columns Resizing for Child Tables {#how-to-prevent-columns-resizing-for-child-tables style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

To prevent the GridGroupingControl\'s child table / grandchild table\'s columns from getting re sized, you must handle the TableControlResizingColumns event of the grid. In the event handler check for the e.TableDescriptor.Name and cancel the event by setting the e.Cancel to true. The following code snippet cancels the re sizing of the Child table.
