## Adding Grid Data Control to the Application {#adding-grid-data-control-to-the-application style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

Now that we have [created a Silverlight application]{style="COLOR: windowtext; TEXT-DECORATION: none; text-underline: none"} and we have seen the basic steps to be followed before adding our Grid controls.

 

In this section, we will see how to add a Grid Data control to the Silverlight application and load data from the database.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

Step-by-step Procedure to Create a Grid Data Control and Load it with Sample Data

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"}
