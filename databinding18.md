## Data Binding {#data-binding style="LINE-HEIGHT: 150%; tab-stops: 0pt"}

Essential Schedule for Silverlight supports full data binding to any type of \"IEnumerable\" source. It has an internal **ICollectionViewAdv** that keeps in sync with the Schedule UI appointments. You must specify the attributes for the **AppointmentMapping** property, to map the properties in the underlying data source to the Schedule appointments.

The various attributes of the **AppointmentMapping** property are as follows.

[·      ]{style="FONT-FAMILY: Symbol"}Subject

[·      ]{style="FONT-FAMILY: Symbol"}Location

[·      ]{style="FONT-FAMILY: Symbol"}StartTime

[·      ]{style="FONT-FAMILY: Symbol"}EndTime

[·      ]{style="FONT-FAMILY: Symbol"}AllDay

[·      ]{style="FONT-FAMILY: Symbol"}ID

[]{style="LINE-HEIGHT: 150%; FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"}
