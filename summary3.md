#### Summary {#summary style="tab-stops: 0pt"}

 

CalcQuickBase is the simplest way to add calculation support to your code. You can create an instance of it, and then just start by using it through either its ParseAndCompute method, or by indexing it with a variable name. You can use CalcQuickBase either in manual calculation mode or in an automatic calculation mode. Automatic calculations will require you to either handle certain events or use the**[ ]{style="COLOR: black"}**RegisterControlArray method for Windows Forms text box and combo box controls.
