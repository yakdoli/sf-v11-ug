##### Creating Multiple Slides {#creating-multiple-slides style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

This tutorial shows how to display multiple slides in a Rotator using XML file. 

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

[·      ]{style="FONT-FAMILY: Symbol"}Multiple slides can be displayed in vertical as well as in horizontal direction.

[·      ]{style="FONT-FAMILY: Symbol"}This can be done by simply setting the height / width of Rotator control as multiple times to that of the corresponding slide\'s height / width (This is just to make sure that the required number of slides are displayed in a single frame). The controls height depends upon the height / width of individual slides you are using inside Rotator.

[·      ]{style="FONT-FAMILY: Symbol"}The **RotationType** must be set to either ContentScroll or SmoothScroll, to display multiple slides.

[·      ]{style="FONT-FAMILY: Symbol"}Horizontally, multiple slides can be displayed by setting **ScrollDirection** to Left or Right.

[·      ]{style="FONT-FAMILY: Symbol"}Vertically, multiple slides can be displayed by setting **ScrollDirection** to Up or Down.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"}
