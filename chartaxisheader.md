### Chart Axis Header {#chart-axis-header style="tab-stops: 0pt"}

[]{#p88}[]{style="COLOR: #15428b"} 

Essential Chart for silverlight provides support for Chart Axis Titles with the help of the Header property. This property can hold the all other elements. We can assign any other silverlight element to this Header property to display the data in chart area.
