#### Dimension - CSS {#dimension---css style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

The Dimension properties are used to specify the size for an HTML element.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

[·      ]{style="FONT-FAMILY: Symbol"}**height**: Specifies the height of an element. The HTMLUI control allows the user to specify the height either in percentage or units.

[·      ]{style="FONT-FAMILY: Symbol"}**width**: Specifies the width of an element

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

The width of an element in HTMLUI can be mentioned in terms of units.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"}
