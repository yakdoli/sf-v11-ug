##### Scrolling Menus {#scrolling-menus style="tab-stops: 0pt"}

**[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"}** 

Scrolling Static Menu

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

When there is more number of menu items added to the menu, scroll arrows can be set to scroll through the items. The **ScrollingEnabled** property must be set to True to automatically create scroll arrows to accommodate and scroll through the menu items. In order to scroll the static menus, you have to set the **Width** of the Menu control for horizontal layout menu and **Height** of the Menu control for vertical layout menu.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

Scrolling Dynamic Menu

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

When there are more number of menu items added to the menu, scroll arrows can be set to scroll through the items. The **ScrollingEnabled** property must be set to True to automatically create scroll arrows to accommodate and scroll through the menu items.

You can set the Height and Width of the menu panel in horizontal and vertical layout using the PopupHeight and PopupWidth property respectively. When menu items exceed the given height and width of the panel, scroll arrows appear on the menu automatically. Also, PopupHeight and PopupWidth property can be set specific to each dynamic menu. You can also use default height or width for all dynamic menu using the DefaultPopupHeight and DefaultPopupWidth properties.

Custom scroll arrow images can be used by assigning the user defined images to the **PopupScrollUpImage** and **PopupScrollDownImage** properties. Also, Menu allows to change the scroll images on mouse over using PopupScrollUpImageHover and PopupScrollDownImageHover properties. This replaces the scroll images with the new ones whenever the mouse pointer passes over the arrows.

 

The scroll area where the scroll buttons are placed can be customized using **ScrollDownLook** and **ScrollUpLook** properties.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"}
