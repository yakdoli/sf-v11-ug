### Adding a Captcha to the MVC Application {#adding-a-captcha-to-the-mvc-application style="tab-stops: 0pt"}

 

Refer to the [Getting Started]{.UGHyperlink} section to know the pre-requisites before stepping in to add a captcha control to the MVC application.

Apart from the above mentioned pre-requisites, adding a captcha to an application requires registering the image handlers under httphandler and handler tags. These image handlers are different from the the MvcResource handler mentioned in the [Getting Started]{.UGHyperlink} section. These image handlers work together with the MvcResourceHandlers.

The following steps explain the addion of a Captcha to an application.

1.   In **View**, create an input field with respect to which the captcha is to be validated and invoke the captcha helper with the control id as argument.
