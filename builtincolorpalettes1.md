### Built-in ColorPalettes {#built-in-colorpalettes style="tab-stops: 0pt"}

ColorPalette is similar to ShapeFill, ShapeStroke and ShapeStrokeThickness.The different between Color Palettes and ShapeFill,ShapeStroke,ShapeStrokeThickness is that the ColorPalette is a set of predefined colors and each shape in the Map has different colors.But if the ShapeFill,ShapeStroke and ShapeStrokeThickness are set, it will be applied to all the shapes in the Map.The available ColorPalette are as follows:

[·      ]{style="FONT-FAMILY: Symbol"}None

[·      ]{style="FONT-FAMILY: Symbol"}ColorPalette1

[·      ]{style="FONT-FAMILY: Symbol"}ColorPalette2

[·      ]{style="FONT-FAMILY: Symbol"}ColorPalette3

[·      ]{style="FONT-FAMILY: Symbol"}ColorPalette4

[·      ]{style="FONT-FAMILY: Symbol"}ColorPalette5

[·      ]{style="FONT-FAMILY: Symbol"}CustomColorPalettePalette

 

In order to show the ColorPalette, the EnableColorPalette property should be true.

ColorPalette Mode

There are two types of ColorPaletteModes. They are:

[·      ]{style="FONT-FAMILY: Symbol"}Sequential

[·      ]{style="FONT-FAMILY: Symbol"}Random

 

In sequential color palette mode, mode color will be applied sequentially, in random mode colors will be applied in a random manner.
