PDF Compression

Compression is the process of reducing the size of data in order to save space or transmission time. Essential PDF controls the compression level of the document by using the PdfCompressionLevel class with the help of the LZW and zlib/deflate compression algorithms. Both LZW and Flate algorithms compress either binary data or ASCII text and always produces the binary data. Image compression is done by adjusting the image quality there by reducing image size.

Features:

The following compression levels are demonstrated in this sample:

  • Above Normal
  • Below Normal
  • Best
  • Best Speed
  • None
  • Normal

Here is an image of the output.

Here is the code snippet to control compression levels:

// Set compression level to the document will control text compression
document.Compression = PdfCompressionLevel.Best;
            
// Handle image compression
image.Quality = 50;
