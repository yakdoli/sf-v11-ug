PDF-A

Features

    Creating a PDF/A-1b document is now easier.  All you have to do is set PdfConformanceLevel as PdfA1B while creating an instance to the PDF document. The PDF/A standard imposes some restrictions regarding the use of colors, fonts, annotations, and other elements. They are listed below.

    • Usage of JavaScript is not allowed by the PDF/A1-B standard.
    • It does not support text containing HTML tags.
    • It does not support attaching files to a PDF document.
    • Adding hyperlinks to a non-PDF file is not supported.
    • It does not support including security features.
    • It does not support form fields.
    • It supports the use of TrueType fonts only, it does not support Type1 fonts.
    • It supports the use of RGB colors, it does not support CMYK colors.

    Here is the code snippet used in this sample:

     //Create a new document with PDF/A standard.
PdfDocument doc = new PdfDocument(PdfConformanceLevel.Pdf_A1B);
 //Add a page
PdfPage page = doc.Pages.Add();            
 //Create Pdf graphics for the page
PdfGraphics g = page.Graphics;                        
 //Create a solid brush
PdfBrush brush = new PdfSolidBrush(Color.Black); 
 // Creating TrueFont
Font f = new Font("Helvetica", fontSize, FontStyle.Regular);            
//Set the font
PdfFont font = new PdfTrueTypeFont(f, true);            
//Draw the text
g.DrawString("PDF/A1-b", font, brush,new PointF(20,20));                    
doc.Save("Sample.pdf");
