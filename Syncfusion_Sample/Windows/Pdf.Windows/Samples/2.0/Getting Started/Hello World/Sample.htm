Features:

    Drawing text in a PDF document is now easier. Now, it is very similar to drawing in .NET GDI. This section demonstrates how a string is drawn in a PDF page using Essential PDF. The process is very similar to drawing any other object on a PDF page. The string is created using the DrawString method of the Graphics class. Be sure to specify the font and brush with which you want the string to be drawn.

    Given below is an image of the output using Graphics.DrawString.

    Here is the code snippet used in this sample:

    //Create a new document.
PdfDocument doc = new PdfDocument();

//Add a page
PdfPage page = doc.Pages.Add();
            
//Create Pdf graphics for the page
PdfGraphics g = page.Graphics;
                        
//Create a solid brush
PdfBrush brush = new PdfSolidBrush(Color.Black);
          
//Set the font
PdfFont font = new PdfStandardFont(PdfFontFamily.Helvetica, fontSize);
            
//Draw the text
g.DrawString("Hello world!", font, brush,new PointF(20,20));

//Save the document.           
doc.Save("Sample.pdf");
