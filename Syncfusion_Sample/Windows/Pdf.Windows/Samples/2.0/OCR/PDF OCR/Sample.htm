OCR Process in PDF

This sample illustrates the process of a Optical Character Recognition in PDF format.

Essential PDF allows to process a OCR in PDF document. Use the Process method of the OCRProcessor class to process a OCR in a PDF.

Below is an illustration of the processed file.

Here is a code snippet to process a OCR in PDF document using Essential PDF.

                //Load the PDF document 
                PdfLoadedDocument lDoc = new PdfLoadedDocument("Region.pdf");
               
               //Process OCR by providing loaded PDF document, Data dictionary and language
                processor.PerformOCR(lDoc, @"../../Tessdata/");
				
                //Save the OCR processed PDF document in the disk
                lDoc.Save("Sample.pdf");
                lDoc.Close(true);

Features:

  • Essential PDF allows you to convert a non-searchable PDF document into a searchable PDF document.

Interactive Features:

  • Click the Process OCR button to convert the Non-searchable PDF document into searchable PDF document
  • Click Yes on the message box to view the generated Pdf file.
