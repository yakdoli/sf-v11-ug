Image Extraction

Features

Essential PDF supports extracting images from a PDF document. You can extract images using the ExtractImages method in the PdfLoadedPage class.

Below is an image of the output.

Here is the code snippet used for conversion:

   // Load an existing PDF
PdfLoadedDocument ldoc = new PdfLoadedDocument("Manual.pdf");
  // Loading Page collections
PdfLoadedPageCollection loadedPages = ldoc.Pages;
   // Extract text from PDF document pages
foreach (PdfLoadedPage lpage in loadedPages)
{
    Image[] img = lpage.ExtractImages();      
}

Interactive Features

  • Choose an existing PDF document using the button near the Source textbox.
  • Click the Extract Image button to extract images from the specified document.
  • Click Yes on the message box to view the extracted images.
  • The displayed images are extracted using Essential PDF source code.
  • All features in the preceding list are illustrated in the extracted images.
