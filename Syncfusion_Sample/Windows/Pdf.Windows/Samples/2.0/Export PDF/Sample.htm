This section contains samples that illustrate using Essential PDF to extract content.

Here is a list of samples with descriptions of each.

  • Image Extraction - This sample demonstrates Essential PDF's support for extracting images from an existing PDF document.

  • Text Extraction - This sample demonstrates Essential PDF's support for extracting text from an existing PDF document.

  • Export as Image - This sample demonstrates exporting PDF pages as raster image using Pdf Viewer.
