Text Extraction

Features

    Essential PDF supports extracting text from a PDF document. You can extract text using the ExtractText method in the PdfLoadedPage class.

    Below is an image of the output.

    Here is the code snippet used for conversion:

       // Load an existing PDF
PdfLoadedDocument ldoc = new PdfLoadedDocument("Manual.pdf");
  // Loading Page collections
PdfLoadedPageCollection loadedPages = ldoc.Pages;
   // Extract text from PDF document pages
foreach (PdfLoadedPage lpage in loadedPages)
{
        s+=lpage.ExtractText();
}

    Note: Essential PDF will not extract text from the meta file that is rendered with a PDF document.

Interactive Features:

  • Choose an existing PDF document using the button near the Source textbox.
  • Click the Extract Text button to extract text from the specified document.
  • Click Yes on the message box to view the text file with extracted text.
  • The displayed text file with text was extracted using Essential PDF source code.
  • All features in the preceding list are illustrated in the text file with extracted text.
