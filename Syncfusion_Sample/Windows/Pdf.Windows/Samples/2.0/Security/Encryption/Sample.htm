Encryption

A PDF document can be encrypted to protect its content from unauthorized access. Encryption applies to all strings and streams in a document.

Features

  • Support for 40-, 128- and 256-bit encryption.    
  • Support for security options like user-owner passwords, digital signatures, and the following restricted-document operations:
    • Allow EditContent
    • Allow CopyContent
    • Allow EditAnnotations
    • Allow AccessibilityCopyContent
    • Allow AssembleDocument
    • Allow Print
    • Allow FullQualityPrint

Here is the sample code used to set document encryption to 128-bit.

// Set encryption key.
doc.Security.KeySize = PdfEncryptionKeySize.Key128Bit;
//Specify the encryption algorithm
doc.Security.Algorithm = PdfEncryptionAlgorithm.AES;

// Setting username and  password to protect the document.
doc.Security.OwnerPassword = "syncfusion";
doc.Security.UserPassword = "password";

// Setting Security properties to the document.
// Allow print with full quality
doc.Security.Permissions = PdfPermissionsFlags.Print | PdfPermissionsFlags.FullQualityPrint;
