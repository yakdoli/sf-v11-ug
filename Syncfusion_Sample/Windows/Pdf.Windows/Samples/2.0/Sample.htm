Essential PDF

Essential PDF is a non-UI component that can be used in ASP.NET and Windows Forms and WPF applications. The usage is common for all the environments, except for the part where the created document is saved to disk, as is the case for a Windows Forms and WPF applications, or streamed to the client browser, as is the case for ASP.NET applications. The Windows section of this documentation contains information on saving the generated document to disk. It also contains information on the samples that are shipped with the product.

Essential PDF is a full-fledged object for creating of PDF files, providing complete control over positioning elements in a PDF document. This model has APIs similar to .NET GDI.

Here is a list of samples with descriptions of each.

  • Product Showcase: This category contains samples demonstrating real-world applications.
  • Getting Started: This category contains a sample that demonstrates how a new PDF document is created and how text can be drawn into it.
  • Drawing Text: This category focuses on demonstrating how the text of various formats can be drawn into a PDF document.
  • Graphics: This category contains samples that explain how various shapes and layers can be created with rich brushes and pens.
  • Tables: This category contains samples that demonstrate the various functionalities supported by the PDFLightTable model.
  • Settings: This category contains samples that deal with various document settings, security settings, and page settings.
  • Security: This category demonstrates various security options available in Essential PDF.
  • User Interaction: This samples demonstrates the creation of various user-interacting features like bookmarks, annotations, actions, links, and forms.
  • Import to PDF: This category contains samples that demonstrates how formats like Web page, MS Word, MS Excel, RTF can be converted to PDF using Essential PDF.
  • Export PDF: This category contains samples that demonstrate how to extract text and images from an existing PDF document.
  • Modify Documents: This category contains samples that demonstrate how to read and modify existing PDF documents using Essential PDF.
