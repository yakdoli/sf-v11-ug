Graphics

This section deals with drawing shapes with different brushes and layers, using Essential PDF.

Here is a list of samples with descriptions of each.

  • Barcode - This sample demonstrates how to draw various sequences of vertical bars and spaces to represent numbers and other symbols. It typically consists of five parts: a quiet zone, a start character, data characters, a stop character, and another quiet zone.

  • Drawing Shapes - This sample demonstrates how various shapes, like line, ellipse, circle, rectangle etc., can be drawn and paginated. It also explains various color spaces that are supported by Essential PDF.

  • Graphic Brushes - This sample demonstrates the support for creating and filling shapes with various types of brushes that are supported by Essential PDF.

  • Image Insertion - This sample demonstrates how to insert images in a PDF document using Essential PDF.

  • Layers - This sample demonstrates how multiple layers can be added to a page and how elements are drawn over each layer.
