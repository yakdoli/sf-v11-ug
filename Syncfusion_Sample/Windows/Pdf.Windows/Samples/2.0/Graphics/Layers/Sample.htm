Layers

Features

    Layers are virtual plates associated with one page. They are stored in separate PDF structures. Each layer has its own graphics, which can be accessed through the Graphics property. You may insert or append layers, which allows you to draw new graphic objects underneath existing layers. The Layers property was created to enable you to access the layer collection of a page.

    This sample focuses on demonstrating the creation of layers in Essential PDF.

    Below is an image of the output.

    Here is the code snippet used to insert layers in a PDF document:

    //Add the first layer
page.Layers.Add();

//Set it as default layer to draw the graphic elements
PdfGraphics graphics = page.DefaultLayer.Graphics;

//Draw Arc 
graphics.DrawArc(pen, rect, 360, 360);

//Add another layer on the page
page.Layers.Add();

//Increment the layer
page.DefaultLayerIndex += 1;
graphics = page.DefaultLayer.Graphics;
        
//Draw another set of elements
pen = new PdfPen(Color.Red, 50);
graphics.DrawArc(pen, rect, 360, 360);
