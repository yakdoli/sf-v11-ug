Viewer

This section demonstrates Essential PDF Viewer's ability to render PDF files.

Here is a list of samples and a description of each of them.

  • Pdf Viewer Demo - This sample demonstrates how to load a PDF into Pdf Viewer and how to get started with the built-in toolbar.

  • Custom Toolbar - This sample demonstrates how to associate a custom toolbar to the Pdf Viewer.

  • Multi Tabbed Viewer - This sample demonstrates Pdf Viewer's support for rendering more than one PdfDocumentView as MDI child window.
