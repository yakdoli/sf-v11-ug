Invoice

This sample focuses on creating an application using Essential PDF and an ADO connection. It demonstrates how to create rich reports, in just a few lines of code, containing styles, formulas, and database connectivity.

Features:

    An invoice has been generated for the given invoice ID; this is done by fetching the data that corresponds to the particular invoice ID from the Northwind database. Price calculation is executed using simple formulas.  The end result is a new PDF document created from scratch.

    This sample demonstrates the PdfGrid features and the gradient brushes that allow users to create rich reports with simple APIs.

Interactive Features:

  • Select the Sales Id from the drop-down list to generate a report for the particular Id.
  • Click Generate Invoice to generate the Pdf document.
  • Click Yes on the message box to view the generated Pdf file.
