Job Application

Essential PDF provides flexible APIs and interactive features that allow creating any real-time application with a PDF. This sample generates a rich application with the following features.

Features:

  • Form creation
  • Form field formatting
  • Image insertion
  • Form action
  • Text insertion and formatting
  • Fonts
  • Page settings

Interactive Features:

  • Click the PDF button to generate the Pdf document.
  • Click Yes on the message box to view the generated Pdf file.
