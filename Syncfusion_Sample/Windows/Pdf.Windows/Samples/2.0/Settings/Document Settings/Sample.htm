Document Settings

This sample illustrates the support for document settings in Essential PDF. It also illustrates adding custom metadata to a PDF document.

Document settings help store information about a document. Extensible Metadata Platform (XMP) is a technology that enables you to embed data that describes a file, known as metadata, into the file itself. It uses XML as the syntax for metadata description.

Features:

 

    XMP is provided with the following schemas:

    • Basic Schema
    • Dublin Core Schema
    • Rights-Management Schema
    • Basic-Job-Ticket Schema
    • Paged-Text Schema
    • PDF Schema

    The document properties of Adobe can be set by the user, either by using the XMP's PDF Schema or the DocumentInformation method of the PDF document.

    The properties that can be set are given below.

    • Author
    • Creation Date
    • Keywords
    • Producer
    • Subject
    • Title

    Here is the code snippet to set document properties:

    //Setting various Document properties.
pdfDoc.DocumentInformation.Title = "Document Properties Information";
pdfDoc.DocumentInformation.Author = "Syncfusion";
pdfDoc.DocumentInformation.Keywords = "PDF";

// XMP Basic Schema.
BasicSchema basic = xmp.BasicSchema;
basic.Advisory.Add( "advisory" );
basic.BaseURL = new Uri( "http://google.com" );

    Custom Metadata

    Essential PDF allows you to define custom metadata attributes, set values for these attributes, and add these custom metadata attributes to a PDF document.

    Here is the code snippet to set custom metadata:

    //Defines and set values for Custom metadata and add them to the Pdf document
CustomSchema custom = new CustomSchema(xmp, "custom", "http://www.syncfusion.com/");
custom["Company"] = "Syncfusion";
custom["Website"] = "http://www.syncfusion.com/";
custom["Product"] = "Essential PDF";

    Custom metadata can be viewed with Adobe Acrobat in the metadata window, which can be reached via Document Properties->Description Tab->Additional Metadata, or you can read the values through Essential PDF.
