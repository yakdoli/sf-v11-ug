Settings

This section contains samples that demonstrate various settings in a PDF document using Essential PDF.

Here is a list of samples with descriptions of each.

  • Document Settings - This sample demonstrates Essential PDF's support for setting document properties. It also explains how XMP properties can be set using Essential PDF.

  • Page Settings - This sample demonstrates the support for setting page-oriented properties such as page size, orientation, viewer preference, page view mode, and page transition. It also explains how transparency can be set.

  • Headers and Footers - This sample demonstrates how headers and footers with page numbers and images can be inserted using Essential PDF.
