Page Settings

Essential PDF has support for various page-setting options like page orientation, page size, page layout, page mode, and page transition.

Features:

    • Page Sizes supported are A0 to A9, B0 to B5, ArchA to ArchE, half letter, ledger, letter, legal, note, and letter11x17.

    • Page Layouts supported are one column, single page, two-column left, two-column right, two-page right, and two-page left.

    • Page Modes supported are full screen, Use Attachment, Use None, Use OC, Use Outlines, and Use Thumbs.

    Here is a sample code to set the page orientation:

    // To set landscape page orientation.
doc.PageSettings.Orientation = PdfPageOrientation.Landscape;

    Here is a sample code to set the page mode:

    // To set UseOC page Mode.
doc.ViewerPreferences.PageMode = PageMode.UseOC;

    Here is a sample code to set the page layout settings:

    // To set two column left page layout.
doc.ViewerPreferences.PageLayout = PageLayout.TwoColumnLeft;

    Here is a sample code to set the page transition:

    //Sets page transition.
doc.PageSettings.Transition.PageDuration = 1;
doc.PageSettings.Transition.Duration = 1;
