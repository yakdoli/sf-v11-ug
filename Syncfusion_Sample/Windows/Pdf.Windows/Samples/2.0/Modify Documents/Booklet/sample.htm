Booklet

Features

    A PDF booklet is very useful  for viewing a book before the document is actually printed. It orders the pages so that after printing, the user will have a book with sequential pages. Those pages can be printed on one side or both.

    This sample demonstrates Essential PDF's support for creating a booklet from an existing PDF document.

    Below is an image of the output.

    Here is the code snippet used to create a booklet:

    //Load a PDF document
PdfLoadedDocument ldoc = new PdfLoadedDocument("Essential_Studio.pdf");

//Create booklet with two sides               
PdfDocument doc = PdfBookletCreator.CreateBooklet(ldoc, new SizeF(500, 500), true);

//Save the document
doc.Save("Sample.pdf");

Interactive Features

  • Choose any existing PDF document using the button near the URL textbox.
  • Choose the page settings option.
  • Click the Create PDF button to generate the PDF document.
  • Click Yes on the message box to view the generated PDF file.
