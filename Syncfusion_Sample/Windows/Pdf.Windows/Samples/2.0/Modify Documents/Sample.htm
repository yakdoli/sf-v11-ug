Modify Documents

This category contains samples demonstrating various editing capabilities of Essential PDF. Below is a list of samples discussed in this section:

  • Merge Documents - This sample demonstrates how to merge multiple PDF documents using Essential PDF.

  • Split Document - This sample demonstrates how to split an existing PDF document into multiple PDF documents.

  • Stamp Document - This sample demonstrates Essential PDF's support for importing a page from an existing PDF and stamping the PDF file.

  • Overlay Documents - This sample demonstrates Essential PDF's support for importing PDF documents as templates.

  • Booklet - This sample demonstrates how to create a booklet from an existing PDF document.
