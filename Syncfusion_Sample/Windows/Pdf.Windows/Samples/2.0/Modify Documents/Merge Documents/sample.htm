Merge Documents

Features:

  • Essential PDF has built-in support for merging multiple PDF documents into a single document.
  • While merging, it reorganizes bookmarks and outlines while preserving forms and annotations that reflect its current state.
  • This sample demonstrates Essential PDF's support for merging multiple PDF documents.

Given below is an image of the output.

Here is a sample code snippet:

string[] paths = {txtDoc1.Text , txtDoc2.Text };
PdfDocument doc = PdfDocument.Merge(paths);

Interactive Features:

  • Choose existing PDF documents using the browse buttons near Select a document and Select another document textboxes.
  • Click the Merge PDF button to merge the specified documents.
  • Click Yes on the message box to view the generated Pdf file.
  • The displayed PDF file was created using Essential PDF source code.
