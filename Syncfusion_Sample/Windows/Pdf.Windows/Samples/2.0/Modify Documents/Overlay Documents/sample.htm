Overlay Documents

Features:

    This sample demonstrates Essential PDF's support for importing pages, from existing PDF documents, as templates. This feature can be used to overlap two PDF pages, one above the other. In this sample, two template documents are used where one stores a set of images and another contains a margin. When you overlap pages from these two documents, the resultant page will contain the images aligned within the specified margin.

    Below is an image of the output.

    Here is the code snippet used to overlay the two documents and create a new one:

    //Create a new document.
PdfDocument doc = new PdfDocument();
PdfPage page = doc.Pages.Add();
PdfGraphics g = page.Graphics;

//Get the first template
PdfPageBase lpage = document1.Pages[0];
PdfTemplate template = lpage.CreateTemplate();

g.DrawPdfTemplate(template, PointF.Empty, page.GetClientSize());

//Get the second template
lpage = document2.Pages[0];
template = lpage.CreateTemplate();

g.DrawPdfTemplate(template, PointF.Empty, page.GetClientSize());

//Save the new document
doc.Save("NewPDF.pdf");
