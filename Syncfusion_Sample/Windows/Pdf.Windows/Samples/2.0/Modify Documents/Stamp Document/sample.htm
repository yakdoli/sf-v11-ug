Stamp Document

Features:

  • Essential PDF provides support for importing content from an existing PDF document and stamping it.
  • This sample demonstrates the import and stamp functionality of Essential PDF.

Below is an image of the output.

Here is the code snippet used in this sample:

//Load the document
PdfLoadedDocument lDoc = new PdfLoadedDocument("Essential_Studio.pdf");
PdfPageBase lPage = lDoc.Pages[0];
PdfGraphics g = lPage.Graphics;

//Stamp text
PdfGraphicsState state= g.Save();
g.SetTransparency(0.25f);
g.RotateTransform(-40);
g.DrawString("Imported using Essential PDF", font, PdfPens.Red, PdfBrushes.Red, new PointF(-150, 450));

//Image watermark
PdfImage image = new PdfBitmap("Ani.gif");
g.Restore(state);
g.SetTransparency(0.25f);
g.DrawImage(image, new PointF(0,0), lPage.Graphics.ClientSize);

Interactive Features:

  • Choose any existing PDF document using the Browse button.
  • Provide a Stamping Text.
  • The default setting generate a document with a text stamping.
  • Choose the Image Watermark option to add an image watermark to the generated PDF document.
  • Click the PDF button to generate the PDF document.
  • Click Yes on the message box to view the generated PDF file.
  • The displayed PDF file was created using Essential PDF source code.
  • All features in the preceding list are illustrated in this file.
