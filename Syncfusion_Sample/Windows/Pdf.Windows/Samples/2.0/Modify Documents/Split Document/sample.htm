Split Document

Often a document needs to be split up, or the parts of its content may need to be created as a new PDF document. This can be easily accomplished with the help of Essential PDF.

Features

  • Documents can be split by page number or by the page itself.
  • This sample demonstrates Essential PDF's support for splitting an existing PDF document into multiple PDF documents.

Below is an image of the output.

Here is a sample code snippet:

//Create two pdf documents
PdfDocument doc1 = new PdfDocument();
PdfDocument doc2 = new PdfDocument();

int splitAtPage = (int)nudSplitPage.Value;

//Split the source document into two based on the page to split-at.
for (int i = 0; i < splitAtPage; i++)
   doc1.ImportPage(srcdoc, i);

for (int j = splitAtPage; j < srcdoc.Pages.Count; j++)
   doc2.ImportPage(srcdoc, j);

//Save pdf document1
doc1.Save("Document1.pdf");

//Save pdf document2
doc2.Save("Document2.pdf");

Interactive Features:

  • Choose any existing PDF document using the button near Document to split up textbox.
  • Choose the Page to Split at option.
  • Click the Split PDF button to split the document.
  • Click Yes on the message box to view the generated PDF documents.
  • The displayed PDF files are created using Essential PDF source code.
  • All features in the preceding list are illustrated in these files.
