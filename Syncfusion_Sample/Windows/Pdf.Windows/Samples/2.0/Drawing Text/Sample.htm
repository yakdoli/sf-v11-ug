Drawing Text

This section demonstrates Essential PDF's support for drawing text in a PDF document.

Here is a list of samples and a description of each of them.

  • Text Flow - This sample demonstrates how large text from a text file can span multiple pages in a PDF document. This sample also explains events that are raised when each page begins and ends.

  • RTL Support - This sample demonstrates how the text of right-to-left (RTL) languages, like Arabic, can be rendered in a PDF document. It also demonstrates various alignments of text.

  • Bullets and Lists - This sample demonstrates Essential PDF's support for creating ordered and unordered lists. It also demonstrates how formatting can be applied to the lists.

  • Multi Column HTML Text - This sample demonstrates Essential PDF's support for rendering a HTML-formatted string in a multicolumn format.
