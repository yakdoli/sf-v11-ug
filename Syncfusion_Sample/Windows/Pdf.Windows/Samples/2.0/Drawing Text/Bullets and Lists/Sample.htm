Bullets and Lists

Lists are used to list items in a collection in order to enhance readability.

Features:

    The types of lists are given below.

    • Ordered list
    • Unordered list

    Essential PDF supports both types of lists and their formatting.    

    This sample demonstrates the various list features of Essential PDF. The PdfOrderedList and PdfUnorderedList classes are used for creating ordered and unordered lists, respectively.

    Here is an image of the output.

    Here is the code snippet to create lists:

    //Create a unordered list
PdfUnorderedList list = new PdfUnorderedList();
            
//Set the marker style
list.Marker.Style = PdfUnorderedMarkerStyle.Disk;

//Apply formattings to list
list.Font = font;
list.StringFormat = format;
            
//Set list indent
list.Indent = 10;

//Add items to the list
list.Items.Add("Item1");
list.Items.Add("Item2");

//Create Ordered list as sublist of parent list
PdfOrderedList subList = new PdfOrderedList();
subList.Marker.Brush = PdfBrushes.Black;
subList.Indent = 20;
list.Items[0].SubList = subList;

//Add items
subList.Items.Add(string.Concat("subitem1"));
