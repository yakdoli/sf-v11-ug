Northwind Report

Essential PDF has support for inserting tables into a PDF page. This is useful for drawing tabular data into the page. Also, there is built-in support for importing an ADO.NET data table into a PDF page.

Features:

  • This sample demonstrates Essential PDF's support for inserting and formatting ADO.NET tables into PDF documents.
  • The PDFLightTable class represents simple tables that are used for publishing structured data from arrays, data tables, or data columns. There are no real cells or rows in those tables, all the data is taken from the data source using the DataSource property.

Steps in inserting an ADO.NET data table:

  • Create a light table to be inserted into the PDF document.
  • Get the data table that is to be inserted from the database and assign it to the data source.
  • Set styles and colors to the table.
  • Set the layout format of the created table to span multiple pages. Set the header to repeat in all pages of the PDF document.
  • Draw the table into the created PDF document.

Here is an image of the output.

Code snippet:

//Create Pdf table
PdfLightTable table = new PdfLightTable();

//Set Data source
table.DataSource = dataTable;

//Set table alternate row style
table.Style.AlternateStyle = altStyle;

//Set header row style         
table.Style.HeaderStyle = headerStyle;
          
//Show the header row
table.Style.ShowHeader = true;

//Repeat header in all the pages
table.Style.RepeatHeader = true;

//Set header data from column caption
table.Style.HeaderSource = PdfHeaderSource.ColumnCaptions;

//Set layout properties
PdfLayoutFormat format = new PdfLayoutFormat();
format.Break = PdfLayoutBreakType.FitElement;
format.Layout = PdfLayoutType.Paginate;
                 
//Draw table
table.Draw(page, PointF.Empty,format);
