XPS to PDF

This sample illustrates the conversion of a XML Paper Specification (XPS) document into PDF format.

Essential PDF allows exporting a XPS document to a PDF document. Use the Convert method of the XPSToPdfConverter class to convert the XPS document to a PDF.

Here is a code snippet to convert a XPS document to a PDF document using Essential PDF.

                // Initialize XPS converter.
                XPSToPdfConverter converter = new XPSToPdfConverter();
               
                // Convert XPS document into PDF document.
                PdfDocument document = converter.Convert("sample.xps");
                
                // Save & close the pdf file.
                document.Save("Sample.pdf");
                document.Close(true);

Features:

  • Essential PDF allows you to convert a XPS document to PDF format.

Interactive Features:

  • Choose any existing XPS document using the button near the textbox.
  • Click the XPS to PDF button to convert the specified XPS document to Pdf file format.
  • Click Yes on the message box to view the generated Pdf file.
