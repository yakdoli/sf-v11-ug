Doc to PDF

This sample illustrates the conversion of a Microsoft Word document into PDF format.

Essential PDF allows exporting a Word document to a PDF document. Use the ConvertToPDF method of the DocToPDFConverter class to convert the Word document to a PDF.

Note:You need to have Essential DocIO installed in order to run this sample.

Below is an illustration of the exported file.

Here is a code snippet to convert a Word document to a PDF document using Essential PDF.

                //Open the word document to convert
                WordDocument wordDoc = new WordDocument("sample.doc");
                DocToPDFConverter converter = new DocToPDFConverter();
                //Convert word document into PDF document
                PdfDocument pdfDoc = converter.ConvertToPDF(wordDoc);
                pdfDoc.Save("DoctoPDF.pdf");

Features:

  • Essential PDF allows you to convert a Word document to PDF format.

Interactive Features:

  • Choose any existing Word document using the button near the textbox.
  • Click the DOC to PDF button to convert the specified Word document to Pdf file format.
  • Click Yes on the message box to view the generated Pdf file.
