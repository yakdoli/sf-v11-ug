Excel to PDF

This sample illustrates the conversion of a MS Excel document to a PDF format.

Essential XlsIO allows to export the Excel document into PDF document. Use the Convert method of ExcelToPDFConverter class to convert the Excel to PDF and save the PDF document. This enabling user easily converts the Excel document to PDF document.

Note:You need to have Essential XlsIO installed in order to run this sample.

Below is the illustration of the exported file.

Here is a code snippet to convert Excel document to PDF document using Essential XlsIO and Essential PDF

                //Open the Excel document to convert
                ExcelToPdfConverter converter=new ExcelToPdfConverter("Sample.xlsx");
                //Convert Excel document into PDF document
                PdfDocument pdfDoc = converter.Convert();
                pdfDoc.Save("ExceltoPDF.pdf");

Features:

  • Supports conversion of a Excel document to PDF document
  • Installation of Essential PDF is necessary to run this sample

Interactive Features:

  • Choose Pdf Page Setup Options to set the Layout of the Pdf page.
  • Click Excel to PDF button to convert the Excel document to PDF file format.
  • Click Yes on the message box to view the generated Pdf file.
  • The displayed PDF file was created using Essential PDF source code.
  • All features in the preceding list are illustrated in this file.
