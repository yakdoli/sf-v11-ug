RTF to PDF

Rich text format (RTF) is a standard specification for formatting documents. RTF files are ASCII files with special commands to indicate formatting information, such as margins and fonts.

Features:

    Essential PDF supports drawing RTF text into a PDF document by converting it into a bitmap or metafile image. Converting RTF text into a bitmap file will generate excellent performance, while converting RTF text into a metafile image provides great resolution and searchable text.

    Given below is an image of the output.

    Here is the code snippet used in this sample:

    //Draw RTF as metafile
PdfMetafile metafile = ( PdfMetafile )PdfImage.FromRtf( text, bounds.Width, PdfImageType.Metafile );
PdfMetafileLayoutFormat format = new PdfMetafileLayoutFormat();

//Allow pagination without any breaks at page breaks.
format.SplitTextLines = false;
metafile.Draw( page, 0, 0, format );

//Draw RTF as Bitmap
image = PdfImage.FromRtf( text, bounds.Width, PdfImageType.Bitmap);
page.Graphics.DrawImage( image, PointF.Empty );
