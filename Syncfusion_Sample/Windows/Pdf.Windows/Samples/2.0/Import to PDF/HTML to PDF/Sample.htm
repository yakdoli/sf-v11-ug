HTML to PDF

Web pages and HTML pages can be imported to a PDF with the help of the HTML converter. The HTML converter converts an HTML Web page to a bitmap or metafile image using MSHTML. MSHTML is a rendering library used to render HTML documents. The MSHTML library is the engine used to drive Internet Explorer.

Features:

    Essential PDF renders the converted image into a PDF. You can convert any Web page to a PDF using either of the above options. Rendering Web pages as bitmaps will produce excellent performance, and as metafiles, you will be provided with an attractive, high-resolution document.

    Essential PDF now provides support for the tagged PDF during HTML to PDF conversion. Tagged PDF is a stylized use of PDF that defines a set of standard structures and attributes to allow the page contents such as text, graphics, and images, to be extracted and reused for other purposes.

    Note: You should have the Syncfusion.HtmlConverter.Base assembly installed to run this sample.

    Here is the code snippet used in convert when this sample runs:

    //Convert web page to image
HtmlConverter html = new HtmlConverter();
Image img = html.ConvertToImage("www.syncfusion.com", ImageType.Metafile)

//Create metafile image
PdfMetafile metafile = (PdfMetafile)PdfImage.FromImage(img);

//Set the layout format
PdfMetafileLayoutFormat format = new PdfMetafileLayoutFormat();
format.Break = PdfLayoutBreakType.FitPage;
format.Layout = PdfLayoutType.Paginate;

//Prevent text getting break at the page breaks
format.SplitTextLines = false;

//Draw the converted image to PDF
metafile.Draw(page, new RectangleF(0, 0, img.Width, img.Height), format);

    Here is the code snippet to generate tagged PDF after conversion:

    PdfDocument document = new PdfDocument();
HtmlConverter html = new HtmlConverter();
html.ConvertToTaggedPDF(document, "www.syncfusion.com");
