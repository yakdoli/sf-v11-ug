Product Showcase

The samples in this section illustrate a wide range of controls and framework components available in Essential Tools.

Word Demo- This sample illustrates the implementation of an application, similar to Word 2007, using advanced ribbon control and other office 2007 controls.

Outlook Demo - The Outlook demo recreates the Outlook 2007 user interface using a group bar and other controls included in Essential Tools.

VS Demo - Visual Studio.NET introduced several new features in terms of Windows user interfaces, such as multiple document interface tabs and refined previously popular features such as docking windows. The Visual Studio .NET demo recreates the Visual Studio.NET user interface using the docking manager, MDI tabs, menus, tool bars, and other controls in Essential Tools.

Tools Overview- Simple overview which will demonstrate in metro theme.
