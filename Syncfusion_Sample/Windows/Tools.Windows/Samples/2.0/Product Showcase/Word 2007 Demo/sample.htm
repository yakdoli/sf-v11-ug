Word 2007 Demo

This sample illustrates the implementation of an application, similar to Word 2007, using ribbon controls.

 

  • RibbonControlAdv

     

    The main feature of Office 2007 is the ribbon control. Essential Tools has introduced RibbonControlAdv, which has functionalities similar to the ribbon control in Office 2007.

    RibbonControlAdv provides support for creating an Office 2007 look for applications. It provides a blue UI ribbon, ToolStripEx, Super ToolTips, and Super Accelerators as part of the Office 2007 controls.

    Features

    • Ribbon Form

      This feature has curved corners that resemble the Office ribbon form.

    • Office Menu Button

      RibbonControlAdv has an Office menu button at the top-left corner of the form. Controls can be added to this button through a designer or through code.

    • Quick-Access Toolbar

      RibbonControlAdv has a quick-access toolbar where quick-access buttons can be added through smart tags in the designer. The quick-access toolbar can be customized at run time using the "Customize Quick Access Toolbar" dialog.

    • Tool Strip Tab Item

      Tab items can be added to RibbonControlAdv, according to a user's needs, using a smart tag in the designer. Tabs, at run time, can be hidden by double-clicking the tab header (a feature of Office 2007).

    • ToolStripEx

      Tool strips can be added to a tab item in the designer by clicking the Add Tab menu item that appears when right clicking the context menu.

    • Look and Feel

      RibbonControlAdv has the same visual styles as Office 2007. Users can choose between two colors, Office Blue and Office Silver, which can be set in the OfficeColorScheme property of the RibbonControlAdv class.

    • Super ToolTip

    Super ToolTip

    The background gradient color can be changed by using a color from the BackColor property. Images can be associated with the header, body, and footer of the Super ToolTip. Image position and text position of the ToolTip can be customized by the ImageAlign and TextAlign properties. A separator can be displayed between the body and the footer.

  • Super Accelerator

    Accelerator keys are shortcut keys that allow users quick access to click-events in the RibbonControlAdv.

    The accelerator key shortcut for controls can be enabled by pressing the "Alt" key. These accelerators can be enabled for all items in the ribbon, including the Office Menu button.
