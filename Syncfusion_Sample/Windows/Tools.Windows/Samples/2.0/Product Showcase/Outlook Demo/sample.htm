Outlook Demo

This sample shows the implementation of a Microsoft Outlook 2007-like interface using Essential Tools and other components from Essential Studio.

XP Menus have been extended to support all the Office 2007 themes (Blue, Black, and Silver) with the appearance of Outlook 2007.

The stacked-mode feature of the group-bar control allows you to implement a navigation bar, similar to that of Microsoft Office Outlook 2007.

The scheduler control included in this sample demonstrates the functionalities and the look of Microsoft's Outlook Scheduler.

RibbonControlAdv is used in this sample to demonstrate the support for Office 2007 themes. Click New -> Mail Message to show a dialog window similar to that of the New Mail Message found in Outlook 2007.

The Essential Grid Grouping feature that enables you to provide custom grouped views of data is also highlighted.

 
Features:

  • Office 2007 themes (Blue, Black, and Silver)

  • Grouping of columns in the "All Mail" display area; right-click on the column headers and select "ArrangeBy," to arrange by that column

Note: In order for this sample to execute, you must have the following Syncfusion assemblies along with the Syncfusion Tools Package assemblies.

  • Syncfusion.Grid.Grouping.Base

  • Syncfusion.Grid.Grouping.Windows

  • Syncfusion.Grouping.Base

  • Syncfusion.Grouping.Windows

  • Syncfusion.Tools.Windows
  • Syncfusion.Schedule.Windows
  • Syncfusion.Schedule.Base
