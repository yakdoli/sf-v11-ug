VS Demo

This sample illustrates the implementation of an application similar to Visual Studio .NET (VS.NET) using Syncfusion menus, toolbars, docking windows, tabbed MDIs, and edit controls.

Features:

  • The Docking Windows framework, included in Syncfusion Essential Tools, provides full-fledged support for VS.NET-like docking, which requires little or no code to implement.
  • The VS.NET Toolbox is created by using the Syncfusion Group View-Group Bar controls included in Essential Tools.

Note: For its execution, this sample depends on the following Syncfusion assemblies along with the Syncfusion Tools Package assemblies.

  • Syncfusion.Edit.Windows
