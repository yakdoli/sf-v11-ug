Owner Draw

This sample illustrates how nodes in the TreeViewAdv control can be owner-drawn.

The OwnerDraw property permits an application to alter the appearance of a control.

Features:

Concatenation of two texts

The concatenation of two texts can be performed by using the BeforeNodePaint and AfterNodePaint events in the TreeViewAdv.

Users can alter the second text without changing the first. Also, the color and font of the second text can be changed without changing the first.    

The OwnerDraw mode must be set to true in order to use the BeforeNodePaint and AfterNodePaint events. This will allow you to customize the background color of the node and the TreeViewAdv control.

Alteration of Colors

The colors of the tree lines, plus-minus symbols, rectangle bounds, and node text can be changed by setting the OwnerDraw property to true and by handling the BeforeNodePaint event handler.

The TreeViewAdv's NodeBackgroundPaint event is also used to color the background of a selected node.

private void treeViewAdv1_NodeBackgroundPaint(object sender,Syncfusion.Windows.Forms.Tools.TreeNodeAdvPaintBackgroundEventArgs e)
{
if (e.Selected == true)
e.Node.Background = new Syncfusion.Drawing.BrushInfo(Color.Azure);
else
e.Node.Background = new BrushInfo(GradientStyle.None, SystemColors.Window, SystemColors.Window);
}

Disable Dotted Rectangle

If a user does not wish to have a dotted rectangle appear when a node is selected, in order to set the focus on the TreeViewAdv, then the following code can be used.

1. Set the TreeViewAdv's OwnerDrawNodes property to true.

this.treeViewAdv1.OwnerDrawNodes = true;

2. Handle the TreeViewAdv's BeforeNodePaint event as shown in the following code.    


	private void treeViewAdv1_BeforeNodePaint(object sender, Syncfusion.Windows.Forms.Tools.TreeNodeAdvPaintEventArgs e)
	{
	e.Active = false;
	}

Hot Tracking

This sample illustrates the concept of HotTracking to highlight a node, giving it the appearance of a hyperlink.

When the HotTracking property is set to true, the node label takes on the appearance of a hyperlink when a pointer hovers over a node. In this sample, the foreground and underline colors are set as blue.
