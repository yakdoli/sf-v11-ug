Drag-Drop

This sample illustrates the implementation of the drag-and-drop feature of the TreeViewAdv control.

Features

  • Initialization

    • Initialization of OLE drag-and-drop on mouse down-and-drag


  • Listening and Accepting

    • Listening to drag-over messages and accepting drag-and-drop operations


  • Show Drag NodeCu

    • Allows users to specify whether a semi-transparent image of a selected node should be drawn beside the cursor during a drag-and-drop operation


  • Drag on Text

    • Indicates whether a drag-drop operation will occur if a node is dragged onto a text area.


  • Tracker

    • The Tracker class can be used to support a drop operation adjacent to a node; otherwise, listening to the DragOver and DragDrop events are enough. The tracker will draw indicators above or below a node while dragging an item over the node.

    The usage semantics for the tracker are given as follows.

    1.Create an instance of the tracker class and listen to the QueryAllowedPositionForNode event. A user can listen to this event and prevent drawing highlights for specific nodes.

    2.The TreeViewAdv's DragOver event updates the highlighted node by calling the SetHighLightNode method.

    3.The TreeViewAdv's DragLeave event clears the highlighted node by calling the ClearHighlightNode method.

    4.The QueryAllowedPositionsForNode event handler specifies whether a computed adjacent position should be allowed or not.

    5.The TreeViewAdv's DragDrop event allows a user to retrieve a highlighted node and drop position from the tracker, helping to insert a new node.


  • Drag Highlight Tracker's “QueryDragInsertInfo”

    • The TreeViewAdvDragHighlightTracker pen can be modified by the added event TreeViewAdvDragHighlightTracker.QueryDragInsertInfo. This allows users to select the color of the TreeViewAdvDragHighlightTracker pen based on the current background.


  • Selection Mode

    • There are three types of selection modes in the TreeViewAdv; they are as given below.

    Single - Users can select only one node at a time and implement the drag-drop operation in the TreeViewAdv.

    Multi-Select Same Level - Users can select only the nodes of the same level, for example, only child nodes or only parent nodes.

    Multi-Select All - Users can select multiple nodes used for implementing the drag-drop operation in the TreeViewAdv.


  • Scrolling Property

    • AutoScrolling of the TreeViewAdv happens when dragging the mouse pointer near the border while a multiple selection is being performed.
