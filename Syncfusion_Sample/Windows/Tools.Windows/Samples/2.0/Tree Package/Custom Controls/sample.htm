Custom Controls

This sample illustrates how custom controls are added in a TreeViewAdv control.

Features:

With the custom controls feature, any control can be included in the nodes of a Tree ViewAdv control. This provides a wider scope of functionality and expands the view of the tree-view nodes.

In this sample, the controls added to the Tree ViewAdv control are XP Tool Bar, Combo BoxAdv, Date-Time Picker, and Label and Grid controls.
