ProgressBar Features

Features:

This sample demonstrates a few of the properties of the Progress BarAdv control.

 

Interactive Features:

  • Create a new Visual C# application or VB .NET application in Visual Studio .NET.
  • Drag and drop a Progress BarAdv control object from the toolbox onto the form and resize it to the desired dimension.
  • Set the properties through the controls on the form and then notice the changes made to the demo Progress BarAdv control.
