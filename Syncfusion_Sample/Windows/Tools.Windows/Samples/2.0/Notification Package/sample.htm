Notification Package

The Notification Package consists of controls used to provide visual feedback regarding the status of specific events in an application. Common applications of the Notification Package include enhanced progress bars, splash screens, and customized message boxes with animation.

The different samples included in the Notification Package are listed below:

  • Splash Panel - This sample demonstrates the implementation of a splash panel in Essential Tools.
  • Splash Control - This sample demonstrates the implementation of the splash control in Essential Tools.
  • StatusBarAdv Panel - This sample demonstrates the implementation of a StatusBarAdv Panel control in Essential Tools.
  • StatusBarAdv - This sample demonstrates the implementation of the StatusBarAdv control in Essential Tools.
  • ProgressBarAdv Features - This sample demonstrates a few of the properties of the ProgressBarAdv control.
  • ProgressBarAdv Demo - This sample demonstrates the implementation of the ProgressBarAdv control in Essential Tools.
