StatusBarAdv

This sample demonstrates the implementation of the StatusBarAdv control in Essential Tools.

Features :

  • The StatusBarAdv is an advanced status bar control that can display status-bar panels with enhanced backgrounds and appearances.
  • It can be dragged and dropped from the toolbox onto a form during design time. Users can then drag and drop controls onto the StatusBarAdv control.
  • The panels or child controls added to the StatusBarAdv can be aligned to Center, Near, Far and ChildConstraints using the Alignment property.
  • Themes can be enabled for the StatusBarAdv control.

Interactive Features :

  • KeyState can be displayed in StatusBarAdv using 'KeyStatePanels' tab.
  • Short date or long date can be displayed in StatusBarAdv using 'DatePanels' tab.
  • Short time or long time can be displayed in StatusBarAdv using 'TimePanels' tab.
  • Custom text can be set to child control of StatusBarAdv using 'CustomPanels' tab.
