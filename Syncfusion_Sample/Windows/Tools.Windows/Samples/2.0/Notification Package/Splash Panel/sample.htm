Splash Panel

This sample demonstrates the implementation of a splash panel in Essential Tools.

Interactive Features:

  • Drag and drop a splash panel control onto the form.

  • By setting the ShowAnimation property to true, the splash panel will appear with animation.

  • The SlideStyle property specifies the different sliding styles: Horizontal, Vertical, Left-to-Right, Bottom-to-Top, Right-to-Left, Top-to-Bottom, or Fade In.

  • In order to place a bitmap file on a splash panel, add a picture box control on top of the panel and set the Docking property of the picture box to Fill. Then set the Image property of the picture box to the bitmap you want display.

     

  • Enabling the CloseOnClick property of the splash panel will close it when clicked.
