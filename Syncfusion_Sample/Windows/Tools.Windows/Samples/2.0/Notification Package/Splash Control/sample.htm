Splash Control

This sample demonstrates the implementation of the splash control in Essential Tools.

Interactive Features:

  • Place an instance of the splash control onto the form.

  • Set the AutoMode property to true in order to automatically launch the splash screen.

  • Set the start position for the splash screen by using the DesktopAlignment property.

  • The splash screen can be animated by enabling the ShowAnimation property.

  • By setting the UseCustomSplashPanel property to true, a custom splash panel can be implemented if the default one is not used.

  • The time period for which the splash should appear can be set through the TimerInterval property.
