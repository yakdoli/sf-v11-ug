Office Style Form

Features:

This sample shows you the Office Style form that can be used as a stand-alone form without the dependency of the ribbon control.

Features :

  • This Office Style form, besides having the normal properties, supports the color schemes Blue, Silver, and Black.
  • It supports help button, Right To left feature everything similar to the normal form with the OfficeStyle look and feel.

Interactive Features :

  • Help button can be shown in the title bar of the OfficeStyleForm by checking 'Show Help Button' checkbox.
  • Icon in the  title bar of the OfficeStyleForm can be shown/hidden using 'Show Icon' checkbox.
  • Right To left feature can enabled by checking 'Right To Left' checkbox.
  • OfficeStyle  color schemes can be changed by the radio button.
  • Caption in the title of the OfficeStyleForm can be controlled by Caption Setting options provided in the bottom of the form.
