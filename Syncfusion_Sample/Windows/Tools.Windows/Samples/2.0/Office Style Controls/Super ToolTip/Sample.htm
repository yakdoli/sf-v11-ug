Super ToolTip

This sample illustrates how the Super ToolTip is used to provide extended ToolTip support.

Features:

Color Schemas

Users can set any background color for the ToolTip. They can also set a border color for the Super ToolTip by using the BorderColor property.

ToolTip parts

ToolTips have three parts: header, body (description area), and footer; all can be hidden with the Hide property. ToolTips also have the Font, ForeColor, Image, Text Align, and ImageScalingSize properties.

Image in ToolTip

A Super ToolTip has image support, which can be assigned for the header, body, and footer.   

Various Controls Support

A Super ToolTip supports any .NET control; examples include menu strips, tool strips, Syncfusion ToolStripEx and RibbonTabControlAdv.

Right-to-Left Support

A Super ToolTip provides support for right-to-left languages.   

Text Margin

A margin can be provided around the text in header, body, or footer by using the TextMargin property.
