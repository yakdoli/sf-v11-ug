Office Style Controls

This sample illustrates the use of various controls providing an Office Style look.

Features:

StatusStripEx

StatusStripEx can be docked to the bottom of a form. It can hold controls like TrackBarEx, ProgressBar and StatusStripButton.

Context MenuStripEx

The Context MenuStripEx control, which appears when right-clicking during run time, has numerous advanced features. Its items can be set as RightToLeft, and the shadow of the Context MenuStripEx can be made visible or hidden. It supports Office Style color schemes as well, and it can also have a header associated with it.

TrackBarEx

This control allows sliding between the minimum value and maximum value specified through a pointer. It also supports full customization of its appearance, including orientation.
