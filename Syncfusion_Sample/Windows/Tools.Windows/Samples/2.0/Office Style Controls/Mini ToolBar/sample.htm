Mini Toolbar

This sample shows you a mini-toolbar that can be used in various scenarios for quick and easy access to items.

The following is a sample image.

Features :

  • Essential Tools comes with MiniToolBar control with similar look and feel of MS Office 2007.
  • The mini-toolbar can be associated with any control by using the AssociatedControl property.
  • It provides options to customize the selected text.
  • MiniToolbar supports all the three ColorSchemes of Office2007.

Interactive Features :

  • Select some text in 'Text Related Controls' tab and right click to view the minitoolbar.
  • Select some text within Messenger textbox in 'Container controls' tab  to view the minitoolbar.
  • Right click on any cell in the grid present in the 'Other controls' tab and change the color scheme with the options given in the minitoolbar.
