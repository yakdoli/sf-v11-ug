Ribbon Merge

This sample illustrates how the ribbon panel in a child form can be merged with the RibbonControlAdv in the parent form.

Features:

  • The items in the child form can be merged with the parent form if the ribbon-panel merge container is used in the child form to host the ToolStripEx.
  • In this sample, the ribbon-panel merge containers named "Formulas" and "Format" in the child form will be merged without having to use code.
