MessageBoxAdv

This sample illustrates the MessageBoxAdv control that provides support for custom colors and images.

Features :

  • MessageBoxAdv supports standard color schemes and custom color schemes in Office 2007 style, for consistent User Interface look and feel.
  •  Custom Icons support is also included in MessageBoxAdv.

Interactive Features :

  • To display MessageBoxAdv with standard color schemes, select Blue or Silver or Black radio button and select button to display MessageBoxAdv.
  • To display MessageBoxAdv with custom color scheme, select Managed radiobutton, select custom color using ColorUIAdv and select button to display MessageBoxAdv.
  • To display MessageBoxAdv with default icon, select default icon from combobox and display MessageBoxAdv.
  • To display MessageBoxAdv with custom image, select custom image from combobox and display MessageBoxAdv.
