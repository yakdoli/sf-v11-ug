SDI Demo

This sample illustrates how to incorporate XP-style customizable menus and toolbars into an application in a SDI scenario. It also illustrates the usage of a pop-up control container to design custom pop-ups.

This version includes a Super ToolTip for menus, which can be seen when hovering over bar items.

Features

  • DropDownBarItem.PopupControlContainer - This property specifies the pop-up control container that will drop down from an item when placed in a menu or a toolbar.

  • In this sample, a Font Dialog control is placed inside a pop-up control container and is associated to the drop-down bar item.

  • The View Parent Bar item lists the toolbars present in the form by using ToolListBarItem.

  • Different pop-up menus are associated to different controls using PopupMenuManager.
