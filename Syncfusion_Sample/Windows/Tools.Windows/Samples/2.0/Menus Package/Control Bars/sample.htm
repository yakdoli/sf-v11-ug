Control Bars

This sample illustrates the Docking-Floating feature of a control bar. It also demonstrates how various controls can be hosted within the control bar.

Features

  • CaptionControl - This property specifies the control to be displayed in the control bar's caption region.

  • DisableDocking - This property allows enabling or disabling the docking feature of the control bar.

  • DisableFloating - This property allows a user to enable or disable the floating state of the control bar.

  • CardLayout - Adds different pages, panels with controls, to the control bar.

  • Control bars can be floated by dragging the form's corners, and docked again by double-clicking. The docked side can be disabled by using the AllowDockBorders property.
  •  The gripper, close, and drop-down buttons in the control-bar header can be hidden using the HideGripperButton, HideCloseButton, and HideDropDownButton properties, respectively.
  •  Users can add a custom pop-up control container to the control bar's drop-down in the header by associating it to the ControlBar.PopupControlContainer property. A pop-up menu can be associated by using the ControlBar.PopupMenu property.
  • Caption height can be adjusted using the ControlBarCaptionHeight property.
