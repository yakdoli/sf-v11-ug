Custom Colors

This sample illustrates how to incorporate custom colors and backgrounds for menus and toolbars.

Feature :

  • Custom Colors can be applied for Menus and toolbars.

Interactive Features :

  • Custom background can be seen when mouse hover the bar items in the form.
  • Right click to show the popupmenu and mouse hover each baritem to see the custom background.

This sample uses the MenuColors class and sets a custom color for the bar items. A custom background has been set using the DrawBackground event handler.

// SelColor - Color for the background of active barItem in Menu dropdown and ToolBar.
MenuColors.SelColor            = Color.FromArgb(179, 71, 59);

// SelTextColor - Forecolor used by the active barItem.
MenuColors.SelTextColor        = Color.White;

// SelBorderColor - BorderColor to be drawn around the active BarItem.
MenuColors.SelBorderColor      = Color.FromArgb(183, 214, 242);

// MenuBGColor - Color used to draw the background of dropdown.
MenuColors.MenuBGColor         = Color.White;

// MenuLeftStripColor - Color used to draw the left strip of menu dropdown.
MenuColors.MenuLeftStripColor  = Color.White;

// DropDownBorderColor - BorderColor used to draw the dropdown border.
MenuColors.DropDownBorderColor = Color.FromArgb(166, 205, 226);

// PressedSelColor - Color used for the background of pressed barItem.
MenuColors.PressedSelColor     = Color.FromArgb(166, 201, 232);

// CheckedSelColor - Color used to draw the background of checked barItem when mouse hovers over it.
MenuColors.CheckedSelColor     = Color.FromArgb(215, 236, 255);

// FloatingCommandBarCaptionColor - Color of caption of floating command bar.
MenuColors.FloatingCommandBarCaptionColor = Color.FromArgb(183, 214, 242);
