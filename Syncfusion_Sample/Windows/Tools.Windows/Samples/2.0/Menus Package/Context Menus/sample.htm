Context Menus

This sample illustrates pop-up menus used to display a context menu with items.

Features:

  • A pop-up menu can easily be displayed and associated with any control using the Pop-Up Menus Manager component.
  • Adding the Pop-Up Menus Manager to a form will add an extended property for the controls.
  • PopupMenu.ParentBarItem - This property specifies the parent-bar item that lists items in a drop-down menu.

// To cancel the popupmenu before showing up
private void popupMenu2_BeforePopup(object sender, CancelMouseEventArgs e)
{
	e.cancel = true;
}
