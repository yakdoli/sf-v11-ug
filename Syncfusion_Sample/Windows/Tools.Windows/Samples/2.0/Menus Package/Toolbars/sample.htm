Toolbars

This sample illustrates the XP ToolBar class that is used to provide a toolbar-like look that can be used outside of the XP Menus framework.

Features

  • Bar Items - Supports a wide variety of bar items.

  • Styles - Supports visual styles such as Default, Office XP, Office 2003, VS 2005, Office 2007, Office 2007 Outlook, and Vista.

  • XPToolBar.BeginGroupAt - This method specifies a separator in the bar items list. The separator will be placed just before a specified bar item.

  • BarItem.XPToolBar.ThemesEnabled - This property specifies whether XP themes should be used.
