Menus Package

The Essential Tools Menus package comes with the XP Menus framework, which provides users with Office XP, Office 2003, VS 2005, and the latest Office 2007 themes and functionalities for menus, toolbars, and context menus. It also supports complete localization.

The different samples included in Menus Package are listed below:

  • SDI Demo - This sample illustrates how to incorporate XP-style customizable menus and toolbars into an application in a SDI scenario.
  • MDI Demo - This sample illustrates the usage of XP Menus in a MDI scenario.
  • Custom Colors - This sample illustrates how to incorporate custom colors and backgrounds for menus and toolbars.
  • Toolbars - This sample illustrates the XP ToolBar class that is used to provide a toolbar-like look that can be used outside of the XP Menus framework.
  • Control Bars - This sample illustrates the Docking-Floating feature of a control bar.
  • Context Menus - This sample illustrates pop-up menus used to display a context menu with items.
