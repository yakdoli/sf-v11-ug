Constrained Layout Bounds

Layout managers typically lay out child controls within a container's client rectangle—except for a flow layout, which might position the child controls beyond the client rectangle.

This sample shows how the layout manager is better at laying out child controls than the client rectangle.

Features:

  • The minimum layout width and height are restricted to 300 and 200, respectively. When the form is resized beyond that, a scroll bar appears.
  • This sample uses a grid layout manager for illustration; however, the premise illustrated here applies to any layout manager.
