Tabbed MDI Package

The Tabbed MDI Package provides a new tabbed MDI layout mode as an alternative to the default Cascade and Tiled modes.

This framework was built with great consideration for ease-of-use. It is intended to avoid having to modify an existing MDI application in order to enable the Tabbed MDI mode. With a single method call, users can switch between Tabbed and regular MDI layout modes.

The different samples included in Tabbed MDI Package are listed below:

  • Tabbed MDI - This sample illustrates how to control tab group, tab-group heights, and child-form associations when using the Tabbed MDI Manager.
  • Advanced Tabbed MDI - This sample illustrates how to incorporate the Tabbed MDI mode into your applications using the TabbedMDIManager class.
