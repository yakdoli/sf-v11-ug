Command Bars

This sample illustrates the usage of the Command Bars framework for creating dockable toolbars.

Note: The Command Bars framework is meant to be used with the default .NET menus. If you plan to use the XP Menus framework in Essential Tools, do not use this framework, instead use the XP Menus framework. Take a look at the XP Menus sample.

Features

  • CommandBar.HideChevron - This property specifies whether the command bar will be drawn with a chevron.

  • CommandBar.PopupMenu - This property specifies the pop-up menu to be displayed when the drop-down button is clicked.

  • CommandBar.PopupContainer - This property specifies the list of categories under which the bar items will be grouped.
