Localization Demo

This sample explains how Syncfusion controls and components—like Docking Manager, XP Menus, Group Bar, and Color UI Picker—can be localized.

Note: To use the German Resources in this application, copy the satellite assemblies from the "de-DE" folder in the sample path to the "bin\debug\de-DE" folder.

Localized parts in the sample are given as follows.

  • For docking, the strings in the context menu and caption button's ToolTips are localized.

  • For XP Menus, the Customization dialog strings in the Quick Customize button are localized to German.

  • For Group Bar, the strings in the navigation pane pop-up menu are localized.

  • For Color UI Picker, the strings in the tab control are localized.


For more information, refer to the localization document found in C:\Program Files\Syncfusion\Essential Studio\(INSTALLED VERSION)\Windows\Tools.Windows\Localization\Localization.

Note: The German text was translated using online tools.
