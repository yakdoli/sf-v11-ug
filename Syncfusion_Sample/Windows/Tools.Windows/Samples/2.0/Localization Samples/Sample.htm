Localization Samples

Localization is the process of customizing an application to adapt to culture-specific resources; this includes language translation as well as information formatting such as date, time, and currency.

Syncfusion supports two different localization procedures that are demonstrated in the following samples.

  • Localization Demo - This sample explains how Syncfusion controls and components - like the docking manager, XPMenus, GroupBar, and ColorUIPicker - can be localized.
  • New Localization Procedure - This sample illustrates how Syncfusion controls and components can be localized programmatically.
