New Localization Procedure

This sample demonstrates how Syncfusion controls and components can be localized through code.

The following sections in the sample have been localized.

  • For XP Menus, the Customization dialog strings in the Quick Customize button are localized to German.

  • For the Color UI Picker, the strings in the tab control are localized.

  • For the MessageBoxAdv, the strings in the buttons are localized.

Note:The German text was translated using Google Translate.
