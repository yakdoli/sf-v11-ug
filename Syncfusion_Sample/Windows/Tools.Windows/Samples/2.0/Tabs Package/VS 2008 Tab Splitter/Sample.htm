VS 2008 Style Tab Splitter

The Tab Splitter Container control supports a VS 2008-style split view of tabbed groups, helping users render different views of the same document.



VS 2008-Style Horizontal Split View

Interactive Features:

Simply fill your form with the tab-splitter container and drag-and-drop the controls onto the different tabs to populate the forms.

The control also comes with several built-in functionalities for rich user-interaction support. They are as follows.

  • Swap Tab Groups.
  • Collapse a Tab Group.
  • Toggle between Vertical and Horizontal Layouts.
