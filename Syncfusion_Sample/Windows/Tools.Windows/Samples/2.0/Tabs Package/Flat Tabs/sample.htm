Flat Tabs

This sample illustrates how to customize the TabControlAdv to create VS .NET Start Page-like flat tabs with a transparent tab strip area and a gradient background.

Features :

  • This sample also shows how to create a custom tab renderer from the TabRenderer2D class in order to customize certain aspects of the tabs.

Interactive Features :

  • OnPaint method of TabControlAdv is overridden to remove the border around the tabcontrol.
  • Customized SingleLineTabPanelRenderers to achieve certain effects in the tab strip portion.
  • OnPaint method of  SingleLineTabPanelRenderers  is overridden draw the gradient background in the tab-strip area.
