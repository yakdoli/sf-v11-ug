Tabs Package

The XP Tabs (TabControAdv) feature provided as part of this package is an advanced tab control with plenty of appearance styles and configuration capabilities. It makes creating custom tabs with custom shapes and styles easy.

The different samples included in Tabs Package are listed below:

  • TabControlAdv Demo - This sample illustrates the design-time setup of the TabControlAdv class.
  • Custom Tab Controls - This sample illustrates the implementation of a custom tab control using an owner-draw technique and deriving from default rendering-base classes.
  • Flat Tabs - This sample illustrates how to customize the TabControlAdv to create VS .NET Start Page-like flat tabs with a transparent tab strip area and a gradient background.
  • VS 2008 Tab Splitter - The Tab Splitter Container control supports a VS 2008-style split view of tabbed groups, helping users render different views of the same document.
  • TabBarSplitter Demo - TabBarSplitter control supports Office 2007 style tab grouping at the bottom.
