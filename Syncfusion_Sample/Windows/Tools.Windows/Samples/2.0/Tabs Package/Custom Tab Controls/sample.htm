Custom Tab Control

Features:

This sample illustrates the implementation of a custom tab control using an owner-draw technique and deriving from default rendering-base classes.

Interactive Features:

In Yahoo Messenger(R), the active tab is the only one that has a border around it; the rest are borderless. This is achieved by setting the TabStyle to "3D" (typeof(TabRenderer3D)), handling the DrawItem event, delegating the background and interior drawing to the default drawing logic, and delegating the border drawing logic to the default drawing logic, but only if the corresponding tab is active.
