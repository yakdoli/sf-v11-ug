Window Fill

This sample mainly demonstrates the usage of the properties, DockToFill and DisallowFloating. It also describes the CloseEnabled property of the docking manager.

 

Features:

DockToFill Property

  • The DockToFill property specifies whether there should be any free space after we have closed or hidden docked windows. If we set this to true, there will be no more free space on the host form after we have changed the dock-state of any of the docked controls. The neighboring control will take up the newly freed space.
