Drag Provider Styles

Features:

This sample illustrates different styles of drag providers, which will be shown when you drag a docked window.

Given below is a sample image.

Features :

The different styles of drag providers are listed below.

  • Default - Provides a standard style of a helper frame for docking windows.
  • Whidbey - Enables Visual Studio 2005 beta 2-style docking.
  • VS 2005 - Enables Visual Studio 2005-style docking.
  • VS 2008 - Enables Visual Studio 2008-style docking.

Interactive Features :

  • Click 'Default' menuitem for enabling standard drag provider.
  • Click 'Whidbey' menuitem for enabling Visual Studio 2005 beta 2-style docking.
  • Click 'VS 2005'  menuitem for enabling Visual Studio 2005-style docking.
  • Click 'VS 2008'  menuitem for enabling Visual Studio 2008-style docking.
