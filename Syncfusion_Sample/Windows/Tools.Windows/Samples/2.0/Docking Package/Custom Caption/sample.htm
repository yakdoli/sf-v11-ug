Custom Caption

This sample shows you how to add custom caption buttons to the docked controls and how to customize the caption bar.

Given below is a sample image.

Features :

  • DockingManager supports adding custom caption buttons to docked controls.

  • Different caption buttons can be added to each docking control.

  • The caption bar can be made visible or invisible.

  • The caption bar can be customized by applying different colors, text alignments or showing/hiding images.

Interactive features :

  • Select the baritem 'Show Caption' to show/hide the caption.

  • Select the baritem 'Show Caption Images' to show/hide the image in the caption bar.

  • Select the baritem 'Enable SuperTooltip' to enable/disable the supertooltip on the caption buttons.

  • Visual styles of the docking windows can be set using ComboBox in the Form.

  • The background of the caption bar can be conttrolled using 'Active Caption Background' and 'Inactive Caption Background' baritems.

  • The forecolor of the caption bar can be conttrolled using 'Active Caption Foreground' and 'Inactive Caption Foreground' baritems.

To add a custom caption button to the docking manager, follow the code snippet given below.



     Syncfusion.Windows.Forms.Tools.CaptionButton captionButton = new  
     Syncfusion.Windows.Forms.Tools.CaptionButton();
     captionButton.ImageIndex = -1;
     captionButton.Name = "CaptionButton1";
     captionButton.Type = Syncfusion.Windows.Forms.Tools.CaptionButtonType.Custom;
     captionButton.TransparentImageColor = System.Drawing.Color.Transparent;
     this.dockingManager1.CaptionButtons.Add(captionButton);
