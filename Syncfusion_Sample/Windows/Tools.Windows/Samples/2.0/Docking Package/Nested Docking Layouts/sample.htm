Nested Docking Layouts

The Docking Windows framework included in Syncfusion Essential Tools provides full-fledged support for nested docking layouts; this requires no code to be included in the application to implement this feature.

This sample illustrates the implementation of nested docking layouts using user controls.

 

Features :

  • Nested Docking Layouts can be created using UserControls.
  • Visual styles for the inner layout and outer layout can be changed.
  • Drag provider style for the inner and outer layouts can be changed.

Interactive Features :

  • The panels  Marron, Orange and Blue can be dragged and docked inside the usercontrols ColorPanel1 and ColorPanel2.
  • Visual styles for the inner layout and outer layout can be applied using 'Visual Style' baritem.
  • Drag provider style or the inner and outer layouts can be changed using 'Drag Provider Style' baritem.
  • UserControls ColorPanel1 and ColorPanel2 can be tabbed together by checking the option 'Nested tabbing Off'.
