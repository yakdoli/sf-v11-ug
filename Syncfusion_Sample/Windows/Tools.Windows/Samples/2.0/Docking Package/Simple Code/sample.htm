Simple Code

This sample shows you how to initialize and set up a docking layout using the docking manager's programmatic API.

Features:

  • Creation and initialization of the docking windows are done inside the InitializeDockingWindows() method, which is invoked from its constructor. Unlike the other Docking Windows samples, the docking manager is not added to the form during design time.

  • The BeginInit() and EndInit() methods are used to indicate that the initialization has started and ended. All the initialization code should be placed between these method calls.

  • In order to convert a control as a dockable window, here we are using the DockControl method of the docking manager. Similarly, to set a control as a floatable window, the FloatControl method is used.

    The Syncfusion.Windows.Forms.Tools.DockingStyle enumeration is used to specify the dock type and position when calling the DockingManager.DockControl method. As the docking manager is only initialized programmatically, it is necessary to explicitly set its host control.
