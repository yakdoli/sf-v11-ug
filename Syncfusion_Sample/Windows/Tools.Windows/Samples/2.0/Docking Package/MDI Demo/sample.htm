MDI Demo

This sample demonstrates the behavior of the Docking architecture in a MDI environment.

Features:

Docking Control to MDI Child Interchangeability

  • The Syncfusion docking architecture provides a facility to convert the docked control to a MDI Child and vice versa. The Form.IsMdiContainer property must be set to true, which will indicate that the form is the container for MDI Child forms. 
    
	this.IsMdiContainer = true;
	this.dockingManager.SetAsMdiChild(Control ctrl,bool value);

    Value: True converts a Docked control to an MDIChild form, False converts an MDIChild form to a Docked control.
