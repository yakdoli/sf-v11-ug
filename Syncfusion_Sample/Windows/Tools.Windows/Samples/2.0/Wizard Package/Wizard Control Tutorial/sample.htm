Wizard Control Tutorial

This sample illustrates the creation and set up of a basic wizard with a set of pages using the Wizard control.

The Wizard package is a collection of controls implementing the Wizard interface—a series of tabs laid out with Next, Back, and other buttons—popularized by many Windows-based applications.

Features:

Package Components

  • Gradient Panel

    • This panel contains the "Title and Description" label and a picture box. It can be customized with gradient colors.

  • Wizard Control Page

    • Wizard control pages are added in a wizard container, which is the control where the selected wizard control page displays. Controls are laid out within the wizard control pages.
