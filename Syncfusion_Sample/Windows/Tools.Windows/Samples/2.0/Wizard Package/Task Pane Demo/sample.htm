XP Task Pane Demo

This sample illustrates how the XP Task Pane control can be used to develop a Microsoft Word-like task pane.

Features:

The XP Task Pane is a container control that can host a number of pages that can be browsed using a drop-down menu and the arrow buttons in the task pane's header. It resembles the task pane in Office XP 2002.

The XP Task Pane control was inspired by the Office XP Task Pane feature. The control internally uses a CardLayout to show one page at a time. The header panel also allows selecting one page at a time. The control provides complete design-time support.

It also supports Office 2007 visual styles including all three color schemes.
