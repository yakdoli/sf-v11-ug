Wizard Package

The Wizard package provides controls that use the Wizard interface, a series of tabs laid out with Next, Back, and other buttons, popularized by many Windows-based applications. The Wizard control lets users create applications quickly. The XP Task Pane provides a different view of Wizards than seen in Office XP.

The different samples included in the Wizard Package are listed below:

  • Wizard Control Tutorial - This sample illustrates the creation and set up of a basic wizard with a set of pages using the Wizard control.
  • Wizard Control Demo - This sample illustrates how the Wizard control can be used to create full-fledged Wizards.
  • Task Pane Demo - This sample illustrates how the XP Task Pane control can be used to develop a Microsoft Word-like task pane.
