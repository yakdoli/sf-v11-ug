ComboBoxAdv

This sample illustrates the implementation of ComboBoxAdv in Essential Tools.

Features:

  • Based on the Combo Box Base control, the ComboBoxAdv pre-binds a list box in its drop-down area and includes additional APIs to provide a framework combo box-like object-model and data binding support.

  • This control exposes the text box and the list box controls, used internally, to let you customize different parts of the control.

  • NumbersOnly - Use this property to set the input for the combo box as only numbers
    • .

  • AutoComplete - Set this property to true to enable the AutoComplete feature in the combo box.

  • CaseSensitiveAutoComplete - This property enables or disables the AutoComplete feature with case sensitivity.

  • Sorted - Use this property to show the drop-down list in a sorted manner.

  • Banner Text Settings - Provides customization options for banner text.

  • Styles - Office XP, Office 2003, VS 2005, Office 2007, and Office 2007 Outlook
    • .

  • Office 2007 Color Scheme - Supports all the three color schemes, Blue, Silver, and Black.
