Option Controls

This sample illustrates the various Option Controls and their features.

The Option controls that are shown in this sample are as follows.

  • CheckBoxAdv

  • RadioButtonAdv

    Features:

  • Office 2007 Style - Sets the Office 2007 Visual Style to Blue, Silver, or Black color schemes

  • Border Style - Sets the border style as Fixed Single, Fixed 3-D, or None

  • Border 3-D Style - Sets various 3-D border styles

  • Border Color - Sets the color of a border

  • Background settings - Allows a control to have gradient background settings

  • Read Only - Sets the text to read-only mode

  • Tri-State - Allows a check box to have three states, Checked, Unchecked, and Undetermined

  • Themes Enabled - Specifies whether or not use the XP Themes when the border style is set to Fixed3D

  • Image Check Box - Creates a check box by using the provided images

  • Glyph Align - Sets the alignment of a check box inside a control

  • Text Align - Sets the alignment of text inside a control

  • Database Binding - Binds values from a database
