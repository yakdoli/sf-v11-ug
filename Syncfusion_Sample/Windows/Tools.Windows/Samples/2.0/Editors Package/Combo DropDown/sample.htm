Combo Drop-Down

This sample illustrates the usage of Combo Drop-Down control.

Features:

  • The Combo Drop-Down control is a lightweight, combo box-like control that can host any control in the drop-down area.

  • This control can be used to create a combo box that shows, for example, a tree-view or a list-view with multiple columns, or any other control that helps in displaying data appropriately.

  • Visual styles - This control supports various visual styles including the Office 2007 style with all three color schemes.

  • Banner Text - This is used to provide cue-banner help text for the combo box drop-down editor.
