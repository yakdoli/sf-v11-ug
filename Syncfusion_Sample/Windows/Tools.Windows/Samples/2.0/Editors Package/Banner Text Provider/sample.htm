Banner Text Provider

This sample introduces a new component named Banner Text Provider.  

Given below is a sample screenshot that displays the banner text for different input controls.

Features :

  • Banner Text provider allows the user to display a cue-banner help text for various input controls like textbox-based, combo box-based, and up-down controls.
  • It also provides properties to customize the look and feel of banner text.
  • It supports two display modes, the Edit Mode and the Focus Mode.
  • In Edit mode, the banner text disappears only when the associated input control is not empty; while in Focus mode, the banner text disappears once the input control receives the focus.

Interactive Features :

  • At runtime we can see the help text displayed in all the input controls.
  • When start typing in the input controls, the banner text disappears.
  • Font styles can be changed using the comboboxes displayed.
