Font Combo Box

This sample demonstrates the implementation of the Font Combo Box control.

 

Features :

  • The Font Combo Box lists the fonts installed on a system. 
  •  It can display in Simple, Drop Down, and Drop-Down List modes.
  • The FontComboBox control provides auto completion from the list.
  • Customization options available for the ComboBox DropDown.
  • Office2007 look and feel is available for the FontComboBox control with all the three color schemes.

Interactive Features :

  • Number of items in  the dropdown can be controlled using numeric up down.
  • Dropdown style can be changed using radio buttons.
  • Autocomplete can be enabled by checkbox.
  • Banner text can be provided using 'BannerText' button.
  • The name of the selected font will be displayed in the textbox shown in the right side.
