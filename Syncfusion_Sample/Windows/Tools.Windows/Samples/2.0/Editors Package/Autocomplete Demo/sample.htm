AutoComplete Demo

This sample illustrates the usage of the AutoComplete control from a XML-based data source with multiple columns of data but without any external data source. The sample also shows how users can display and filter information based on any column in the data source.

Features:

  • The AutoComplete control also supports images for each row in a list of matching entries.

  • Use the DataSource property to set the data binding with an external source. After setting the DataSource property, use the SetDataTable() method to set the content from the DataSource property to the AutoComplete instance.

  • These images can be customized to fit the application's needs.

  • In this sample, different images are used to differentiate between different types of Web addresses (HTTP, e-mail, FTP).

  • Various modes that define the type of AutoCompletion being used on the controls are AutoSuggest, AutoAppend, Both, MultiSuggest, and MultiSuggestExtended.

  • This kind of usage is ideal for displaying history items and most recently used items.

  • Multiple text box controls share the same history information.

  • You can type a word into any of the text boxes, hit the "Enter" key, and then see that the word has been added to the history list of all the text box controls.
