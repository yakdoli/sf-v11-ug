ColorUI Controls

This sample shows the usage of the Color UI Controls (ColorUIControl, ColorPickerButton, ColorPickerUIAdv) present in Essential Tools.

Features:

Color UI Control

  • Its easy to use the color-picker tool.
  • Tab names can be set using the properties.
  • Visibility of the color groups (Pallete, Web, System, User Colors) can be set.
  • Use the ColorSelected event to apply color settings when any other color is selected.

Color Picker Button

  • It is a color UI control embedded inside a button. It shows the color UI control when clicked.
  • Tab names can be set using the properties.
  • A default color group can be set to show when the color-picker button is clicked.
  • Visibility of color groups (Pallete, Web, System, User Colors) can be set.
  • A selected color can be set as the background for the button.
  • A selected color's name or hexadecimal value can be set for the button's text property.
  • Use the ColorSelected event to apply color settings when any other color is selected.

ColorPickerUIAdv

  • It is a themed color UI control specifically made for Office 2007 applications.
  • It has Office 2007-theme colors (Blue, Black, and Silver).
  • Visual styles for the Office 2007 theme can be enabled or disabled.
  • Custom color groups can be added using the Groups Collection property. In this sample, select the "Show Groups" button to add the custom color groups.
  • The order for the groups collection present in the ColorPickerUIAdv can be changed.
