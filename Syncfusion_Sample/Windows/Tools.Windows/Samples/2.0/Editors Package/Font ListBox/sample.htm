Font List Box

This sample demonstrates the usage of Font List Box control.

Features :

  • The Font List Box lists the fonts installed on a system.

  • It has options in selecting the fonts such as single-select, multi select or not selectable.

  • Vertical and horizontal scrollbars available for the FontListBox control.

Interactive Features :

  • In this sample, the calculator button's font changes according to the font selected from the Font List Box.
