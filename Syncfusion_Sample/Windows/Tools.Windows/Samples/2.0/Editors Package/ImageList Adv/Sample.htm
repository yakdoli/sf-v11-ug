Image ListAdv

This sample illustrates the Image ListAdv control used to display images in high clarity.

Features:

The Image ListAdv control cannot be associated to the controls in the designer. However, it makes the task easier by assigning images at run time by using a single line code.

Controls with an image property that directly associates images can use the Image ListAdv's Image property.

For controls that have to be associated with an image list, the Image ListAdv should be converted via the ImageListAdv.ToImageList method.
