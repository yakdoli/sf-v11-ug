Editor Controls

This sample shows the various Editor Controls with their features.

The Editor controls are comprised of the below listed controls.

  • Currency Text Box
  • Integer Text Box
  • Double Text Box
  • Percent Text Box
  • Text BoxExt
  • Domain Up-DownExt
  • Numeric Up-DownExt
  • Currency Edit
  • Editable List
  • Masked Edit

Features:

  • Border Style - Sets the border style as fixed single, fixed 3-D, or none.

  • Border 3-D Style - Sets various 3-D boarder styles.

  • Positive Color - Sets the color of the positive value.

  • Negative Color - Sets the color of the negative value.

  • Border Color - Sets the color of the border.

  • Banner Text - Provides cue-banner help text for input, up-down, and text-box based controls. For the extended controls—such as Currency Text Box, Integer Text Box, Double Text Box and Percent Text Box—the banner text will work only when their UseNullString property is set to true and its NullString property is empty.

  • Text Align - Sets the alignment of the text inside the control.

  • Culture Settings - Sets the culture that is to be used for formatting the numeric display.

  • Themes Enabled - Specifies whether or not to use the XP Themes when border style is fixed 3-D.

  • Show Overflow Indicator - Indicates that the length of the text entered has gone over its original capacity.

  • Overflow Indicator ToolTip - Shows ToolTip text for the overflow indicator.

  • Database Binding - Binds the value from a database.

  • Office 2007 Style - Sets the Office 2007 Visual Style with Blue, Silver, or Black color schemes.

  • Intercept Arrow Keys - Indicates whether the up-down control will increase or decrease the value when the "UP ARROW" and "DOWN ARROW" keys are pressed.

  • Multiline - Allows a control to have multiple lines of text.

  • Word Wrap - Wraps text around the edges of the text box and shows the full content.

  • Scroll Bars - Sets the scroll bars to scroll through text.

  • Character Casing - Sets the case for the characters present in the text box.

  • Right-to-Left - Sets the text display from right to left.

  • Read Only - Sets the text to read-only mode.

  • Show Calculator - Shows or hides the calculator button.

  • Transfer from Calculator - Specifies whether to transfer the calculated value to the editor control.

  • Transfer to Calculator - Specifies whether to transfer the value from the editor control to the calculator.

  • Want Button - Specifies whether a button should be visible when the control is in Editable mode.
