Combo Box Base

This sample demonstrates data binding and adding list controls to the combo box base.

The Combo Box Base is an advanced combo box control that provides many improvements over the standard combo box control. It essentially separates the edit portion from the drop-down portion, thereby making this architecture powerful and flexible. Users may plug-in any list control-derived class as the list for the portion of the combo box using the ListControl property.

Features:

  • The Text property of the Combo Box Base can be bound to a database as shown below.
    

this.comboBoxBase.DataBindings.Add("Text",this.dataSet11.Products,"Discontinued");
  • Use the ListControl property to set any list control to the Combo Box Base.

  • Any list control that is already bounded can be attached to the Combo Box Base. Use the SelectionChangeCommitted event when the text property of the Combo Box Base changes.
