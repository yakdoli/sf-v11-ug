Button Edit

This sample illustrates the use of the Button Edit control.

Features:

  • The Button Edit control combines a text-box control with a collection of button controls that can be customized to create many commonly used interfaces such as a file-folder browser or a drop-down text control.


  • This sample shows a file picker and a folder browser using the Button Edit control.


  • Drop-down pop-up controls are also shown in this sample using the Button Edit control and the Pop-up Control Container control.
