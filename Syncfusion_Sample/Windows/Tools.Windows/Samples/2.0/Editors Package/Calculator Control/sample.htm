Calculator Control

This sample shows the implementation of a calculator control.

Features:

  • Calculator controls are used to perform standard arithmetic operations.

  • They can be displayed in two display modes: Windows Standard layout and Financial layout.

  • There is also a pop-up calculator that is shown in this sample. The calculator control provides full keyboard support in both regular and pop-up modes.

  • Vertical spacing and horizontal spacing between the calculator buttons can be set through the numeric up-down controls.

  • TextAlign can be set to left, right, or center by toggling the radio button controls.

  • Also, this sample demonstrates the two different layouts, Financial and Windows Standard, supported by the Syncfusion Calculator control.
