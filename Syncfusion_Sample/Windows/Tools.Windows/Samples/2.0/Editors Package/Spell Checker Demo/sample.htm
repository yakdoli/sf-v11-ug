Spell Checker Demo

This sample demonstrates how a SpellChecker can be associated with the RichTextBox control, adding proofreading functionality.

Features:

  • A custom dictionary editor that can add or remove words from the dictionary during run time.

  • Options that can be customized during run time with the Options dialog.

  • The ability to make editable controls compatible with the spelling checker by wrapping them with SpellEditorWrapper, which takes an instance of TextBoxBase as an argument, and passes it as a parameter to the Spellchecker.
