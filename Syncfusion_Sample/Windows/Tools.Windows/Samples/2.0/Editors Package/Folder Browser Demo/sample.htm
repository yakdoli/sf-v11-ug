Folder Browser Demo

This sample illustrates the usage of the Folder Browser control.

Features:

The FolderBrowser class implements a convenient and easy-to-use wrapper for the folder browser Win 32 Shell API.

This sample shows how you can use Enums for StartLocation and Styles.

The different kinds of styles present are given below.

  • ShowAdminShares - Shows the administrator shares, if the user has privileges to view it

  • BrowseForEverything - Shows all the files and folders in a dialog box

  • BrowseForPrinter - Shows the path to find printers connected to the system or in the network.

  • BrowseForComputer - Allows you to browse for other computers present in the network

  • RestrictToFileSystem - Restricts viewing files related only to the File System

  • AllowUrls - Allows users to enter a URL to browse

  • RestrictToSubFolder - Allows users to view only up to sub-folders

  • ShowTextBox - Shows a text box used to enter the path of a file or folder

  • Validate - Validates an entered path

  • RestrictToDomain - Allows users to view files allotted only to their domain

  • FolderBrowserFolder.SelectLocation - Used to automatically scroll and highlight a path
