Split ContainerAdv

This sample illustrates the use of the SplitContainerAdv control. This control is compatible with 1.1 and higher frameworks only.

The SplitContainerAdv control is similar to the split-container control in the .NET 2.0 framework. It has two panels separated by a splitter. The SplitContainerAdv control allows you to set all the child controls that could share a vertical and horizontal space.

Features:

Collapsible Panels

Each panel can be hidden using the Panel1Collapsed and Panel2Collapsed properties. The Panel1MinSize and Panel2MinSize properties are used to specify the minimum size of the splitter panel instances in the SplitContainerAdv, effectively controlling how close the splitter bar can be placed to either edge of the control.

Splitter

A splitter can be fixed by setting the IsSplitterFixed property to true. Also, the FixedPanel property allows you to select which panel in the control should not resize as the SplitContainerAdv instance resizes. SplitterDistance and Orientation properties allow you to set the starting location and direction of the splitter bar.

Users can add more SplitContainerAdv controls onto any one of the panels in another SplitContainerAdv control and can change the orientation to get a desired look.
