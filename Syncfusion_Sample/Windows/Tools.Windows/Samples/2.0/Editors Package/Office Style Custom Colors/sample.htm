Office 2007 Custom Colors

 This sample illustrates the custom colors feature.

Features :

  • Essential Tools comes with the ability to customize the color theme of Office 2007-styled controls.

Interactive Features :

  • Custom color can be applied at runtime using 'Custom Colors 'dropdown button.
  • Custom color can be set to all the controls in a form by using the following single line of code.

//Applies custom colors for syncfusion controls whose color scheme
//is set as Managed and Visual styles set as Office2007
Office2007Colors.ApplyManagedColors(this, Color.Violet);

Note: To apply custom colors, the visual styles of the control need to be set as Office2007 and the color scheme should be set as Managed.
