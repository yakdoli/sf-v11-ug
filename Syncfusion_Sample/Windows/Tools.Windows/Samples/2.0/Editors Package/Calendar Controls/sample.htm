Calendar Controls

This sample demonstrates the usage of MonthCalendarAdv and DateTimePickerAdv controls in Essential Tools.

The MonthCalendarAdv control is an advanced calendar control that can display all the months of a year.

Features:

This sample shows the various features of MonthCalendarAdv as listed below.

  • Culture settings
  • Ability to set first day of the week
  • Show week numbers
  • Support for multiple selection and mouse-drag selection
  • Visual styles
  • Themes support
  • Ability to set images for left and right scroll buttons
  • Ability to set colors for various units of the control

The DateTimePickerAdv control can be set to any culture available on the host system and customized as per requirements.

This sample shows the various features of DateTimePickerAdv as listed below.

  • Visual styles
  • Enhanced menu support.
  • Themes support
  • Ability to show check box and up-down buttons
  • Various default and custom formats.
  • Culture settings
  • Null mode key reset
  • Read-only and right-to-left support
  • Custom pop-up window
  • Data-binding support
