Editors Package

The Editors Package is a comprehensive collection of controls commonly used in creating data-entry forms. All the editor controls have been architected to provide a simple and powerful means for capturing data input, validating data, and displaying it with the appropriate formatting. They also come with full data-binding, internationalization, and localization support.

The different samples included in this Package are listed below:

  • AutoComplete Demo - This sample illustrates the usage of the AutoComplete control from a XML-based data source.
  • ButtonAdv - This sample demonstrates the implementation of a ButtonAdv control in Essential Tools.
  • Button Edit - This sample illustrates the use of the Button Edit control.
  • Calendar Controls - This sample demonstrates the usage of MonthCalendarAdv and DateTimePickerAdv controls in Essential Tools.
  • Calculator Control - This sample shows the implementation of a calculator control.
  • Color UI Controls- This sample shows the usage of the Color UI Controls present in Essential Tools.
  • Banner Text Provider - This sample illustrates the usage of the Banner Text Provider. This component allows the user to display cue-banner help text for various input controls.
  • ComboBoxAdv - This sample illustrates the implementation of ComboBoxAdv in Essential Tools.
  • Combo Box AutoComplete - This sample demonstrates the usage of the Combo Box AutoComplete control.
  • Combo Box Base - This sample demonstrates data binding and adding of list controls to the combo box base.
  • Combo Drop-Down - This sample illustrates the usage of Combo Drop-Down control.
  • Container controls - The Editors Package has a set of container controls. These samples illustrate the container controls in Editors Package.
  • Editor Controls - This sample shows the various editor controls with their features.
  • Folder Browser Demo - This sample illustrates the usage of the Folder Browser control.
  • Font Combo Box - This sample demonstrates the implementation of the Font Combo Box control.
  • Font List Box - This sample demonstrates the usage of Font List Box control.
  • Gradient Label - This sample demonstrates the implementation of the Gradient Label control.
  • ImageListAdv - This sample illustrates the ImageListAdv control used to display images in high clarity.
  • Multicolumn Combo Box - This sample illustrates the usage of a Multicolumn Combo Box control.
  • Option Controls - This sample illustrates the various Option Controls and their features.
  • Office 2007 Custom Colors - Essential Tools comes with the ability to customize the color theme of Office 2007-styled controls. This sample illustrates the custom colors feature.
  • Spell Checker Demo - This sample demonstrates how to associate a SpellChecker control with a rich text box.
  • SplitButton Demo - This sample demonstrates how to use the SplitButton Control.
