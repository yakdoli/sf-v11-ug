Custom Task Bar

This sample illustrates how the XP TaskBar can be customized to get an MSN Messenger-like look; it also illustrates how to place images in the "Expander" button.


   
MSN Messenger-Like Task Bar    Custom Task Bar with Image in Expander Button

Features:

  • MSN Messenger Appearance

    This shows how to add an image to the XP TaskBar background and how to get a gradient task-bar background.

    Three properties (GradientStartColor, GradientEndColor, BlendImage) are created to change the gradient color and image during design time.

    Also, this sample shows how to customize the XP Task Bar Box header using the ProvideHeaderBackgroundBrush event.

  • Custom XP Task Bar Box with Image in Expander Button

    The XP TaskBar Box is customized to show images in the "Expander" button. The header is filled with gradient colors by using the BrushInfo class; borders can be drawn in its Paint event.
