Navigation View

Breadcrumbs or breadcrumb trails are a navigation technique used in user interfaces. Their purpose is to give users a way to keep track of locations within programs or documents; they are mainly used in the Vista environment.

Essential Tools provides a NavigationView control that uses the breadcrumbs technique to keep track of locations and folders.

This sample navigates the directories while keeping track of the paths. Also, the sample exposes the features of NavigationView control.

   

Features:

  • Images Support

    The NavigationView control provides image support for both root bars and child bars. The NavigationView control displays the selected bar's image in the left of the control.

  • History Manager

    Using HistoryManager, we can keep track of each folder we traverse. Each folder that is navigated through is shown in the address bar separated by a forward arrow. If we click on any one of these arrows, it will instantly turn into a drop-down list showing us all the folders directly below the one you currently have open.

  • Custom Button Support

    You can customize the NavigationView by adding custom buttons to it.

  • Visual Styles

    NavigationView supports the Office 2007 visual style and the Vista visual style.
