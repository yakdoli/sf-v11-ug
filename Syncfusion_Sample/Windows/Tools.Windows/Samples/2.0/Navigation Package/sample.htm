Navigation Package

The Navigation package includes controls—such as the Group Bar, Group View, and the XP TaskBar—that are primarily used for implementing command lists where each command is represented by an image or text. Popular examples of command lists are the Microsoft Outlook Bar, VS .NET Toolbox, Windows 2000/XP Folder Selectors, and the Windows XP Task Bar. 

A group bar implements a tab-type container that can serve as a host for a collection of group views or other Windows Forms controls. A group view is an extensively customizable list-type control capable of cloning popular command interfaces. The XP TaskBar can be used to display standard command items or as a container that can host several other controls.

The different samples included in the Navigation Package are listed below:

  • Group Bar Demo - This sample illustrates the usage of a Group Bar control that creates clones of the Microsoft Outlook Bar and VS .NET Toolbox windows.
  • GroupView Demo - This sample illustrates the implementation of list-type controls that facilitate navigation within an application's user-interface.
  • Custom Task Bar - This sample illustrates how the XP TaskBar can be customized to get an MSN Messenger-like look; it also illustrates how to place images in the "Expander" button.
  • Navigation View - This sample navigates the directories and keeps track of the path we are navigating. Also, the sample exposes the features of NavigationView control.
  • Task Bar - This sample illustrates the use of the XP TaskBar and XP TaskBar boxes to simulate the latest task-bar feature in XP.
  • Range Slider - This sample illustrates the use of the Range sliders ability of filtering the records with dual thumbnails.
