XP TaskBar

This sample illustrates the use of the XP TaskBar and XP TaskBar boxes to simulate the latest task-bar feature in XP.

XP Task Bar is a part of the Navigation Package that is used to implement list-type controls for facilitating navigation within an application's user-interface. This is frequently encountered in the Windows XP Task Bar.

The XP Task Bar can be used to display both standard command-type items or as a container hosting other controls.

Features

  • Show-Hide Task Bar Items

Task bar items can be hidden by setting the Visible property to false.

  • Use Additional Animation

Using the XP TaskBar box's UseAdditionalAnimation property, you can provide animation when XP TaskBar items are added, removed, or hidden.

  • Enable-Disable Task Bar Items

Task bar Items can be enabled or disabled by using the Enabled property.

  • ToolTip Functionality

ToolTips can be provided for XP TaskBar items as well as disabled task bar items.

  • Drag-Drop Operation

 Task bar items can easily be dragged and dropped between XP TaskBar boxes.

  • Header Text Align and Header Direction

Header text alignment can be set to Near, Middle, or Far.
Header direction can be right-to-left or left-to-right.

  • Large Image Support

Large icons, such as 16x16 icons or 32x32 icons, can be used in the XP TaskBar header as well as in XP TaskBar items.

  • Animation Delay and Animation Position Count

This feature specifies the animation delay and the number of animation positions during expand-collapse operations.

  • Style

The new Office 2007 style has been added with three color schemes (Blue, Silver, and Black).

  • Support for Gradient Background Colors

This feature provides events, such as ProvideHeaderBackgroundBrush and ProvideItemsBackgroundBrush, that are used to set the header gradient background color and item gradient background color, respectively.
