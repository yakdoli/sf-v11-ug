Group Bar Demo

The Group Bar is a part of the Navigation package that is used to implement list-type controls for facilitating navigation within an application's user interface. This function is frequently seen in the Microsoft Outlook Navigation bar.

It implements a tab-type container that can serve as a host for a collection of group views or other Windows Forms controls.

This sample illustrates the usage of a Group Bar control that creates clones of the Microsoft Outlook Bar and VS .NET Toolbox windows.

Appearance Settings

  1. Background Color
    The background color for the Group Bar and the Group View is set using the BackColor property, as shown. 

    
 this.gbVisualStudio.BackColor = cpBack.SelectedColor;
 this.GeneralGroupView.BackColor = Color.White;

  2. Border

    • Group Bar Border - The border for the Group Bar can be specified in the BorderStyle property. This border is drawn as the outer rectangle for an entire Group Bar control. 
      
this.gbVisualStudio.BorderStyle =  BorderStyle.FixedSingle
    • GroupBar ClientBorder - The client area of each Group Bar item can have borders of different colors for each side by using the DrawClientBorder property. 
      
this.gbVisualStudio.DrawClientBorder =  this.cbDrawClientBorder.Checked;
            foreach (GroupBarItem gbItem in  this.gbVisualStudio.GroupBarItems)
            {
                gbItem.ClientBorderColors = new  Syncfusion.Windows.Forms.Tools.BorderColors(cpBack.Sele ctedColor, cpBack.SelectedColor, cpBack.SelectedColor,  cpBack.SelectedColor);
            }

  3. Visual Styles
    The Group Bar provides options for various visual styles such as Default, Office 2003, and Office 2007. 

    
this.gbVisualStudio.VisualStyle =  VisualStyle.Office2003

  4. Color Schemes
    The Group Bar supports all three Office 2007 color schemes as well as custom colors. 

    
this.gbOutlook.Office2007Theme = Office2007Theme.Blue;

Behavior Settings

  • Animated Selection
    Switching between two Group Bar items can be animated by using the AnimatedSelection property. 
    
 this.gbVisualStudio.AnimatedSelection  = true;

  • Integrated Scrolling
    Integrated scroll buttons of the Group Bar can scroll items in the group view by setting the IntegratedScrolling property. 
    
this.gbVisualStudio.IntegratedScrolling = true;

  • Stacked Group Bar
    This is a full-featured implementation of a "Navigation Pane Options" dialog that allows users to show, hide, or move Group Bar items; it is similar to the functionality available in Microsoft Outlook 2007.

    • Stacked Mode
      Group Bar items can be stacked, similar to the Outlook Bar, by using the StackedMode property. 

      
this.gbOutlook.StackedMode = true;

    • Chevron in Group Bar
      When the Group Bar is in Stacked mode, the chevron button can be visible by using the ShowChevron property. 

      
 this.gbOutlook.ShowChevron = true;

    • Item Image in Header
      Images associated with a Group Bar item can be displayed in the Group Bar header when a particular item is selected; this is done using the ShowItemImageInHeader property. 

      
this.gbOutlook.ShowItemImageInHeader = true;

  • Collapsible Group Bar
    A stacked Group Bar can be collapsed or expanded to save application space. The Essential Group Bar provides various customization options for this feature.

    • AllowCollapse indicates whether a group bar can be collapsed.
    • Collapsed indicates whether a group bar is collapsed.
    • CollapsedText indicates what text is to be shown in the collapsed client area of the group bar.
    • CollapsedWidth allows setting the width of a collapsed group bar.
    • CollapseImage allows setting the image of a collapse button when in expanded state.
    • ExpandButtonToolTip is the ToolTip for the collapse button when the control is collapsed.
    • ExpandImage allows setting the image of the collapse button when in the collapsed state.
    • MinimizeButtonToolTip is the ToolTip for the collapse button when the control is expanded.
    • NavigationPaneToolTip is the ToolTip for a navigation pane.
    • PopupAutoClose indicates whether a pop-up should close when clicking on another Group Bar item.
    • PopupClientSize is used to set the size of the pop-up window.
    • PopupResizeMode indicates how the pop-up window can be resized. It allows horizontal resizing, vertical resizing, both, or none of these options.
    • ShowPopupGripper is used to show a pop-up windows gripper.
    • BeforePopup is an event that occurs before the Group Bar item’s pop-up is shown.
    • StateChanging is an event that is triggered when a state is about to change.
    • StateChanged is an event that occurs when a state is changed.

Events

The Group Bar provides an option to add, remove, or rename Group Bar items at run time. A context menu can be associated with a Group Bar to perform user-defined actions. Various events supported by the Group Bar are as follows.

  • GroupBarItemAdded
  • ProvideGroupBarItemBrush
  • GroupBarItemSelectionChanging
  • GroupBarItemSelected
  • GroupBarItemAdded
  • GroupBarItemRenamed
  • GroupBarItemRemoved
  • ShowContextMenu
  • NavigationPaneDropDownClick
