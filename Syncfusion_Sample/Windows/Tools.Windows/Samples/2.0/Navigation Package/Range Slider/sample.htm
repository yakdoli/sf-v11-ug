Range Slider

A range slider allows users to dynamicly change results by narrowing a range. Changes can be made with the dual thumbs of the slider to lower, upper, or both values.

This sample illustrates the various properties of the range slider, which will easily fetch records within a particular range.

Features

Settings

  • The size of the channel and slider can be changed.
  • Tick frequency can be set and slider ticks can be shown or hidden.
  • The ValueChanged event of the slider occurs when the thumb is moved, enabling users to get the slide minimum and maximum value range.

Colors

  • The range slider provides various properties to customize the colors of the range, thumb, or channel.

Interactive Features

  • Move the thumb in the range slider to select a particular range of records from the database and display those records in the grid.
  • Use the Settings tab to change the size of the slider and thumb.
  • Use the Color tab to customize the range slider in various colors.
