Group View Demo

The Group View is a part of the Navigation package that may be used for implementing list-type controls that facilitate navigation within an application's user-interface.

This functionality is often seen in the Microsoft Outlook Navigation bar, the VS.NET toolbox, the Windows 2000/XP Folder Selector list, and the Windows XP Task Bar.





Features:

Group View Properties and Events

The Group View comes with a large set of properties and events for extensive customization.

  1. Color Settings

    Other than the default color settings for backgrounds and foregrounds, the Group View provides an option for setting custom colors for actions like selecting and highlighting.

  2. Text Settings

    • It provides an option to wrap text when the Group View width is reduced.
    • It has an option used to set underlining text when an item is highlighted.
    • It also provides an option to set text offset and text spacing.

  3. Items Settings

    • Flow View
      Group View items can be ordered in flow-view or list-view by using the FlowView property.
    • Button View
      Group View items can also be drawn with a button-type flat look using the ButtonView property.

  4. Group View Appearance

    • Highlight Image and Text
      When Group View items are highlighted, the Group View provides an option to highlight the text area, the image area, or both.
    • ToolTips
      The ToolTip of a Group View item can be made visible with the ShowToolTips property.
    • Integrated Scrolling
      Items in the Group View can be scrolled by integrated scrollers of the Group View control. If the Group View is associated with any of the parent controls, such as the group bar, the IntegratedScrolling property will delegate the scrolling functionality to the parent control. When IntegratedScrolling is set to false, Group View items will be scrollable.

  5. Group View Events

    The Group View provides a wide range of events, from default events to advanced collections such as GroupViewItemHighlighted, GroupViewItemRenamed, GroupViewItemsReordered, ShowContextMenu, and GroupViewItemSelected.

Group View Modes

The Group View control can be customized to create a wide range of interfaces by using various appearance and behavior attributes.  Such interfaces include lists with large or small images, regular or flow views, button-type selections, flat looks, and selective item highlighting or underlining.

The Group View has the ability to clone the following.

  • Microsoft Outlook Bar with Large Icon View
  • Microsoft Outlook Bar with Small Icon View
  • VS.NET Tool Box List View
  • VS.NET Tool Box Flow View
  • Office 2000 Folder Browser
  • Office XP Folder Browser
  • MSN Messenger Side-Bar List
  • XP Task-Bar List
