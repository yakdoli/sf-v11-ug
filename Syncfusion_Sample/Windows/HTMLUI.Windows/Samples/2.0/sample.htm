HTMLUI Control Samples

There are some Essential HTMLUI samples that are shipped with the product. They step you through the important features, their functionality, and their implementation process.

The samples in Essential HTMLUI are categorized based on their functionality.

Loading HTML

The HTMLUI control is so flexible that HTML documents from various sources are easily loaded into it. This section focuses on the important methods of loading HTML documents into the HTMLUI Control.

HTMLUI Tags

The HTMLUI control supports a wide range of HTML tags. This section focuses on the different tags and customizable bookmarks support that come with the Essential HTMLUI.

HTMLUI Appearance

Not only does HTMLUI support HTML tags but also popular Web browsers, their attributes, and style references. The HTMLUI control’s support for applying styles to a document in different ways is the focus of this section.

HTML Layout

The HTMLUI control reduces the difficulties of the developers in creating custom, rich user-interactive interfaces. The samples in this section show how flexible day-to-day applications are developed with the help of HTMLUI and a HTML document alone.

HTMLUI Renderer

The HTML documents available from the resources like URI or HTML documents in the form of string can be loaded into the HTMLUI control. Also, the documents available in the other complex formats can be converted into customized HTML documents and loaded into the HTMLUI control.

HTMLUI Events

The HTMLUI control is made as user interactive as possible. The events section focuses on the different events that are associated with the HTMLUI control that are not only user interactive, but also informatory. With the help of such powerful events support, HTMLUI helps the developer in creating advanced UI applications easily.

HTMLUI Custom Controls

With the HTMLUI control, it is much easier to use the user-defined controls with the user-defined HTML tags in the HTML document. This type of custom controls usage helps in developing an advanced user-interactive application with the coordination of different user-defined controls.

HTMLUI Format

HTMLUI provides rich formatting options to the HTML elements with the help of the HTML "Format" object. Not only is formatting done easily, but with the HTMLUI control, it is easier to access the characteristics of the HTML elements like location, size, etc.; this helps the developer to decide on the length and the complexity of the application.

HTMLUI User Interaction

The HTMLUI control is completely user interactive. The HTML elements in the HTML document, when rendered in the HTMLUI control, are made to support various UI events like a common Windows Forms control. Also, some common options like flexible keyboard shortcuts, customizable scroll options, advanced text selection options, make the control elastic in any circumstance.

HTMLUI Exporting

This section deals with the conversion of the documents in the HTMLUI control to different formats. A HTML document available in the HTMLUI control can be converted to an image and exported to various formats. Also, the HTMLUI control supports printing along with the print preview options.
