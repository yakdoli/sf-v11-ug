Data-Bind HTML

This sample illustrates how the CSS styles are loaded from a database and applied to the HTMLUI control.

Features:

  • This sample applies the CSS styles to the HTML file.

  • The CSS styles are accessed from the database and applied to the .htm file.

  • There are various CSS styles supported by the HTMLUI control.
