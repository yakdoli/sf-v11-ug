Formatting Demo

This sample shows how the styles are applied using the HTMLFormat object.

Features:

  • The HTML format is applied to the elements through the HTMLFormat object.

  • The styles to the HTML elements, like font, color, background color, etc., can be defined at run time and applied to the elements.

  • The HTMLUI control also allows changing the visibility of HTML elements at run time using the xVisible attribute.


Here is the code to define the HTML format with styles that can be applied to the HTML elements in the HTMLUI control.

 
	
	HTMLFormat format = new HTMLFormat("FormatGreen");

	format.Font = new Font("Verdana",12);

	format.ForeColor = Color.Blue;

	div.Format = format;
