Element Coordinates

This sample shows how to determine the location and the size of the HTML element in HTMLUI.

Features:

  • The HTMLUI control allows the user to access the position of the elements displayed on the control.

  • The user can also access the size and location of elements using the Size, Width, and Height properties associated with the element.


Here is the code to get the HTML element's X and Y coordinates of the selected description tag.

 
	private void description_Click( object sender, EventArgs e )
	{
		BaseElement element = sender as BaseElement;
		beginPoint = element.Location;
		endPoint = new Point(beginPoint.X+element.Size.Width,beginPoint.Y+element.Size.Height);
		this.label2.Text = "Element begins at " + beginPoint.ToString() + " and ends at " +  endPoint.ToString();
		this.label3.Text = "Element's Height\n" + element.Height.ToString() + "\n\nElement's Width\n" + element.Width.ToString();
		this.htmluiControl1.Refresh();
	}
