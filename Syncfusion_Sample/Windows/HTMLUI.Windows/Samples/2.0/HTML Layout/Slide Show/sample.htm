Slide Show

This sample shows how HTMLUI can be used as an advanced presentation engine that executes a slide-show presentation at a specified time delay.

Features:

  • This sample uses the Timer class to generate an event at fixed intervals.

  • HTML documents are loaded sequentially into the HTMLUI control between the specified time intervals.

  • Also, it features the use of Start, Stop, and Reset applications with the use of Timer in presentation applications.

Here is the code to start the timer. 
    

//To Start the timer.
timeDelay = 1000;
if(this.textBox1.Text!="")
{
	timeDelay = (Convert.ToInt32(this.textBox1.Text)*1000);
}
int secTimeDelay = (timeDelay/1000);
this.textBox1.Text = secTimeDelay.ToString();
Console.WriteLine("Timer Started. Time delay is " + secTimeDelay.ToString());
timer.Interval = timeDelay;
timer.Start();
timer.Tick += new EventHandler(timer_Tick);


//To stop the timer.

timer.Stop();
timer.Tick -= new EventHandler(timer_Tick);
