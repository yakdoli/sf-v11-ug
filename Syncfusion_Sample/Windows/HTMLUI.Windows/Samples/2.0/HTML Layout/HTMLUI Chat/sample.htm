HTMLUI Chat

This sample shows how HTMLUI can be used to create a chat application.

Features:

  • HTMLUI is helpful in creating cool UI applications.
  • The HTMLUI control helps the user to change text content at run time. The user can convert text to an image as needed.
  • Also, with the HTMLUI control, the user can save changes made to the rendered document as XML and retrieve the same whenever needed.
  • The user can also customize the position of the Scroll control in UI applications according to the needs of those applications.

//To get the HTML Element present in the .htm file.


	this.htmluiControl1.LoadFromString(OuterXML);
	IHTMLElement[] element = this.htmluiControl1.Document.GetElementsByName("body");
	IHTMLElementsCollection collection = element[0].Children;

	foreach(IHTMLElement elem in collection)
	{
		WriteElement(elem);
	}
	return this.htmluiControl1.Document.Document.OuterXml;
