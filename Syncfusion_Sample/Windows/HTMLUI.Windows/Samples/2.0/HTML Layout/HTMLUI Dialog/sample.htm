HTMLUI Dialog

This sample shows how to create a form-based dialog using the HTMLUI control.

Features:

  • Only one form with a HTMLUI control is used to create a multiple dialog application.
  • The application separates the display definition into HTML and the logic into C# code.
  • The flexible nature of HTML makes it easy to display validation messages.
