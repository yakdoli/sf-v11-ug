Scripting

This sample illustrates the support of self-contained HTML documents by the HTMLUI control.

Features:

  • HTMLUI supports scripting that helps the user in writing self-contained codes, thereby improving the usability of the application.
  • This sample contains a script to replace the given text to an image on the click of a button.
  • HTML elements can be made to interact with the help of the script embedded inside the HTML document.
