HTML Renderer

HTMLUI supports the rendering of HTML contents from various resources.

  • HTMLUI Editor: This sample shows how the HTMLUI control renders HTML content  available as text in the control.
  • HTMLUI Explorer: This sample shows how the HTML content available in the URI is rendered in the HTMLUI control.
  • Scripting: This sample illustrates the support of self-contained HTML documents by the HTMLUI control.
  • XSL Transform: This sample shows how a XML document can be converted to HTML using XSLT and loaded into the HTMLUI control.
