HTMLUI Editor

This sample shows how the HTMLUI control renders HTML content available as text in the control.

Features:

  • The HTML content available in the form of text can be loaded in the HTMLUI control with the help of the LoadFromString(string html) method of the HTMLUI control.
  • With the HTMLUI control, users can access the HTML elements individually at run time and they can even change the text with the InnerHTML() method that is available in the control.
  • The HTMLUIControl.Document class returns the XML document object that is specific to the HTML content available in the control. This can also be accessed and changed in order to change the text available in the control.

Here is the code to load from the string.

    this.htmluiControl1.LoadFromString(htmlString);
