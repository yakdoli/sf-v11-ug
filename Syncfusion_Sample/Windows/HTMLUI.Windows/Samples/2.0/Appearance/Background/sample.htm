Background

This sample shows how to add background color to an application in HTMLUI.

Features:

  • HTMLUI allows the customization of background color in the application by either including it within the tag or using the styles.

  • Background color in the HTML document can be applied at run time by including a style attribute or linking it to an external style sheet.

Based on the check box values, the corresponding .htm files with their styles are loaded into the HTMLUI control.

 
	if(this.checkBox2.Checked)
