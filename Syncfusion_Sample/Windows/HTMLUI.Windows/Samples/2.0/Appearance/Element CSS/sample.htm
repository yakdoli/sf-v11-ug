Element CSS

This sample shows how a style is applied to a HTML element in the HTMLUI control at run time.

Features:

  • The HTMLUI control allows the user to add an attribute to a HTML element and also access its values at runtime with the help of the Attributes property associated with the HTML element.

  • By accessing this property, the user can check whether the specified attribute is present in the HTML element with the help of the elem.Attributes.Contains(string attributeName) method.

  • If the specified attribute is not present, the user can add the attribute with the help of the Attributes.Add(string attributeName) method and apply the required style.


Here is the code that adds the attribute at run time and sets the style for the element.

   
	BaseElement elem = this.htmluiControl1.GetElementAtClientPoint(e.X,e.Y) as BaseElement;
	if(elem.Attributes.Contains("class")==false)
	{
		elem.Attributes.Add("class");
	}
	if(elem.Attributes["class"] != null)
	{
		if(radioButton1.Checked == true)
		{
			elem.Attributes["class"].Value = "tttRed";
		}
		else if(radioButton2.Checked == true)
		{
			elem.Attributes["class"].Value = "tttBlack";
		}
		else
		{
			MessageBox.Show("Check a style for the elements display.");
		}
	}
