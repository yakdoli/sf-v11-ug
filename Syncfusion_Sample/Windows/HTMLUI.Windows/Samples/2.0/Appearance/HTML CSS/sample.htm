HTML CSS

This sample illustrates the Cascading Style Sheets support in HTMLUI.

    Features:

  • The support for style sheets in HTMLUI enables the user to define styles for HTML elements and decide on the appearance of the HTML elements in the application.

  • HTMLUI supports the use of three types of style sheets. They are: (1) External Style Sheets (2) Internal Style Sheets (3) Inline Style Sheets.

  • External Style Sheets are linked to the file through the Link tag. Internal Style Sheets are applied with the help of the Style tag inside the head section while Inline style sheets are applied as the values of the style attributes of the specific HTML element.
