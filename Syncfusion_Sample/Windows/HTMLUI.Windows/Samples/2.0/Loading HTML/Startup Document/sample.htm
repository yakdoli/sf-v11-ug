Startup Document

This sample shows how a startup HTML document can be loaded into the HTMLUI control.

Features:

  • The startup documents are front-end presentations that are loaded at the time of startup.
  • The user can specify the startup HTML file name with its location in the StartupDocument property either in the properties window or through code.

Here is the code to set startup_page.htm as the startup document for the HTMLUI control.

    this.htmluiControl1.StartupDocument = @"C:\MyProjects\Startup\startup_page.htm";
