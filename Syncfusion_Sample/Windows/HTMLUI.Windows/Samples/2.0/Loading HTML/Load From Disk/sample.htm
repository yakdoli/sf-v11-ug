Load from Disk

With the HTMLUI control, any HTML file from the local drive can be easily loaded into the control.

Features:

  • Any HTML file from the user drive can be loaded into the HTMLUI control by specifying the location of the document as a string in the LoadHTML() method.

  • The HTMLUI control can also load MHTML files from the local drive.
  • The user can customize the HTMLUI control to load the documents into the control by dragging and dropping them onto it.

Here is the sample code that gets the file path of the given file name from the disk and loads it into the HTMLUI control.


    string filePath = GetFilesLocation() + fileName;
this.htmluiControl1.LoadHTML(filePath);



    //GetFilesLocation is the method used to get the full path of the file 
located in the disk.


    //Filename is the name of the file to be loaded from disk.




    static string GetFilesLocation()
{
	if(filePath != null)
		return filePath;

	string path;
	path = System.IO.Path.GetDirectoryName(System.Reflection.Assembly.GetExecutingAssembly().GetName().CodeBase );
	int slashPos = path.LastIndexOf("\\",path.Length);
	slashPos = path.LastIndexOf("\\",slashPos-1);
	slashPos = path.LastIndexOf("\\",slashPos-1);
	if(slashPos > 0)
		path = path.Substring(0,slashPos);
	path = path.Replace("file:\\", "");
	filePath = path + "\\files\\";
	return filePath;
}
