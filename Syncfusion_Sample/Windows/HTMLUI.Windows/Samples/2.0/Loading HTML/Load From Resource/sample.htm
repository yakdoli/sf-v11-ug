Load from Resource

This sample shows how a HTML file that is embedded in a resource is loaded into the HTMLUI control.

Filtering Tutorial

  • A HTML file that is loaded into the embedded resource is accessed in HTMLUI by loading the HTML file from the specified stream where the HTML document is placed.
  • The LoadHTML() method of the HTMLUI control can load HTML documents from the specified stream.

Here is the code to load the specified HTML file that is marked as the project's embedded resource.

     
Assembly assembly = Assembly.GetExecutingAssembly();

Stream htmlStream = (Stream)Assembly.GetExecutingAssembly().GetManifestResourceStream(LoadingFileFromResource.resfile.htm"); 
                    
this.htmluiControl1.LoadHTML(htmlStream);
