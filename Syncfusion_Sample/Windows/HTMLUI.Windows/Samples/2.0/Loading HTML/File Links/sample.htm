File Links

This sample shows how links are supported in HTMLUI.

Features:

  • HTMLUI supports linking to various documents either from the user drive or from a URI, which helps in developing user friendly applications like bookmarking, referencing, etc.

  • HTMLUI allows the use of Tab key shortcut to navigate through the links, and the Enter key to open the link that is focused.

  • Images instead of text can be used in the HTMLUI control as links to the other files.

  • Another important feature that HTMLUI provides with links is that the user can decide the course of action forwarded by the link after being clicked, with the help of the HTMLUI control's LinkClicked Event.

The loaded file contains links to other documents. The HTMLUI control will allow you to navigate through the links in the HTML page.

    this.htmluiControl1.LoadHTML("model.htm");
