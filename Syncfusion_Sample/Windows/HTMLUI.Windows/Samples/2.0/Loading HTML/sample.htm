Loading HTML

HTML documents available at various resources can be easily loaded into the HTMLUI control.

Some of the resources from which the HTML documents are loaded are given below.

  • Load from Disk: With the HTMLUI control any HTML file from the local drive can be easily loaded into the control.
  • Load from Resource: This sample shows how a HTML file that is embedded in a resource is loaded into the HTMLUI control.
  • File Links: This sample shows how a HTML file that is embedded in a resource is loaded into the HTMLUI control.
  • Startup Document: This sample shows how a startup HTML document can be loaded into the HTMLUI control.
