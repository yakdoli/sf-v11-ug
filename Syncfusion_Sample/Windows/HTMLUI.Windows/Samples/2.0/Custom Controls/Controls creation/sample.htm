Controls Creation

This sample shows how custom controls are created in HTMLUI with the help of HTML code alone.

Features:

  • The custom controls are not standard HTML elements but user-defined controls that are included by the user in order to improve the application’s richness and productivity.

  • The Custom tag is used to include the Windows Forms controls in the HTML document. The custom tag comes with two attributes: assembly and class.

  • The assembly attribute refers to the namespace where the control is located, and the class attribute represents the control.


The following code renders the control in HTMLUI:

 
     Hashtable htmlelements = new Hashtable();
	htmlelements = e.Document.ElementsByUserID;

	BaseElement monthCalendarElement1 = htmlelements["monthCalendar1"] as BaseElement;
	new CustomControlBase( monthCalendarElement1, this.monthCalendar1  ); 

	BaseElement dataGridElement1 = htmlelements["dataGrid1"] as BaseElement;
	new CustomControlBase( dataGridElement1, this.dataGrid1  );
