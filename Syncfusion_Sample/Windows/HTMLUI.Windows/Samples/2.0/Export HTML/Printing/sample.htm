Printing

This sample shows how a HTML document available in the HTMLUI control can be printed.

Features:

  • HTMLUI control supports the printing of the document available in the control.

  • Along with printing, print preview is also present in HTMLUI, which provides a preview of the document before it is printed.

  • Customized keyboard shortcuts can be enabled for the printing options in HTMLUI with the help of the control’s key events.


Here is the code to show the print preview of the document rendered in the HTMLUI control.


        //To initialize the document before printing it.

	//Document initialization

	HTMLUIPrintDocument pd = new HTMLUIPrintDocument(this.htmluiControl1.Document );

	//To preview the page before the page gets printed.

	//Print Preview

	PrintPreviewDialog dlg = new PrintPreviewDialog() ;

	dlg.Document = pd;

	dlg.ShowDialog();

Here is the code to print the document rendered in the HTMLUI control.

 
        //To print the page rendered in the HTMLUI control.

	PrintDialog dg = new PrintDialog();

	dg.AllowSomePages = true;

	dg.Document = pd;

	if(dg.ShowDialog() == DialogResult.OK)

	pd.Print();
