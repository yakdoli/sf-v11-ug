Bookmarks

This sample shows how bookmarks are supported in HTMLUI for content available inside a document and also for the content available in other documents.

Features:

  • Bookmarks are very important features as they help users revisit their favorite pages.
  • HTMLUI supports bookmarking content within a page and other related files.
  • The ID attribute and the href attribute of the anchor element are used for this purpose.

Here is the code used for bookmarking.

    

<a href="Newfile.htm#bookmark"> New file Book mark </a>
<a href="#bookmark"> Bookmark in same file </a>
......
......
<div id="bookmark"> Syncfusion's HTMLUI Control </div>
