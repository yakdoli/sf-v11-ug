Tables

This sample shows how HTML tables can be used in HTMLUI.

Features:

  • The HTMLUI control has rich table support that lets the user decide on the table’s dimensions and design.

  • The HTMLUI table support also comes with different alignments of text within the table.

  • HTMLUI control provides support for the standard table attributes like the colspan, rowspan, and nowrap. These features help in developing UI applications with rich, custom design implementations.

Here is the HTML code that renders the table in a HTMLUI control.

     
<table id="Table1" width="100%" border="1">
		<tr>
			<td id="td11" colspan="3">This is a column with colspan=3</td>
		</tr>
		<tr>
			<td id="td21">Column 1</td>
			<td>Column 2</td>
			<td>Column 3</td>
		</tr>
		<tr>
			<td height="40" id="td31" valign="bottom">Column 1 with custom height</td>
			<td id="td32">Column 2</td>
			<td id="td33">Column 3</td>
		</tr>
</table>
