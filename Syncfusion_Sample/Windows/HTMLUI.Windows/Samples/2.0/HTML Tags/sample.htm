HTML Tags

HTMLUI supports a rich set of HTML Tags. Tag elements are rendered through managed code similar to that of the popular Web browsers.

  • Bookmarks: This sample shows how bookmarks are supported in HTMLUI for the content available inside a document and also for the content available in other documents.
  • Tables: This sample shows how HTML tables can be used in HTMLUI.
  • Tags support: This sample shows the various HTML tags that are supported in HTMLUI.
