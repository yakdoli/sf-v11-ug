Scroll Bar Demo

Scroll Bar Demo illustrates the usage of detached scrollbars with Essential Grid. You can drop the windows' scrollbars onto your form and make the grid to use them instead of using the Grid’s standard scrollbars.

Features

  • Setting UseSharedScrollBars to true enables this functionality by synchronizing the external scrollbars.

  • For the options of HScrollBehavior and VScrollBehavior, either disable or share the existing one.

  • The external scrollbars are set through InnerScrollBar.
