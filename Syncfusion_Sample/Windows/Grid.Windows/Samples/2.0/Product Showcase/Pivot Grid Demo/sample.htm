Pivot Grid Demo

This sample simulates the pivot table feature of Excel. The pivot grid, as the name implies, pivots data through the drag-and-drop operation, to organize the data in a cross-tabulated form. The major advantage with pivot grid is that you can extract the desired information from a large list within seconds. Along with presenting the data, it also enables you to summarize and group data. It finds its main application in the financial domain, where it is used to organize and analyze business data.

The pivot grid comprises of the following components.

  • Display Grid - Displays the data extracted from the underlying database.

  • Pivot Table Field List - Lists the available fields from the database. Provides a way to add/remove the fields to/from the grid.

  • Drag-Drop Areas - Serves as a view state of the pivot grid, where you can rearrange the fields by doing a drag-and-drop operation between the row and column label areas.

Features

  • Data-binding support
  • AutoCalculation of total summary
  • Filters
  • Grouping Support
  • Support to customize the appearance of the grid
  • Support for XML and binary serialization
