Excel-Like UI Demo

This sample shows the implementation of a Microsoft Excel 2007-like UserInterface by using the Essential Grid control.

Features

  • Simulates MS Excel

  • It contains a "name box" that shows the current selection range, and a "formula bar", which supports the formula cells.

  • The row and column headers of the selected range are highlighted on the UI. This feature is similar to Excel.
