Editor Cell Demo

Editor Cell Demo exhibits different options and styles for editing a cell. You will learn how to make to place an image with text in a textbox and applying Currency, Password, MaskEdit styles to the cells

Given below is an image of the sample.

Features:

BrowseOnly

Browse-only property is permit controls to be more easily set to un-editable without affecting background appearance, no greyed-out effect, etc. in GridControl.

  • The Grid is set as uneditable using Browseonly property.

   this.gridControl1.BrowseOnly = true;

  • To enable the grid as editable set the BrowseOnly property. to false.

   this.gridControl1.BrowseOnly = false;

MaskEdit

The various available options allow to mask any cell that permits valid inputs and also control any invalid input in the cell. For example, you can use a mask edit cell to facilitate the entry of a formatted Social Security number, phone number, or a three-character alpha code

  • The commonly used mask symbols can be implemented using MaskEdit.

   GridStyleInfo style1 = gridControl1[2, 4];
   GridMaskEditInfo maskedEditStyle1 = style1.MaskEdit;

  • Masking symbols can be set to style1.MaskEdit.Mask property. The character at the position between the comparison operators (>C<) will be displayed in upper-casing. The code below shows the first character in upper-casing.

   maskedEditStyle1.Mask = ">C<CCCCCCCCCCCC";

Password

This cell allows you to type a password, but only a masking character like "*", will be displayed as you type

To create a password cell, do the following:

  • Set the CellType as OriginalTextBox.

gridControl1.TableStyle.CellType = "OriginalTextBox";

  • Set Style.PasswordChar to the character that needs to replace the password.

GridStyleInfo style;
style = gridControl1[row, 2];
style.PasswordChar = '*';

  • Additionally, you can specify casing for the character to be "Lower" or "Upper", using the below code.

style.CharacterCasing = CharacterCasing.Lower;

Curreny

The various options in this sample include setting patterns to display positive and negative numbers; grouping the digits; setting decimal separator such as ".", and more options.

  • To have currency cells, you should set the cell type as Currency.

 GridStyleInfo style = gridControl1[row, 2];
 style.CellType = "Currency";

  • Using the CurrencyEdit property, the currency symbol, pattern, decimal digits, and text color for positive and negative numbers can be changed.

       style.CurrencyEdit.CurrencyDecimalDigits = 2;
       style.CurrencyEdit.CurrencyNegativePattern = 1;                
       style.CurrencyEdit.CurrencyPositivePattern = 0;
       style.CurrencyEdit.CurrencySymbol = (row % 2 == 0) ? "$ " : "DM ";
       style.CurrencyEdit.NegativeColor = System.Drawing.Color.Red;

Numeric UpDown

Numeric Up-Down Cell employs custom-generated, enhanced NumericUpDown control in grid cells. The NumericUpDown control is just a Windows spin box used to display numeric values.
