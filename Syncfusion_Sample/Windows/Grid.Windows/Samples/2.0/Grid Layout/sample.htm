Grid Layout

The different samples listed below illustrate the various features of the grid layout:

Multiple Views Demo - This sample illustrates how easy it is to share data among grids

Resize to Fit Demo- This sample illustrates optimal fitting of columns and rows based on the content of a cell

Selection Mode Demo- This sample illustrates the different selection modes of grid cells

Error provier Demo - This sample illustrates how to set the error icon and custom error in GridControl

Custom Cell AutoSizing Demo- This sample illustrates automatic resizing of grid cells depending on the size of custom controls placed inside the cell

Sorting Demo  - This sample illustrates how to implement header-click sorting in Essential Grid control
