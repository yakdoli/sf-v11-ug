Resize to Fit Demo

This sample illustrates optimal fitting of columns and rows based on the content of cells.



Features:

  • ColWidths.ResizeToFit method is used to resize the column widths. 

            this.gridControl1.ColWidths.ResizeToFit(GridRangeInfo.Cols(1,5));

  • RowHeights.ResizeToFit is used to resize the row heights.

            this.gridControl1.RowHeights.ResizeToFit(GridRangeInfo.Rows(1,5));

AutoFit:

When entering characters in cell, if the contents exceeds the cell width, then the cell is displayed with characters by using AutoFit property.

An enum AutoFitOptions has been implemented which enables to make the contents to turn into characters on checking whether the content is any one of the following:

  • Alphabet.
  • Numeric.
  • Both Alphabet and Numeric.
  • None.

Alphabet:

After choosing the option as Alphabet, the alphabets will alone be converted into the specified characters.

             this.gridControl1[RowIndex, ColIndex].AutoFit = AutoFitOptions.Alphabet;

Numeric:

After choosing the option as Numeric the Numbers will alone be converted into the specified characters.

             this.gridControl1[RowIndex, ColIndex].AutoFit = AutoFitOptions.Numeric;

Both:

After choosing the option as Both, the alphabets will alone be converted into the specified characters.

             this.gridControl1[RowIndex, ColIndex].AutoFit = AutoFitOptions.Both;

None:

After choosing the option as None, it will retrieve the normal functioning of the Grid.The Contents will be displayed as it is present.

             this.gridControl1[RowIndex, ColIndex].AutoFit = AutoFitOptions.None;

Display with Desired Characters:

The characters that are displayed could be changed by using AutoFitChar option. After specifying the characters, the characters will be displayed in the cell. By default, the contents are filled with #.
