Print Page Layout Demo

This sample illustration shows the watermark page layout on the grid according to the pages that divide the grid for printing.

Features

  • This behavior is implemented to easily identify the divided layout of a grid at run time. The layout will be shown with the help of a line between the divided pages, and the page number will be displayed on each layout as text. The line color and the text color can also be changed using the helper properties. By default, it will be set to Blue.
