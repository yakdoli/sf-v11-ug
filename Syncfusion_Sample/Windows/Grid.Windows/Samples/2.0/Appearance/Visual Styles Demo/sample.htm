Visual Styles Demo

Essential GridControl offers a wide range of support for the appearance of the Grid. This sample illustrates the application of different skins by using the color scheme GridVisualStyles in the Essential GridControl. You will be able to switch between themes, while running this sample.



Features:

  • Styles can be set to the GridControl by assigning a GridVisualStyles enumeration value to the GridVisualStyles property.

The following sample code sets an Office2010Black skin theme to the Essential GridControl by using GridVisualStyle.

        gridControl1.GridVisualStyles= Syncfusion.Windows.Forms.GridVisualStyles.Office2010Black;

The new GridVisualStyle themes will be applied only when the EnableLegacyStyle is False, by default the EnableLegacyStyle is True. The following sample code sets the LegacyStyle as false to apply the new GridVisualStyle color scheme.

         gridControl1.Model.EnableLegacyStyle = false;
