Appearance

The different samples that illustrate the appearance properties of Essential Grid are listed below:

Visual Styles Demo- Illustrates the implementation of VisualStyle support in the GridControl

Text Format Demo- Illustrates data formatting in a grid

Grid Style  Demo- Illustrates the relationship between a standard style, a table style, a base style, a row style, a column style, and a cell style

SkinManager Demo- This sample illustrates the implementation of Skin Manager to the Syncfusion Grid controls.

Cell Customization Demo- Demonstrates custom painting on grid cells

Grid Properties Demo- Explores the properties of the Essential Grid control
