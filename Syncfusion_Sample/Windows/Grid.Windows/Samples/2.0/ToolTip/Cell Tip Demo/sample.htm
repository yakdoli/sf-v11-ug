Cell Tip Demo

This sample demonstrates the GridStyleInfo.CellTipText property that allows you to specify a ToolTip for a range of cells or individual cells.

Features:

  • Cell tip text can be set for TableStyles, RowStyles, ColStyles, and for Individual Cells.

                this.gridControl1[2,2].CellTipText = "TipText for cell 2,2";
                this.gridControl1[2,2].BackColor = Color.Aquamarine;

                this.gridControl1[2,4].CellTipText = "TipText for cell 2,4";
                this.gridControl1[2,4].BackColor = Color.Aquamarine;

                this.gridControl1.ColStyles[3].CellTipText = "TipText for column 3";
                this.gridControl1.ColStyles[3].BackColor = Color.Bisque;

                this.gridControl1.RowStyles[5].CellTipText = "TipText for row 5";
                this.gridControl1.RowStyles[5].BackColor = Color.LightBlue;
