Syncfusion Essential Diagram
Essential Diagram is an extensible, high-performance .NET diagramming framework for Windows Forms and ASP.NET applications. It can be used for developing Microsoft Visio-like interactive 2-D graphics and diagramming applications. It stores graphical objects in a node graph and renders them onto a screen. You can create organization layouts, flow diagrams, and complex vector-based graphics, or you can populate a diagram based on an external data source, or you can implement your own logic to drive a display.
