Layers

This sample demonstrates the "Layers" feature available in Essential Diagram. The layers are transparent sheets that can be added to the model and the objects are added to the layer. Layers allow to categorically arrange a set of nodes on the diagram.

Features

  • Add/Remove - You can add/remove the layers with the given layer name to/from the diagram.The nodes contained in the layer will be removed when layer is removed.
  • Enable/Disable - Layer can be enabled/disabled by using its Enabled property.
  • Visiblility - You can use the layer's Visible property to change the visibility of the layer.

    To create and add a layer to the diagram model, use the following code.

    			// Initializes a new instance of the "Layer" class.
            Layer layer = new Layer(this.diagram1.Model, "Layer1");
            // Gets or sets a value indicating whether the layer is enabled or disabled.
            layer.Enabled = false;
            // Gets or sets a value indicating whether the layer is visible.
            layer.Visible = true;
            // Adds the specified layer to the Model.Layer's collection.
            this.diagram1.Model.Layers.Add(layer);
            // Adds the specified layer to the Model.ActiveLayers's collection.
            diagram1.Model.ActiveLayers.Add(layer);
