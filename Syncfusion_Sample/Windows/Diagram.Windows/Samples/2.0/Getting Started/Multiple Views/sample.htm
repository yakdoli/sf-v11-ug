Multiple Views

The Multiple Views sample highlights the modularity and power of the Model View Controller (MVC) architecture used by Essential Diagram.

This sample demonstrates using different views on the same diagram model by using two diagram controls sharing the same model. Each diagram control will have its own view and controller components.

Note: Any changes to the diagram model will simultaneously be refreshed on all views attached to that model.

Features

  • To trigger a diagram-click event, use the following code.

    •            this.diagram1.Click += new System.EventHandler(this.diagram1_Click);
           this.diagram2.Click += new System.EventHandler(this.diagram2_Click);

  • To assign a model and property editor to a diagram when it is clicked, employ the following code.

    •            private void diagram1_Click(object sender, System.EventArgs e)
		{
                 this.SelectDiagram(this.diagram1);
                }

              private void diagram2_Click(object sender, System.EventArgs e)
		{
                 this.SelectDiagram(this.diagram2);
                }

             private void SelectDiagram(Syncfusion.Windows.Forms.Diagram.Controls.Diagram diagram)
		{
	         this.selectedDiagram = diagram;
                 this.selectedDiagram.Model = this.model;
	         this.propertyEditor.Diagram = this.selectedDiagram;
		}
