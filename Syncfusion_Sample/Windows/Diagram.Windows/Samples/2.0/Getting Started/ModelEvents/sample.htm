Model Events

This sample illustrates the various "Diagram.Model.EventSink"s events supported by Essential Diagram.

Features

  • The model events are listed below:
    • NodeCollectionChanged
    • NodeCollectionChanging
    • ConnectionsChanging
    • ConnectionsChanged
    • PinPointChanging
    • PinPointChanged
    • SizeChanging
    • PropertyChanging
    • PropertyChanged
    • FlipChanging
    • FlipChanged
    • ZOrderChanging
    • ZOrderChanged
    • VertexChanging
    • VertexChanged

    Use the following code to trigger the PinpointChanged/Changing event when a node is moved from one position to another in diagram control.

          
				// Occurs when a node's pin point is changing.
	            diagram1.Model.EventSink.PinPointChanging += new PinPointChangingEventHandler(EventSink_PinPointChanging);
	            // Occurs when a node's pin point is changed.
	            diagram1.Model.EventSink.PinPointChanged += new PinPointChangedEventHandler(EventSink_PinPointChanged);
				
				void EventSink_PinPointChanged(PinPointChangedEventArgs evtArgs)
	        	{
	             listBox1.Items.Add("\nPinPointChanged event Fired.\n"); 
	        	}
	
				void EventSink_PinPointChanging(PinPointChangingEventArgs evtArgs)
				{
	             listBox1.Items.Add("\nPinPointChanging event Fired.\n"); 
				}
