Custom Command

The Custom Command sample demonstrates how to perform undo-redo operations in diagram applications.

Click History -> Undo/Redo, to perform the undo-redo operation on actions performed on diagram nodes.

Features

  • To perform undo and redo operations, use the following code.
    this.diagram1.Model.HistoryManager.Undo();
this.diagram1.Model.HistoryManager.Redo();
  • To start and end a custom action, use the following code.
    this.diagram1.Model.HistoryManager.StartAtomicAction( "Custom Action" ); 
this.diagram1.Model.HistoryManager.EndAtomicAction( );
