Save to Database

The Save to Database sample illustrates the process of saving a diagram as a binary in the database.

Features

  • Creating a new diagram
  • Saving a diagram into the database
  • Loading the diagram from the database
  • Updating an existing diagram
  • Deleting a diagram in the database

    The following code illustrates how to save and load the diagram using a stream.

    
//Save diagram.
MemoryStream dStream = new MemoryStream();
diagram1.SaveBinary(dStream);

//Load diagram.
diagram1.LoadBinary(dStream);
