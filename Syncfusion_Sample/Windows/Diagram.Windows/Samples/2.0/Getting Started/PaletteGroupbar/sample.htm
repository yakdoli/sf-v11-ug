Palette Group Bar Demo

The PaletteGroupbar control displays a collection of symbol palettes that the symbol models contain in a Groupbar control.

This sample demonstrates the various type of features available in the PaletteGroupbar Control.

Features

Appearance Settings

  • Background Color
    The background color for the Palette Group Bar and the Palette Group View is set using the BackColor property, as shown. 
    
			this.pgbVisualStudio.BackColor = cpBack.SelectedColor;
            foreach (GroupBarItem gbItem in this.pgbVisualStudio.GroupBarItems)
            {
                gbItem.BackColor = cpBack.SelectedColor;
            }

  • Border
    • Palette Group Bar Border - The border for the Palette Group Bar can be specified in the BorderStyle property. This border is drawn as the outer rectangle for an entire Palette Group Bar control. 
      
this.pgbVisualStudio.BorderStyle =  BorderStyle.FixedSingle
    • Palette GroupBar ClientBorder - The client area of each Palette Group Bar item can have borders of different colors for each side by using the DrawClientBorder property. 
      
this.pgbVisualStudio.DrawClientBorder =  this.cbDrawClientBorder.Checked;
            foreach (GroupBarItem gbItem in  this.pgbVisualStudio.GroupBarItems)
            {
                gbItem.ClientBorderColors = new BorderColors(cpBack.Sele ctedColor, cpBack.SelectedColor, cpBack.SelectedColor,  cpBack.SelectedColor);
            }
  • Visual Styles
    The Palette Group Bar provides options for various visual styles such as Default, Office 2003,Office 2007 and Metro. 
    
this.pgbOutLook.VisualStyle =  VisualStyle.Office2003
  • Color Schemes
    The Palette Group Bar supports all three Office 2007 color schemes as well as custom colors. 
    
this.pgbOutLook.Office2007Theme = Office2007Theme.Blue;

Behavior Settings

  • Animated Selection
    Switching between two Palette Group Bar items can be animated by using the AnimatedSelection property. 
    
 this.pgbVisualStudio.AnimatedSelection  = true;

  • Integrated Scrolling
    Integrated scroll buttons of the Palette Group Bar can scroll items in the palette group view by setting the IntegratedScrolling property. 
    
this.pgbVisualStudio.IntegratedScrolling = true;

  • Stacked Group Bar
    This is a full-featured implementation of a "Navigation Pane Options" dialog that allows users to show, hide, or move Palette Group Bar items; it is similar to the functionality available in Microsoft Outlook 2007.

    • Stacked Mode
      Palette Group Bar items can be stacked, similar to the Outlook Bar, by using the StackedMode property. 

      
this.pgbOutlook.StackedMode = true;

    • Chevron in Palette Group Bar
      When the Palette Group Bar is in Stacked mode, the chevron button can be visible by using the ShowChevron property. 

      
 this.pgbOutlook.ShowChevron = true;

    • Item Image in Header
      Images associated with a Palette Group Bar item can be displayed in the Palette Group Bar header when a particular item is selected; this is done using the ShowItemImageInHeader property. 

      
this.pgbOutlook.ShowItemImageInHeader = true;

  • Collapsible Palette Group Bar
    A stacked Palette Group Bar can be collapsed or expanded to save application space. The Essential Palette Group Bar provides various customization options for this feature.

    • AllowCollapse indicates whether a group bar can be collapsed.
    • Collapsed indicates whether a Palette group bar is collapsed.
    • CollapsedText indicates what text is to be shown in the collapsed client area of the Palette group bar.
    • CollapsedWidth allows setting the width of a collapsed group bar.
    • CollapseImage allows setting the image of a collapse button when in expanded state.
    • ExpandButtonToolTip is the ToolTip for the collapse button when the control is collapsed.
    • ExpandImage allows setting the image of the collapse button when in the collapsed state.
    • MinimizeButtonToolTip is the ToolTip for the collapse button when the control is expanded.
    • NavigationPaneToolTip is the ToolTip for a navigation pane.
    • PopupAutoClose indicates whether a pop-up should close when clicking on another Palette Group Bar item.
    • PopupClientSize is used to set the size of the pop-up window.
    • PopupResizeMode indicates how the pop-up window can be resized. It allows horizontal resizing, vertical resizing, both, or none of these options.
    • ShowPopupGripper is used to show a pop-up windows gripper.
    • BeforePopup is an event that occurs before the Group Bar item’s pop-up is shown.
    • StateChanging is an event that is triggered when a state is about to change.
    • StateChanged is an event that occurs when a state is changed.
