Labels

This sample demonstrates the creation of label and how to add the labels to the node. A label is a text objects which bounds inside the node. The node has collection of labels and you can modify the style, position and format the label using its various properties.

Features

  • A label has positioning, appearance and formatting features.
  • It has the various types of positioning options to position the container such as BottomCenter, BottomLeft, BottomRight, TopCenter, TopLeft, TopRight,MiddleLeft, MiddleRight and Center.
  • You can change the properties of the label by using the properties provided in the property editor.

    To create and add a label for the node, refer to the following code.

    

			//Initializes a new instance of the label class.
	        Syncfusion.Windows.Forms.Diagram.Label lbl = new Syncfusion.Windows.Forms.Diagram.Label(rect, "Top Left");
	        //Sets the colour of the font.
	        lbl.FontColorStyle.Color = Color.White;
	        //Sets the position of the label.
	        lbl.Position = Position.TopLeft;
	        // Gets or sets whether default positioning has to be used.
	        lbl.UpdatePosition = true;
	        // Adds the specified label to the node.
	        rect.Labels.Add(lbl);
