Expander

The Expander sample illustrates expandable and collapsible nodes found in Essential Diagram.

Features

  • A custom ExpanderSymbol symbol class that extends the Essential Diagram symbol-base type and adds functionality for containing and displaying child symbols.

  • The child symbols, or sub-nodes, are shown or hidden by double clicking the parent symbol. The expanded or collapsed state of the symbol is indicated using + / - entities drawn by the expander symbol.


  • Employee information sourced from a XML-format Access database file is used to create the organization diagram.


  • A DirectedTreeLayoutManager implementing a simple vertical tree-type layout is used for laying out the employee hierarchy diagram.
