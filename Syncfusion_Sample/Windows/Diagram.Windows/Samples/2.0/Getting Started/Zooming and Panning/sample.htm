Zooming and Paning

This sample demonstrates the zoom and pan features implemented in Essential Diagram.

Features

  • ZoomIncrement - Sets the zoom increment value applied to the diagram view. By default it is set to 20.
  • ZoomIn - Zooms in the diagram document.
  • ZoomOut - Zooms out the diagram document.
  • ZoomToActual - Zooms to the actual size of the document.
  • ZoomTool - Lets you to zoom-in and zoom-out the diagram document.
  • Pan Tool - Lets you to pan the diagram document.

    To change the zoom increment value, you can use the below code.

    
		// Gets or sets the zoom increment value applied to the view attached document.
        diagram1.View.ZoomIncrement = 10;

    To zoom in the diagram document, you can use the below code.

    
diagram1.View.ZoomIn();

    To zoom out the diagram document, you can use the below code.

    
diagram1.View.ZoomOut();

    To zoom the diagram document to its actual size, you can use the below code.

    
diagram1.View.ZoomToActual();

    To activate the zoom tool, use the following code.

    
	diagram1.Controller.ActivateTool("ZoomTool");

    To activate the pan tool, use the following code.

    
	diagram1.Controller.ActivateTool("PanTool");

Note

    You can also use the Diagram.View"s Magnification property to zoom-in/zoom-out the diagram document.

    Use the below code to zoom-in/zoom-out the diagram document using the Magnification property.

    
	//ZoomIn
	diagram1.View.Magnification += 10;
	//ZoomOut
    diagram1.View.Magnification -= 10;
