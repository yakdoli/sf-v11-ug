Line Bridging and Routing

This sample demonstrates the "Line Routing" and "Line Bridging" features available in Essential Diagram.

Features

  • LineBridgingEnabled - Indicates whether the line bridging is enabled for the model.
  • BridgeStyle - Sets the bridge style for the model. The various types of BridgeStyle are Arc, Gap, Square, Sides2, Sides3, Sides4, Sides5, Sides6 and Sides7.
  • LineBridgeSize - Sets the size of the bridges added to the line to jump over.
  • LineRoutingEnabled - Indicates whether the line routing is enabled for the model.
  • RoutingMode - Sets the routing mode for the model. The different types of routing modes are InActive, Automatic and SemiAutomatic.
  • DistanceToObstacles - Sets the distance to the obstacles.

    Use the following code to enable the "Line Bridging" and "Line Routing" in the diagram model.

    //Enables LineBridging
diagram1.Model.LineBridgingEnabled = true;
//Enables LineRouting 
diagram1.Model.LineRoutingEnabled = true;
//Sets the routing mode of the LineRouter as Automatic.
diagram1.Model.LineRouter.RoutingMode = RoutingMode.Automatic;
