Diagram Events

This sample illustrates the various Diagram.EventSink"s events supported by Essential Diagram.

Features

  • The diagram events are listed below:
    • NodeCollectionChanged
    • NodeCollectionChanging
    • NodeSelected
    • NodeDeselected
    • NodeClick
    • NodeDoubleClick
    • NodeMouseEnter
    • NodeMouseLeave
    • SelectionListChanged
    • MagnificationChanged
    • OriginChanged

    Use the following code to trigger the NodeCollectionChanged/Changing event, when a node is added to the Diagram control.

          
			// Occurs when the node collection is changing.
			diagram1.EventSink.NodeCollectionChanging += new CollectionExEventHandler(EventSink_NodeCollectionChanging);
            // Occurs when the node collection is changed.
            diagram1.EventSink.NodeCollectionChanged += new CollectionExEventHandler(EventSink_NodeCollectionChanged);   

			void EventSink_NodeCollectionChanged(CollectionExEventArgs evtArgs)
			{
	             listBox1.Items.Add( "\nNodeCollectionChanged event Fired.\n"); 
	        }
	
	        void EventSink_NodeCollectionChanging(CollectionExEventArgs evtArgs)
	        {
	            listBox1.Items.Add("\nNodeCollectionChanging event Fired.\n"); 
			}
