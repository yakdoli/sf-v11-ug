Controls Galore

The Controls Galore sample demonstrates the new ControlNode class that enables embedding Windows Forms controls in a diagram.

This sample features symbols hosting various Windows Forms controls and also a new control-node tool that allows control nodes to be interactively created at run time.

Features

  • Hosting Control - It is used to host the control to inside the node.

  • Activate Style - It specifies an activation style for the ControlNode node type. ControlNode has various types of activation style such as Click,ClickPassThrough,DoubleClick and SelectedClick

  • The following code illustrates how to insert a control node in the diagram control.

    
private void InsertButton()
{
    ButtonAdv btn = new ButtonAdv();
    btn.Text = "Click to activate me";
    btn.UseVisualStyle = true;
    btn.UseVisualStyleBackColor = true;
    btn.Office2007ColorScheme = Office2007Theme.Blue;
    btn.Appearance = ButtonAppearance.Office2007;
    btn.Click += new EventHandler( btn_Click );
        	
    ControlNode ctrlnode = new ControlNode( btn, new RectangleF( 100, 200, 150, 50 ) );
    ctrlnode.ActivateStyle = ActivateStyle.ClickPassThrough;
    this.diagram1.Model.AppendChild( ctrlnode );
}
