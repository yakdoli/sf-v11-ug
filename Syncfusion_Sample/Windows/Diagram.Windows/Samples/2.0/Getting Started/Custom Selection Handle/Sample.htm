Custom Selection Handle

Essential Diagram provides built-in support for customizing the appearance of selection handles. You can easily define your own handle styles by inheriting the UserHandleRenderer class and overriding the appropriate methods. This class acts as an interface between the user and the diagram by providing various style options that can be customized.

Features

  • Diagram node selection handle color or outline color can be customized using the "Handle Color" or "Handle Outline Color" button option in the sample.

    The following code illustrates how to change the color of the selection handle.

    
this.diagram1.View.HandleColor = this.colorPickerButton2.SelectedColor; 
this.diagram1.View.HandleOutlineColor = this.colorPickerButton1.SelectedColor;
