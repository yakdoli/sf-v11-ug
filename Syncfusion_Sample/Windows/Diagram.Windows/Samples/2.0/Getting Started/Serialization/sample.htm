Serialization

The Serialization sample demonstrates Essential Diagram's built-in serialization support. This sample shows the approach involved in creating a simple diagram and saving it to disk as an Essential Diagram document (*.edd file). Loading a diagram from a persisted diagram file is also featured.

Features

  • The File Save/Open menu commands may be used to save and load diagrams as binary documents.
  • The binary serialization API used within the sample can be replaced by the Essential Diagram SOAP serialization API for XML format serialization.

    To load the diagram file in binary format, use the following code.

               this.diagram1.LoadBinary(this.openFileDialog1.FileName);

    To save the diagram file in binary format, use the following code.

               this.diagram1.SaveBinary(this.saveFileDialog1.FileName);
