Grid

This sample demonstrates the grid features of Essential Diagram.

Features

  • Color - Sets the color of the grid.
  • DashStyle - Sets the line style of the grid.
  • GridStyle - Sets the appearance of the grid.
  • HorizontalSpacing - Sets the horizontal distance between the grid points.
  • VerticalSpacing - Sets the vertical distance between the grid points.
  • MinPixelSpacing - Sets the minimum spacing between the grid points in the device units.
  • SnapToGrid - Sets a value indicating whether the snap to grid feature is enabled or disabled.
  • Visible - Sets a value indicating whether the grid is visible.

    To change the appearance of the Grid view, use the following code.

    
			// Gets or sets the appearance of the grid.
            this.diagram1.View.Grid.GridStyle = GridStyle.Line;
            // Gets or sets the DashStyle value for dashed lines.
            this.diagram1.View.Grid.DashStyle = System.Drawing.Drawing2D.DashStyle.Dot;
            // Gets or sets the color for drawing the grid.
            this.diagram1.View.Grid.Color = Color.LightGray;
            // Gets or sets the vertical distance between the grid points.
            this.diagram1.View.Grid.VerticalSpacing = 15;
            // Gets or sets the horizontal distance between thegrid points.
            this.diagram1.View.Grid.HorizontalSpacing = 15;
