Commands

This sample illustrates the various types of commands supported by Essential Diagram

Features

  • Cut - Cuts the selected node in the diagram and moves it to the clipboard.

    • Use the below code to cut the selected node from the Diagram control.

    // Remove the currently selected nodes from the diagram and move it to the clipboard.
diagram1.Controller.Cut();

  • Copy - Copies the selected node in the diagram and moves them to the clipboard.

    • Use the below code to copy the selected node from the Diagram control.

    // Copy the currently selected nodes to the clipboard.
diagram1.Controller.Copy();

  • Paste - Pastes the node in the clipboard to the diagram.

    • Use the below code to paste the nodes to the diagram.

    // Paste the contents of the clipboard to the diagram.
diagram1.Controller.Paste();

  • Align Left - Aligns the selected node to the left side of the previously selected node

    • Use the below code to align the node to the left side of the previously selected node in the diagram.

    // Align the selected nodes along the left edge of the first node.
diagram1.AlignLeft();

  • Align Center - Aligns the selected node to the center side of the previously selected node

    • Use the below code to align the node to the center side of the previously selected node in the diagram.

    // Align the selected nodes along the vertical center of the first node.
diagram1.AlignCenter();

  • Align Center - Aligns the selected node to the right side of the previously selected node

    • Use the below code to align the node to the right side of the previously selected node in the diagram.

    // Align the selected nodes along the left edge of the first node.
diagram1.AlignRight();

  • Align Top - Aligns the selected node at the top of the previously selected node

    • Use the below code to align the node to the top of the previously selected node in the diagram.

    // Align the selected nodes along the top edge of the first node.
diagram1.AlignTop();

  • Align Middle - Aligns the selected node to the middle of the previously selected node

    • Use the below code to align the node to the middle of the previously selected node in the diagram.

    // Align the selected nodes along the horizontal center of the first node.
diagram1.AlignMiddle();

  • Align Bottom - Aligns the selected node at the bottom of the previously selected node

    • Use the below code to align the node to the bottom of the previously selected node in the diagram.

    // Align the selected nodes along the bottom edge of the first node.
diagram1.AlignBottom();

  • Rotate - Rotates the selected nodes in a diagram with the specified number of degrees

    • Use the below code to rotate the node in the diagram

    // Rotate the selected nodes about their logical origin by the specified number of degrees.
diagram1.Rotate(90);

  • Flip Vertical - Flip the selected nodes relatively to the vertical axis

    • Use the below code to flip the node to vertical axis

    // Flips the selected nodes about their vertical(Y) axis.
diagram1.FlipVertical();

  • Flip Horizontal - Flip the selected nodes relatively to the Horizontal axis

    • Use the below code to flip the node to horizontal axis

    // Flips the selected nodes about their horizontal(X) axis.
diagram1.FlipHorizontal();

  • Space Acorss - Positions the selected node for equal horizontal spacing

    • Use the below code to postion the node horizontally

    // Positions the selected node for equal horizontal spacing
diagram1.SpaceAcross();

  • Space Down - Positions the selected node for equal vertical spacing

    • Use the below code to postion the node vertically

    // Positions the selected node for equal vertical spacing
diagram1.SpaceDown();

  • Same Width - sets the width of all selected nodes to be equal

    • Use the below code to set the width of all selected nodes to be equal

    // sets the width of all selected nodes to be equal
diagram1.SameWidth();

  • Same Height - sets the height of all selected nodes to be equal

    • Use the below code to set the height of all selected nodes to be equal

    // sets the height of all selected nodes to be equal 
diagram1.SameHeight();

  • Same Size - sets the width and height of all selected nodes to be equal

    • Use the below code to sets the width and height of all selected nodes to be equal

    // sets the width and height of all selected nodes to be equal
diagram1.SameSize();
