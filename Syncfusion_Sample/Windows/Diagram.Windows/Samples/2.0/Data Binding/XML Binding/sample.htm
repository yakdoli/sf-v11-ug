XML Binding

This sample illustrates how to generate/bind the diagram based on the given XML data source programmatically. It uses the HierarchicalLayoutManager to lay out the diagram nodes.

Features

  • It generates the diagram with the given data source as a XML format data file.
  • It uses the pre-defined shapes and connectors to draw the diagram.
  • It uses a HierarchicalLayoutManager to control the layout of nodes on the diagram.
