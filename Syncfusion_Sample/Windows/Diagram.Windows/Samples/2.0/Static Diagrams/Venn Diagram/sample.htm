Venn Diagram

Venn diagrams or Set diagrams are diagrams that show all possible logical relations between finite collections of sets (aggregation of things).

The Venn diagrams normally comprise overlapping circles. The interior of the circle symbolically represents the elements of the set, while the exterior represents the elements that are not members of the set.

Features

  • A serialized soda machine state diagram which is designed in the diagram builder is loaded into the Diagram control.
  • It is a read-only diagram.
  • The objects in the diagram are not movable and editable.
