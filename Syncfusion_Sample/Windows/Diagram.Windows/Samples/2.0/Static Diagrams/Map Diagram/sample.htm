Map Demo

The highlight sample demonstrates, mouse hover effects implemented in Essential Diagram.

Features

  • It is a read-only diagram.
  • The tooltip displays the node name when the pointer is hovering over it.
  • The objects in the diagram are not movable and editable.
  • Part of the diagram can be highlighted when the pointer hovers over it.

    Using the GetNodeUnderMouse method, you can retrieve a node under the pointer.

    
// Retrives node under the mouse action.
    
Node node1 = (Node)this.diagram1.Controller.GetNodeUnderMouse(new Point(e.X, e.Y));

    We can highlight a node by changing the LineStyle and FillColor effects.

    
// Convert node as polygon.
    
Polygon poly = globelNode as Polygon;
    
Random r = new Random();  
    
          
    
// Setting fill style for the ploygon.
    
poly.FillStyle.Color = Color.FromArgb(r.Next(255), r.Next(255), r.Next(255));           

    
globelNode.LineStyle.DashStyle = System.Drawing.Drawing2D.DashStyle.Dash;
    
globelNode.LineStyle.LineWidth = 3;
    

    
// Resetting the node with default values.
    
poly.FillStyle.Color = defaultColor;
    
globelNode.LineStyle.DashStyle = System.Drawing.Drawing2D.DashStyle.Solid;
    
globelNode.LineStyle.LineWidth = 1;
