Clock

This sample shows the implementation of an analog and digital clock using Essential Diagram component from Essential Studio.

Features

    In order to display the time in an analog clock, the timer is enabled to change the node's pinpoint and rotation angle in an interval.

        		 private void timer2_Tick(object sender, EventArgs e)
		     {		            
		        secondsNode.RotationAngle = DateTime.Now.Second * (360 / 60) + 270;
		        minutesNode.RotationAngle = DateTime.Now.Minute * (360 / 60) + 270;  
		        hoursNode.RotationAngle = passedMinutes / 720F * 360 + 270;            
		     }

    In order to display the time in a digital clock, the timer is enabled to change the visibility of the digits (nodes) in an interval.

            private void DrawClock(int p, Group displayNode)
        {
            diagram1.BeginUpdate();

            int[] bits = bitList[p] as int[];
            
            for (int i = 0, j = 0; i < displayNode.Nodes.Count; i++)
            {
                displayNode.Visible = true;
                if (i == bits[j] - 1)
                {
                    Node node = displayNode.Nodes[bits[j] - 1];
                    if (node != null)
                        node.Visible = true;
                    if (j < bits.Length - 1)
                        j++;
                }
                else
                {
                    Node node = displayNode.Nodes[i];
                    if (node != null)
                        node.Visible = false;
                }
            }
            diagram1.EndUpdate();
        }
