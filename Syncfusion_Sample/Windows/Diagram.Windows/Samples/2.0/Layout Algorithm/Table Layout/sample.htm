Table Layout

The "Table Layout" sample demonstrates the TableLayoutManager layout management class in Essential Diagram. The TableLayoutManager's pre-built layout algorithm arranges the nodes quickly in a table-like manner without implementing tedious node layout routines.

Features

  • Expand modes that specifies how the nodes are arranged.
    • Horizontal - Places the nodes horizontally in a specified number of columns. In this case, row count is automatically calculated. The nodes are arranged in a table-like manner; the table containing the specified number of columns will place the nodes in a new row when the maximum column count is reached.
    • Vertical - Places the parent vertically in the specified number of rows. In this case, column count is automatically calculated. The nodes are arranged in a table-like manner; the table containing the specified number of rows will place the nodes in a new column when the maximum row count is reached.
  • CellSizeMode - Aligns the cells of a table if the nodes are of different sizes.
  • There are three different types of CellSizeMode as given below.
    • EqualToMaxNode
    • b. Minimal
    • c. MinimalTable
  • Horizontal Spacing - Sets the horizontal spacing between the adjacent nodes.
  • Vertical Spacing - Sets the vertical spacing between the adjacent nodes.
  • Top Margin - Sets the top margin of the layout manager.
  • Left Margin - Sets the left margin of the layout manager.
  • MaxSize - Sets the size of the cell.
  • MaxRowsCount - Sets the vertical cell count in the table.
  • MaxColumnCount - Sets the horizontal cell count in the table.
  • AutoLayout - Updates the layout of the model automatically.

    Use the following code, to create TableLayoutManager with the Rows and Columns count as 5.

    			TableLayoutManager tlLayout = new TableLayoutManager(this.diagram1.Model, 5, 5);
            //Sets the cell size mode of TableLayoutManager
            tlLayout.CellSizeMode = CellSizeMode.EqualToMaxNode;
            //Sets the orientation way of lay out.
            tlLayout.Orientation = Orientation.Horizontal;
            //Sets the Max-size of the Layout area.
            tlLayout.MaxSize = new SizeF(1300, 600);
            //Sets the TableLayoutManager as responsible for updating the layout of the diagram.
            this.diagram1.LayoutManager = tlLayout;
            // Updates the layout of the nodes in the model.
            this.diagram1.LayoutManager.UpdateLayout(null);
