Org Chart Layout

The OrgLayout sample uses Essential Diagram to create and display an organizational chart using the employee data sourced from the XML data file.

Features

  • Horizontal Spacing - Sets the horizontal spacing between the adjacent nodes.
  • Vertical Spacing - Sets the vertical spacing between the adjacent nodes.
  • Top Margin - Sets the top margin of the layout manager.
  • Left Margin - Sets the left margin of the layout manager.
  • RotationAngle - Sets the rotation angle for the layout manager.
  • OrgChartLayoutManager supports two different type of LayoutType.
    • Horizontal
    • Waterfall
  • AutoLayout - Updates the layout of the model, automatically.
  • The EmployeeSymbol custom symbol class extends Essential Diagram Symbol base type, and adds the functionality for containing and displaying the child symbols. The expanded-collapsed state of the symbol is indicated by using +/- entities or RightDownTriangle with ellipse or rectangle as backgroundShape drawn by the EmployeeSymbol.

    To create OrgChartLayoutManager with the LayoutType as "Waterfall", the tree rotation as "TopToBottom" (i.e.RotationAngle =0), horizontal spacing and vertical spacing as 40 and 50 respectively, use the following code.

    			OrgChartLayoutManager dtLayoutMgr = new OrgChartLayoutManager(this.diagram1.Model, RotateDirection.TopToBottom, 40, 50, LayoutType.Waterfall, 1);
            //Sets the DirectedTreeLayoutManager as the responsible for updating the layout of the diagram.
            this.diagram1.LayoutManager = dtLayoutMgr;
            // Updates the layout of the nodes in the model.
            this.diagram1.LayoutManager.UpdateLayout(null);
