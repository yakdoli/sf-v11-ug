Seat Reservation

This sample demonstrates the flexibility of the diagram nodes by simulating a seat reservation system.

Features

  • You can select and deselect the desired seats by simply clicking on the seats and then click the "Submit" button to reserve a selection.

    The following code illustrate how to reserve your selection.

    
 //Reserve your selection
 private void btnSubmit_Click(object sender, EventArgs e)
 {
   diagram1.BeginUpdate();
   lblSeatsReserved.Text = "";
   foreach (object val in selectedSeats)
   {
     Node node1 = diagram1.Model.Nodes.FindNodeByName(val.ToString());
     seatsReserved.Add(node1.Name, "Reserved");
     Syncfusion.Windows.Forms.Diagram.Rectangle rect = node1 as Syncfusion.Windows.Forms.Diagram.Rectangle;
     rect.FillStyle.Color = Color.FromArgb(143, 160, 194);
     if (lblSeatsReserved.Text != "")
        lblSeatsReserved.Text += ", " + node1.Name;
     else
        lblSeatsReserved.Text = node1.Name;
   }
   selectedSeats.Clear();
   diagram1.EndUpdate();
   newSelection = true;
 }
