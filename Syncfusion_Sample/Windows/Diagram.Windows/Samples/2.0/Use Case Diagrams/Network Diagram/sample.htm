Network Flow Model

The Network Model sample demonstrates the use of layers in a diagram created using the Diagram Windows control. The depicted diagram is composed of three different layers separated along functional lines. Symbols associated with a specific function are added to the layer representing that category.

Layers can be added/removed, enabled/disabled and shown/hided by the options provided in the sample.

Features

  • Add/Remove Layer - You can add/remove the layer with the given layer name to the diagram. The nodes contained in the layer will be removed when layer is removed.

    • To create and add layer to the diagram model, use the following code.

    		// Initializes a new instance of the Layer class.
	    Layer layer = new Layer(this.diagram1.Model, “Layer1”);
	    //Adds the created layer to the Model's layer collection
	    this.diagram1.Model.Layers.Add(layer);
	    //Adds the created layer to the Model's Active Layer collection
        diagram1.Model.ActiveLayers.Add(layer);

  • Enable/Disable - Layer can be enabled/ disabled by using its Enabled property.

    • To enable/disable a layer use the following code.

    		// Gets or sets a value indicating whether the layer is enabled or disabled.
        layer.Enabled = false;

  • Visiblility - You can use layer visible property to change the visibility of the layer.

    • To hide/show layer in a diagram model, use the following code.

    	// Gets or sets a value indicating whether the layer is visible.
            layer.Visible = true;
