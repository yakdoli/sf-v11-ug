Styling and Formatting

Various styles of the table control have been supported in the Grid Grouping control, according to the different kinds of elements. The control can be easily formatted with a back color and a variety of appearances.

The different samples included in this feature are listed below:

Table Style Demo - This sample demonstrates how you can customize the appearance of nested tables in a hierarchical data set through the designer.

Column Style Demo - This sample demonstrates how to format columns.

Group Style Demo - This samples demonstrates how to customize the layout of groups through the designer.

Record Style Demo - This sample demonstrates how to apply different styles to alternate rows in a  Grid Grouping control.

Skin Customization Demo - This sample illustrates the implementation of VisualStyle support in the GridGroupingControl.

Conditional Formatting Demo - This sample demonstrates how to selectively apply styles to table cells.

Dynamic Formatting Demo - This sample shows how to apply style settings dynamically.

Cell Formatting Demo - This sample simulates the FormatCells dialog feature of Excel in GridGroupingControl.

Table Indent Formatting Demo - This sample allows you to display nested tables in a Grid Grouping control without indenting the children of the records in the parent table.

Custom Cell Range Demo - This sample demonstrates covered ranges, bannered ranges, and drawing background images.

Cell Comment Tip Demo - This sample demonstrates how to show Cell Comment Tip Demo in GridGroupingControl.

Error provider Demo - This sample demonstrates how to show the error icon and set custom error .

Custom Views Demo - This sample demonstrates how to filter, group, sort, and summarize data.
