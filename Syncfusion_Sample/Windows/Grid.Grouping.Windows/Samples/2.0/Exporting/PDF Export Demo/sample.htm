PDF Export Demo

This sample demonstrates converting content in a Grid Grouping control into a PDF file.

Features:

  • Implementation allows multiple options to modify a PDF.
  • Modify according to features such as Add Header, Add Footer, and Specify Margin for all pages.



  • PDF libraries are used to support converting grid content to a PDF page.
  • The dependent DLLs are Syncfusion.Pdf.Base and Syncfusion.GridHelperClasses.Windows.
