Dynamic Object Binding Demo

This sample illustrates the support for Dynamic Object Binding (New feature supports only in and above .Net 4.0) in GridGroupingControl

Features:

Dynamic properties will get populated along with data in GridGroupingControl. All the following operations can be performed with this bounded source as well.

  • Sorting
  • Grouping
  • Add New Record
  • Filtering
  • Summaries
  • Editing and Saving

ExpandoObject and DynamicObject can also be bound to the GridGroupingControl.

Here is the image of the sample

Interactive Features:

  • IsDynamicData have to be enabled before binding datasource to GridGroupingControl.

  • Also supports Complex binding support with dynamic data.

  • You will be able to see the binding of complex properties in both Expando and Dynamic objects

Note : Keyword "dynamic" is not supported by VB, so a VB project for this feature is not applicable
