Nested-Table Group Options Demo

This sample exposes some of the properties of the GridGroupOptionsStyleInfo class.

Features:

  • Properties set through NestedTableGroupOptions are applied only to related child tables in a hierarchy.

  • Properties demonstrated in this sample:

    • Show Caption Plus-Minus

    • Show Caption

    • Add New Record before Details

    • Add New Record after Details

    • Caption Text

  • Changes to these properties can be seen in the Orders table (child table of Customers table in this sample).

  • Other properties of NestedTableGroupOptions:

    • Show Group Footer

    • Show Group Header

    • Show Filter Bar
