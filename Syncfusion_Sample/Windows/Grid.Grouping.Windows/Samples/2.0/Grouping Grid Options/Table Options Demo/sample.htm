Table Options Demo

This sample illustrates the properties of the GridTableOptionsStyleInfo class.

Features:

  • Properties set through the table options are applicable table wide.

  • Some properties demonstrated in this sample:

    • Allow Selection

    • Show Tree Lines

    • Show Row Header

  • Other properties in the table options:

    • Allow Multicolumn Sort

    • Allow Sort Columns

    • Allow Drag Columns

  • Height of the column header row, caption row, record row, summary row, and record-preview row can be set.

  • Default column width, row header width, and indent width can also be controlled through the table options.
