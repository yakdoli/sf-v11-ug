Top-Level Group Options Demo

This sample demonstrates some of the properties of the GridGroupOptionsStyleInfo class.

Features:

  • Properties set using the TopLevelGroupOptions apply only to the top table and not to nested-child tables or child-group tables.

  • Some properties demonstrated in this sample:

    • Show Caption

    • Show Group Footer

    • Show Group Header

    • Add New Record before Details

    • Add New Record after Details
