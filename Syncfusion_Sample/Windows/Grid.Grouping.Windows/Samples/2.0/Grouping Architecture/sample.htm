Grouping Architecture

The content of the cells or any kind of element can be retrieved, and the various events can be tracked to hold the control.

The different samples included in this feature are listed below:

Elements View Demo - Demonstrates how you can access all the elements in a hierarchical grid from top to bottom.

Current Element Access Demo - Demonstrates how you can access the current element and current cell in a Grid Grouping control.

Table Control Events Demo - Allows you to display an event log that will show the order of the various table control events that are raised.

Grouping Engine Events Demo - Allows you to display an event log that will show the order of the various events, related to the grouping engine, that are fired.
