Real-Time Update Demo

This sample shows you how to optimize the refreshing of individual cells or summaries when an underlying data source is frequently changed.

Features:

The sample after execution is illustrated above.

Interactive Features:

  • When the Start Timer button is clicked, it starts a timer that changes the Freight column repeatedly.
  • The Grid caches the graphics context between updates and draws only columns and summaries that were changed using the GridControlBase.DrawClippedGrid method.
