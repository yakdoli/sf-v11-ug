GridVsChart Demo

This sample demonstrates the how to bind a chart based on the selection/ actual value in a Grid.

Features:

Code to invoke the chart control:

Syncfusion.GridHelperClasses.GroupingChartHelper charthelp
        = new Syncfusion.GridHelperClasses.GroupingChartHelper();
charthelp.WireGrid(this.gridGroupingControl1, this.chartControl1,
    "CustomerID", new ArrayList { "Freight" });

 //To remove chartcontrol
 charthelp.UnWire();

  • There are two type of view:

    • Actual View - It is used to show the all value what are in the grid

    • Selection View - It is used to show what are selected in the grid

    Here is the code for ViewType

    charthelp.ViewType = ViewType.SelectionView

  • We can enable the Real Time Update in chart by using to the following code:

    charthelp.RealTimeUpdate
