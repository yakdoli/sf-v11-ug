Outlook Demo

This sample shows the implementation of a Microsoft Outlook like interface using the Essential Grid grouping control and other components from Essential Studio.

The following is a sample image.


Essential Grid has been extended to support all the Office themes (Blue, Black, and Silver) with the look and feel of Outlook.

The scheduler control is also included in this sample, which demonstrates the functionalities and look of Microsoft's Outlook Scheduler.

Features:

  • Applying Office themes --Blue, Black, and Silver--to see the effects of the application

  • Grouping columns in the mails display area (right click on the column headers and select ArrangeBy in the column)

  • Custom selection navigation with all arrow keys

  • Custom grouping for dates, size, priority, and flag fields

  • Dynamic formatting when narrowly switching between column-set view and normal view

  • Customized current view

  • Outlook appearance and functionalities

  • Locates selection based on pressed keys
