Doc to PDF

This sample illustrates Essential DocIO's and Essential PDF's support for conversion from MS Word document to PDF document.

Note: Installation of Essential PDF is necessary to run this sample.

Below is the illustration of the exported file.

Here is a code snippet to convert word document to PDF document using Essential DocIO and Essential PDF

                //Open the word document to convert
                WordDocument wordDoc = new WordDocument("sample.doc");
                DocToPDFConverter converter = new DocToPDFConverter();
                //Convert word document into PDF document
                PdfDocument pdfDoc = converter.ConvertToPDF(wordDoc);
                pdfDoc.Save("DoctoPDF.pdf");
