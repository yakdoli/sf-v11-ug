RTF to DOC

This sample illustrates Essential DocIO's support for converting an RTF file to a Microsoft Word document.

Here's an example of what the conversion looks like.

Here is a code snippet to convert RTF document to Word document using Essential DocIO

                //Open the RTF document to convert
                WordDocument document = new WordDocument();
                document.Open("RTFToDoc.rtf",FormatType.Rtf  );
				//Save the document with FormatType Doc or Docx
            	document.Save("Sample.doc", FormatType.Doc);
