Clone and Merge

This sample illustrates Essential DocIO's support for cloning and merging the MS word documents. The cloning-and-merging feature makes it possible to combine sections from several documents into a single document.

Below is the illustration of the output document.

Features:

  • Combine sections from several documents into a single document
  • Ability to clone sections, tables, paragraphs etc.,.
  • Ability to import Contents from one document to another with options similar to MS Word Copy paste functionality.
