Header and Footer

This sample illustrates Essential DocIO's support for header and footer.

Below are some illustrations demonstrating headers and footers.

Features:

  • Insert headers and footers to include text, images, page numbers, dates, document title, file name, and author name.
  • Add page numbers in formats including Arabic, Roman Upper, and Roman Lower.
  • Restart page numbers from any section and change starting page number for each section.
