Document Protection

This sample illustrates Essential DocIO's support for protecting the Word document.

Below is the illustration of the output document.

Here is a code snippet for protecting the Word document using Essential DocIO

document.Protect(ProtectionType.AllowOnlyRevisions, "password");

Features:

  • Support for protecting the Word documents from editing by third party.
  • DocIO provides the support document protection types - comments only, form fields only, track changes only and reading only.
