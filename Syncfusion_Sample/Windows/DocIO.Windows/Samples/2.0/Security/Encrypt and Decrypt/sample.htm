Encrypt and Decrypt

This sample focuses on the encryption and decryption of a Word document using Essential DocIO.

Below are some illustrations of the output document.

Here is a code snippet:

// Encrypt the document by giving password
document.EncryptDocument("syncfusion");  

//Decrypt the document by giving the correct password
WordDocument document = new WordDocument("c:\\sample.doc","syncfusion");

Features:

  • Supports encryption and decryption of a Word document.
