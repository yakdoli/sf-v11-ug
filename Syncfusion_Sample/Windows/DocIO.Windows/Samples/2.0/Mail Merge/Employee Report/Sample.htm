Employee Report

This sample illustrates how to perform mail merge operations to create an employee report using Essential DocIO.

Below is the illustration of the output document.

Features:

  • Supports for mail merge and events
  • supports mail merging for dynamically growing portions of a document
  • All content with group markers will be automatically repeated for every record in the data source
  • A data table, data view, or the data in a data reader can be used to execute mail merge
