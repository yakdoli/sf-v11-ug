Mail Merge

Mail merge is a word-processing feature that can be used to create many documents at once where only certain portions of each document vary. This helps simplify repetitive tasks.

This section contains the below samples.

  • Letter Format-This samples demonstrates Essential DocIO's support for generating a report in a letter format.
  • Employee Report-This samples demonstrates Essential DocIO's support for generating an employee report.
  • Mail Merge Event-This samples demonstrates Essential DocIO's support for a mail merge event.
  • Nested Mail Merge-This samples demonstrates Essential DocIO's support for a nested mail merge.

Before looking at the samples in this section, let's discuss how a document with a merge field is created and how it works with DocIO.

Mail merge primarily consists of two files, the main document or template and the merge fields. The template contains information that will remain the same in each record, and the merge fields are placeholders for the information that will change the template document according to records from the data source. This includes two main steps; they are as follows:

  1. Merge field creation

    -Create a new Word document with merge fields. This can be done by inserting merge fields through Insert--> Fields--> Merge.

    Provide the merge-field name and text to be inserted before and after the field.

    Note: The merge-field name should match the field name of the data source.

  2. Data Binding

    3. Essential DocIO provides support for merging data from various sources, such as a data set, data reader, data table, or data view. The data from the data source is merged into the template document based on the field name of the data source, which matches the field name of the merge field.
