Styles

This sample demonstrates built-in and custom styles using Essential DocIO.

Below is the illustration of the output document demonstrating built-in styles applied via DocIO.

Here is a code snippet for applying built-in styles using Essential DocIO

doc.LastParagraph.AppendText("Heading 1");
doc.LastParagraph.ApplyStyle(BuiltinStyle.Heading1);

Custom Styles

Below is the illustration of the output document that demonstrates custom styles applied using Essential DocIO.

Here is a code snippet for creating and applying custom styles.

WParagraphStyle style = (IWParagraphStyle)doc.AddParagraphStyle("Normal");
style = (IWParagraphStyle)doc.AddParagraphStyle("UserStyle_Heading1");
style.CharacterFormat.Bold = true;
style.CharacterFormat.FontName = "Verdana";
style.CharacterFormat.FontSize = 25;
IWParagraph paragraph = section.AddParagraph();
paragraph.AppendText("[ Style Applied ]: ");
paragraph.ApplyStyle(style.Name);

Features:

  • Supports built-in styles available in MS Word.
  • Supports user-defined custom styles.
