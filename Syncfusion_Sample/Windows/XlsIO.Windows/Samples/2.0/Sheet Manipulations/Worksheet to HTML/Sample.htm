Worksheet to HTML

A worksheet / workbook can be converted to HTML. The output file can be saved to disc and also as a memory stream.

Here is an illustration of the output.

Features:

  • Worksheet to HTML conversion
  • Workbook to HTML conversion

Interactive Features:

  • Select option from the --Worksheet, Workbook -- to convert into HTML.
  • Click Convert to generate and view the HTML.
  • Click Input Template to view the Input Template file.
  • All features in the preceding list are illustrated in this file.

Here is a code snippet:

// Convert the worksheet to image
sheet.SaveAsHtml("Sample.html");

// Convert the worksheet to image
workbook.SaveAsHtml("Sample.html", HtmlSaveOptions.Default);
