Sheet Manipulations

This category contains samples that deal with various manipulations that can be performed with a worksheet using Essential XlsIO.

Here is a list of samples with a short description of each.

  • Row-Column Manipulations - This sample deals with various operations that can be done in rows and columns in Excel. This includes features like inserting or deleting rows or columns, grouping or ungrouping rows or columns, and setting row height and column width.

  • Worksheet Management - This sample demonstrates the various worksheet management capabilities of XlsIO. You can perform actions like add, move, copy, or delete worksheets. You can copy worksheets with or without formatting preserved, depending upon the user's needs.

  • Worksheet to Image - This sample demonstrates how to convert a worksheet to an image.

  • Worksheet to HTML - This sample demonstrates how to convert a worksheet to HTML.
