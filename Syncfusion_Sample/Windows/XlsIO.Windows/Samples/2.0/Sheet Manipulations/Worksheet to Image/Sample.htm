Worksheet to Image

A worksheet can be converted to an image that is subject to few limitations. The image can be of any format, and it can also be saved as a memory stream.

For limitations, please check Convert to Image.

Here is an illustration of the output.

Features:

  • Worksheet to image conversion
  • Saving workbooks as images of Bitmaps and metafiles

Interactive Features:

  • Choose the option Bitmap/Metafile to convert the worksheet to the required image type.
  • Click Convert to generate and view the document.
  • Click Input Template to view the Input Template file.
  • All features in the preceding list are illustrated in this file.

Here is a code snippet:

// Convert the worksheet to image
Image img = sheet.ConvertToImage(1, 1, lastRow, lastColumn, ImageType.Bitmap, null);

// Save the image.
img.Save("Sample.png", ImageFormat.Png);
