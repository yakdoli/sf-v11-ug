Styles and Formatting

Excel 2007 is known for its various styles and formatting that give it an attractive appearance. Microsoft brings up the gradient styles and a number of new colors. Essential XlsIO supports formatting cells with gradient and RGB colors. You can also set styles and formatting that are supported in Biff8 format.

This sample demonstrates Essential XlsIO's support for various styles and formatting in Excel 2007 format. Both built-in styles and custom styles using the interior method IStyle are used for setting gradient styles.
Note: This can apply gradient styles only for the Excel 2007 format.

Features:

  • Creating a calendar using XlsIO
  • Built-in styles
  • Custom styles
  • Gradients

Interactive Features:

  • Choose one from the options--Excel 2007,Excel 2010--to generate a document in Excel 2007,Excel 2010 respectively.
  • Click Create button to generate and view the document.
  • Click Input Template button to view the Input Template file.
  • All features in the preceding list are illustrated in this file.

Here is code a snippet used to apply a gradient:

// Set builtin style
sheet["K7"].BuiltInStyle = BuiltInStyles.Accent2_20;
sheet["K8"].BuiltInStyle = BuiltInStyles.Accent4_20;

// Set Gradient pattern for the cell.
IStyle style = sheet.Range["B3"].CellStyle;
style.HorizontalAlignment = ExcelHAlign.HAlignCenter;
style.Interior.FillPattern = ExcelPattern.Gradient;
style.Interior.Gradient.TwoColorGradient(ExcelGradientStyle.Diagonl_Up, ExcelGradientVariants.ShadingVariants_3);
style.Interior.Gradient.ForeColorIndex = ExcelKnownColors.White;
style.Interior.Gradient.BackColor = colorArray[monIndex - 1];
