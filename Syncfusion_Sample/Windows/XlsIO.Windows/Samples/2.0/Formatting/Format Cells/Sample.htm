Format Cells

Formatting worksheets makes them more attractive and easier to use. Excel provides various options for setting font, font size, color, and background.

The "Format Cells" dialog in Excel has six tabs, each with several characteristics that allow you to set various formats for a cell. You can set number formats, horizontal and vertical alignment, text control operations, font and font styles, and borders for the cells.

Below is an illustration in Excel showing formatting options.

Features:

  • Various styles such as colors, font settings, borders, and alignments
  • Number formats
  • Patterns
  • Conditional formatting
  • RTF
  • Image insertion

Interactive Features:

  • Choose one from the options--Excel 2003,Excel 2007,Excel 2010--to generate a document in Excel 2003, Excel 2007,Excel 2010 respectively.
  • Click Create to generate and view the document.
  • All features in the preceding list are illustrated in this file.

Here is a code snippet:


//Set Number formatting
sheet.Range["F24"].Number =5000;
sheet.Range["F24"].NumberFormat = "$#,##0.00";

//Set alignment
sheet.Range["D3"].HorizontalAlignment = ExcelHAlign.HAlignCenter;

//Set Text wrap
sheet.Range["D29"].WrapText = true;

//Set font
sheet.Range["D7"].CellStyle.Font.FontName = "Arial Black";

//Set borders
sheet.Range["F9"].CellStyle.Borders.LineStyle = ExcelLineStyle.Medium;
sheet.Range["F9"].CellStyle.Borders[ ExcelBordersIndex.DiagonalDown ].ShowDiagonalLine= false;
sheet.Range["F9"].CellStyle.Borders[ ExcelBordersIndex.DiagonalUp].ShowDiagonalLine= false;

//Set color and pattern
sheet.Range["B7:J7"].CellStyle.FillPattern = ExcelPattern.DarkDownwardDiagonal;
sheet.Range["B7:J7"].CellStyle.Color = Color.FromArgb(163,245,245);
