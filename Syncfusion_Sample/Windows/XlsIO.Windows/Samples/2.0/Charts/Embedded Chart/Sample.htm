Embedded Chart

Essential XlsIO has APIs for creating an embedded chart. The IChartShape interface represents the embedded chart in memory. This object can be used to format and modify chart settings; it is similar to modifying chart properties using Excel.

Features:

This sample demonstrates the following features:

  • Creation of embedded chart types
  • Chart creation from values fetched from a template document

Interactive Features:

  • Choose one from the options --Excel 2003,Excel 2007,Excel 2010--to generate a document in Excel 2003, Excel 2007,Excel 2010 respectively.
  • Click Create to generate and view the document.
  • All features in the preceding list are illustrated in this file.

Below is an illustration of the output document generated by XlsIO.

Here are a few code snippets used in this sample:

//Clustered Column Chart
IChartShape chart = sheet.Charts.Add();

//Set Chart Type
chart.ChartType = ExcelChartType.Pie;

//Set DataRange.
chart.DataRange = sheet.Range["B4:D10"];

//Chart Title
chart.ChartTitle = "Sales comparison";
