Sparklines

Essential XlsIO has APIs for creating sparkline. The SparklineGroup class represents the sparklines in memory. This object can be used to format and modify the sparkline settings; it is similar to modifying sparkline properties using Excel.

Features:

This sample demonstrates the following features:

  • Creation of Sparkline types
  • Sparkline creation from values fetched from a template document

Interactive Features:

  • The generated document will be in Excel 2010 format,because sparkline is an exclusive feature of Excel 2010 and it is not supported in Excel 2003 and 2007 formats
  • Click Create to generate and view the document.
  • All features in the preceding list are illustrated in this file.

Below is an illustration of the output document generated by XlsIO.

Here are a few code snippets used in this sample:

//A new Sparkline group is added to the sheet sparklinegroups
SparklineGroup sparklineGroup = sheet.SparklineGroups.Add();

//Set the Sparkline group type as line
sparklineGroup.SparklineType = SparklineType.Line;

//The sparklines are added to the sparklinegroup.
Sparklines sparklines = sparklineGroup.Add();

//Set the Sparkline Datarange .
IRange dataRange = sheet.Range["D6:G17"];
//Set the Sparkline Reference range.
IRange referenceRange = sheet.Range["H6:H17"];

//Create a sparkline with the datarange and reference range.
sparklines.Add(dataRange,referenceRange);
