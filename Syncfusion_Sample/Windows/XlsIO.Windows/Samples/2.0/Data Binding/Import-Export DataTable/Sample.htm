Import-Export Data Table

In this sample, data from an Excel worksheet is exported to the data grid and the same data is imported into a new worksheet with styles.

The ImportDataTable and ExportDataTable methods make it very easy to work with ADO.NET data tables. It takes just one line of code to import data from a spreadsheet to a data table and export the data from a data table into a spreadsheet.

Features:

This sample demonstrates the following features:

  • Usage of helper methods for working with ADO.NET
  • Importing data from a spreadsheet to a data table
  • Exporting data in a data table to a worksheet
  • Applying column and row styles

Interactive Features:

  • Click Import to import data from the template worksheet to a grid.
  • Choose one from the options --Excel 2003,Excel 2007,Excel 2010--to export a document in Excel 2003, Excel 2007,Excel 2010 respectively.
  • Click Export to export data from a grid to a worksheet with styles applied.
  • All features in the preceding list are illustrated in this worksheet.

Here is a code snippet:

//Read data from spreadsheet and export to datagrid
DataTable customersTable = sheet.ExportDataTable(sheet.UsedRange,ExcelExportDataTableOptions.ColumnNames);
this.dataGrid1.DataSource = customersTable;

//Import the data from grid to excel sheet.
sheet.ImportDataTable((DataTable)this.dataGrid1.DataSource,true,3,1,-1,-1);
