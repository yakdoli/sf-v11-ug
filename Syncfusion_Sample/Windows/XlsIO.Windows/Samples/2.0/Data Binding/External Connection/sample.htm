External Connection

This sample implement the connection between Excel and DB.

To export the selected coutry customer's from the DB and import to the worksheet by using refresh()

Features:

  • To add the connection to workbook
  • To add the query table to worksheet using the connection
  • Using the refresh() to import the data to worksheet from the DB

Interactive Features:

  • Select the customer country from the list.Multiple selection also available.
  • Choose one from the options --Excel 2007,Excel 2010--to generate a document in Excel 2007,Excel 2010 respectively.
  • Click Create to generate and view the external connection generated for the corresponding customer country.
  • All features in the preceding list are illustrated in this file.

            //Add the connection to workbook
