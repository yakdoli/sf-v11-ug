Invoice

This sample focuses on creating an application using an Essential XlsIO and ADO connection.

An Excel template is read by XlsIO and an invoice is generated for the given invoice ID; this is done by fetching data corresponding to the particular ID from the Northwind database. Price, total, and other calculations are performed with simple formulas using Essential XlsIO.

Features:

  • Creating rich reports with styles
  • ADO .NET connectivity
  • Formulas

Interactive Features:

  • Select the invoice ID from the list.
  • Choose one from the options --Excel 2003,Excel 2007,Excel 2010--to generate a document in Excel 2003, Excel 2007,Excel 2010 respectively.
  • Click Create to generate and view the invoice generated for the corresponding invoice ID.
  • All features in the preceding list are illustrated in this file.
