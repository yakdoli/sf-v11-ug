Data Binding

This category contains samples that can be used to create large reports with custom formats.

Here is a list of samples that can be used to create reports.

  • External Connection -This sample demonstrates how Essential XlsIO implement the connection between MsExcel and DB(MsAccess,SQL..etc).

  • Import-Export Data Table - This sample demonstrates importing and exporting data from a data grid using Essential XlsIO. This can be done with a single line of code, and you can apply custom styles when viewing the data in the data grid of an Excel sheet.

  • LINQ Wrapper -This sample demonstrates how Essential XlsIO can be used in association with LINQ.

  • Template Marker -This sample demonstrates the functionality of a marker that can be used to apply formatting.

  • Invoice - This sample demonstrates how an Invoice can be generated for a given customer ID using Essential XlsIO.
