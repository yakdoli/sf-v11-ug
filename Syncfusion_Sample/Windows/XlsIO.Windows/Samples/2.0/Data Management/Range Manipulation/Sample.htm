Range Manipulation

This sample demonstrates Essential XlsIO's support for manipulating a range of cells.

A range is a group or block of cells in a worksheet that have been selected or highlighted. Normally, there is only one cell in a worksheet with a black outline. Ranges are identified by cell references or addresses of the cells in the upper-left and lower-right corners of the range. These two references are separated by a colon (:), which tells Excel to include all the cells between these start and end points. An adjacent cell's range can be depicted as "A5:F10."

Essential XlsIO provides various ways to manipulate ranges.

Features:

  • Named Range Usage

    • Named ranges are a powerful tool in Excel that allows you to assign a meaningful name to a single cell or a range of cells. For example, you can assign the name "Cell1" to cell "C1" and then use the name "Cell1" anytime you would normally use the cell C1. This allows the user to name a collection of cells or a single cell, making it more readable.

    Below are images in Excel for creating named ranges.

  • IRange interface usage

    • This is the common interface used to work with a range.  It allows the user to perform a number of operations, like copying a range of cells from one row or column to another, merging cells, applying formats, assigning text or various formats, and deleting a range of cells.

  • IMigrantRange interface usage

    • This is an optimal way for writing simple strings and strings with styles, which improves performance.

  • AutoFilter

    • This is a method of sub-setting data, which simply removes unwanted data from view. This allows the user to view selected records based on certain constraints, reducing the time it takes to search for a record in a large report. This sample also demonstrates AutoFilter usage in Essential XlsIO.

Here is a code snippet:


//Creates an AutoFilter in the specified range.
sheet.AutoFilters.FilterRange = sheet.Range["B14:E91" ];

//Copies range of cells from source to destination with the styles preserved.
source.CopyTo(des,ExcelCopyRangeOptions.None|ExcelCopyRangeOptions.CopyStyles);

//Defining the Range
IName lname1 = sheet.Names.Add("One");

//Another way to refer range of cells.
lname1.RefersToRange = sheet.Range[98,4,98,5];

//Merges the range of cells
sheet.Range["One"].Merge();

//Creates migrant range and assigns value.
IMigrantRange migrantRange = sheet.MigrantRange;
migrantRange.ResetRowColumn(6,4);
migrantRange.Value = "INVENTORY REPORT";
