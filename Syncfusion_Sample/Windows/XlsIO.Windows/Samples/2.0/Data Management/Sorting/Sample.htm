Data Sorting

Essential XlsIO can sorts a range of cells dynamically, at runtime without any dependency on MS Excel.
It supports to sorts the data based on values and colors.


Features:


  • Supports to sort the large data.
  • Sorts based on the selected algorithm.
  • Sorts the data based on the cell and font color.

Interactive Features:


  • Choose the Columns to sort.
  • Choose the order by in specify the sort order.
  • Choose the column index to sort. Click Sort Values to sort based on values and view the document.
  • Click Sort Colors to sort based on colors and view the document.
