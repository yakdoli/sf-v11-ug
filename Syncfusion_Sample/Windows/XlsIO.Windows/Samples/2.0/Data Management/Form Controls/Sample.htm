Form Controls

Essential XlsIO can read and write form controls such as text boxes, check boxes, and combo boxes.

Note: The option button is supported in XLSX format alone.

Features:

  • Creating rich reports with text boxes, check boxes, and combo boxes using XlsIO
  • Text box formatting
  • Check boxes with Cell Link
  • Comboboxes boxes with formatting
  • Option button with formatting

Interactive Features:

  • Choose one from the options--Excel 2003,Excel 2007,Excel 2010--to generate a document in Excel 2003, Excel 2007,Excel 2010 respectively.
  • Click Create to generate and view the document.
  • All features in the preceding list are illustrated in this file.

Here are code snippets that show how to insert text boxes, check boxes, combo boxes and option buttons:

// Create a textbox and apply backcolor to it
ITextBoxShape textBox1 = sheet.TextBoxes.AddTextBox(15, 5, 23, 190);
textBox1.Fill.FillType = ExcelFillType.SolidColor;
textBox1.Fill.ForeColor = Color.FromArgb(255, 255, 204);

// Create a checkbox
ICheckBoxShape checkBox1 = sheet.CheckBoxes.AddCheckBox(25, 5, 20, 75);
checkBox1.Text = "Studio";

 // Create a combobox
IComboBoxShape comboBox1 = sheet.ComboBoxes.AddComboBox(27, 5, 20, 100);

// Assign value
comboBox1.ListFillRange = sheet2["A1:A12"];

// Select item
comboBox1.SelectedIndex = 6;

// Create an option button
IOptionButtonShape optionButton1 = sheet.OptionButtons.AddOptionButton(46, 5);

// Assign value
optionButton1.Text = "American Express";
