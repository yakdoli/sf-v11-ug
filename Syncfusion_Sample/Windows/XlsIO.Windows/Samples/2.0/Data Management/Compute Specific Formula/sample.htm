Compute Specific Formula

This sample demonstrates how Essential XlsIO can compute the values of a formula entered during run time using the calculate engine.

Features:

  • Exporting a data table to a grid using XlsIO
  • Run-time formula value calculation using an integrated calculate engine

Interactive Features:

  • Click Import Data from Excel to import data and formulas from an Excel worksheet to a grid.
  • Click any formula cell to view the computed value in the text box below.
  • Click Open Sample Workbook in Excel to view the template worksheet with data and formulas.
