Interactive Features

Excel has the following interactive features:

  • Comments
  • Hyperlinks

Comments are notes used to provide context to your data in Excel cells. They are basically used to denote the content in a cell. Text in the comments can be richly formatted to emphasize a comment concerning the cell.

Below is an image in Excel shown when inserting a comment.

A hyperlink is a rich feature in Excel that allows users to instantly access another place in the current workbook, another workbook, or a file associated with another application. A hyperlink can be inserted in a cell or a shape in Excel. This allows you to enter a cell reference in the current workbook. You can also browse to another workbook, a different file, or a Web page.  You can even enter an e-mail address.

Below is an image in Excel showing the insertion of a hyperlink.

  • The insertion of various hyperlinks

Here is a code snippet:

//Insert Comments
//Adding comments to a cell.
sheet.Range["B21"].AddComment().Text= "XlsIO is a Backoffice product with high performance!";

//Add RichText Comments
IRange range = sheet.Range["B23"];
range.AddComment().RichText.Text = "Great Sample!";
IRichTextString rtf = range.Comment.RichText;

//Formatting first 4 characters
IFont greyFont = workbook.CreateFont();
greyFont.RGBColor = Color.FromArgb(333365);
rtf.SetFont( 0, 3, greyFont );

//Formatting last 4 characters
IFont greenFont = workbook.CreateFont();
greenFont.RGBColor = Color.FromArgb(0x85, 0xbf, 0x75);
rtf.SetFont( 4, 7, greenFont );


//Create Hyperlink for a Website
IHyperLink hyperlink = sheet.HyperLinks.Add(sheet.Range["C5"]);

//Set the hyperlink type
hyperlink.Type = ExcelHyperLinkType.Url;

//Set the target address
hyperlink.Address = "http://www.syncfusion.com/products/reporting-edition/xlsio";

//Set the screen tip text
hyperlink.ScreenTip = "To know more About XlsIO go through this link";
