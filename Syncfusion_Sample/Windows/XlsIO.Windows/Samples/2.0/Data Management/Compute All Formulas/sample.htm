Compute All Formulas

Essential XlsIO can compute the values of a formula entered during run time using the calculate engine that is integrated in XlsIO. This makes it possible to retrieve the values of  formulas that were entered using XlsIO during run time. Currently, over 150 functions are supported, which covers all common usage scenarios.

Features:

  • Writing formulas using XlsIO
  • Formula calculation using the integrated calculate engine

Interactive Features:

  • Choose one from the options--Excel 2003,Excel 2007,Excel 2010--to generate a document in Excel 2003, Excel 2007,Excel 2010 respectively.
  • Click Create to generate and view the document.
  • All features in the preceding list are illustrated in this file.
  • View the calculated values in output window.
