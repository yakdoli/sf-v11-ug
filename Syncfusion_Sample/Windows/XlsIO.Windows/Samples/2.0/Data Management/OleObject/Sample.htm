OleObject

Essential XlsIO now supports the insertion of OLE objects. This sample demonstrates how an OLE object can be inserted. It is possible to embed the object, or insert it as a link. For both types, location and size can be adjusted.

Features:

  • Ability to Insert OLE objects.
  • Ability to adjust position.
  • Ability to adjust size.

Interactive Features:

  • Choose from the options Excel 2007 or Excel 2010 to generate a document.
  • Click Create to generate the document.
  • All features will be illustrated in the generated file.

Below is an image of the output.

Here is a code snippet:

// Insert OleObject
IOleObject oleObject1 = sheet.OleObjects.Add(@"FaxOrderForm.pdf", System.Drawing.Image.FromFile(@"pdfIcon.jpg"), OleLinkType.Embed);

// Specify the position
oleObject1.Location = sheet[8, 11];

// Specify the size of the icon
oleObject1.Size = new Size(100, 100);
