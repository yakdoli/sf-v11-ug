Pivot Table

When there is a need to summarize large reports, Excel allows you to summarize data through pivot table insertion. Excel 2007 supplies various built-in styles for pivot tables, allowing the creation of pivot tables with rich formats and more flexibility in summarizing reports.

Features:

  • Creating pivot tables from template data
  • Customizing pivot tables with various properties
  • Formatting pivot tables with built-in styles

Note: This feature is currently not supported for the .xls format.

Interactive Features:

  • Choose one from the options --Excel 2007,Excel 2010--to generate a document in Excel 2007,Excel 2010 respectively.
  • Click Create pivot table to generate and view the document.
  • Click Customize pivot table to generate and view the document with customized pivot table.

Here is a code snippet used to create and format the pivot table.

//Create Pivot table
IPivotCache cache = workbook.PivotCaches.Add(sheet["A1:H50"]);
IPivotTable pivotTable = sheet1.PivotTables.Add("PivotTable1", sheet1["A1"], cache);
pivotTable.Fields[2].Axis = PivotAxisTypes.Row;
pivotTable.Fields[4].Axis = PivotAxisTypes.Row;
IPivotField field = sheet1.PivotTables[0].Fields[5];
pivotTable.DataFields.Add(field, "Sum of Units", PivotSubtotalTypes.Sum);

//Apply built-in style
pivotTable.BuiltInStyle = PivotBuiltInStyles.PivotStyleDark12;
