Stock Portfolio

The Stock Portfolio sample creates a stock report selected by the user at run time. Data for the stocks will be retrieved from a database and populated in the Excel spreadsheets, which are created using Essential XlsIO. The generated Excel sheet will contain hyperlinks, comments, intuitive charts, and images.

Features:

  • Chart creation with rich formatting
  • Hyperlinks
  • Images
  • Worksheet copying
  • Chart creation from ADO.NET connectivity

Interactive Features:

  • Select one more stocks from the list.
  • Select the start and end date timeline.
  • Select a background image.
  • Select the check box for a curved border in the chart.
  • Click Create Report to view the stock details for the selected option.
  • Click on the links in the output worksheet to navigate and view the particular stock details.
  • Click Input Template to view the Input Template file.
