Settings

This category contains samples that apply various settings to an Excel document.

Here is the list of samples that demonstrate the various settings of Excel using Essential XlsIO. Microsoft Excel provides various settings for security and protection in order to control whom access is given to and what can be changed.

  • Encrypt and Decrypt - This sample demonstrates how a XLSX file can be encrypted or decrypted using XlsIO.

  • Document Settings - This sample demonstrates how various properties that affect the document, such as page-setup options and document properties, can be set using Essential XlsIO.

  • Worksheet Protection - This sample demonstrates how an Excel worksheet can be protected and unprotected with a password. It also explains how to lock and unlock a range of cells.

  • Workbook Protection - This sample demonstrates how an Excel workbook can be protected and unprotected with a password.
