Essential ProjIO Essential ProjIO is a .Net class library that can read and write Microsoft Project xml files. It does not make of COM interop and is built from scratch in C#. It can even be used on systems that do not have Microsoft Project installed

This sample demonstrates the creation of a simple project from the scratch using few lines of code with Essential ProjIO

Features:

  • Provides read and write for project xml format documents.
  • Create and manipulate the tasks in a project
  • Create and update Resources in a project
  • Manipulate resource assignments in a project
