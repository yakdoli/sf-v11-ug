Export

Grid control contents can be exported to an XLS, PDF and MS Word files.

The exporting feature is illustrated in the following samples:

GridDataBoundGrid XLS Export Demo - This sample demonstrates how GridDataBoundGrid content can be exported to an XLS file.

GridDataBoundGrid PDF Export Demo - This sample demonstrates how GridDataBoundGrid content can be exported to a PDF file.

GridDataBoundGrid Word Converter Demo - This sample demonstrates how GridDataBoundGrid content can be exported to a Word file.
