The Card View Layout support the view of record in card view format. At present, it was implemented like Wiring the GridDataBoundGrid into GridCardViews object.

The below image shows the Card View Layout.

Card View Demo

Features

The Card View layout represents data in a card format, where records are represented by cards and record fields are represented by card fields.Card View display data records as cards consisting of card fields. The Card View support is higly enhanced or customized by using the events such as CardQueryCellInfo, CardSaveCellInfo, CellClick etc.

Please refer to the following code. 

//Wire the GridDataBoundGrid.
GridCardView card = new GridCardView();
card.WireGrid(this.gridDataBoundGrid1);

Interactive Features

  • CardCellBorders - Show/hide the card view record cells border's.

  • ShowCaption - Show/hide the cardcaption's.

  • ApplyRoundedCorner - The card view layout can be customized, i.e: the card view corners can be represented normally, or with rounded corners.

  • BrowseOnly - The card view edit mode can be enabled/disabled by using this property.

  • CardVisualstyle - Applies the card visual styles like Office2007Blue, Office2010Black, Metro etc.

  • CardStyle - Card View style are represented in two way which is StandardLables and MergedLables. In StandardLables the card view is shown as the header field and records are represented within a card. In MergedLables the card view contain only the records, the header fields are represented commonly.
