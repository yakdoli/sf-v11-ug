Customization

This section consists of the following samples:

Drop Grid Demo - This sample displays hierarchical data using nested drop grids.

Events Demo - This sample illustrates tracing the order of events fired when operating a data bound grid control.

GDBG Combo Columns Demo - This sample illustrates Essential Grid's support for the foreign-key combo boxes in grid cells.

GDBG Multi-Header Demo - This sample illustrates the implementation of multiple row and column headers in a GDBG.

Multi-Row Record Demo - This sample illustrates how to switch between the records from the Northwind Customers table and simultaneously displaying a single row per record and multiple rows per record.

Repeater User Control Demo - This sample illustrates the definition of a user control in order to edit a row in a data table.
