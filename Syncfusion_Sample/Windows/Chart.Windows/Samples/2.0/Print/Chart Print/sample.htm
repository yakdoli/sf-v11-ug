Chart Printing

Previewing and printing a chart is very easy in Essential Chart.

Features:

  • Print Preview

    A print-preview dialog can be opened by using the following code. 

    
PrintDocument printPreviewDocument = this.chartControl1.PrintDocument;
this.printPreviewDialog1.Document = printPreviewDocument;
this.printPreviewDialog1.FormBorderStyle = FormBorderStyle.Fixed3D;
this.printPreviewDialog1.ShowDialog();
  • Printing Chart

    A chart can be printed in gray scale, color, or "Check Printer" mode. The following code shows how a chart can be printed using color mode.

    
PrintDialog printDialog1 = new PrintDialog();
printDialog1.Document = this.chartControl1.PrintDocument;
if (printDialog1.ShowDialog() == DialogResult.OK)
{
   this.chartControl1.PrintColorMode = ChartPrintColorMode.Color;
   this.chartControl1.PrintDocument.Print();
}
  • Printing Toolbar

    The chart control supports restricting the printing of a toolbar using the PrintToolbar Boolean property.

    
 this.chartControl1.PrintDocument.PrintToolBar =true;

    Also, a chart control can support printing a series with textures when printing in black-and-white mode.

    
this.chartControl1.PrintColorMode =ChartPrintColorMode.GrayScale;
