Spark Line Chart

This sample illustrates the various types of sparkline charts supported by Essential sparkline.

Sparkline Chart Demo

Features

  • Sparkline Chart Types

    The sparkline chart type is set as shown below 

    
//sets the Sparkline Type as Line
this.sparkLine1.Type = SparkLine.SparkLineType.Line;
//sets the Sparkline Type as Column
this.sparkLine1.Type = SparkLine.SparkLineType.Column;
//sets the Sparkline Type as WinLoss
this.sparkLine1.Type = SparkLine.SparkLineType.WinLoss;
  • Marker Support

    The markers are visual indicator to represent the location of data points in the Sparkline graph. 

    
//To enable the markers should be display at every data point location in line sparkline graph
this.sparkLine1.Markers.ShowMarker = true;
//To enable the markers for highest value shown in the all types of sparkline.
this.sparkLine1.Markers.ShowHighPoint = true;
//To enable the markers for lowest value shown in the all types of sparkline.
this.sparkLine1.Markers.ShowLowPoint = true;
//To enable the markers for start value shown in the all types of sparkline.
this.sparkLine1.Markers.ShowStartPoint = true;
//To enable the markers for end value shown in the all types of sparkline.
this.sparkLine1.Markers.ShowEndPoint = true;
//To enable the markers for negative value shown in the all types of sparkline.
this.sparkLine1.Markers.ShowNegativePoint = true;
  • Color Setting

    Customized the markers , line, column and background colors for sparkline charts through the below properties. 

    
//Gets or Sets color to line, column and backgroung. 
this.sparkLine1.LineStyle.LineColor  = this.colorPickerButton1.SelectedColor;
this.sparkLine1.ColumnStyle.ColumnColor = new Syncfusion.Drawing.BrushInfo(this.colorPickerButton2.SelectedColor);
this.sparkLine1.BackInterior = new Syncfusion.Drawing.BrushInfo(Color.White);
//Gets or Sets marker color to line type sparkline. 
this.sparkLine1.Markers.MarkerColor = new Syncfusion.Drawing.BrushInfo(this.colorPickerButton6.SelectedColor);
//Gets or Sets marker color to High, Low, Start,End and Negative points
this.sparkLine1.Markers.HighPointColor = new Syncfusion.Drawing.BrushInfo(this.colorPickerButton6.SelectedColor);
this.sparkLine1.Markers.LowPointColor = new Syncfusion.Drawing.BrushInfo(this.colorPickerButton6.SelectedColor);
this.sparkLine1.Markers.StartPointColor = new Syncfusion.Drawing.BrushInfo(this.colorPickerButton6.SelectedColor);
this.sparkLine1.Markers.EndPointColor = new Syncfusion.Drawing.BrushInfo(this.colorPickerButton6.SelectedColor);
this.sparkLine1.Markers.NegativePointColor = new Syncfusion.Drawing.BrushInfo(this.colorPickerButton6.SelectedColor);
