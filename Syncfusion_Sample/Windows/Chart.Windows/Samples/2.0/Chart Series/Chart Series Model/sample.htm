Chart Series Model

This sample demonstrates the implementation of a basic chart series model and chart series indexed model.

  

Features:.
  • Indexed Mode

    Indexed models do not have an associated x-axis value for their points. The x-axis value is the position of data within a series. The chart control can be switched to the indexed mode by using the Indexed property.

    • Custom Series Model

      The CustomSeriesModel class that implements the IChartSeriesModel interface provides both the "X" and "Y" values for plotting chart points.

      
series.SeriesModel = new CustomSeriesModel();
this.chartControl1.Indexed = false;
    • Custom Indexed Series Model

      The CustomIndexedSeriesModel class implements the IChartSeriesIndexedModel interface and is used in special cases where the "X" values are not needed. 

      
series.SeriesIndexedModelImpl = new CustomIndexedSeriesModel();
this.chartControl1.Indexed = true;

  • Label Model

    The label for a chart series is added by customizing IChartAxisLabelModel.
