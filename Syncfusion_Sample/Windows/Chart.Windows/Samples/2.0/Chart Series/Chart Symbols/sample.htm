Chart Symbols

Essential Chart features a rich styles architecture that allows point-level formatting as well as series-level formatting; it also has the ability to control the manner in which individual settings are inherited.

This sample demonstrates symbols in a series as well as in individual series points.

Chart Symbols

Features:

  • Adding symbols to points in a series

    Symbols are added to a series as shown below.

    
this.chartControl1.Series[14].Style.Symbol.Shape = ChartSymbolShape.Star;
this.chartControl1.Series[14].Style.Symbol.Size = new Size(10,10);
this.chartControl1.Series[14].Style.Symbol.Color = Color.Yellow;

    Note: adding a symbol to a series, as shown above, will apply to all the points in a series.

  • Adding symbols to an individual point

    Symbols are added to individual points using the PrepareStyle event, as shown below.

    
this.chartControl1.Series[14].PrepareStyle += new ChartPrepareStyleInfoHandler(ChartControlSeries_PrepareStyle);
