Chart Custom Points

Custom points can be plotted on a chart to annotate interesting data. A custom point can display text, backgrounds, borders, symbols, and markers. This sample illustrates both custom points and empty points in a chart control.

Chart Custom Points

Features:

  • Custom Points

    Custom points are created as shown below.

    
ChartCustomPoint cp1=new ChartCustomPoint();
cp1.Symbol.Shape = ChartSymbolShape.Circle;
cp1.Symbol.Color = Color.Red;
cp1.Font.Size = 9f;
this.chartControl1.CustomPoints.Add(cp1);
  • Custom Point Position

    There are various options that can be used to set the position of a custom point on a chart; this is performed using the CustomType property.
    The various ChartCustomPointType options are:

    • Percent
      The coordinates of a custom point are taken as a percentage of the chart area.
      
cp1.CustomType = ChartCustomPointType.Percent;
// Point that follows a percent type
cp1.XValue = 68;
cp1.YValue = 47;
    • Pixel
      The coordinates of a custom point are taken in pixels. 
      
cp1.CustomType = ChartCustomPointType.Pixel;
// Point that follows a pixel type
cp1.XValue = 190;
cp1.YValue = 135;
    • Chart Coordinates

      Chart coordinates are taken as coordinates of a custom point. These are not tied to a series.

      
cp1.CustomType = ChartCustomPointType.ChartCoordinates;
// Point that uses chart coordinates 
cp1.XValue = 2003;
cp1.YValue = 289;
    • Point Follow
      A custom point will follow the regular point of any series it is assigned to. 
      
cp1.CustomType = ChartCustomPointType.PointFollow;
// Point that follows a series point:
cp1.PointIndex = 0;
cp1.SeriesIndex = 1;
  • Empty Points

    A chart point can be made into an empty point by setting the IsEmpty property. 

    
 this.chartControl1.Series[0].Points[1].IsEmpty = true;
  • Symbol Border

    The chart control supports setting a border style for ChartStyleInfo.Symbol.

    
 this.chartControl.Series[0].Style.Symbol.Border.Width = 2;
 this.chartControl.Series[0].Style.Symbol.Border.Color = Color.Yellow;
 this.chartControl.Series[0].Style.Symbol.Border.DashStyle=DashStyle.Solid;
