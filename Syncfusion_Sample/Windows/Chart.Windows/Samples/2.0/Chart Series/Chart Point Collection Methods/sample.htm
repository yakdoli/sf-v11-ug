Chart Point Collection Methods

This sample shows how to find the minimum and maximum values from a chart control series collection.

Chart Point Collection Methods

Features

  • Find Value

    The chart control supports four overloading methods for FindValue, as given below:

    • FindValue( Double)

      This method returns the first chart point in the collection that has a specified first Y-value. The search always starts at the beginning of the collection.

      	 this.chartControl1.Series[0].Summary.FindValue(dbl);
    • FindValue (Double, String)

      This method returns the first chart point in the collection that has a specified X- or Y-value.

      	 this.chartControl1.Series[0].Summary.FindValue(dbl, str);
    • FindValue (Double, String, Index )

      This method returns the first chart point with the specified X- or Y-value and starts the search at the specified index to last point.

      	 this.chartControl1.Series[0].Summary.FindValue(dbl, str, ref staIndx);
    • FindValue (Double, String, Index, Index )

      This method returns the first chart point with a minimum value. The search starts and ends at the specified index.

      	 this.chartControl1.Series[0].Summary.FindValue(dbl, str, ref staIndx, endIndx);

  • FindMaxValue

    The chart control supports four overloading methods for FindMaxValue, as given below

    • FindMaxValue()

      This method returns the first chart point in the collection with a maximum first Y-value. The search always starts at the beginning of the collection.

      	 this.chartControl1.Series[0].Summary.FindMaxValue();
    • FindMaxValue(String)

      This method returns the first chart point in the collection with a maximum first X- or Y-value. The search always starts at the beginning of the collection.

      	 this.chartControl1.Series[0].Summary.FindMaxValue(str);
    • FindMaxValue(String, index)

      This method returns the first chart point in the collection with a maximum first X- or Y-value. The search always starts at the specified index.

      	 this.chartControl1.Series[0].Summary.FindMaxValue(str, ref staIndx);
    • FindMaxValue (String, Index , Index)

      This method returns the first chart point with a maximum value. The search always starts and ends at the specified index.

      	 this.chartControl1.Series[0].Summary.FindMaxValue(str, ref staIndx, endIndx);

  • FindMinValue

    The chart control supports four overloading methods for FindMinValue, as given below

    • FindMinValue()

      This method returns the first chart point in the collection with a minimum first Y-value. The search always starts at the beginning of the collection.

      	 this.chartControl1.Series[0].Summary.FindMinValue();
    • FindMinValue (String)

      This method returns the first chart point in the collection with a minimum first X- or Y-value. The search always starts at the beginning of the collection.

      	 this.chartControl1.Series[0].Summary.FindMinValue(str);
    • FindMinValue (String, index)

      This method returns the first chart point in the collection with a minimum first X- or Y-value. The search always starts at the specified index.

      	 this.chartControl1.Series[0].Summary.FindMinValue(str, ref staIndx);
    • FindMinValue (String, Index , Index)

      This method returns the first chart point with a minimum value. The search always starts and ends at the specified index.

      	 this.chartControl1.Series[0].Summary.FindMinValue(str, ref staIndx, endIndx);
