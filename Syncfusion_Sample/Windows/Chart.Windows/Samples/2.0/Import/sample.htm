Importing

Essential Chart supports different ways to import series data.

  • Data from an Array-This sample demonstrates how to retrieve chart data from an array.

  • Data from an Indexed Model-This sample demonstrates how to retrieve chart data using an indexed model.

  • Data from CSV-This sample demonstrates loading and displaying chart data from a CSV file.

  • Data from Database-This sample demonstrates loading and displaying chart data from a database.

  • Data from Excel-This sample demonstrates loading and displaying chart data from an Excel file.

  • Data from XML- This sample demonstrates loading and displaying chart data from a XML file.
