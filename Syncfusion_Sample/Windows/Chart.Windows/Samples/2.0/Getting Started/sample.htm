Getting Started

  • Getting Started-This sample illustrates how to create a simple column chart using Essential Chart.

    • A wizard interface is included in Essential Chart.

    • Using the wizard interface property, settings can be changed easily.

    • With the wizard interface displayed, properties can also be changed dynamically.

    • Essential Chart can be displayed on a form by dragging and dropping it there.

    • A chart can be initialized by creating a series and adding it to the series using the Form_Load event.
