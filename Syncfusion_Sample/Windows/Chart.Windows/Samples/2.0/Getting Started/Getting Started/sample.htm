Getting Started

This sample illustrates how to get started quickly by populating a chart with data points and configuring it with common settings.

The chart control used in this sample is dragged and dropped from the toolbox onto the form. A set of series is then created and added to the chart control.

Features

  • Creating a series

    This chart series was created using the following code.

    
ChartSeries series = new ChartSeries("Series 1");
series.Points.Add(0, 500);
series.Points.Add(1,340);
series.Points.Add(2,240);
series.Points.Add(3,440);
series.Type = ChartSeriesType.Column;
series.Text = series.Name;

    The above code creates a series with four points, sets the chart type, and assigns a name to the series.

  • Adding a series to a chart

    The above series was added to the chart as follows:

    this.chartControl1.Series.Add(series);

    Similarly, three different series were created and added to the chart.

  • Customize Chart

    Basic customization of the appearance and behavior of the chart area, legend, and axes occurs in the InitializeControlSettings method.
