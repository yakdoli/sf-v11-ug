Chart Styles Customization

Essential Chart features a rich-styles architecture that allows point-level formatting as well as series-level formatting with the ability to control how individual settings are inherited.

This sample demonstrates series styles and series-point styles.

Custom Chart Styles

Features

Chart points can be customized individually to show specific points of importance. Interior color, style, label color, label format, symbol properties, and other appearance-related properties can be customized using the Styles property.
