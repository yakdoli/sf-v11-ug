Right-to-Left Support

This sample demonstrates the right-to-left support of both text and axes.

RTL Chart

Features

By default, a chart control displays text and axes in left-to-right alignment. For axes customization, the chart control has the Inversed property for x-axis and OpposedPosition property for y-axis.

Essential Chart also supports right-to-left languages, like Hebrew and Arabic. To work with RTL languages, you need to install those languages locally.

    
// Implementing RightToLeft support to axes.
this.chartControl1.PrimaryXAxis.Inversed = true;
this.chartControl1.PrimaryYAxis.OpposedPosition = true;
