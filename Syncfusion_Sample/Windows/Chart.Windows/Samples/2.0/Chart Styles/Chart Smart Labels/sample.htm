Chart Smart Labels

This sample shows how to customize the appearance of smart labels in the chart control.

Chart Smart Labels

Features

    The appearance of the smart labels is customized through various properties, as shown below.

  • Border settings

    • Smart labels Border

      Border width and border color of the smart tag can be set using the code below.

      this.chartControl1.SmartLabelsBorderColor= Color.Orange;
this.chartControl1.SmartLabelsBorderWidth = 1;
