Zoom and Scrolling

The "Zooming and Scrolling" samples illustrate the zooming and scrolling operations supported by Essential Chart.

  • Key zooming can be performed by using the ZoomIn and ZoomOut properties of a chart control.

  • Scrolling in all directions is enabled by using the properties ZoomRight, ZoomLeft, ZoomUp, and ZoomDown.

  • Essential Chart also supports mouse zooming and scrolling.
