Chart Utility Functions

The samples under Statistics show how to perform a statistical analysis using Essential Chart.

Chart with Utility Function

Features

The result of the utility functions performed in this sample, like normal distribution, t-cumulative distribution, and f-cumulative distribution, can be calculated using the UtilityFunctions class. The code below demonstrates the calculation of normal distributions and inverse normal distributions. 

    
double values = UtilityFunctions.NormalDistribution(0.3);
double values = UtilityFunctions.InverseNormalDistribution(0.3);
