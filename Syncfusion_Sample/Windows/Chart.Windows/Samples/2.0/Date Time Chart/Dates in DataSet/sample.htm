Dates in a Data Set

This sample demonstrates binding a chart series to a data set with date-time values.

Data-Bound Chart

Features:

  • Chart Data-Bind Model

    ChartDataBindModel allows a data source to bind with a chart series. 

    
ChartDataBindModel model = new ChartDataBindModel(this.prodDs1,"Products");
model.XName = "ExpiresDate";
model.YNames = new string[]{"Quantity"};
series.SeriesModel = model;

    Since the values in the data set are of the date-time type, the value type of the primary x-axis is set as DateTime. 

    
this.chartControl1.PrimaryXAxis.ValueType = ChartValueType.DateTime;
