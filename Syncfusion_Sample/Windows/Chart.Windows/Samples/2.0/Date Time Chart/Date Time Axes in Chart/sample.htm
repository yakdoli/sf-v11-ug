Date Time Axes in Chart

This sample demonstrates date handling in Essential Chart. The chart series data can be defined with date-time values. The primary x-axis and y-axis of the control can be set with date-time values.

Chart Date Handling

Features:

  1. Populate Date-Time Values

    Use the following code to populate the series with date-time values. 

    
DateTime start = new DateTime(2003, 11, 1);
series1.Points.Add(start.AddDays(7), 363);
series1.Points.Add(start.AddDays(14), 417);
series1.Points.Add(start.AddDays(21), 263);
series1.Points.Add(start.AddDays(28), 453);
series1.Points.Add(start.AddDays(35), 378);

  2. Date-Time Value Type

    The data type for the axis value on the chart is set to date-time using the ValueType property. 

    
// Specify the value type of the X-axis as DateTime.
this.chartControl1.PrimaryXAxis.ValueType = ChartValueType.DateTime;

  3. Date-Time Range and Interval

    The date-time range of the axis is set using the DateTimeRange property. 

    
//Sets the range for the Axes.
this.chartControl1.PrimaryXAxis.RangeType=ChartAxisRangeType.Set;
this.chartControl1.PrimaryXAxis.DateTimeRange=new ChartDateTimeRange(start,start.AddDays(42),7,ChartDateTimeIntervalType.Days);

    The date-time interval is set using the axis IntervalType property. 
    
this.chartControl1.PrimaryXAxis.IntervalType = ChartDateTimeIntervalType.Days;

  4. Date-Time Format

    The date-time format is set by using the DateTimeFormat property. 

    
this.chartControl1.PrimaryXAxis.DateTimeFormat = "d MMM, ddd";
