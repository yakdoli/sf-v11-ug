Real Time

Essential Chart provides a real-time chart with scrolling that can be implemented using either data or an array model.  The chart control updates itself when data changes.  Essential Chart allows you to draw a real-time chart recorder by implementing the IChartSeriesModel.

  • Chart Recorder-This sample shows a real-time chart recorder application implemented using Essential Chart.
