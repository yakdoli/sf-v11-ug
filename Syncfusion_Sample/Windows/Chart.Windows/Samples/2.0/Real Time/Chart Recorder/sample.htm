Chart Recorder

This sample shows a real-time chart recorder application implemented using Essential Chart.

Chart Recorder

Features:

  • The chart data points are held outside the chart control, and the chart control updates itself when data changes.


  • The length of time represented on the x-axis is controlled through a top-array-model implementation providing only the required number of points to the chart.


  • Zooming and scrolling are supported for user-selected portions of the chart.
