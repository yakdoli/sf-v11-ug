Filtered Data

This sample shows how to improve the performance of a chart when loading a large number of data points using IChartSeriesModel.

Chart Filtering

Features:

  • In this sample, a new set of data points is added every 800 milliseconds.  Data points added before five minutes are filtered out by the grouping engine. The range along the primary x-axis is held for three minutes during run time in order to improve the performance of the chart.

    
// Data points are added as per the Timer Interval.

if (this.chartControl1.Series[7].Points[this.chartControl1.Series[7].Points.Count - 1].DateX  > start)
{
this.chartControl1.PrimaryXAxis.DateTimeRange = new ChartDateTimeRange(DateTime.Now.AddMinutes(-1),DateTime.Now.AddMinutes(2),1,ChartDateTimeIntervalType.Minutes);
start = this.chartControl1.PrimaryXAxis.DateTimeRange.End;
}

  • The styles of the series points can be enabled or disabled. 

    
//Point Symbols and Point Text will not be visible if 'EnableStyles' is set to false.
series.EnableStyles = false;
