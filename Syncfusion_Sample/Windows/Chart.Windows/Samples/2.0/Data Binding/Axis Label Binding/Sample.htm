Axis Label Binding

This sample demonstrates loading and displaying chart data from a database. It also illustrates how to bind axis labels in pie, funnel, and pyramid charts.

Features:

  • Chart Data-Bind Model

    Essential Chart exposes a flexible data model, ChartDataBindModel, that can easily display data from any source (file, database, real time).
    The .mdb file is first converted to a data set; ChartDataBindModel then uses this as a source for populating data.

    
// The Access database
string fileName = WinFormsUtils.FindFile(@"..\..\..\..\..\Data\","ChartData.mdb"); 
string myConnectionString = "PROVIDER=Microsoft.Jet.OLEDB.4.0;Data Source=" + fileName;

// Define the database query
string mySelectQuery = "SELECT City, ID, Population FROM Demographics";

// Create a database connection object using the connection string    
OleDbConnection myConnection = new OleDbConnection(myConnectionString);

// Create a database command on the connection using query
OleDbCommand myCommand = new OleDbCommand(mySelectQuery,myConnection);

myCommand.Connection.Open();

// Create a database reader
OleDbDataReader myReader = myCommand.ExecuteReader(CommandBehavior.CloseConnection);

//Load the contents into a dataset.
DataSet dataSet = ConvertToDataSet(myReader,"Demographics");

//Initializes new chart series.
ChartSeries series = new ChartSeries();
series.Name = "Products";

ChartDataBindModel model = new ChartDataBindModel(dataSet,"Demographics");
model.XName = "ID";
model.YNames = new string[]{"Population"};

series.SeriesModel = model;

//Adds the series to the ChartSeriesCollection.
this.chartControl1.Series.Add(series);

// Close the reader and the connection
myReader.Close();
myConnection.Close();
  • Chart Data-Bind Axis Label Model

    The labels for the x-axis are loaded from the database using an implementation of the ChartDataBindAxisLabelModel class.

    
this.xAxisLabelModel = new ChartDataBindAxisLabelModel(dataSet,"Demographics");
this.xAxisLabelModel.LabelName = "City"; 

//this.chartControl1.PrimaryXAxis.AutoValueType = false;
this.chartControl1.PrimaryXAxis.LabelsImpl = this.xAxisLabelModel;

////Sets the value type of the axis.
this.chartControl1.PrimaryXAxis.ValueType = ChartValueType.Custom;
  • ShowDataBindLabels

    The following code illustrates how to bind axis labels to pie, funnel, and pyramid charts.

    
this.chartControl1.Series[0].Style.DisplayText=true;
this.chartControl1.Series[0].ConfigItems.PieItem.ShowDataBindLabels = true;
this.chartControl1.Series[0].ConfigItems.FunnelItem.ShowDataBindLabels = true;
this.chartControl1.Series[0].ConfigItems.PyramidItem.ShowDataBindLabels = true;
