Huge Data in Multi-Chart

This sample shows how selections in a high-level view can be expanded to a low-level view when they are bound to large data.

Features:

The top chart shows the entire set of 100,000 data points while the bottom chart shows a subset of the points based on the selection in the top chart.

Both charts are bound to a custom-data model that automatically compresses (by averaging) multiple chart points into one; this makes it possible to plot all the points within the available chart space.   

The custom-data model automatically increases or decreases this compression ratio as the chart's size changes because of resizing or zooming.
