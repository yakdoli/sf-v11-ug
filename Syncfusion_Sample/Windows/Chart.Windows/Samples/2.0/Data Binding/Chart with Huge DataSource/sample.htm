Chart with Huge Data Source

This sample illustrates rendering 100,000 data points on a chart.

Features

  • A custom-chart data model, ChartOversizedDataBindModel, is used to render data points from a large data source. This model uses Essential Grouping to compress (by averaging) the number of data points rendered, automatically, based on chart width.

  • The custom-chart data model also listens to the chart resize and zooming events, and it automatically shows more or less points in the chart.

Interactive Features:

  • According to the combo box selection, the chart control will render chart series points in the sample.
