Multiple Axes Layout

This sample demonstrates the usage of a multiple axes layout in Essential Chart.

Multiple Axes Layout

Features:

  • Multiple Axes

    Multiple axes are created in the chart as shown below. 

    
//Initialize a ChartAxis
private ChartAxis secYAxis = new ChartAxis();

    
// To render this axis at a position opposite to that of the primary Axis.
secYAxis.OpposedPosition = true;
secYAxis.Orientation = ChartOrientation.Vertical;
//Add the ChartAxis to the Axes collection
this.chartControl1.Axes.Add(secYAxis);
// Specify that the 3rd series should be plotted against this secondary Y axis
this.chartControl1.Series[2].YAxis = secYAxis;
  • Layout Mode

    The layout mode of an axis is set using the YAxesLayoutMode property. 

    
 this.chartControl1.ChartArea.YAxesLayoutMode = ChartAxesLayoutMode.SideBySide;

    The chart control supports combining both stacking and side-by-side chart axes layouts when multiple axes are used.

    
//Created chart axes.
ChartAxis axis = this.chartControl1.PrimaryYAxis;
ChartAxis axis0 = new ChartAxis(ChartOrientation.Vertical);
ChartAxis axis1 = new ChartAxis(ChartOrientation.Vertical);

//Added chart axes into the chart:
chartControl1.Axes.Add(axis0);
chartControl1.Axes.Add(axis1);

//Created chart axis layout using ChartAxisLayout class.
ChartAxisLayout layout1 = new ChartAxisLayout();

//Added the secondary and Primary axes into the layout.
layout1.Axes.Add(axis);
layout1.Axes.Add(axis0);
layout1.Axes.Add(axis1); 

//Added the layout into ChartArea.
chartControl1.ChartArea.YLayouts.Add(layout1);
    All axes with the same orientation must be added to ChartAxisLayout  and PrimaryAxis.
