Chart Custom Axis Origin

This sample shows a custom origin used for a chart axis in Essential Chart.

Features:

  • Custom Origin

     A custom origin can be implemented for both the x-axis and y-axis. Any double-type value can be set as the origin.

     A custom origin is turned on through the CustomOrigin property of the ChartAxis class. 
    
this.chartControl1.PrimaryYAxis.CustomOrigin = true;

    A custom origin can be set as shown below. 

    
this.chartControl1.PrimaryYAxis.Origin = 68;
  • Origin Customization

    The RangeType property is used to explicitly set custom bounds and intervals. 

    
this.chartControl1.PrimaryXAxis.RangeType = ChartAxisRangeType.Set;
this.chartControl1.PrimaryXAxis.Offset = 2;

     An axis-padding value can be calculated using the RangePaddingType property. 

    
this.chartControl1.PrimaryXAxis.RangePaddingType = ChartAxisRangePaddingType.Calculate;
  • Inversed Axis

    If the Inversed property of the ChartAxis class is set, then the corresponding axis range will be inverted (right-to-left or top-to-bottom). 

    
this.chartControl1.PrimaryXAxis.Inversed = true;
this.chartControl1.PrimaryYAxis.Inversed = true;
  • Opposed Axis

    An axis can be rendered on the opposite side of its usual position by setting the OpposedPosition property in the ChartAxis class.

    
this.chartControl1.PrimaryXAxis.OpposedPosition = true;
this.chartControl1.PrimaryYAxis.OpposedPosition = true;
