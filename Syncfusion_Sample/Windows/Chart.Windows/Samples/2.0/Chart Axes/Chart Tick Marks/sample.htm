Chart Tick Marks

This sample illustrates the various customization options for a chart control's axis tick marks.

Chart Tick Marks Demo

Features:

  • Tick Label Grid

    The axis-label grid is made visible by the DrawTickLabelGrid property.

    
 this.chartControl1.PrimaryXAxis.DrawTickLabelGrid = true;
  • Tick Label Grid Padding

    Padding of the tick-label grid is set using the TickLabelGridPadding property.

    
  this.chartControl1.PrimaryXAxis.TickLabelGridPadding = 5;
  • Tick Color

    The color of the ticks rendered on an axis is set using the TickColor property.

    
 this.chartControl1.PrimaryXAxis.TickColor = Color.Red;
  • Tick size

    Major tick size of the axis is set using the TickSize property, as shown in the following.

    
 this.chartControl1.PrimaryXAxis.TickSize = new Size(8,2);
  • Small Ticks Per Interval

    The number of small ticks displayed per major interval is set using the SmallTicksPerInterval property.

    
 this.chartControl1.PrimaryXAxis.SmallTicksPerInterval = 5;
  • Small Tick Size

    The size of small ticks displayed on the axis is set using the SmallTickSize property. By default, small ticks are not visible.

    
 this.chartControl1.PrimaryXAxis.SmallTickSize = new Size(5,1);
  • Tick Drawing Operation Mode

    To set the tick-drawing operation mode while the chart is zoomed, the TickDrawingOperationMode property can be used.

    
//Intervals remains unchanged during zooming when the mode is set as "IntervalFixed".
this.chartControl1.PrimaryXAxis.TickDrawingOperationMode = ChartAxisTickDrawingOperationMode.IntervalFixed;
//Intervals are kept constant when the mode is set as "NumberOfIntervalsFixed".
this.chartControl1.PrimaryXAxis.TickDrawingOperationMode = ChartAxisTickDrawingOperationMode.NumberOfIntervalsFixed;
