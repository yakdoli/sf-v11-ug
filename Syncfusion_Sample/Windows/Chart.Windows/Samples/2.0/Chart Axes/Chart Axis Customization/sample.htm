Chart Axis Customization

This sample demonstrates customizing the axis lines and axis grid lines of a chart control.

Customized Chart Axis

Features:

  • Axis Position

    An axis can be reversed using the Inversed property so that its points render from right to left, if horizontal; and top to bottom, if vertical. The OpposedPosition property sets the axis on the opposite side of the primary axis

    
//Sets the axis points as reversed
this.chartControl1.PrimaryXAxis.Inversed = true;
this.chartControl1.PrimaryYAxis.Inversed = true;
//Moves the axis to the opposite side of Primary Axis
this.chartControl1.PrimaryXAxis.OpposedPosition = true;
this.chartControl1.PrimaryYAxis.OpposedPosition = true;
  • Dash Style

    The Dash Style property is used to set the style of a line. 

    
//Sets the style of axis lines
this.chartControl1.PrimaryXAxis.LineType.DashStyle = DashStyle.Solid;
this.chartControl1.PrimaryYAxis.LineType.DashStyle = DashStyle.Solid;
//Sets the style of axis grid lines
this.chartControl1.PrimaryXAxis.GridLineType.DashStyle = DashStyle.Solid;
this.chartControl1.PrimaryYAxis.GridLineType.DashStyle = DashStyle.Solid;
  • Line Width

    Width of the axis lines and axis grid lines are set using the code below. 

         
//Sets the axis lines width
this.chartControl1.PrimaryXAxis.LineType.Width = 1;
this.chartControl1.PrimaryYAxis.LineType.Width = 1;
//Sets the axis grid lines width
this.chartControl1.PrimaryXAxis.GridLineType.Width = 1;
this.chartControl1.PrimaryYAxis.GridLineType.Width = 1;
  • Tick Size

    The size of the ticks rendered on the primary axis lines are set using the TickSize property.

    
//Sets the axis tick size
this.chartControl1.PrimaryXAxis.TickSize = 1;
this.chartControl1.PrimaryYAxis.TickSize = 1;
  • Foreground Color

    Foreground color of the axis lines and axis grid lines can be set separately using the LineType.ForeColor property and the GridLineType.ForeColor property, as shown below.

    
//Sets the fore color of axis lines
this.chartControl1.PrimaryXAxis.LineType.ForeColor = Color.Black;
this.chartControl1.PrimaryYAxis.LineType.ForeColor = Color.Black;
//Sets the fore color of axis grid lines
this.chartControl1.PrimaryXAxis.GridLineType.ForeColor = Color.Black;
this.chartControl1.PrimaryYAxis.GridLineType.ForeColor = Color.Black;
  • Axes Titles

    The TitleColor and TitleFont properties are used to set the color and font of axes titles. 

    
//Sets the Title color of axis title
this.chartControl1.PrimaryXAxis.TitleColor=Color.Blue;
this.chartControl1.PrimaryYAxis.TitleColor=Color.Red;
//Sets the Title font of axis title
this.chartControl1.PrimaryXAxis.TitleFont=new Fot("Arial",10);
this.chartControl1.PrimaryYAxis.TitleFont=new Fot("Arial",10);
  • Axes Label Placement

    The AxisLabelPlacement property of axis used to set the label position in axis.It can be placed Inside or Outside the plotted chart area using ChartPlacement enum. 

    
//Set the label placement inside of plotted chart area
this.chartControl1.PrimaryYAxis.AxisLabelPlacement = ChartPlacement.Inside;
//Set the label placement outside of plotted chart area
 this.chartControl1.PrimaryYAxis.AxisLabelPlacement = ChartPlacement.Outside;
