Chart Breaks

This sample demonstrates how breaks can be drawn in a chart when chart-point values differ greatly.

Features

To enable breaks in a chart, the MakeBreaks property should be set to true.        

this.chartControl1.PrimaryYAxis.MakeBreaks = true;

Breaks Mode

There are three possible modes in which breaks are set. They are:

  • ChartBreaksMode.None
    If the breaks mode is set to None, then breaks are not used for the specific axis in the chart control. 
    this.chartControl1.PrimaryYAxis.BreakRanges.BreaksMode = ChartBreaksMode.None;
  • ChartBreaksMode.Manual
    If the breaks mode is set to Manual, the break range can be specified manually. The following methods are available to set breaks manually: 
    this.chartControl1.PrimaryYAxis.BreakRanges.BreaksMode = ChartBreaksMode.Manual;
    
//Clears all break ranges
this.chartControl1.PrimaryYAxis.BreakRanges.Clear();
//Adds a break range 
this.chartControl1.PrimaryYAxis.BreakRanges.Union(new DoubleRange(500,600));
//Removes a break range 
this.chartControl1.PrimaryYAxis.BreakRanges.Exclude(new DoubleRange(500,600));
  • ChartBreaksMode.Auto
    In auto-mode, the break range will be computed automatically by the chart control itself, and the break amount is used to specify the minimal relative difference between values. 
    //Sets the mode to auto
this.chartControl1.PrimaryYAxis.BreakRanges.BreaksMode = ChartBreaksMode.Auto;
//Sets the value of Break Amount 
this.chartControl1.PrimaryYAxis.BreakRanges.BreakAmount = 0.3;
    In auto-mode, only primary axes can be enabled with breaks, but this mode is not suitable for zooming and stacked charts.

Break-Line Settings

There are various options available for customizing the appearance of a break line. They are discussed below.

  • Line Type
    The type of line used in a break line can be customized. The types offered are: Wave, Straight, and Randomize.
    this.chartControl1.PrimaryYAxis.BreakInfo.LineType = ChartBreakLineType.Wave;
  • Line Spacing
    Spacing of the break line is set using the LineSpacing property. 
    this.chartControl1.PrimaryYAxis.BreakInfo.LineSpacing = 5;
  • Line Color
    The border color of the break line is set using the LineColor property. 
    this.chartControl1.PrimaryYAxis.BreakInfo.LineColor = Color.Black;
  • Line Width
    The border width of the break line is set using the LineWidth property. 
    this.chartControl1.PrimaryYAxis.BreakInfo.LineWidth = 1;
  • Line Style
    The line style of the break line is set using the LineStyle property. 
    this.chartControl1.PrimaryYAxis.BreakInfo.LineStyle = DashStyle.Dot;
  • Spacing Color
    The color that fills the inside of the break line is set using the SpacingColor property. 
    this.chartControl1.PrimaryYAxis.BreakInfo.SpacingColor = Color.White;
