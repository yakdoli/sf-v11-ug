Chart Strip Line

Strip lines are background bands rendered in a chart area. This sample demonstrates the implementation of strip lines in Essential Chart.

Strip Line with Image in X-Axis Strip Line with Gradient Interior in Y-Axis
Fixed-Width Strip Line

Features:

  • Strip Lines

    A strip line can be created with the code below.

    
ChartStripLine thermoStripLine = new ChartStripLine();
thermoStripLine.Enabled = true;
thermoStripLine.BackImage = img;
thermoStripLine.Period = 50;
thermoStripLine.Width = 0.4;
  • Strip Line Axis

    A strip line can be added to a chart control's primary x-axis as shown below. 

    
this.chartControl1.PrimaryXAxis.StripLines.Add(thermoStripLine);
    It can also be added to the y-axis using the code given below. 
    
this.chartControl1.PrimaryXAxis.StripLines.Add(StripLine);
  • Strip Line Customization

    Various appearance settings for a strip line can be customized. The strip line used to mark "Historical High Temp" can be customized, as shown below. 

    
Color[] stripLineColor = new Color[] { Color.FromArgb(229, 150, 32), Color.FromArgb(252, 247, 235), Color.FromArgb(229, 150, 32) };
stripLine1.Vertical = false;
stripLine1.Text = "Historical High: 93'F";
stripLine1.Font = new System.Drawing.Font("Arial", 7.5F, System.Drawing.FontStyle.Regular, System.Drawing.GraphicsUnit.Point, ((byte)(161)));
stripLine1.Interior = new BrushInfo(GradientStyle.Vertical, stripLineColor);

  • Strip Line Position

    A strip line is configured by setting its start, end, period, and width with the same value-type as the axis that holds it.

    • Strip Lines with a Primary X-Axis of the Date-Time Type

      • Strip Line Start
        Sets the start point of a strip line
        
 StripLine.Start = 3;
      • Strip Line End
        Sets the end range of a strip line
        
StripLine.End = 4;
      • Strip Line Width
        Sets the width of each strip line
        
StripLine.Width = 1;

    • Strip Lines with a Primary X-Axis of the Date-Time Type

      • Start at Axis Position
        Sets whether or not the strip line will start at the beginning of the axis range
        
StripLine.StartAtAxisPosition = true;
      • Date Offset
        Sets the offset of a strip line if the primary x-axis is of the date-time type and StartAtAxisPosition is set to true
        
StripLine.DateOffset = new TimeSpan(3, 0, 0, 0, 0);
      • Width Date
        Sets the width of each strip line as a time span
        
StripLine.WidthDate = new TimeSpan(1, 0, 0, 0, 0);
      • Period Date
        Sets the period over which the strip line will appear
        
StripLine.PeriodDate = new TimeSpan(4, 0, 0, 0, 0);
      • Fixed-Width Strip Line

        The chart control supports setting a fixed width for strip lines, irrespective of the axis range. The default value of the fixed-width property is zero.

        
StripLine.FixedWidth=10;
