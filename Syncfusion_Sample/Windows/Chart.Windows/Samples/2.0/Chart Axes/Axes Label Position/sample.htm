Customize Label Position

This sample demonstrates customize the y-axis individual label position of a chart control.

Customize Label Position

Features:

  • Customize Label Position

    An axis individual label position can be customize using the AxisLabelPlacement property by ChartFormatAxisLabel event of chart so that its labels render on right or left side of axis, if horizontal; and top to bottom, if vertical. The OpposedPosition property sets the axis on the opposite side of the primary axis.

    
//Sets the AxisLabelPlacement property in ChartFormatAxisLabel event
private void chartControl1_ChartFormatAxisLabel(object sender, ChartFormatAxisLabelEventArgs e)
        {
          if (e.Label == "1")
                {
                    if (customizeEvent)
                    {
                        if (ser.Points[(int)e.Value - 1].YValues[0] > 0)
                            e.AxisLabelPlacement = ChartPlacement.Outside;
                        else
                            e.AxisLabelPlacement = ChartPlacement.Inside;
                    }
                    e.Label = "Canada";
                }
        }
