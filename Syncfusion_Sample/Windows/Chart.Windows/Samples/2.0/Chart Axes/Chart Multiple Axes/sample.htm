Chart Multiple Axes

This sample demonstrates the usage of multiple axes in Essential Chart.

Multiple Chart Axes

Axes Title Draw Mode as Ellipsis

Features:

  • Multiple Axes

    Multiple axes are created in the chart as shown below. 

    
//Initialize a ChartAxis
private ChartAxis secYAxis = new ChartAxis();

    
// To render this axis at a position opposite to that of the primary Axis.
secYAxis.OpposedPosition = true;
secYAxis.Orientation = ChartOrientation.Vertical;
//Add the ChartAxis to the Axes collection
this.chartControl1.Axes.Add(secYAxis);
// Specify that the 3rd series should be plotted against this secondary Y axis
this.chartControl1.Series[2].YAxis = secYAxis;
  • Layout Mode

    The layout mode of an axis is set using the YAxesLayoutMode property.

    this.chartControl1.ChartArea.YAxesLayoutMode = ChartAxesLayoutMode.SideBySide;
  • Axes Title Draw Mode

    When the title exceeds the axis length, it is possible to display the partial text with ellipses. The Axes Title Draw mode of an axis is set using the TitleDrawMode property. 

    
 this.chartControl1.PrimaryXAxis.TitleDrawMode = ChartTitleDrawMode.Ellipsis;
 this.chartControl1.PrimaryXAxis.TitleDrawMode = ChartTitleDrawMode.None;
 this.chartControl1.PrimaryXAxis.TitleDrawMode = ChartTitleDrawMode.Wrap;

    
 this.chartControl1.PrimaryYAxis.TitleDrawMode = ChartTitleDrawMode.Ellipsis;
 this.chartControl1.PrimaryYAxis.TitleDrawMode = ChartTitleDrawMode.None;
 this.chartControl1.PrimaryYAxis.TitleDrawMode = ChartTitleDrawMode.Wrap;
