Chart Axes

The Chart Axis samples show the various ways to customize axes, lines, and ticks.

  • Chart Axis Customization-This sample demonstrates customizing the axis lines and axis grid lines of a chart control.

  • Chart Breaks-This sample demonstrates how breaks can be drawn in a chart when chart-point values differ greatly.

  • Chart Custom Axis Origin-This sample shows a custom origin used for a chart axis in Essential Chart. A series point can be given as an origin; the custom origin can be turned on or off by setting the CustomOrigin property of the ChartAxis class.

  • Chart Grouping Labels-This sample demonstrates how to create axes grouping labels using the ChartAxisGroupingLabel class..

  • Chart Labels Customization-This sample illustrates various customization options for chart axis labels. The chart axis labels are customized by setting TickLabelsDrawingMode of the ChartAxis class to UserMode and adding a new set of labels to the collection using the ChartAxisLabel class.

  • Chart Multiple Axes-This sample demonstrates the usage of multiple axes in Essential Chart.

  • Chart Strip Line-This sample demonstrates the implementation of strip lines in Essential Chart. The position of the date strip-line can be adjusted by customizing the following data: DateOffset, Width Date, and Period Date.

  • Chart Tick Marks-This sample illustrates the various customization options for a chart control's axis tick marks.

  • Multiple Axes Layout-This sample demonstrates the usage of multiple axes layouts in Essential Chart.

  • Axis Crossing-This sample illustrates the crossing of X and Y axes.

  • Axes Label Position-This sample demonstrates customize the y-axis individual label position of a chart control.
