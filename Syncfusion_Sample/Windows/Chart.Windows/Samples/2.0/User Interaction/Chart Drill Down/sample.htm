Chart Drill Down

This sample illustrates how to simulate a drill-down chart by reinitializing the chart with two different sets of data.
Column Chart to Drill Down Drilled-Down Pie Chart

Interactive Features:

  • To view the drill-down chart, you can click on any column and it will show a pie chart of that particular column.

  • To go back to the column chart, click on any one of the segments in the pie chart.

Chart Events

  • ChartRegionClick

    The ChartRegionClick event is used to get the chart region point. Accordingly, chart data and chart type will change.

  • ChartRegionMouseHover

     The ChartRegionMouseHover event is used to set the ToolTip of the column chart series when the pointer hovers over the series points.
