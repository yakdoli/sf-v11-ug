Chart Series Highlight

This sample demonstrates the Series Highlighting feature of Essential Chart.

Highlighting Series

Features:

  • Series Highlight

    To highlight a series as a whole, the chart control has a property called SeriesHighlight which must be set to true. On assigning this property as true will highlight the specific series on which the mouse hovers. The series can also be highlighted by hovering the mouse on particular series in the legend, the other series in the chart will be transparent. The transparency can be controlled by using “DimmedInterior” property of the series.



    • 
//Turns on Series Highlight
this.chartControl1.SeriesHighlight = true;

  • Series Highlight Color

    The highlight color for any series can be changed by adjusting the color in the HighlightInterior property. 

    
this.chartControl1.Series[0].Style.HighlightInterior = new BrushInfo(Color.Gold);
  • Hidden Series Color

    While mouse hovering on particular series all the other series in the chart will be set to the color given in DimmedInterior property. 

    
BrushInfo bi = new BrushInfo(GradientStyle.Vertical, Color.Red, Color.Red);
this.chartControl1.Series[0].Style.DimmedInterior = new BrushInfo(0, bi);
  • SeriesHighlightIndex:

    • If you want to highlight only a particular series alone then you can specify the value for this property. By default it will be -1.

    If AutoHighlight property is set then the SeriesHighlight will have no effect.
