User Interaction

Essential Chart provides various ways to interact with a chart.

  • Chart AutoHighlight-This sample demonstrates the AutoHighlight feature of Essential Chart.

  • Chart Custom Trend Lines-This sample demonstrates drawing custom trend lines in Essential Chart.

  • Chart Drag and Drop-This sample demonstrates dragging and dropping a data point to a new location within a plot.

  • Chart Drill Down -This sample illustrates how to simulate a drill-down chart by reinitializing the chart with two different sets of data.

  • Chart Interactive Cursor-This sample shows how to implement an interactive cursor using Essential Chart. Interactive cursors can be moved from one data point to another by dragging.

  • Chart Point- This sample shows the usage of the GetPointByValue and GetValueByPoint methods.

  • Chart Region Events -This sample shows how to use chart-region events, raised by a chart control, that were started via user interactions.

  • Chart Series Highlight -This sample demonstrates the SeriesHighlight feature of Essential Chart.

  • Chart Toolbar-This sample illustrates the toolbar feature in Essential Chart.

  • Chart ToolTips- This sample demonstrates how to create and customize ToolTips for Essential Chart.
