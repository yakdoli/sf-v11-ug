Chart Interactive Cursor

This sample shows how to implement an interactive cursor using Essential Chart. Interactive cursors can be moved from one data point to another by dragging or move to chart area by specifying the values in XInterval and YInterval.

Interactive Charts

Features:

  • Interactive cursor

    An Interactive cursor is used to indicate the x-axis and y-axis values of a data point. Interactive cursors are created as shown in the following code. 

    
ChartInteractiveCursor cursor1 = new ChartInteractiveCursor(this.chartControl1.Series[0]);
this.chartControl1.ChartArea.InteractiveCursors.Add(cursor1);
this.chartControl1.ChartArea.InteractiveCursors[0].XPosition = 2;
this.chartControl1.ChartArea.InteractiveCursors[0].YPosition = 5;
  • Cursor Color

    The color of an interactive cursor can be set to any system-defined color. 

    
cursor1.Color = Color.Yellow;
  • MoveTo ChartArea

    An Interactive curosr canbe moved to chart area by using MoveToChartArea property of cursor.

    
this.chartControl1.ChartArea.InteractiveCursors[0].MoveToChartArea = true;
  • XInterval

    Cursor on the X Axis can be moved from left to right or right to left based on the value provided in XInterval property of interactive cursor. 

    
this.chartControl1.ChartArea.InteractiveCursors[0].XInterval = 1;
  • YInterval

    Cursor on the Y Axis can be moved from top to bottom or bottom to top based on the value provided in YInterval property of interactive cursor. 

    
this.chartControl1.ChartArea.InteractiveCursors[0].YInterval = 2;
  • Moving the Cursor

    The VerticalMove and HorizontalMove methods are used to move a cursor to the next closest data point in a series.

    • 
//Returns the series index value of the closest point to the specified Y value.
this.chartControl1.ChartArea.InteractiveCursors[0].VerticalMove(true);
//Returns the series index value of the closest point to the specified X value.
this.chartControl1.ChartArea.InteractiveCursors[0].HorizontalMove(true);
