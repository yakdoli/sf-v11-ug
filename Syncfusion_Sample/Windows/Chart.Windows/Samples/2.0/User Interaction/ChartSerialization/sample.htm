This sample illustrates the serialization feature in Essential Chart.

Chart Serialization

Using a ChartTemplate Save method,We can save chart series points and styles in XML file.

Features:

  • Save
    Components of a Chart like ChartSeries and ChartArea etc., are stored in a template which can be loaded later into another different ChartControl. 
    
ChartTemplate.Save(this.chartControl1, "Chart.xml");
  • Load
    It provides Load method to load an already saved Chart template. Data series will be applied in a sequential order, i.e., the first set of data series will be applied to the destination Chart's first series and the second set of the data series will be applied to the destination Chart's second series and so on. 
    
ChartTemplate.Load(this.chartControl3, "Chart.xml");
