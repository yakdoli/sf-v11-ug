Chart Auto Highlight

This sample demonstrates the AutoHighlight feature of Essential Chart.

Highlight Chart

Features:

  • AutoHighlight

    Points on a chart can be highlighted when the pointer hovers over them. This is accomplished by enabling the AutoHighlight property of the chart control.



    • 
//Turns on AutoHighlight
this.chartControl1.AutoHighlight = true;

  • Highlight Color

    The highlight color for any series can be changed by adjusting the color in the HighlightInterior property. 

    
this.chartControl1.Series[0].Style.HighlightInterior = new BrushInfo(Color.Gold);
