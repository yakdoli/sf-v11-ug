Chart ToolTips

This sample demonstrates how to create and customize ToolTips for Essential Chart.
Formatted Chart ToolTips Fancy Chart ToolTips

Features:

  • ToolTips

    ToolTips can be enabled in a chart control using the ShowToolTip property. By enabling this property, ToolTips associated with all the chart regions, such as Chart, Chart Area, Chart Series, and ToolBar, will be visible when the pointer hovers on the corresponding region. 

    
this.chartControl1.ShowToolTips = true;
    1. Chart ToolTip

      The ToolTip for a chart control is set using the ChartTooltip property. 

      
this.chartControl1.ChartToolTip = "Chart ToolTip";
    2. Chart Area ToolTip

      The ToolTip for the chart-area control is set using the ChartAreaTooltip property. 

      
this.chartControl1.ChartAreaToolTip = "Chart Area ToolTip";
    3. Series ToolTip

      This gives information about the series of a chart control.

      It has the following formats:

      "{0}" - Series Name

      "{1}" - SeriesStyle.Tooltip

      "{2}" - SeriesStyle[i].Tooltip

      "{3}" - SeriesPoints[index]Tooltip

      "{4}" - SeriesPoints[index].YValues

      The ToolTip format is set using the series PointsTooltipFormat property. 
      
series1.PointsToolTipFormat = "{0}";

  • Fancy ToolTip

    Essential chart also supports fancy ToolTips for a chart series.

    A fancy ToolTip, as the name implies, displays the ToolTip in an attractive manner, which is different from other default ToolTips. The appearance of fancy ToolTips can be customized.

    1. Fancy ToolTip Visibility

      Fancy ToolTips can be visible, as shown in the following code. 

      
series1.FancyToolTip.Visible = true;
series2.FancyToolTip.Visible = true;
    2. Fancy ToolTip Color

      The background color and foreground color of a fancy ToolTip can be set using the following code.

      
series.FancyToolTip.BackColor = Color.Yellow;
series.FancyToolTip.ForeColor = Color.Green;
    3. Fancy ToolTip Border

      The border width and color of a fancy ToolTip can be customized.

      
series.FancyToolTip.Border.Width = width;
series.FancyToolTip.Border.ForeColor = this.BorderColorPicker.SelectedColor;
    4. Fancy ToolTip Style

      The style of a fancy ToolTip can be set as Ellipse, Rectangle, or SmoothRectangle, using the Style property.

      
series.FancyToolTip.Style = MarkerStyle.Ellipse;
    5. Fancy ToolTip Symbol

      The symbol of a chart ToolTip can be set using the Symbol property.

      
series.FancyToolTip.Symbol = ChartSymbolShape.Circle;
