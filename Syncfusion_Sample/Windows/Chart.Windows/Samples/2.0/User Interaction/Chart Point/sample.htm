Chart Point

This sample shows the usage of the GetPointByValue and GetValueByPoint methods.

Get Chart Point Values

Features:

  • Get Point by Value

    This method retrieves the corresponding point of a value in a chart region.

  • Get Value by Point

    This sample also demonstrates the GetValueByPoint method that can be used to calculate chart coordinates at a given client point.

    
ChartPoint chpt = chartControl1.ChartArea.GetValueByPoint( new Point( e.X, e.Y ) );
Point pt = chartControl1.ChartArea.GetPointByValue( chpt );
