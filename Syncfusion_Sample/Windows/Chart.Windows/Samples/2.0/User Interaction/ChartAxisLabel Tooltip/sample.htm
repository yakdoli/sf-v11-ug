ChartAxisLabel ToolTip

This sample demonstrates how to create and customize ToolTips for ChartAxisLabels.

Features:

  • ToolTips

    ToolTips can be enabled in a chart control using the ShowToolTip property. By enabling this property, ToolTips associated with all the chart regions, such as Chart, Chart Area, Chart Series, and ToolBar, will be visible when the pointer hovers on the corresponding region. 

    
this.chartControl1.ShowToolTips = true;

  • Same above ShowToolTips property enables the Tooltip for ChartAxisLabels

    • Interactive Features:

  • We can cutsomize the AxisLabel tootip by using Tooltip property in ChartFormatAxisLabel event.

  • We can also customize the Tooltip via ChartRegionMouseMove event of chart.
