Multiple Pie Chart

This sample demonstrates how to implement multiple pie chart using Essential Chart.

Multiple Pie Chart

Features

The chart control supports to add multiple pie in a chart using 'MultiplePies' boolean property and also set specific doughnut co - efficient value for each pie chart series to get this complete functionality of this feature.


this.chartControl1.ChartArea.MultiplePies = true;

This implementation will work only in SingleArea Chart, will not work when chart is in DivideArea.

  • Pie Chart Types

    The various types of pie charts are set as shown in the following code. 

    
this.chartControl1.Series[0].ConfigItems.PieItem.PieType = ChartPieType.Bevel;
this.chartControl1.Series[0].ConfigItems.PieItem.PieType = ChartPieType.Custom;
this.chartControl1.Series[0].ConfigItems.PieItem.PieType = ChartPieType.InSide;
this.chartControl1.Series[0].ConfigItems.PieItem.PieType = ChartPieType.OutSide;
this.chartControl1.Series[0].ConfigItems.PieItem.PieType = ChartPieType.Round;
  • Doughnut, PieHeight structure

    A pie chart can be rendered to a doughnut structure by making use of the ChartConfigItem.DoughnutCoeficient property. Height of the chart can also be adjusted, as shown in the following code.

    
this.chartControl1.Series[0].ConfigItems.PieItem.DoughnutCoeficient = 0.1;
this.chartControl1.Series[0].ConfigItems.PieItem.PieHeight=0.5;
  • DrawItemText Event

    The DrawItemText event to customize the rendering of the legend item text using the following code snippets. 

    
this.chartControl1.Legend.DrawItemText += new LegendDrawItemTextEventHandler(Legend_DrawItemText);
