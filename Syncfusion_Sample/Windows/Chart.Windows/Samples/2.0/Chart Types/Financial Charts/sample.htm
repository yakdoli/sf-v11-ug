Financial Charts

This sample illustrates candle, high-low, open-high-low-close, Kagi, point-and-figure, Renko, and three-line-break charts.

   

Features

Financial data will usually contain more than one y value; therefore, more than two values are added to the points of a series in a chart control.


ChartSeries series1 = new ChartSeries(" Series 1",ChartSeriesType.Candle);
series1.Text = series1.Name;
DateTime date1=new DateTime(2006,2,12);
series1.Points.Add(date1.AddDays(1), 456, 214, 364, 386);
series1.Points.Add(date1.AddDays(2), 491, 234, 321, 378);
series1.Points.Add(date1.AddDays(3), 482, 193, 352, 302);
series1.Points.Add(date1.AddDays(4), 437, 243, 354, 391);
series1.Points.Add(date1.AddDays(5), 421, 223, 357, 317);
series1.Points.Add(date1.AddDays(6), 434, 263, 339, 385);
			
this.chartControl1.Series.Add(series1);
  • Financial Chart Types

    Various financial charts are applied to a chart control by using the following code.

    • 
this.chartControl1.Series[0].Type = ChartSeriesType.Candle;
this.chartControl1.Series[0].Type = ChartSeriesType.HiLo;
this.chartControl1.Series[0].Type = ChartSeriesType.HiLoOpenClose;
this.chartControl1.Series[0].Type = ChartSeriesType.Kagi;
this.chartControl1.Series[0].Type = ChartSeriesType.PointAndFigure;
this.chartControl1.Series[0].Type = ChartSeriesType.Renko;
this.chartControl1.Series[0].Type = ChartSeriesType.ThreeLineBreak;
  • Chart 3-D Mode

    A chart can be rendered in 3-D by enabling the Series3D property.

    • this.chartControl1.Series3D = true;
  • Open-Close-Draw Mode

    The OpenCloseDrawMode property will set the chart series in open, close, or both modes. This property is applicable to open-high-low-close charts. 

    • 
//sets the Chart series in Open mode
this.chartControl1.Series[0].OpenCloseDrawMode = ChartOpenCloseDrawMode.Open;
//sets the Chart series in Close mode
this.chartControl1.Series[0].OpenCloseDrawMode = ChartOpenCloseDrawMode.Close;
//sets the Chart series to have both open and close mode
this.chartControl1.Series[0].OpenCloseDrawMode = ChartOpenCloseDrawMode.Both;
