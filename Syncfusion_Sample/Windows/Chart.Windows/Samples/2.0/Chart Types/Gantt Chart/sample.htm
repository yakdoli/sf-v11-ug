Gantt Chart

This sample illustrates Gantt charts.

Gantt Chart

Features:

  • Gantt Chart Types

    A Gantt chart is set using the following code. 

    
//Sets the Chart Type as Gantt
this.chartControl1.Series[0].Type = ChartSeriesType.Gantt;

  • Drag and Drop

    Dragging and dropping within a chart can be enabled by handling the appropriate Chart Region Mouse events.

  • Chart Custom Points

    Chart custom points are used to set custom points for a series so they can show employees' names.

    • 
ChartCustomPoint ccp = new ChartCustomPoint();
ccp.XValue = this.chartControl1.Series[1].Points[0].X;
ccp.YValue = this.chartControl1.Series[1].Points[0].YValues[1];
ccp.CustomType = ChartCustomPointType.ChartCoordinates;
ccp.Text = "John";
ccp.Color = Color.Black;
ccp.Font.Facename = "Tahoma";
ccp.Font.Size = 7f;
this.chartControl1.CustomPoints.Add(ccp);
  • Chart Strip Lines

    Strip lines are used to highlight the weekends on a calendar. 

    • 
ChartStripLine stripLine1 = new ChartStripLine();
stripLine1.Enabled = true;
stripLine1.Vertical = false;
stripLine1.Start = 6;
stripLine1.End = 7;
stripLine1.Interior = new BrushInfo(GradientStyle.Horizontal, stripLineColor);
this.chartControl1.PrimaryXAxis.StripLines.Add(stripLine1);
