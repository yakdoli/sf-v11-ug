Error Bar Chart

This sample demonstrates error bars in a line chart.

  
Error Bar Chart

Features

  • Error Bars

    Error bars are drawn by enabling the DrawErrorBars property of the chart series.

    

this.chartControl.Series[0].DrawErrorBars = true;

  • Error Bar Symbol

    The error bar symbol is customized through the ErrorBarsSymbolShape property.

    

this.chartControl.Series[0].ErrorBarsSymbolShape = ChartSymbolShape.Circle;

  • Series Point Symbol

    This sample also includes options for setting symbols in series points by using the following code.

    
 this.chartControl1.Series[0].Style.Symbol.Shape = ChartSymbolShape.Star;
