Pyramid and Funnel Charts

This sample illustrates pyramid and funnel charts, also known as accumulation charts.

Features:

    • Chart 3-D Mode

      A chart can be viewed in 3-D mode by enabling the Series3D property.

      • this.chartControl1.Series3D = true;
    • Label Style

      The LabelStyle property is used to customize the position of data-point labels. 

      this.chartControl1.Series[0].ConfigItems.PyramidItem.LabelStyle = ChartAccumulationLabelStyle.OutsideInColumn;

    • Label Placement

      The LabelPlacement property is used to customize the alignment of a label when rendered inside a pyramid or funnel chart. 

      this.chartControl1.Series[0].ConfigItems.PyramidItem.LabelPlacement = ChartAccumulationLabelPlacement.Left;

    • Gap Ratio

      To control the gap between points, the GapRatio property can be used. 

      this.chartControl1.Series[0].ConfigItems.PyramidItem.GapRatio= 0.1;

    • Figure Base

      The FigureBase property is used to allow the end user to render the base of the pyramid or funnel chart in different modes. 

      this.chartControl1.Series[0].ConfigItems.PyramidItem.FigureBase=ChartFigureBase.Square;
