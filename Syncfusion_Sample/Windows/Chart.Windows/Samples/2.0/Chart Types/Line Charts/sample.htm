Features

  • Line Chart Types

    There are four types of line charts you can make use of, as shown in the following code.

    
//sets the Chart Type as Line
this.chartControl1.Series[0].Type = ChartSeriesType.Line;
//sets the Chart Type as Spline
this.chartControl1.Series[0].Type = ChartSeriesType.Spline;
//sets the Chart Type as StepLine
this.chartControl1.Series[0].Type = ChartSeriesType.StepLine;
//sets the Chart Type as RotatedSpline
this.chartControl1.Series[0].Type = ChartSeriesType.RotatedSpline;
  • Border Settings

    Border width and border style of a line chart are customized through the Width and DashStyle properties.

    		
this.chartControl1.Series[0].Style.Border.Width = 1f;
this.chartControl1.Series[0].Style.Border.DashStyle = DashStyle.Dot;
  • Point Color Settings

    Data points in a series can be set to different colors by listening to the PrepareStyle event.

    		
ChartSeries series = sender as ChartSeries;
if (series != null)
{
if (args.Index == 0)
	args.Style.Interior = new Syncfusion.Drawing.BrushInfo(Color.Red);
else if (args.Index==1)
	args.Style.Interior = new Syncfusion.Drawing.BrushInfo(Color.Green);
else if (args.Index == 2)
	args.Style.Interior = new Syncfusion.Drawing.BrushInfo(Color.Blue);
else if (args.Index == 3)
	args.Style.Interior = new Syncfusion.Drawing.BrushInfo(Color.Yellow);
else if (args.Index == 4)
	args.Style.Interior = new Syncfusion.Drawing.BrushInfo(Color.Violet);
else if (args.Index == 5)
	args.Style.Interior = new Syncfusion.Drawing.BrushInfo(Color.BlanchedAlmond);
else if (args.Index == 6)
