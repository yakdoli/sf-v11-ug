Combination Chart

Features

  • Combination Chart

    Combination charts use two or more chart types to emphasize that a chart contains different kinds of information. The chart types combined in this sample are column and spline. The chart types of a series in a combination chart are set as shown. 

    
//sets the Chart Type as Histogram
this.chartControl1.Series[0].Type = ChartSeriesType.Column;
this.chartControl1.Series[1].Type = ChartSeriesType.SplineArea;
    The type of column chart is set as follows. 
    
this.chartControl1.Series[0].ConfigItems.ColumnItem.ColumnType = ChartColumnType.Cylinder;

  • Chart 3-D Mode

    Combination charts can be rendered in 3-D by enabling the Series3D property.

    • this.chartControl1.Series3D = true;
