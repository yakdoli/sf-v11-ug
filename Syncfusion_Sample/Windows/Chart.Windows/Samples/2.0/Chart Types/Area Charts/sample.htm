Area Charts

This sample illustrates the various types of area charts supported by Essential Chart.

  
  
 

Features

  • Area Chart Types

    There are four types of area charts you can use. They are as follows.

    
//sets the Chart Type as Area
this.chartControl1.Series[0].Type = ChartSeriesType.Area;
//sets the Chart Type as StepArea
this.chartControl1.Series[0].Type = ChartSeriesType.StepArea;
//sets the Chart Type as SplineArea
this.chartControl1.Series[0].Type = ChartSeriesType.SplineArea;
//sets the Chart Type as StackingArea
this.chartControl1.Series[0].Type = ChartSeriesType.StackingArea;
  • Series Color Settings

    Background and foreground colors for area charts are customized through the Interior property of the ChartStyleInfo class.

    • this.chartControl1.Series[0].Style.Interior = new BrushInfo(200, new BrushInfo(GradientStyle.ForwardDiagonal,Color.Gray,Color.White));
  • Chart 3-D Mode

    A chart is rendered in 3-D mode by enabling the Series3D property.

    • this.chartControl1.Series3D = true;
  • Border settings

    Border color and border width of an area chart can be changed through the Border.Color and Border.Width properties.

    this.chartControl1.Series[0].Style.Border.Color=Color.Red.
this.chartControl1.Series[0].Style.Border.Width = 1;

Interactive Features

  • Chart series type can be changed when selecting the chart type combo box option in the application.
  • Chart series color can be customized when select the Forecolor and Backcolor buttons in the application.
  • Series border width has been changed when changing the numeric up-down button option in the sample.
