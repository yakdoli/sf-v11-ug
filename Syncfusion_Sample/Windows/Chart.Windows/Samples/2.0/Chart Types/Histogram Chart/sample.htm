Histogram Chart

This sample illustrates a histogram chart.

Histogram Chart

Features:

  • Histogram Chart Type

    A histogram chart can be set using the following code. 

    
//sets the Chart Type as Histogram
this.chartControl1.Series[0].Type = ChartSeriesType.Histogram;

  • Chart 3-D Mode

    A histogram chart can be rendered in 3-D mode by enabling the Series3D property.

    • this.chartControl1.Series3D = true;
