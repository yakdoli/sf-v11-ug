Bar Charts

This sample illustrates bar and stacked bar charts.

Bar Chart - 3-D View    Stacked Bar - 3-D View
  

Features:

  • Bar Chart Types

    Bar chart types are set as follows. 

    
//sets the Chart Type as Bar
this.chartControl1.Series[0].Type = ChartSeriesType.Bar;
//sets the Chart Type as StackingBar
this.chartControl1.Series[0].Type = ChartSeriesType.StackingBar;
  • Chart 3-D Mode

    A chart is rendered in 3-D mode by enabling the Series3D property.

    • this.chartControl1.Series3D = true;
  • Chart Series Points

    Chart series points are displayed by using the DisplayText property in a chart control. 

    this.chartControl1.Series[0].Style.DisplayText = true;

  • Series Color Settings

    Chart series colors can be changed by using the Interior property in a chart control. 

    this.chartControl1.Series[0].Style.Interior = new BrushInfo(200, new BrushInfo(GradientStyle.ForwardDiagonal,Color.Gray,Color.White));
