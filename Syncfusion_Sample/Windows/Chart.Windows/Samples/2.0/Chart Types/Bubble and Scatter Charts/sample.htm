Bubble and Scatter Charts

This sample demonstrates bubble and scatter charts supported by Essential Chart.

Bubble Chart    Scatter Chart
  
Bubble Image Chart    Scatter Spline Chart
  

Features:

  • In this sample, the bubble-type of a plot can be selected from the ChartSeriesConfigCollection.BubbleItem class.

    
//Renders the bubbles as per the specified type.

this.chartControl1.Series[0].ConfigItems.BubbleItem.BubbleType =ChartBubbleType.Circle;

  • Symbol shape, color, and size of a scatter chart can be customized using the various properties of the ChartStyleInfo class. The connect-type of a plot can be changed using the ChartSeries.ScatterConnectType property.

    
//The scattered points are connected with the type specified.

this.chartControl1.Series[i].ScatterConnectType  = ScatterConnectType.Spline;

  • Scatter-line tension value is customizable.

    
// Gets / sets the tendion required for the ScatterLine of the chart .

this.chartControl1.Series[i].ScatterSplineTension=0.1;
