Polar and Radar Charts

This sample demonstrates polar and radar charts created by Essential Chart.

Features:

  • Polar and Radar Chart Types

    Polar and radar charts are set using the following code. 

    
//sets the Chart Type as Polar
this.chartControl1.Series[0].Type = ChartSeriesType.Polar;
//sets the Chart Type as Polar
this.chartControl1.Series[0].Type = ChartSeriesType.Radar;
  • Draw-Type

    The draw-type of a chart can be changed using the RadarItem property of ConfigItems, as shown in the following code. 

    
// Renders the Chart such that points are connected and the enclosed region filled.
this.chartControl1.Series[0].ConfigItems.RadarItem.Type = ChartRadarDrawType.Area;
// Renders the Chart such that points are connected but the enclosed region is not filled.
this.chartControl1.Series[0].ConfigItems.RadarItem.Type = ChartRadarDrawType.Line;
// Renders the Chart such that points are rendered witht the associated symbol.
this.chartControl1.Series[0].ConfigItems.RadarItem.Type = ChartRadarDrawType.Symbol;

  • Radar Axis Style

    The style of a radar axis can be set using the RadarStyle property of a chart control.

    
//Axes are rendered as Polygonal.
this.chartControl1.RadarStyle = ChartRadarAxisStyle.Polygon;
//Axes are rendered as circles.
this.chartControl1.RadarStyle = ChartRadarAxisStyle.Circle;

  • Inversed Radar and Polar Chart

    Essential Chart support Inversed Radar and Polar charts. This feature used to render the Polar and Radar charts in Clockwise direction. 

    
//Inverted X-Axis.
this.chartControl1.PrimaryXAxis.Inversed = true;
//Inverted Y-Axis.
this.chartControl1.PrimaryYAxis.Inversed = true;

  • Polar and Radar line style customization

    Chart control support to customize the solid circle lines of the Polar and Radar chart. Primary X and Y axes pens property is used to set the style of the solid circles.

    
//XY-Axes style customization.
 Pen[] mypens = new Pen[3];
 mypens[0] = new Pen(Color.Red, 2f);
 mypens[1] = new Pen(Color.Green, 2f);
 mypens[2] = new Pen(Color.DarkGoldenrod, 2f);
 mypens[0].DashStyle = DashStyle.DashDotDot;
 mypens[1].DashStyle = DashStyle.Dot;
 this.chartControl1.PrimaryXAxis.Pens = mypens;
 this.chartControl1.PrimaryYAxis.Pens = mypens;

    PrimaryXAxis.Pens collection property used to draw the spokes and PrimaryYAxis.Pens used to draw the inner circles of the chart.

    PrimaryXAxis.LineType property is used to customize the outermost of circle and also PrimaryYAxis.LineType is used to customize the vertical Y Axis.

    For proper output, Both the Pens property of Axes should be filled by pens collection.
