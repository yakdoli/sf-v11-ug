Range Area Chart

This sample illustrates the range area chart type supported by Essential Chart.

Range Area Chart

Features:

  • Range Area Chart

    A range area chart type is set using the following code. 

    
this.chartControl1.Series[0].Type = ChartSeriesType.RangeArea;
  • Series Points

    A range area chart requires one x value and two y values.

    
series1.Points.Add(0.3, 30, 50);
series1.Points.Add(1.2, 32, 54);
series1.Points.Add(2, 39, 62);
series1.Points.Add(3, 46, 72);
series1.Points.Add(4, 55, 79);
series1.Points.Add(5, 64, 86);
series1.Points.Add(6, 69, 89);
series1.Points.Add(7.1, 67, 87);
series1.Points.Add(8.2, 61, 81);
series1.Points.Add(9.3, 48, 72);
series1.Points.Add(10.4, 40, 62);
series1.Points.Add(11.5, 33, 53);
  • Chart Custom Points

    Chart custom points are used to set custom points in a series. The custom points used in this sample have images and text to represent series-point data.

    
ChartCustomPoint ccp = new ChartCustomPoint();
ccp.XValue = this.chartControl1.Series[0].Points[i].X;
ccp.YValue = this.chartControl1.Series[0].Points[i].YValues[1];
ccp.CustomType = ChartCustomPointType.ChartCoordinates;
ccp.Text = this.chartControl1.Series[0].Points[i].YValues[1].ToString();
ccp.Symbol.Shape = ChartSymbolShape.Image;
ccp.Images = new ChartImageCollection();
ccp.Images.Add(this.imageListAdv1.Images[1]);
ccp.Symbol.ImageIndex = 0;                
ccp.Symbol.Size = new Size(32, 32);
this.chartControl1.CustomPoints.Add(ccp);
