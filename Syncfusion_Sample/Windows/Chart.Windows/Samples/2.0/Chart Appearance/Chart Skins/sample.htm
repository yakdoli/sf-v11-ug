Chart Skins

Features

Essential Chart supports Fourteen different Skin styles. The interior color and the back interior of the chart control can be customized.

The skins of a chart can be customized using the below properties,

  • ChartInterior- Indicates the Chart Interior color.
  • BackInterior- Indicates the Back interior of the chart.ie, outside the chart area.
  • Skins- Build- in skins can be set using the skins property of the chart control.
