Chart Template

This sample illustrates the chart template features supported by Essential Chart.

Chart Template

Features

Chart control appearance settings are edited as required and saved as a template that can be applied to other chart controls.

  • Save Chart Template

    The code below shows how the appearance of a chart can be saved to a XML template file.

    
//create an instance of ChartTemplate, scan the appearance related settings in the chart control and save it to an xml file
ChartTemplate ct = new ChartTemplate(typeof(ChartControl));
ct.Scan(this.chartControl1);                        
ct.Save("MyTemplate.xml");
  • Load Chart Template

    The code below shows how the appearance of a chart can be applied from a XML template file.

    
//create an instance of ChartTemplate, scan the appearance related settings in the chart control and save it to an xml file
ChartTemplate ct = new ChartTemplate(typeof(ChartControl));
ct.Load("MyTemplate.xml");                    
ct.Apply(this.chartControl1);

Interactive Features

  • New and existing chart appearances can be saved as XML template files using the Save button in the application.
