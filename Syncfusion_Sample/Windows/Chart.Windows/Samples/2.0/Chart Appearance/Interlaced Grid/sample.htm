Interlaced Grid

Essential Chart supports interlaced grids for both the x-axis and the y-axis. 

Features

The interlaced grid's color can be customized.


this.chartControl1.PrimaryXAxis.InterlacedGrid = true;
this.chartControl1.PrimaryXAxis.InterlacedGridInterior = new Syncfusion.Drawing.BrushInfo(System.Drawing.Color.FromArgb(166, 184, 21);

this.chartControl1.PrimaryYAxis.InterlacedGrid = true;
this.chartControl1.PrimaryYAxis.InterlacedGridInterior = new Syncfusion.Drawing.BrushInfo(System.Drawing.Color.FromArgb(124, 144, 179));

Interactive Features

  • Chart axes interlaced grid visibility can be changed according to the check box selection in the application.
