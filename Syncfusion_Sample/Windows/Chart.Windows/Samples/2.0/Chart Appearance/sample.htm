Chart Appearance

Essential Chart's appearance can be controlled through the various elements of the chart display area that are exposed by the control through its properties.

  • Chart Appearance-This sample illustrates how to customize chart control appearance.

    • The chart interior color can be shown or hidden by setting or clearing the ChartInterior property of the ChartControl.

    • Chart Font Style - Text displayed in a series can be shown or hidden with a selected font face by the Display Text property.

    • Chart Rotate - The rotation of the axis is handled by the Rotation property of the chart area.

    • Chart Background - The chart background image can be set either at run time or at design time using the ChartAreaBackImage property.

    • The DisplayShadow property is used to enable a shadow for a chart series exposed by the ChartStyleInfo class.

  • Chart Skins-This sample illustrates how to customize with build in skins.

  • Chart Borders-This sample illustrates how to customize a chart control border. The ChartStyleInfo object is used to provide style information for chart elements.

  • Chart Custom Drawing-This sample illustrates how to customize a chart's appearance by drawing designs over the chart area.

  • Chart Divide Area-This sample shows the divide-area feature of a chart control.

  • Chart Template-This sample illustrates the chart template features supported by Essential Chart.

  • Chart Watermark-This sample shows the Image and Text watermark feature of a chart control.

  • Interlaced Grid-This sample illustrates the interlaced grid's support for both the x-axis and the y-axis; it also shows how to customize those interlaced grids.
