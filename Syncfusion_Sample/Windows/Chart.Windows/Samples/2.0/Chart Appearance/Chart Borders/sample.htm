This sample shows how to customize the appearance of the chart control.

Chart Borders

Features

Essential Chart supports thirteen different border styles. The interior style and the size of the chart border can be customized.

The border of a chart area can be customized using the below border properties,

  • BorderColor- Indicates the border color of the chart area
  • BorderStyle- Indicates the border style
  • Borderwidth- Specifies the width of the border

Chart control border appearance can be customized using the below properties:

  • Interior- Specifies the interior color of the border; this property is only effective when setting the SkinStyle as Sunken, Etched, or Raised
  • SkinStyle- Specifies the skin style of the chart control
  • FrameThickness- Indicates the thickness of the frame; this property is only effective when setting the SkinStyle as Frame.
  • BaseColor- Indicates the base color of the border.

Interactive Features

  • According to the Border Skin Style combo box list selection, the chart control border style will be changed.
