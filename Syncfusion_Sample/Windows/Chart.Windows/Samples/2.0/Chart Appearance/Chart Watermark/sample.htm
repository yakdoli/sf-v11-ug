Chart Watermark

This sample shows the watermark feature of a chart control.

Features

  • The Text property is used to set the watermark. 

    
this.chartControl1.ChartArea.WaterMark.Text="Syncfusion Chart"

  • The Image property is used to display an image as a watermark. 

    
this.chartControl1.ChartArea.Watermark.Image = System.Drawing.Image.FromFile("Logo.bmp");

  • The Opacity property is used to set the opacity of a watermark as a percentage. 

    
this.chartControl1.ChartArea.Watermark.Opacity=60;

  • The HorizontalAlign and VerticalAlign properties are used to set a watermark horizontally or vertically in a chart area. 

    
this.chartControl1.ChartArea.Watermark.HorizontalAlignment=ChartAlignment.Near;
this.chartControl1.ChartArea.Watermark.VerticalAlignment=ChartAlignment.Near;

  • The ZOrder property can be set to display a watermark on top of a chart. 

    
this.chartControl1.ChartArea.Watermark.ZOrder=ChartWaterMarkOrder.Over;
this.chartControl1.ChartArea.Watermark.ZOrder=ChartWaterMarkOrder.Behind;
