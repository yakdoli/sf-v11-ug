Features

This sample demonstrates Essential Chart's support for customizing pie, doughnut, funnel, and pyramid charts when the divide area is enabled.

  • The GetSeriesBounds() method is used to get the bounds of the divided area when ChartArea.DivideArea is set to true. 

    
this.chartControl1.ChartArea.GetSeriesBounds(series);

  • The ShowSeriesTitle property is used to display the series name as a title for each chart in a divided area. 

    
ChartSeries.ConfigItems.PieItem.ShowSeriesTitle=true;
ChartSeries.ConfigItems.FunnelItem.ShowSeriesTitle=true;
ChartSeries.ConfigItems.PyramidItem.ShowSeriesTitle=true;

  • The PieWithSameRadius property is used to draw the pie series with the same radius when the divide area is enabled. 

    
ChartSeries.ConfigItems.PieItem.PieWithSameRadius=true;

Interactive Features

  • The pie chart control is rendered with the same radius when  the "Pie with Same Radius" check box is selected.
