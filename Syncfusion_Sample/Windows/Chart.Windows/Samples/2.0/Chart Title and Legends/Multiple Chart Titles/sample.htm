Multiple Chart Titles

This sample illustrates chart titles in Essential Chart.

Chart Titles

Features:

  • Chart Title

    Chart titles are created and added to a chart as seen below.

    
ChartTitle title = new Syncfusion.Windows.Forms.Chart.ChartTitle();
title.Text = "Essential Chart - Multiple Title";
title.Position = ChartDock.Top;
title.ShowBorder = true;
this.chartControl1.Titles.Add(title);

    Multiple titles can be added to a chart. Each of the chart titles can be docked at the top, bottom, left, right, or they can be floated.

    Chart titles can be visible or hidden by using the Visible property

    
this.chartControl1.Titles[0].Visible = true;
  • Font Settings

    The Font and ForeColor properties are used to customize a chart title's font.

    
this.chartControl1.Titles[0].Font = new Font("Courier New", 12.0f, FontStyle.Bold);
this.chartControl1.Titles[0].ForeColor = Color.Black;
  • Border settings

    The border of a chart title can be displayed or hidden using the ShowBorder property. 

    
this.chartControl1.Titles[0].ShowBorder = true;
Sets the style of the border line
this.chartControl1.Titles[0].Border.DashStyle = DashStyle.Solid;
Sets the width of the border line
this.chartControl1.Titles[0].Border.Width = 1;
Sets the forecolor of the border line
this.chartControl1.Titles[0].Border.ForeColor = Color.Black;
