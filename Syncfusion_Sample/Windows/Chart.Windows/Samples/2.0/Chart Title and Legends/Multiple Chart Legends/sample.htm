Multiple Chart Legends

This sample demonstrates how multiple legends are used in a chart control.  It also shows how a specific series can be associated with specific legends.

Multiple Chart Legends

Features:

  • Creating a Chart Legend

    1. Creating a new chart legend

      You can create a new legend and initialize it by giving it text and a name; this is done using the Name and Text properties.

      
ChartLegend legend1 = new ChartLegend(chartControl1);
legend1.Name = "legend1";
legend1.Text = "Zen";

    2. Customizing a Legend

      Various properties are available that will customize the appearance of a legend. The code snippet below illustrates the property settings used to customize the legend in this sample. 

      
legend1.Position = ChartDock.Right;         
legend1.RepresentationType = ChartLegendRepresentationType.SeriesImage;
legend1.Font = new Font("Tahoma", 7.5f, FontStyle.Bold);
legend1.ForeColor = Color.FromArgb(23, 83, 120);
legend1.BackInterior = new BrushInfo(GradientStyle.Vertical, legendColor);
legend1.ShowBorder = true;
legend1.Border.ForeColor = Color.FromArgb(143, 192, 223);
legend1.Border.Width = 1;

  • Associating a Chart Legend

    Legends can be associated with a chart control and a desired series by using the following code. 

    
chartControl1.Legends.Add(legend1);
 series1.LegendName = "legend1";

  • Legend Placement

    A legend can be placed outside or within a chart area by using the LegendsPlacement property as follows. 
    
Specifies the legend to be placed outside the ChartArea
this.chartControl1.LegendsPlacement = ChartPlacement.Outside;
 this.chartControl1.DockingManager.LayoutMode = ChartLayoutMode.Wrap;
