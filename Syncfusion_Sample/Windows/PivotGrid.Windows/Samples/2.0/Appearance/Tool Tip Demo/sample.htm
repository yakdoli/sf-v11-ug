Tool Tip Demo

This Sample illustrates how to show to ToolTip in PivotGrid Control.

Features

  • Support to enable and disable ToolTips for the PivotGrid control.
  • Support to enable and disable ToolTips for individual cell styles (e.g. column header, row header, value cell, summary cell, or summary header)

Interactive Features:

  • ToolTipEnabled is used to enable or disable ToolTips for the PivotGrid control and for individual cell styles.
