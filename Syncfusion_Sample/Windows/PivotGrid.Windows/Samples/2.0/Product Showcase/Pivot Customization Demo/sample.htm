Pivot Customization Demo

This sample illustrates the use of different properties to customize the PivotGrid Control.

Features:

  • PivotTableFieldList

  • GroupBar

  • SubTotals

  • GrandTotal

Features:

PivotTableFieldList - can be support in PivotGrid samples so that PivotGrid can be presented like an ExcelPivotTable

It enables drag and drop feature of fields between different areas like Column, row, value and filter. By using the PivotTable Field List, you can add, rearrange, or remove fields to show data in a PivotGrid exactly the way that you want.  [Default Value : False]

Below is the code to enable TableFieldList in PivotGridControl.

                this.pivotGridControl1.ShowPivotTableFieldList = true;

GroupBar - enables the drag and drop feature of fields between different areas such as column, row, value and filter.  [Default Value : False]

Below is the code to enable GroupBar in PivotGridControl.

                this.pivotGridControl1.ShowGroupBar = true;

Sub Totals - In the case of larger data table, helps the user to have an abstract view of the data by hiding subtotals using the ShowSubTotals property

Below is the code to show SubTotals in PivotGridControl. You can hide the SubTotals by setting its value as False [Default Value : True]

                this.pivotGridControl1.ShowSubTotals = true;

Grand Totals - can be support in PivotGrid samples so that PivotGrid can be presented like an ExcelPivotTable. 

Below is the code to show GrandTotals in PivotGridControl. You can hide the GrandTotal by setting its value as False [Default Value : True]

                this.pivotGridControl1.ShowGrandTotals = true;
