Calculate Windows Samples

Calculate samples are shipped with this Syncfusion product. These samples will guide you through important features, functionalities, and the process of implementation.

The samples included in this section are listed below:

  • Array ICalcData Demo-Adds calculation support to a double array

  • Data Grid Calculator Demo-Demonstrates the implementation of the ICalcData on a data-grid object

  • Working with CalcQuick Demo-Illustrates several tasks associated with using CalcQuick objects

  • Grid Data Bound Grid Calculator Demo-Demonstrates the implementation of ICalcData on a Grid Data Bound Grid object

  • Tabbed Panel Grids Demo-Illustrates several aspects of working with a CalcWorkbook

  • XLS File Using XlsIO Demo-Illustrates the class XlsIOCalcWorkbook that facilitates Essential Calculate working with XLS files
