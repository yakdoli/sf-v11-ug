Syncfusion Essential Edit
Essential Edit is a syntax-highlighting edit control that is designed to be extensible and easy to use. It supports automatic outlining, multilevel undo-redo, rich formatting options, and it is easy to configure. It comes with predefined, XML-based language configurations for coloring the syntax of commonly used languages, and it also supports the creation of custom configurations for other languages. Configuration files can be used to specify outlining blocks and to highlight the syntax of embedded scripts. Essential Edit has rich-text editing features in addition to code highlighting abilities, making it a highly efficient text editor.
