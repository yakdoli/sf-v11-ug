Printing

This sample demonstrates the printing support in the edit control. It also demonstrates the use of print dialog, print preview, and the headers-footers features.

The edit control provides complete support for printing its contents. You can either print the entire document, just the current page, specific pages, or certain selected text only. The printing implementation is very similar to the ones available in standard applications such as Microsoft Office or Visual Studio.NET.

Features:

  • We have a default printer dialog that provides options to customize the printer settings, number of copies, the pages to be printed, etc.

  • The edit control also includes a "print preview" feature that lets you view the document before printing. Moreover, customizable headers and footers are also supported in Essential Edit.
