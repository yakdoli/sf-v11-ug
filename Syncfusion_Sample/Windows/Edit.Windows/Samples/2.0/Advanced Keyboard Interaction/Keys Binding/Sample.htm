Keys Binding

This sample demonstrates the usage of the Keys Binding Dialog to bind a keystroke combination to any standard or custom command. It also illustrates how you can programmatically bind any method to any custom keys combination.

Features

  • Essential Edit supports extensive interaction with the keyboard through the use of shortcut keys to perform certain predefined or custom-bound actions.
  • The insert state of Essential Edit can also be toggled using the Insert key. When InsertMode is set to true, the characters typed are inserted into Essential Edit without overwriting the existing text.
  • Essential Edit has predefined shortcut key combinations that when pressed, will perform certain standard tasks.
  • Essential Edit offers support for the action-keystroke binding functionality, providing the user with the ability to perform advanced customization of action-keystroke bindings to suit his or her preference.
  • Using the Keys Binding dialog, any desired keystroke combination can be bound to a standard (or custom) command like Copy, Cut, Paste, or Find.
  • It is also possible to specify user-defined commands and bind custom keystroke combinations to them through the use of RegisteringKeyCommands and RegisteringDefaultKeyBindings events.

Interactive Features

  • Any desired keystroke combination can be bound to standard or custom commands by clicking the LaunchKeyBinding menu item and opening the Keys Binding dialog and editing it.
