Custom Code Editor

This sample illustrates the implementation of a Custom Code Editor using Syncfusion Menus and the edit control. This sample also illustrates the various features of edit control.

Essential Edit offers fully configurable Syntax Highlighting for general text editing purposes. Syntax highlighting greatly improves the readability of any text, and has found its way into many commercial applications. Syntax highlighting is accomplished in Essential Edit through the use of XML-based configuration files.

Configuration files control how Essential Edit interprets and displays text. The configuration files also support language-based syntax coloring of embedded scripts. If requirements are more complex, our syntax configuration model will easily allow customization of the configuration file to meet any requirements.

We have predefined configuration files for languages like C#, VB.NET, SQL, HTML, XML, Java, JScript, VBScript, Delphi, and plain text. We plan to create more configuration files for other languages in the near future. All the predefined configuration settings are present in the Essential Edit.Configurator.KnownLanguages collection. It is possible to set Essential Edit to use any of the predefined configuration settings by using the ApplyConfiguration method.

Essential Edit also supports a Configuration Editor for customizing configuration settings at run time.

Features :

  • Essential Edit offers advanced text indentation support, which can be customized to suit the requirements of the user.
  • Essential Edit has one of the most powerful and intelligent bracket highlighting and indentation guidelines features.
  • Essential Edit supports the User Margin feature, which can be used to display additional information regarding its contents.
  • Essential Edit provides a view-splitting functionality, similar to that of the code editor of Visual Studio.NET.
  • Essential Edit has the ability to indicate white spaces in its content.
  • WordWrap allows users to view the entire content of a line by wrapping text at the edge of the control (or the text area) into one or more lines that normally would have been outside the line view.
  • The selection margin is a thin vertical margin along the left side of Essential Edit that allows the selection of content from the entire line by simply clicking on the corresponding selection margin area of that particular line.
  • Essential Edit supports the customization of certain sections of text by setting Read-Only.

Interactive Features :

  • Different language configuration can be selected by clicking the respective language in the Language menu.
  • Property Grid can be displayed by clicking Properties menu.
  • We can customize the Text in the EditControl using various options available in the Advanced menu.
  • We can print the text in the edit control using various options available in the Printing menu.
  • We can display split views of the Edit control using Window menu
