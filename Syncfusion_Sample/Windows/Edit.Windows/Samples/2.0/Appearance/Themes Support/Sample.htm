Themes Support

This sample demonstrates the ability to have a Windows XP-like theme for Essential Edit. Features like the Scrollbars, Splitters, Control Borders when in FixedSingle Style, Outlining ToolTips, and IntelliSense Pop-ups using a context ToolTip, Context Choice, Context Prompt, appear themed when using XP style.

Features :

  • The themed appearance in Essential Edit is independent of the underlying operating system. It is not necessary to have Windows XP running to be able to see this feature. It can be seen on any of the earlier operating systems like Windows 2000.

Interactive Features :

  • We can toggle the XPThemes by clicking ToggleXPThemes menu item.
