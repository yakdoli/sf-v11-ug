Appearance

Essential Edit provides support for customizing its appearance through the use of a gradient background, WordWrap, user margin, and Windows XP themes. It also supports multiple views of the same file through the use of vertical and horizontal splitters.

  • Gradient Background - Illustrates the support for gradient colored backgrounds for Essential Edit to enhance its appearance
  • Single-Line Mode - Illustrates the single-line mode of operation in the Essential Edit. The single line mode can be turned on by setting the "SingleLineMode" property to true
  • Split Views - Illustrates how you can split a single document into multiple views using the vertical and horizontal splitters
  • Themes Support - Illustrates the ability to have a Windows XP-like theme for Essential Edit.
