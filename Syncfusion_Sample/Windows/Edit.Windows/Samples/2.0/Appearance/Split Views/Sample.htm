Split Views

This sample illustrates how you could split a single document into multiple views using the vertical and horizontal splitters. A maximum of four split views are supported in the edit control.

Features :

  • Essential Edit provides a view-splitting functionality that facilitate the splitting of a single document into several split views so that you can work with multiple areas of a document at the same time.
  • Essential Edit has built-in support for horizontal and vertical splitters.
  • A maximum of four split views are supported; however, it is possible to perform either a horizontal or vertical split only by optionally hiding a splitter, if you wish to have two views instead of four.

Interactive Features :

  • Clicking Split into Quandrants menu item in SplittersMenu menu will split the Edit control into four split views.
  • Clicking Horizontal halves or Vertical halves menu item will split the edit control into two split views in horizontal or vertical.
  • Clicking Show Horizontal Splitters or Show Vertical Splitters will display horizontal or vertical split views. 

    // To show or hide Horizontal Splitters.
this.editControl1.ShowHorizontalSplitters = true;

// To show or hide Vertical Splitters.
this.editControl1.ShowVerticalSplitters = true;

// To split into Vertical halves.
this.editControl1.SplitVertically();

// To split into Horizontal halves.
this.editControl1.SplitHorizontally();

// To split into Quadrants.
this.editControl1.SplitFourQuadrants();
