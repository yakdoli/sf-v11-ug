Single-Line Mode

This sample illustrates the single-line mode of operation in Essential Edit. The single-line mode can be turned on by setting the "SingleLineMode" property to true.

Features :

  • Essential Edit can be operated in a single-line mode much like a Windows Forms text box with its Multiline property set to false.
  • This enables you to have a simple text box-like control but with all the powerful features of Essential Edit, like syntax highlighting, selection, underlining, and so on.

Interactive Features :

  • We can toggle to single line and multiline mode by clicking ToggleSingleLineMode menu item. 

    editControl.SingleLineMode = true;
