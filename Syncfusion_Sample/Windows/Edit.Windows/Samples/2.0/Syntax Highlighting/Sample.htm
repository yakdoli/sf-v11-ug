Syntax Highlighting

Syntax highlighting and code coloring enhance the readability of code. Essential Edit fully supports this functionality.

Essential Edit supports highlighting of embedded scripts as well as plain text. XML based configuration files are used for syntax highlighting purposes and Essential Edit offers out-of-the-box support for syntax highlighting some of the commonly used languages. The configuration files are extensible and can be configured to highlight any desired language.

  • Custom Config File - Illustrates how you could write a custom configuration file for a LISP-like language and apply syntax coloring, automatic outlining and a rich set of other features.
  • Syntax Coloring - Illustrates the language-based Syntax Highlighting feature.
