Syntax Coloring

This sample demonstrates the language-based Syntax Highlighting feature of Essential Edit, including examples containing embedded scripts. It also lets you view the predefined configuration settings for the mentioned list of languages. Moreover, it illustrates the use of the Configuration Editor to dynamically customize syntax highlighting features at run time.

Essential Edit offers fully configurable syntax highlighting for general text editing purposes. Syntax highlighting greatly improves the readability of any text and has found its way into many commercial applications. Syntax highlighting is accomplished in Essential Edit through the use of XML-based configuration files. Configuration files control how Essential Edit interprets and displays text. The configuration files also support language-based syntax coloring of embedded scripts. If requirements are more complex, our syntax configuration model will easily allow customization of the configuration file to meet any requirements.

Features:

  • We have predefined configuration files for languages like SQL, Pascal, HTML, VB.NET, XML, C# and plan to create more configuration files for other languages in the near future. All the predefined configuration settings are present in the EssentialEdit.Configurator.KnownLanguages collection. It is possible to set Essential Edit to use any of the predefined configuration settings using the ApplyConfiguration method.

  • Essential Edit also supports a Configuration Editor for customizing configuration settings at run time.
