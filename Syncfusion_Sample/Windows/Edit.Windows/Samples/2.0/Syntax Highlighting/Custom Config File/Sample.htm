Custom Config File

This sample illustrates how you could write a custom configuration file for a LISP-like language and apply syntax coloring, automatic outlining, and a rich set of other features to the edit control.

Features:

  • Syntax highlighting is accomplished in Essential Edit through the use of XML-based configuration files. Configuration files control how Essential Edit interprets and displays text. The configuration files also support language-based syntax coloring of embedded scripts. If requirements are more complex, our syntax configuration model will easily allow customization of the configuration file to meet any requirements. 

    • <?xml version="1.0" encoding="utf-8" ?>
<ArrayOfConfigLanguage>
	<ConfigLanguage name="SQL" CaseInsensitive="true" Known="SQl" StartComment="--">
		<formats>
			<format name="Text" Font="Courier New, 10pt" FontColor="Black" />			
			<format name="Whitespace" Font="Courier New, 10pt" FontColor="Black" />
			<format name="KeyWord" Font="Courier New, 10pt" FontColor="Blue" />
			<format name="String" Font="Courier New, 10pt, style=Bold" FontColor="Red" />
			<format name="Number" Font="Courier New, 10pt, style=Bold" FontColor="Navy" />
			<format name="Operator" Font="Courier New, 10pt" FontColor="DarkCyan" />
		</formats>
		<extensions>
			<extension>sql</extension>
		</extensions>
		<lexems>
			<lexem BeginBlock="(" Type="Operator" />
			<lexem BeginBlock=")" Type="Operator" />
			<lexem BeginBlock=""" Type="Operator" />
			<lexem BeginBlock="|" Type="Operator" />
			<lexem BeginBlock="=" Type="Operator" />
			<lexem BeginBlock="." Type="Operator" />
			<lexem BeginBlock="," Type="Operator" />
			<lexem BeginBlock=";" Type="Operator" />
			<lexem BeginBlock="--" Type="Operator" />
			<lexem BeginBlock="boolean" Type="KeyWord" />
			<lexem BeginBlock="varchar2" Type="KeyWord" />	
			<lexem BeginBlock="from" Type="KeyWord" />
			<lexem BeginBlock="group" Type="KeyWord" />
			<lexem BeginBlock="create" Type="KeyWord" />
			<lexem BeginBlock="distinct" Type="KeyWord" />
			<lexem BeginBlock="delete" Type="KeyWord" />
			<lexem BeginBlock="tab" Type="KeyWord" />
			<lexem BeginBlock="select" Type="KeyWord" />
			<lexem BeginBlock="where" Type="KeyWord" />
			<lexem BeginBlock="table" Type="KeyWord" />
			<lexem BeginBlock="rows" Type="KeyWord" />
			<lexem BeginBlock=""" EndBlock=""" Type="String" IsEndRegex="true" IsComplex="true" OnlyLocalSublexems="true">
				<SubLexems>
				<lexem BeginBlock="\" EndBlock=".+" IsEndRegex="true" Type="String" />
				</SubLexems>
			</lexem>
		</lexems>
		<splits>			
			<split>/*</split>
			<split>*/</split>
			<split>--</split>
			<split>@@</split>
		</splits>
	</ConfigLanguage>
</ArrayOfConfigLanguage>

  • Essential Edit supports the creation of custom configuration files for highlighting code, and it is possible to plug-in an external configuration file to Essential Edit using the Configurator.Open method.
