Export Demo

This sample illustrates the ability of the edit control to export its text content and its associated syntax highlighting information into XML, RTF, or HTML format. It also illustrates the ability to provide XML, RTF, and HTML source code for the generated documents.

Features :

  • Essential Edit has the ability to export its contents and its associated syntax highlighting information into XML, RTF, or HTML formats.
  • This allows the user to share the text associated with Essential Edit along with its rich attributes such as syntax highlighting, line numbers, underlines, and many such useful features in universally accepted formats like XML, RTF, and HTML. Essential Edit has SaveAsXML, SaveAsRTF, and SaveAsHTML methods for this purpose.
  • Essential Edit is also capable of providing RTF, XML, and HTML source code for generating these documents using the GetTextAsRTF, GetTextAsXML, and GetTextAsHTML methods.

Interactive Features :

  • To export Edit's contents to various formats can be performed by clicking ExportMenu menu and selecting Save as RTF or Save as HTML or Save as XML menu items respectively.
  • To retrieve RTF, XML, and HTML source code for generating these documents can be done by clicking ExportMenu menu and selecting Get RTF Source or Get HTML Source or Get XML Source respectively.
