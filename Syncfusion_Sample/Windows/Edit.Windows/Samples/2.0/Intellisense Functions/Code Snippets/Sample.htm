Code Snippets

A code snippet is a fragment of code or text that you can store in a XML file and reuse at a later time. The edit control offers support for code snippets in Visual Studio 2005. This includes the ability to load and save VS.NET 2005-compatible XML snippet files. The highlighted field indicates to the user the data that is to be entered to complete the code snippet. Code snippets can be inserted into the edit control by typing the snippet name and pressing the Tab key. Code snippets can also be inserted using the Ctrl + ` combination.

Features :

  • Essential Edit supports code snippets feature where we can load and save VS.NET 2005-compatible XML snippet files.

Interactive Features :

  • Pressing Ctrl + ` displays a list of code snippets that can be inserted.
  • The up or down arrow keys allow selection of items in the list.
  • The Tab key inserts the selected snippet into the edit control.
  • While the caret is in a field, the Enter key can be used to navigate to the next field.
  • Code snippet editing is cancelled if the caret is moved outside the highlighted area.
