Advanced IntelliSense

This sample illustrates the support for the IntelliSense feature in Essential Edit. It also shows how you could add context-based configuration settings to Essential Edit's configuration file, dynamically in code. Another feature this sample demonstrates is the ability to bold the current parameter to be input to the Context Prompt pop-up.

The IntelliSense support in Essential Edit mainly consists of the Context Choice, the Context Prompt, and the Context ToolTip features.

Features:

  • The Context Choice support allows the creation of pop-ups for displaying a list of options used to complete whatever the user is typing. This feature is modeled on the "List Members Intellisense" feature in Visual Studio and is very convenient when editing programming languages. It is also possible to auto-complete a word by using up or down arrow keys and to choose the Context Choice item and pressing the Tab key. The Context Choice pop-up can be dismissed by pressing the Esc. key.


  • The Context Prompt feature allows the creation of pop-ups for displaying variations of syntax or overloads or method signatures. This feature is modeled on the "Parameter Info Intellisense" feature in Visual Studio. Each of the Context Prompt items can have a syntax specifier string and text message, providing additional information on each item. The user is able to scroll through the syntax variations either by using the up or down arrow keys or by clicking on the up or down buttons on the pop-up.


  • The Context ToolTip displays ToolTips with helpful information when the mouse is hovering over a word in Essential Edit. This feature is modeled on the "Quick Info Intellisense" feature in Visual Studio. Whenever the mouse hovers over a word, the Context ToolTip is updated and if some text information is provided, it is displayed in a ToolTip.
