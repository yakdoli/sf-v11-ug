IntelliSense Functions

The IntelliSense support in Essential Edit mainly consists of the Context Choice, the Context Prompt, and the Context ToolTip features. These features are analogous to the List Members, Parameter Info, and Quick Info IntelliSense features in Visual Studio.NET.

  • Advanced IntelliSense - Illustrates the support for above mentioned IntelliSense feature in Essential Edit
  • Code Snippets - Illustrates the support for code snippets in Visual Studio 2005
  • Context Choice and Prompt - Illustrates the support for the Context Choice and the Context Prompt features, which are a part of the IntelliSense support
  • Context ToolTip - Illustrates the support for the Context Tooltip in Essential Edit
  • IntelliSense from XML - Illustrates the edit control's support for populating the Context Choice list and the Context Prompt using a XML file
