Context ToolTip

This sample illustrates the support for a Context ToolTip in Essential Edit. It displays information about the format type of the word (or lexeme) on which the mouse cursor is currently hovering.

Features :

  • The Context ToolTip displays ToolTips with helpful information when the mouse is hovering over a word in Essential Edit.
  • This feature is modeled on the "Quick Info Intellisense" feature in Visual Studio.

Interactive Features :

  • Whenever the mouse hovers over a word, the Context Tooltip is updated, and if some text information is provided, it is displayed in a ToolTip.
