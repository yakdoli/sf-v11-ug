Break Point

This sample illustrates the support for break points in Essential Edit. The sample also has a run-time customization editor that lets you change the indicator image and specify any custom color to the line background.

Features :

  • Essential Edit provides support for adding bookmarks, customizing the bookmark's background, setting custom images as indicators, navigating between bookmarks and indicators, and clearing bookmarks and custom indicators.
  • Essential Edit allows setting custom background colors for individual lines as well as a selected block of text. Customized line background coloring can provide special visual effects for some applications.
  • The BreakPoint feature can be implemented by combining the line background feature with the custom indicator feature. This feature is modeled on the BreakPoint feature in Visual Studio.

Interactive Features :

  • We can customize the images of the book marks and set the background color by clicking Customize menu.
