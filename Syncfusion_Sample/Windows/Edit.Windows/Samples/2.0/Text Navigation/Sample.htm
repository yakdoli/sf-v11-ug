Text Navigation

Essential Edit has extensive support for navigation through its contents. Apart from standard navigation techniques using the keyboard, it also supports bookmarks navigation, bracket highlighting, indentation guidelines, and text drag-drop.

  • Break Point - Illustrates the support for break points in Essential Edit
  • Custom Bookmarks - Illustrates the support for adding, toggling, clearing, and navigating between default bookmarks and custom indicators
  • Indentation Guidelines - Illustrates the support for the matching-brace highlighting functionality and indentation guidelines, including the navigation support between matching braces in an indentation block
  • Positions and Offsets - Illustrates how to get or set the cursor position, features that demonstrate inter-conversion between virtual and physical locations, and the inter-conversion between the offset values and the virtual locations.
  • Text Navigation - Illustrates how the text can be navigated in terms of characters, words, and lines in Essential Edit.
