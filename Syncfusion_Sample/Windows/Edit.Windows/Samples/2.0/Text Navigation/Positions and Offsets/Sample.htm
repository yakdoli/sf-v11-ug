Positions and Offsets

This sample demonstrates the various features available in Essential Edit to get or set the cursor position, features that demonstrate inter-conversion between virtual and physical locations, and the inter-conversion between the offset values and the virtual locations.

Features:

Essential Edit has a wide array of features for handling text operations using Positions and Offsets.    

  • Properties like CurrentColumn, CurrentLine, and CurrentPosition can be used to set the position of the keyboard cursor.


  • Methods like PointToVirtualPosition, PointToPhysicalPosition, ConvertVirtualPositionToPhysical, ConvertVirtualPositionToOffset, and ConvertOffsetToVirtualPosition can be used for inter-conversion between virtual or actual positions and offsets.


  • The PhysicalLineCount property is also another useful API that returns the actual number of the lines in Essential Edit.
