Custom Bookmarks

This sample illustrates the support for adding, toggling, clearing, and navigating between default bookmarks and custom indicators in Essential Edit.

Features :

  • Essential Edit provides support for adding bookmarks, customizing the bookmark’s background, setting custom images as indicators, navigating between bookmarks and indicators, and clearing bookmarks and custom indicators.

Features :

  • Custom Bookmark images can be set by selecting appropriate image in the Custom Indicators GroupBox and clicking the Set Custom Indicator button.
  • Book marks can be navigated through using various buttons available in the BookMark Options GroupBox.
