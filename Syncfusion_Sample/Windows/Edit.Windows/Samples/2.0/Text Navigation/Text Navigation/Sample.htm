Text Navigation

This sample illustrates the text navigation features available in Essential Edit. Text can be navigated in terms of characters, words, and lines in Essential Edit.

Features:

Essential Edit offers ample support for text navigation, and it is possible to navigate at character, word, line, page, or entire document levels.

Here is a brief summary of the APIs available at each level.

  • Methods for character-level navigation - MoveUp, MoveDown, MoveLeft, and MoveRight.


  • Methods for word-level navigation - MoveLeftWord and MoveRightWord
    • .

  • Methods for line-level navigation - MoveToLineStart and MoveToLineEnd.


  • Methods for page-level navigation - PageUp and PageDown.


  • Methods for document-level navigation - MoveToBeginning and MoveToEnd.

Essential Edit offers support for text manipulation operations like insert, delete, append, and insertion of multiple lines through the use of APIs like AppendText, InsertText, DeleteChar, DeleteCharLeft, DeleteWord, DeleteWordLeft, and Lines.
