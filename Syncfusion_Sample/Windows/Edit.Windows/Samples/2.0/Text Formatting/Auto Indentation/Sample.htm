AutoIndentation

This sample illustrates the AutoIndentation support in Essential Edit. The text indentation style can be set to None, Block, or Smart, based upon the requirements of the application.

No indentation of text takes place on pressing the Enter key when the AutoIndentMode is set to None. When the AutoIndentMode is set to Smart, the next line is indented by one tab size from the beginning column of the previous line upon pressing the Enter key. Upon pressing the Enter key, when the AutoIndentMode is set to Block, the next line begins at the same column as the previous line .

Features :

  • The Edit control offers advanced text indentation support, which can be customized to suit the requirements of the user.
  • The AutoIndent property can be used to turn on AutoIndentation, and the AutoIndentMode property can be used to specify the desired mode.

Interactive Features :

  • The text indentation style can be selected by clicking IndentMode menu and selecting None or Block or Smart menu items accordingly.
