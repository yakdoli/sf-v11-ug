Custom Outlining

This sample demonstrates how the automatic outlining functionality in the edit control can be applied to any custom file (not necessarily programming language code) by having appropriate lexeme and split entries in the configuration file.

Features :

  • Essential Edit provides Visual Studio-like support for collapsing and expanding blocks of code through the use of Collapsers (plus-minus buttons).
  • Essential Edit also provides support for outlining ToolTips, as in Visual Studio, that display the content of the collapsed block of text.

Interactive Features :

  • The outlining ToolTip is built-in and appears automatically when the mouse pointer is placed over the collapsed block of text.
