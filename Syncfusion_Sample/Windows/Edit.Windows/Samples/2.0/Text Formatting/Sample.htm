Text Formatting

Essential Edit has text formatting support to layout its content. Common text formatting techniques like WordWrap, indentation, out-dentation, outlining, content dividers, and so on are available in Essential Edit.

  • Custom Outlining - Illustrates how the Custom Outlining functionality in the edit control can be applied to any custom file
  • Keyword Formatting - Illustrates how you can use the Configuration File settings to dynamically custom format the contents of the edit control
