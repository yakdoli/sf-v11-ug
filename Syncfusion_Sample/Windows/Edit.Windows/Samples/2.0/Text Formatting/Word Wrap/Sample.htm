Word Wrap

It can also be performed at the edge of the edit control text area or edge of the edit control. If a text area is being used, the appearance of the WordWrap Margin on the right can be customized.

Features :

  • WordWrap allows users to view the entire content of a line by wrapping text at the edge of the control (or the text area) into one or more lines that normally would have been outside the line view.
  • Essential Edit supports wrapping both in terms of words and characters. It also supports wrapping at the control border or the specified text area border.

Interactive Features :

  • Wrap Mode can be selected using Wrap Mode menu. Using this we can wrap the text at the control border or the specified text area border.
  • Wrap Type can be selected using Wrap Mode menu. Using this we can also wrap the text both in terms of words and characters.
