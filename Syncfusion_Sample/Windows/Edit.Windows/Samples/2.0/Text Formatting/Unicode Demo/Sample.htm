Unicode Demo

This sample illustrates how the edit control supports Unicode text handling, including clipboard operations and the ability to handle non-monospace fonts.

Features:

  • Essential Edit fully supports displaying and serializing Unicode characters.


  • All Unicode text is saved in UTF-8 format by default.


  • Moving Unicode text between the edit control and other word processing software programs is straightforward via copy-paste clipboard functions.


  • Essential Edit also supports the handling of all other text encoding formats specified in the System.Text.Encoding class, like ASCII, UTF7, UTF8 and BigEndianUnicode.
