Status Bar

The status bar is a component often found at the bottom of windows in a graphical user interface. It is very frequently divided into sections, each of which show different information. Its job is primarily to display information about the current state of its window, although some status bars have extra functionality. For example, many Web browsers have clickable sections that pop up a display of security or privacy information.

This sample illustrates the built-in Status Bar feature in the edit control. The status bar in the edit control can be optionally hidden or shown. It hosts a number of specialized status bar panels like Text Panel, Status Panel, Encoding Panel, Coords Panel, and Insert Panel, which provide specific information about various aspects of the edit control application. The text associated with each of these status bar panels can be customized and each of the them can be optionally hidden or shown. The status bar also hosts a sizing gripper, which can be optionally hidden or shown as well.

Features:

  • TextPanel - Used to display descriptive text information about the edit control application. It can also be used to indicate the location of the currently loaded file in the edit control

  • FileNamePanel - Used to display the name of the file loaded in the edit control

  • StatusPanel - Used to display information about the state of the edit control application. It can be used to display the read-only state of a file loaded into the edit control

  • EncodingPanel - Used to display information on the encoding used in the edit control

  • CoordsPanel - Used to indicate the current keyboard cursor position

  • InsertPanel - Used to indicate the state of the insert key

In addition to the above information, any custom text can also be displayed in the status bar panels. Moreover, the status bar also hosts a sizing gripper, which supports the following modes of operation: Smart, Visible, and Hidden.
