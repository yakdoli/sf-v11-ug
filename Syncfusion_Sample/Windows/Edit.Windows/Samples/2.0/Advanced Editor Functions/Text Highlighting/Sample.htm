Text Highlighting

This sample illustrates the support for line and selection backgrounds in the edit control. It is also possible to have borders for a highlighted background. Any desired color and hatch-style can be used for highlighting the background or borders.

Features

  • Essential Edit allows setting a custom background color for individual lines as well as a selected block of text.
  • Customized line background coloring can provide special visual effects for some applications.
  • These can be used to highlight important lines in a document or offending code.
  • In conjunction with the Custom Indicators feature, Essential Edit can also be used to implement the Visual Studio-like BreakPoint feature.

Interactive Features

  • Different hatch styles can be applied by selecting different hatch styles defined in the Hatch Style combo box.
  • When clicking the Highlight Line button, the current line will be highlighted using the colors selected in the Background and Border group boxes.
  • When clicking the Highlight Selection button, the selected text will be highlighted using the colors selected in the Background and Border group boxes.
