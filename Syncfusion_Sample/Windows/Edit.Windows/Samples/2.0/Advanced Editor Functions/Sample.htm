Advanced Editor Functions

Essential Edit can be used as a stand-alone text editing control capable of performing advanced functions like find-replace, line numbers, font customization, clipboard operations, text highlighting, and user margin operations. It also supports a highly customizable built-in context menu. All these features make Essential Edit a very useful control even if syntax highlighting is not required in the application.
