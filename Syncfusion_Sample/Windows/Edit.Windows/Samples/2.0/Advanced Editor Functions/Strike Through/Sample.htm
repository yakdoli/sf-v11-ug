Strike Through

This sample illustrates the strike-through feature of the edit control. It shows how it can be used to strike off various parts of text in the edit control using different color codes where each color code indicates a specific type of text. For example, red can be used to strike through missing text, green to strike through modified text, and blue to strike through newly added text.

Features

  • The StrikeThrough feature allows the user to perform a strikethrough operation on the text contained in Essential Edit.
  • This a useful feature in denoting text that was deleted from the original document or highlighting offending code.
  • It is also possible to specify any custom color for the strike-through line.

Interactive Features

  • When clicking the StrikeLine menu item, you can strike the current line.
  • Select text in the Edit control, and to strike the selected text, click the StrikeSelection menu item.
  • To strike the specified text range, click the StrikeTextRange menu item.

    // To strike the entire line.
this.editControl1.StrikeThrough(this.editControl1.CurrentLine, Color.IndianRed);

// To strike the selected text.
this.editControl1.StrikeThrough(this.editControl1.Selection.Top, this.editControl1.Selection.Bottom, Color.Navy);

// To predefine a text range to be striked out.
int startLineNumber = 12;
int endLineNumber = 17;

// Creating the associated Coordinate Points that indicate the text range.
CoordinatePoint startCoordinatePoint = this.editControl1.Parser.GetCoordinatePoint(startLineNumber,1);
CoordinatePoint endCoordinatePoint = this.editControl1.Parser.GetCoordinatePoint(endLineNumber,this.editControl1.GetLine(endLineNumber).LineLength);

// Strike out the text in the specified text range.
this.editControl1.StrikeThrough(startCoordinatePoint, endCoordinatePoint, Color.Red);
