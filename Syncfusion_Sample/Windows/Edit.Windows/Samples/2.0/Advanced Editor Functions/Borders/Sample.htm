Borders

This sample illustrates the text borders feature in the edit control. This is useful for highlighting specific sections of text in the edit control.

Features:

Essential Edit supports a text borders feature to highlight certain sections of content.   

Text borders can use one of the following styles.

  • Dash
  • Dot
  • Dot Dash
  • Solid
  • Wave Styles

You can also specify one of the following weights for the borders.

  • Thick
  • Double
  • Bold

It is possible to draw text borders for a specified region or for a selected region of text.
