Text Range Customization

This sample illustrates the creation of read-only regions in Essential Edit.

Features

  • Essential Edit supports the customization of certain sections of text. This can be achieved using methods like MarkAsReadOnly and RemoveReadOnly.
  • The text selected and marked as read-only cannot be edited in the Essential Edit. Any section of the text marked as read-only can be made editable again by applying the RemoveReadOnly method.
  • The read-only region can be highlighted using custom foreground and background colors.

Interactive Features

  • Select some text and click the MarkReadOnly menu item. The text will become read-only and the background and the text color will change.
  • Select the read-only text and click the RemoveReadOnly menu item. The read-only text will become editable.
