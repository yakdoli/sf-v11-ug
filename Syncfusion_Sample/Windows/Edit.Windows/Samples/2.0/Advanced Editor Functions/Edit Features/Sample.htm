Edit Features

Features:

  • Line Numbers
  • Custom Indicators
  • Outlining Collapsers
  • User Margins
  • WordWrap
  • Bookmarks
  • Splitters
  • White Spaces Indicators

Features :

  • Essential Edit offers support for the line numbers functionality to easily navigate content. The line numbers automatically get generated and assigned to the content of Essential Edit when the ShowLineNumbers property is set to true.
  • Edit unites all the anchors under the notion of bookmarks and custom indicators. It allows any number of custom images or bookmarks in a document, however, each line can have only one indicator or bookmark associated with it at a time.
  • It provides Visual Studio-like support for collapsing and expanding blocks of code through the use of Collapsers (plus-minus buttons). Sections of code that form the outlining blocks can be specified using configuration settings.
  • WordWrap allows users to view the entire content of a line by wrapping text at the edge of the control (or the text area) into one or more lines that normally would have been outside the line view.
  • It supports the User Margin feature, which can be used to display additional information regarding content. Moreover, information can also be displayed on a line-by-line basis.
  • It provides a view-splitting functionality which facilitate the splitting of a single document into several split views so you can work with multiple areas of a document at the same time.
  • Essential Edit has the ability to indicate white spaces within its content. The default indicators used are dots for spaces, right-arrows for tabs, and a mirrored P for the line feed. Essential Edit also supports custom indicators to highlight white spaces.
  • It also has a Property Grid associated with the edit control that lets you customize the control's properties at run time.

Interactive Features :

  • Using Line no menu item, we can display or hide line numbers.
  • Using Outlining Collapsers menu item, we can display or hide Edit control's collapsers.
  • Using WhiteSpaces menu item, we can hide or display whitespaces.
  • Using Indicator Margin menu item, we can display or hide Indicator margin.
  • Usind Scrollbars menu item, we can display or hide Edit control's Scrollbar.
