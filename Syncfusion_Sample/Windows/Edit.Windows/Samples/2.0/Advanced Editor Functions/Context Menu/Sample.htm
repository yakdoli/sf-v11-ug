Context Menu

This sample illustrates the built-in Context Menu support in the edit control. The context menu is highly customizable and allows the user to add custom menu items or delete existing ones. It can be displayed using the standard or Office 2003 look and feel.

Features:

  • A highly customizable Visual Studio.NET-like context menu is available in Essential Edit.
  • The default context menu has the standard Visual Studio.NET context menu-like look and feel with default menu items. The default menu items can be cleared, and any number of custom menu items can be added to the context menu in addition to the default menu items.
  • The context menu can optionally have an Office 2003 look and feel. Also, the Office 2003 theme is automatically used if the Essential Tools assemblies are referenced in the application.
