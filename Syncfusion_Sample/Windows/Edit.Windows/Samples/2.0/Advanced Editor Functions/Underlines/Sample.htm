Underlines

This sample illustrates the use of the Underlines feature in the edit control.

Features

  • Essential Edit supports Underlines and Wavelines that are mainly used to notify the user about errors or important sections of a document.
  • The underlines can be customized using different styles, colors, and weights.
  • The color of the underlines can be red, green, or blue.
  • The underlines support different styles like Solid, Dash, Dot, Dash Dot, and Wave.
  • It is also possible to specify the weight for the underlines to be single or double.

Interactive Features

  • When clicking the UnderlineSelection menu item, the selected text will be underlined using the style and color defined in the Style and Color group boxes.
  • When clicking the RemoveUnderline menu item, the underlines in the selected text will be removed.
  • When checking the Double checkbox, the underlines will have double lines.
