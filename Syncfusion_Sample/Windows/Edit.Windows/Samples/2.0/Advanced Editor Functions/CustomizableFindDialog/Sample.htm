Customizable Find Dialog

This sample illustrates the Customizable Find Dialog feature of the Edit Control. Now find dialog shows the message box when find operation reached the starting point of search as like in VS editor. It is also possible to customize the find dialog’s properties and methods.

Features :

  • Essential Edit allows setting custom message to find dialog when Find reached the starting point of search.
  • Custom message text will be in any language such as Arabic, Hebrew, German, etc.
  • Essential Edit allows localize the find dialog by any language easily.
  • Once find dialog finished the search in specified selection, Edit control shows that specified selection as like in VS Editor. This behavior did’ not support in previous versions Essential Edit.

Interactive Features :

  • When you click the Set button, own Find dialog which has been inherited from Edit control’s find dialog will be created.
  • After checked the Use Customized Find Dialog option and Click the Set button, press Ctrl+F to display the Customized Find dialog
