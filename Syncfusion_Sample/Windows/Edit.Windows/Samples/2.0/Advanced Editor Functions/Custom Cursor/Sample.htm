Custom Cursor

This sample illustrates the different cursors supported by the edit control. It is also possible to change the cursor dynamically at run time, as illustrated by the sample.

Features

  • Essential Edit supports all the cursors in the .NET framework cursors enumerator.
  • Any desired cursor can be set as the mouse cursor for Essential Edit using the Cursor property.

Interactive Features

  • Click the Cursors menu item and select the cursor type you wish. The Edit control's mouse cursor will be changed to the selected type.
