Clipboard Functions

This sample illustrates the support for basic editing and clipboard operations in Essential Edit like Undo, Redo, Copy, Cut, Paste, Select, and Cancel.

Features

  • Essential Edit supports cutting, copying, or pasting text data using the clipboard.
  • The Copy, Cut, and Paste methods in Essential Edit facilitate these clipboard operations.

Interactive Features

  • Select text in the Edit Control and click the Edit menu item, and then select the Cut or Copy menu item.
  • Click the Edit menu item and select the Paste menu item. The text will be pasted.
  • This operation can also be done in various ways using default keyboard shortcuts.
