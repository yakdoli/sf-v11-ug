Action Grouping

This sample demonstrates how you could group a set of actions as one entity, and then undo or redo these actions using one single Undo or Redo method call. Run this sample and open an action group using the ActionMenu. Perform a set of actions on the edit control, and close the action group. Now you can undo or redo these actions using the Undo-Redo menu items in the Action menu.

Features :

  • The default edit control in Windows Forms supports just one level of undo-redo operations, whereas Essential Edit supports multiple levels.
  • The Undo and Redo methods are used to accomplish undo and redo operations, respectively.
  • This can also be accomplished with default keyboard shortcuts like Ctrl-Z and Ctrl-Y key combinations.
    • .
  • It also supports action grouping which allows the users to specify a set of actions as groups for undo-redo purposes.
  • When an action group is created and a set of actions is added to it, the entire set is considered as one entity.
  • It has methods such as UndoGroupOpen, UndoGroupClose, and UndoGroupCancel, to programmatically manipulate the undo-redo action grouping.

Interactive Features :

  • Click Action Menu and select Open Group menu item.
  • Modify the text more than once in EditControl.
  • Click Action Menu and select Close Group menu item
  • Click Action Menu and select Undo menu item and the modified text will be removed.
  • Click Action Menu and select Redo menu item and the removed text will be replaced.

    // To open a group.
this.editControl1.UndoGroupOpen();

// To close a group.
this.editControl1.UndoGroupClose();

// To cancel a group.
this.editControl1.UndoGroupCancel();

// To save the undo or redo information.
this.editControl1.ResetUndoInfo();
