Selection Margin

This sample illustrates the use of the Selection Margin feature and the Changed Lines marking feature in the edit control. It also shows how the selection margin background and width can be customized.

Features

  • The Selection Margin is a thin vertical margin along the left side of Essential Edit that allows the selection of content for the entire line by simply clicking on the corresponding selection margin area of that particular line.
  • This is a very user friendly feature that simplifies the line selection process.
  • The selection margin can be enabled using the ShowSelectionMargin property in the edit control.
  • The lines that are changed in the edit control can be clearly identified by a difference in color in the selection margin using the Changed lines marking feature.
  • Also it is possible to differentiate the lines that are changed after saving a file.
  • This can be enabled using the SavedLinesMarkingLineColor property in the Edit control.

Intereactive Features

  • Using Background Color and Foreground Color menu items in the Selection Margin menu, you can customize the background color and the foreground color of the Selection Margin.
  • Using the Margin Width menu item in the Selection Margin menu, you can set the Selection Margin's width.
  • Using the Saved lines color and Changed lines color menu items in the Changed Line Marking Menu, you can customize the saved lines color and changed lines color.

// To change the selection margin background color.
this.editControl1.SelectionMarginBackgroundColor = Color.LightBlue;

// To change the selection margin foreground color.
this.editControl1.SelectionMarginForegroundColor = Color.Gray;

// To change the selection margin width.
this.editControl1.SelectionMarginWidth = 20;

// To change the color of changed lines in selection margin.
editControl1.ChangedLinesMarkingLineColor = Color.Yellow;

// To change the color of saved changed lines in selection margin.
editControl1.SavedLinesMarkingLineColor = Color.Green;
