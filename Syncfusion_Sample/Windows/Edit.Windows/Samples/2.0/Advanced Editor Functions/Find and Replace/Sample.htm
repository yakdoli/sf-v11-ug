Find and Replace

This sample demonstrates the use of various run-time dialogs in the edit control, like Find, Replace, and GoTo. As the names indicate, the Find dialog is used to search a specific text, the Replace dialog is used to search and replace a specific text, and the GoTo dialog is used to place the mouse cursor in the specified line. The Find and Replace dialogs have several search options like case matching, whole word matching, hidden text searching, searching above the current cursor position, and using regular expressions for pattern matching.

Features:

  • Essential Edit supports text search-and-replace functionalities both in code and through the use of advanced Find and Replace dialogs. These functionalities can also be accessed programmatically using methods like FindText and ReplaceText.

  • The FindText method finds the first occurrence of the specified text as per the conditions specified - like match case, match whole word, search hidden text, and search up.

  • The ReplaceText method replaces the first occurrence of the specified text with the replacement text as per the conditions specified - like match case, match whole word, search hidden text, and search up.

  • Essential Edit supports the GoTo functionality both through the use of  run-time dialogs and programmatic APIs.

    The GoTo method can be used to position the mouse cursor on any specified line. And it not only positions the cursor on the appropriate line, but it also scrolls the concerned line into view.
