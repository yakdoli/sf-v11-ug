User Margin

This sample illustrates the User Margin customization support in the edit control. It is possible to set custom text as well as a custom background for the user margin region.

Features

  • Essential Edit supports the User Margin feature, which can be used to display additional information regarding content.
  • The user margin is highly customizable and supports custom width, background, and foreground colors including the gradient coloring scheme.

Interactive Features

  • The margin can also be placed in the left or right side by selecting the Left or Right menu item in the UserMarginPlacement menu. Refer to the below illustration where the user margin is placed to the left.
