Right-to-Left Demo

This sample demonstrates the right-to-left (RTL) layout support that can be applied when using RTL languages. This includes line numbers, bookmarks, and print-preview features.

Features:

Essential Edit offers support for Line Numbers, Book Marks and Print Preview features in Right-To-Left.

Here is a brief summary supported features.

  • Line numbers, bookmarks, and selection margins available for RTL text.


  • Context menus, ToolTips, and dialogs that support RTL layouts.


  • Printing and print-previews for RTL layouts.


  • Line borders, underlines, and text-range customizations.


Essential Edit supports dynamic switching between layouts; press Left Ctrl + Left Shift and Right Ctrl + Right Shift to switch.

  • To enable right-to-left rendering, the RenderRightToLeft API of EditControl must be set to true, as follows:

[C#]
       this.editControl1.RenderRightToLeft = true;
[VB .NET]
       Me.editControl1.RenderRightToLeft = True
