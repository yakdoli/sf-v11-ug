Column Guides

This sample illustrates the ability of Essential Edit to support an unlimited number of Column Guides.

Features:

  • Column guides are desired by users to indicate certain columns that have special meaning. For instance, in certain languages (like in Cobol), no text should appear past column 80; hence, it will be useful for the editor if a line is displayed, which indicates the location of column 80.
  • Essential Edit supports an unlimited number of column guides. Each column guide can have a custom color and location.
  • Locations can be specified by a column (calculated based on the type and size of the font being used in Essential Edit).
