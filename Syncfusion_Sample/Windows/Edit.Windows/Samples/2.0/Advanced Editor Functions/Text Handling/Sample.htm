Text Handling

This sample illustrates the use of Text Handling features of the edit control, like the programmatic setting of text, inserting text at a given location, and appending text to the existing content of the edit control. It also demonstrates how to programmatically delete the selected text or delete the text in a specified text range.

Features

  • Essential Edit offers rich support for text manipulation operations like Insert, Delete, Append, and Insertion of Multiple Lines.
  • All these operations can be performed programmatically as well.

Interactive Features

  • When clicking SetText menu item, a dialog opens and the text entered into the dialog will be added to the Edit control, when clicking the Dialog's Accept button.
  • When clicking the AppendText menu item, a dialog opens and the text entered into the dialog will be appended to the Edit control, when clicking the Dialog's Accept button.
  • When clicking the InsertText menu item, a dialog opens and the text entered into the dialog will be inserted at the current cursor point, when clicking the Dialog's Accept button.
  • All the text entered into the Edit control will be deleted when clicking the DeleteAll menu item.
  • The selected text in the Edit control will be deleted when clicking the DeleteSelection menu item.

// To append text.
this.editControl1.AppendText(this.inputString);

// To insert text at a particular line and a column.
this.editControl1.InsertText(this.editControl1.CurrentLine, this.editControl1.CurrentColumn, this.inputString);

// To delete all the text.
this.editControl1.DeleteAll();

// To delete a selection.
this.editControl1.DeleteText(this.editControl1.Selection.Top, this.editControl1.Selection.Bottom);
