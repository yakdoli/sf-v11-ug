Scheduler Demo

This sample uses a schedule control on a form. It also has menu items that illustrate various properties and settings of the schedule control. It loads a sample "Appointments" file that holds appointments for April 2007.

The data provider is found in the SimpleScheduleDataProvider.cs or SimpleScheduleDataProvider.vb that is shipped as part of the sample. The classes in this file implement interfaces used by the schedule control to access the appointments data used by the control.

                           

The initial display showing a month view of appointments loaded as this sample runs.

 

Features:

  • The Navigation Calendar that appears by default to the left of a schedule control can be sized to show more months.

                                            

The navigation calendar has been sized to show four months

 

  • You can click on a single date in the navigation calendar to display the day view for that particular clicked date. You can also drag-select dates or control-click dates in the navigation calendar to display appointments for several days.

                           

 Here we see a day view for April 5th.

 

  • There is a context menu that will allow you to switch the types of views and also edit data in the schedule control.

                           

 Default Context Menu

 

  • An Appearance object allows you to control the look of the schedule control. This sample allows you to change, load, or save Appearance objects from a menu selection.

                        

Loading an Appearance object

 

 

  • Here is an image that demonstrates how an Appearance object has been loaded from disk and applied to the control.
