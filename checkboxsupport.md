##### Check Box Support {#check-box-support style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

The following properties can be used to set check box for the tree nodes and to handle the status of the check box.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

::: {align="center"}
+-----------------------------------+----------------------------------------------------------------------------+
|                                   |                                                                            |
|                                   |                                                                            |
| Item Property                     | Description                                                                |
+-----------------------------------+----------------------------------------------------------------------------+
| Checked                           | Gets / sets check box for the node. Default value is False.                |
+-----------------------------------+----------------------------------------------------------------------------+
| ShowCheckBox                      | Specifies whether to display check box for a node. Default value is False. |
+-----------------------------------+----------------------------------------------------------------------------+
:::

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

Check box can be provided for every single node or you can customize only for the required nodes. Enabling the **ShowCheckBox** property in the TreeView Designer dialog for an item, displays a check box for that node.

Setting the **Checked** property to **True**, by default applies a tick mark indicating the checked state of the check box.

These properties can be programmatically set as follows.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

+--------------------------------------------------------------------------------------------------------+
| **[\[C#\]]{style="FONT-FAMILY: 'Courier New'; FONT-SIZE: 9pt"}**                                       |
|                                                                                                        |
| []{style="FONT-FAMILY: 'Courier New'; COLOR: blue; FONT-SIZE: 9pt"}                                    |
|                                                                                                        |
| [Node.ShowCheckBox = [true]{style="COLOR: blue"};]{style="FONT-FAMILY: 'Courier New'; FONT-SIZE: 9pt"} |
|                                                                                                        |
| [Node.Checked = [true]{style="COLOR: blue"};]{style="FONT-FAMILY: 'Courier New'; FONT-SIZE: 9pt"}      |
+--------------------------------------------------------------------------------------------------------+
