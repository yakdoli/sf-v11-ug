#### Custom Labels and Images[]{style="FONT-SIZE: 10pt"} {#custom-labels-and-images style="tab-stops: 0pt"}

Using the CustomLabels property, you can add text labels to the Essential Gauge. And also we can add images (Gauge Image) in circular gauge to present gauge with more visual image elements. The below are some of the major properties using which we can customize the Essential Gauge.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

Table 13: Property Table

::: {align="center"}
  ------------- ----------------------------------------------------------------------------
  Property      Description
  LabelValue    Text value for the label. Default value is String.Empty.
  Location      Location of the label and image can be specified. Default value is Center.
  TextAngle     Customize the angle of the text rendering. Default value is set to 0.
  ImageWidth    Specifies the image width. Default value is set to 0.
  ImageHeight   Specifies the image height. Default value is set to 0.
  ImageSource   Lets you specify the image source for image element.
  ------------- ----------------------------------------------------------------------------
:::
