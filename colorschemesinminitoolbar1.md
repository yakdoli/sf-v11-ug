##### Color Schemes in MiniToolBar {#color-schemes-in-minitoolbar style="tab-stops: 0pt"}

[]{style="COLOR: #15428b"} 

MiniToolbar supports all the three ColorSchemes of Office2007, which can be set using the **ColorScheme** property.

[]{style="COLOR: #15428b"}
