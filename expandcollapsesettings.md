[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

Expand Settings

 

The content template of a snap element can be collapsed or expanded on clicking html or any element, for which the element\'s id must be set to the[ ]{style="FONT-FAMILY: 'Verdana','sans-serif'; FONT-SIZE: 8pt"}**ExpandCollapseElementIDs**.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

::: {align="center"}
  -------------------------- ---------------------------------------------------------------------------
  Property                   Description
  ExpandCollapseElementIDs   A comma-delimited list of object ids allowed to expand/collapse for Snap.
  -------------------------- ---------------------------------------------------------------------------
:::

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

The id of the element, which on clicking the content template must be collapsed or expanded, should be set to the ExpandCollapseElementIDs property.

Here, the id of the div element of the header template is set, such that the content template is collapsed or expanded, while clicking anywhere on the snap header.

Collapse Settings

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

A snap element\'s content template can be hidden by default, i.e., collapsed by setting the **IsCollapsed** property to **True**.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

::: {align="center"}
  ------------- ----------------------------------------------------------------
  Property      Description
  IsCollapsed   When set to true, the Snap control will collapse on page load.
  ------------- ----------------------------------------------------------------
:::
