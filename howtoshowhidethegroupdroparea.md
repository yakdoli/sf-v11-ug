## How to show / hide the GroupDropArea {#how-to-show-hide-the-groupdroparea style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

To allow grouping at runtime, the GridGroupingControl displays a drop panel (GroupDropArea) onto which users can drag-and-drop the column header. You can use the **ShowGroupDropArea** property to show / hide this GroupDropArea.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

+----------------------------------------------------------------------------------------------------------------------------------------------------------------------+

|                                                                                                                                                                      |
