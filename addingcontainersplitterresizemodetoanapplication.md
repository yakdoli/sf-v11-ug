##### Adding ContainerSplitterResize mode to an Application {#adding-containersplitterresize-mode-to-an-application style="tab-stops: 0pt"}

The user can resize only the elements that share common splitters using the **ContainerSplitterResize** property. To [resize only the elements that share common splitters]{style="COLOR: black"}, set this property to **SplitterResizeMode.EdgeChildren**. To resize all the elements inside the container, set this property to **SplitterResizeMode.AllChildren**. By default the **ContainerSplitterResize** property is set to **SplitterResizeMode.AllChildren**.
