#### Range Visibility {#range-visibility style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

The **RangeVisibility**[ ]{style="FONT-FAMILY: 'Courier New'; COLOR: red"}property specifies the visibility of the selected range. The default value is set to true. When set to false, the slider displays only one thumb and the property value is used to indicate the position of the thumb.
