### TEXTAREA - Text Area Tag {#textarea---text-area-tag style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

The **Text Area** tag is used to define a multiline text box. The HTMLUI control supports the following attributes with the **\<textarea\>** tag for receiving inputs from the user effectively.

[·      ]{style="FONT-FAMILY: Symbol"}**rows**: Specifies the number of rows for the text area

[·      ]{style="FONT-FAMILY: Symbol"}**cols**: Specifies the number of columns for the text area
