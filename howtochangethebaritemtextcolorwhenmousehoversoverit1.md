##### How to change the bar item text color, when mouse hovers over it {#how-to-change-the-bar-item-text-color-when-mouse-hovers-over-it style="tab-stops: 0pt"}

[]{style="COLOR: #15428b"} 

Bar item text color can be changed at run time, when the mouse hovers over it, with the help of [MenuColors]{style="COLOR: windowtext; TEXT-DECORATION: none; text-underline: none"} class.

[]{style="COLOR: #15428b"} 

+----------------------------------------------------------------------------------------------------------------------------------------------------+
| **[\[C#\]]{style="FONT-FAMILY: 'Courier New'; COLOR: black"}**                                                                                     |
|                                                                                                                                                    |
| []{style="FONT-FAMILY: 'Courier New'"}                                                                                                             |
|                                                                                                                                                    |
| [//Set the baItem\'s text Color]{style="FONT-FAMILY: 'Courier New'; COLOR: green"}                                                                 |
|                                                                                                                                                    |
| [Syncfusion.Windows.Forms.[MenuColors]{style="COLOR: teal"}.SelTextColor = [Color]{style="COLOR: teal"}.Blue;]{style="FONT-FAMILY: 'Courier New'"} |
+----------------------------------------------------------------------------------------------------------------------------------------------------+

[]{style="COLOR: #15428b"} 

+----------------------------------------------------------------------------------------------------------------------------------------------------+
| **[\[VB.NET\]]{style="FONT-FAMILY: 'Courier New'; COLOR: black"}**                                                                                 |
|                                                                                                                                                    |
| **[]{style="FONT-FAMILY: 'Courier New'; COLOR: black"}**                                                                                           |
|                                                                                                                                                    |
| [\'Set the baItem\'s text Color]{style="FONT-FAMILY: 'Courier New'; COLOR: green"}                                                                 |
|                                                                                                                                                    |
| [Syncfusion.Windows.Forms.[MenuColors]{style="COLOR: black"}.SelTextColor = [Color]{style="COLOR: teal"}.Blue]{style="FONT-FAMILY: 'Courier New'"} |
