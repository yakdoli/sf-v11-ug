### Chart Axis Range[]{#p82} {#chart-axis-range style="tab-stops: 0pt"}

[]{style="COLOR: #15428b"} 

Chart Axis Range is determined by using the IsAutoSetRange property of the axis. If this property is set to *true*, then the *start value* and the *end value* of the axis is automatically determined using the series value. Otherwise, the value given in the Range property will be the *start value* and the *end value* of the axis.
