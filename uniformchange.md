#### Uniform Change {#uniform-change style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

The **UniformChange** property specifies the distance to be moved by the thumb when Handle Buttons are clicked. The default value is set as 1.
