##### Server-Side Events {#server-side-events style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

A CallbackMultiplexer is non-visual web control that lets you refresh one or more callback controls within the same callback event.

 

The server side event can be invoked to refresh the list of CallbackPanels on client side.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

::: {align="center"}
+-----------------------------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| **[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"}**      | **[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"}**                      |
|                                                                                               |                                                                                                               |
| **[Event]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"}** | **[Description]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"}**           |
+-----------------------------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| Callback                                                                                      | Server side event that is handled with the client side args on calling callback method of client side object. |
+-----------------------------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
:::

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

Handle this event to change the contents of CallbackPanels according to the event args sent by the client.

 

For example, if you want to send custom callback event to the page, where you can either decide to refresh one or more callback panels. You can use **ControlsToRefresh** property of CallbackMultiplexer to add CallbackPanels to this list controls that should be refreshed.

 

The following code sample demonstrates how to refresh CallbackPanels.
