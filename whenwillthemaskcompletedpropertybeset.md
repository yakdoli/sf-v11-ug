#### When will the MaskCompleted property be set {#when-will-the-maskcompleted-property-be-set style="tab-stops: 0pt"}

 

MaskCompleted is set to true if all the required and optional inputs have been entered; otherwise, the false.MaskCompletedChanged event occurs when the MaskCompleted property is changed.
