First, create Customer class with some properties: Name, Company, and so on. In future these properties will serve as fields for GGC. Then create an instance of BindingList class (that implements IBindingList interface) and add some Customer instances to it. Assign the instance of BindingList class as data source for GGC in Page_Load. GGC will step through public properties of Customer class and these properties will be represented as fields in GGC.

You can use any predefined or custom class that implements IBindingList, ITypedList or IList instead of BindingList class. The concept will be the same.
