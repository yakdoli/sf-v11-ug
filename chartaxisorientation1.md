### Chart Axis Orientation {#chart-axis-orientation style="tab-stops: 0pt"}

[]{style="COLOR: #15428b"} 

Chart for Windows Phone enables to set the orientation of the Chart Axis. The default orientation of the ChartAxis is *Horizontal*. This property is mostly used in Multiple Axes scenarios.
