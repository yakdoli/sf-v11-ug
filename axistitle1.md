### Axis Title {#axis-title style="tab-stops: 0pt"}

Essential Chart provides properties to set the custom titles for the axes. Set the title text for an axis by using the Title property. Customize this text by using the TitleColor and TitleFont properties.

 

::: {align="center"}
  --------------------- ------------------------------------------------
  Chart Axis Property   Description
  TitleColor            Sets the color for the title text of the axis.
  TitleFont             Sets the font style for the title text.
  --------------------- ------------------------------------------------
:::

[]{style="FONT-FAMILY: 'Calibri','sans-serif'"}
