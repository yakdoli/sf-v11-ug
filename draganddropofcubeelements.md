## Drag and Drop of Cube Elements {#drag-and-drop-of-cube-elements style="tab-stops: 0pt"}

You can add/delete an element to/from the current report of the OLAP Client using the drag and drop option of Cube Elements feature. You can add an element to the current report by dragging the element from the cube dimension browser and dropping it into the appropriate Axis Element Builder. To delete an element from the current report just drag that element and drop it anywhere inside the Client except over the Axis Element Builders. Moving the element between Axis Element Builders can be done by dragging the element form one Axis Element Builder and dropping it on another axis.

 

Use Case Scenarios

The following tasks can be performed using the Drag and Drop feature:

Adding/Deleting an element to/from the current report.

Moving the elements between the Axis Element Builder.

Dragging and dropping makes interaction easier.
