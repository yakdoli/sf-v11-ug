#### Digital Value Support {#digital-value-support style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

Essential Linear gauge Windows Phone facilitates usage of a Digital gauge component within a Linear gauge component to display the character or numeric value of the pointed reading in the Linear gauge.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"}
