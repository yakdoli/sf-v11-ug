#### Animation Style {#animation-style style="tab-stops: 0pt"}

           

JumpList control provides predefined animation to show the group view. This enables you to enhance the appearance of the control.

           

The following are the predefined animation style for the JumpList control:

[·      ]{style="FONT-FAMILY: Symbol"}Default

[·      ]{style="FONT-FAMILY: Symbol"}Rotate

[·      ]{style="FONT-FAMILY: Symbol"}Fade

[·      ]{style="FONT-FAMILY: Symbol"}Zoom

 

 

Adding Animation Style

You can add animation style using the *AnimationStyle* property. Default value is *Fade*.
