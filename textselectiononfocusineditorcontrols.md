## Text Selection On Focus in Editor Controls {#text-selection-on-focus-in-editor-controls style="tab-stops: 0pt"}

The Editor controls can behave as standard text box when selected. With TextSelectionOnFocus property, the cursor can be placed at the position of the mouse pointer, when Editors are clicked initially.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

[·      ]{style="FONT-FAMILY: Symbol"}The Editor controls when clicked initially ensure selection of the entire text and on subsequent

[·      ]{style="FONT-FAMILY: Symbol"}Click the cursor would be placed at the position of mouse pointer.

[·      ]{style="FONT-FAMILY: Symbol"}When the property TextSelectionOnFocus is set to false, the entire text is not selected initially but the cursor would be placed at the position of mouse pointer.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"}
