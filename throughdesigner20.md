[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

To create a Snap control:

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

1.   Add a new ASP.NET page to your project.

2.   In design view, from the toolbox, drag a Snap control onto the design surface.

3.   In HTML view, insert the following code snippet between the \<cc1:Snap ID=\"Snap1\"\> and \</cc1:snap\> tags.
