#### Data Binding Support for WPF Ribbon {#data-binding-support-for-wpf-ribbon style="tab-stops: 0pt"}

The ribbon control supports data binding. The data template for the ribbon can be applied by using a property called Content Template in the Ribbon tab.

 

Two new properties have been added to the Ribbon Tab control.

[·      ]{style="FONT-FAMILY: Symbol"}**ContentTemplate** -- Is used to retrieve or set a data template for Ribbon tab content. It is a dependency property.

[·      ]{style="FONT-FAMILY: Symbol"}**ContentTemplateSelector** -- Is used to retrieve or set a template selector, which allows custom templates to be applied according to logic. It is a dependency property.
