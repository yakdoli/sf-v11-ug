##### Appointment Importance {#appointment-importance style="tab-stops: 0pt"}

You can set the importance of the appointment using **IsLowImportance** and **IsHighImportance** properties. By default, both are set to be false and that means normal importance. If you can set the **IsLowImportance** to true, it means that automatically **IsHighImportance** property is set to be false. Also, you can set the importance using the appointment window.

The following code shows the setting of  importance as High for an appointment.
