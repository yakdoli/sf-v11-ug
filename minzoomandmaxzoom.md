#### MinZoom and MaxZoom {#minzoom-and-maxzoom style="tab-stops: 0pt"}

 

MinZoom value is used to set the minimum of *ZoomFactor* value. Map cannot be be zoomed-out below the *MinZoom* value.

MaxZoom value is used to set the Maximum of ZoomFactor value. Map cannot be be zoomed-in above the *MaxZoom* value.
