### Find {#find style="tab-stops: 0pt"}

The **Find** finction finds  a portion of a string from a particular text and returns the location of the string.

 

**Syntax:**

**Find(lookfor, lookin, start)**

 

**where,**

[·      ]{style="FONT-FAMILY: Symbol"}**lookfor** is the text you want to search.

[·      ]{style="FONT-FAMILY: Symbol"}**lookin** is the the text in which you want to search.

[·      ]{style="FONT-FAMILY: Symbol"}**start** specifies the starting position of the text from where you want to start searching in the text. This  is optional.
