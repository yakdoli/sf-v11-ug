##### Close Button {#close-button style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

The TabStrip enables you to close tabs by making use of the **ShowCloseButton** property. You can set the Close image to the TabStrip Item using the **CloseImageURL** property. CloseImageURL Property is available for the following.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

[·      ]{style="FONT-FAMILY: Symbol"}StateDataPushed

[·      ]{style="FONT-FAMILY: Symbol"}StateDataHover

[·      ]{style="FONT-FAMILY: Symbol"}StateDataDefault

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

::: {align="center"}
  ----------------- -----------------------------------------------------------------------------
  Property          Description
  ShowCloseButton   Specifies to show the close button in the tab item.
  CloseImageURL     Specifies the corresponding close button image to be used for the tab item.
  ----------------- -----------------------------------------------------------------------------
:::
