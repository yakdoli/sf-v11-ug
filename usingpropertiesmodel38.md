##### Using Properties Model {#using-properties-model style="tab-stops: 0pt"}

1.   In the **Controller**, create an instance of the [MobAccordionModel]{style="COLOR: #2b91af"}, set the client side events and pass the instance through the **view specific data** to **View** as given below:
