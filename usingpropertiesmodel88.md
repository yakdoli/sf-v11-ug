##### Using Properties Model {#using-properties-model style="tab-stops: 0pt"}

The following steps explain how to set the dimesnsions for the scroll bar and target scrollable content:

1.   In the controller, create an instance of **MobScrollPanelModel**.

2.   Define the Height, Width and ScrollBarWidth properties and pass the instance through the view-specific data to the view.
