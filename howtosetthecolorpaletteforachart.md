## How to set the Color Palette for a Chart? {#how-to-set-the-color-palette-for-a-chart style="tab-stops: 0pt"}

[   ]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; FONT-SIZE: 9pt"}
