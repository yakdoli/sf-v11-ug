## Paging {#paging style="tab-stops: 0pt"}

Paging is supported in OlapChart with the help of Pager control. Paging helps to improve the performance of control in displaying or loading large amount of data in OlapChart.

You can associate the Pager control with the OlapControlID. The Pager also has an easy skinning mechanism. You can associate the Css with some default class names.
