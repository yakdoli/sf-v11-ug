##### Client-Side Object Model {#client-side-object-model style="tab-stops: 0pt"}

 

The look-and-feel of the Calendar can be customized using the css properties to enrich the appearance settings which has been discussed below.

 

You need to create new css file and set CustomCSS property of Calendar the path of this css file. After that default css setting will not be applied to the Calendar control.

 

Calendar control is inherited from WebControl, so, you can use WebControl style properties for customization Calendar appearance (BackColor, ForeColor, BorderColor, BorderStyle, BorderWidth, Width, Height and Font properties).
