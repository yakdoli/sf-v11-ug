#### Adding Content to GroupBar Item {#adding-content-to-groupbar-item style="tab-stops: 0pt"}

 

You can add content to a GroupBar Item using a Panel or a GroupView control. Any content can be added to the GroupBar Item by adding any control inside the panel. To view the contents of a GroupBar Item in ListView mode, add the GroupView as content in the GroupBar Item.

Events to handle with GroupBar Item

 

The events of GroupBarItem are **GroupBarItemAdded** and **GroupBarItemRemoved**.

 

[·      ]{style="FONT-FAMILY: Symbol"}**GroupBarItemAdded** -- which is called when GroupBarItem is added. GroupBarItem can be added either by using procedural code or by using context menu to add new item.

[·      ]{style="FONT-FAMILY: Symbol"}**GroupBarItemRemoved** -- which is called when GroupBarItem is removed. GroupBarItem can be removed by using procedural code or by using context menu to remove an existing item.
