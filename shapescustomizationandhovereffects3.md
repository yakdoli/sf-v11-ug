## Shapes Customization and Hover Effects {#shapes-customization-and-hover-effects style="tab-stops: 0pt"}

Shapes in the Map control can be customized using ShapeFill, ShapeStroke, ShapeStrokeThickness and HoverEffects.

 

ShapeFill, ShapeStroke, and ShapeStrokeThickness

ShapeFill property will fill the Shape with the respective color. ShapeStroke property will set the border color of the shape in the Map. And ShapeStrokeThickness property will set the thickness of the shape's border.

Hover Effects

Hover effects for the shape in the Map can be set using the following Properties:

[·      ]{style="FONT-FAMILY: Symbol"}ShapeHoverFill

[·      ]{style="FONT-FAMILY: Symbol"}ShapeHoverStroke

[·      ]{style="FONT-FAMILY: Symbol"}ShapeHoverStrokeThickness

[·      ]{style="FONT-FAMILY: Symbol"}EnableShapeHoverEffects

 

Table 8: Hover Effects API Description

::: {align="center"}
  ----------------------------------------- -----------------------------------------------------------------------------------------------------------------------------------------------------
  Application Programming Interface (API)   Description
  ShapeHoverFill                            This property will fill the shape with the given color during mouse over on the shape.
  ShapeHoverStroke                          This property will set the border color of the shape during  mouse over on the shape.
  ShapeHoverStrokeThickness                 This property will set the border width of the shape color during  mouse over on the shape
  EnableShapeHoverEffects                   This property will enable or disable the Hover effects of the shape. If value is true then Hover effects will be enabled. Else it will be disabled.
  ----------------------------------------- -----------------------------------------------------------------------------------------------------------------------------------------------------
:::
