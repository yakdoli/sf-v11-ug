## Creating a Chart {#creating-a-chart style="tab-stops: 0pt"}

[]{style="COLOR: #15428b"} 

Here are the step-by-step instructions for creating a chart in a Silverlight application. The procedure is elaborated under the following sections:

[]{style="COLOR: #15428b"} 

1.   Creating a Silverlight application.

2.   Deploying Essential Chart to the application.

3.   Intialization of basic chart and its child control.  

4.   Creating a Chart control.

[]{style="COLOR: #15428b"} 

Creating a Silverlight Application

[]{style="COLOR: #15428b"} 

1.   Open Microsoft Visual Studio. Go to File menu and click New Project.

[]{style="COLOR: #15428b"} 

![](ImagesExt/image59_13.jpg){border="0"}

Figure 6: Start Page

[]{style="COLOR: #15428b"} 

[]{style="COLOR: #15428b"} 

2.   In the New Project dialog, select Silverlight Application template, name the project and click OK.

[]{style="COLOR: #15428b"} 

![](ImagesExt/image59_14.jpg){border="0"}

Figure 7: Project Window

[]{style="COLOR: #15428b"} 

A new Silverlight application is created.

[]{style="COLOR: #15428b"} 

**[]{style="COLOR: #15428b"}** 

Deploying Essential Chart to the Application

[]{style="COLOR: #15428b"} 

1.   Go to Solution Explorer. Right-click References folder and click Add Reference.

[]{style="COLOR: #15428b"} 

![](ImagesExt/image59_15.jpg){border="0"}

Figure 8: Solution Explorer

[]{style="COLOR: #15428b"} 

2.   Add the following assemblies to the project References folder.

[]{style="COLOR: #15428b"} 

Syncfusion.Chart.Silverlight.dll

[]{style="COLOR: #15428b"} 

![](ImagesExt/image59_16.jpg){border="0"}

Figure 9: Add Reference

[]{style="COLOR: #15428b"} 

3.   Add Syncfusion.Chart.Silverlight reference in XAML or C# code as follows.
