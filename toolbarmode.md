#### Toolbar Mode {#toolbar-mode style="tab-stops: 0pt"}

This feature enables you to add caption, icon or both to the Toolbar item. You can set different mode for each item of a Toolbar.[]{style="COLOR: #c00000"}

 

Adding Toolbar Mode to an Application
