#### Splitter Controls {#splitter-controls style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

The SplitterBar control represent the borders which can be dragged in order to resize the SplitPanes. Each SplitterBar can have up to two buttons, used to expand / collapse the adjacent panes. Splitter control\'s Layout defines the orientation of its panes and split bars. Each Splitter can have either **Horizontal** or **Vertical** Layout. Splitter control lets you add splitters inside the SplitPanes also leading to nested Splitters.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"}

Sliding Pane

**[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"}** 

SlidingZone and SlidingPane provides the ability to create sliding panes in your Splitter control. The visual elements of the sliding panes and zones are as follows.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

[·      ]{style="FONT-FAMILY: Symbol"}**SlidingZone**: this is the left / right / top / bottom most area of the screen.

[·      ]{style="FONT-FAMILY: Symbol"}**SlidingPane**: special pane which by default will be collapsed, expanded or docked.
