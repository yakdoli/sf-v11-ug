#### Setting HTML Attributes to the Accordion {#setting-html-attributes-to-the-accordion style="tab-stops: 0pt"}

1.   In **View**, create the contents of the accordion with the header and div tags and invoke the accordion helper with the control ID as the first argument, and the HTML attributes as the second argument followed by the TargetContentId method with the id of the accordion contents as argument.
