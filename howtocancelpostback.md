##### How to Cancel Postback? {#how-to-cancel-postback style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

TreeView control has server side events such as NodeSelected, NodeRenamed, NodeExpanded, NodeCollapsed, NodeDropped, NodeCheckChanged events. TreeView has some properties (such as AutoPostOnNodeSelect, AutoPostBackOnNodeRename, AutoPostBackOnNodeExpand, AutoPostBackOnNodeCollapse, AutoPostBackOnNodeDrop, AutoPostBackOnNodeCheckChanged) that indicate whether to postback the page and fire server side event, after the node is changed.

 

For instance, for cancelling postback after the node is selected, AutoPostBackOnNodeSelect property must be set to False.
