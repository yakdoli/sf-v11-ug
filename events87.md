#### Events {#events style="tab-stops: 0pt"}

[]{#p155} 

The following events are associated with the CheckedListBox control.

 

::: {align="center"}
  --------------------- ------------------------------------------------------
  Event                 Description
  SelectionChanged      This event is raised when the selection is changed.
  CheckedItemsChanged   This event is raised when checked items are changed.
  --------------------- ------------------------------------------------------
:::
