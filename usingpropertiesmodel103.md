##### Using Properties model {#using-properties-model style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Calibri','sans-serif'; COLOR: black"} 

The following steps explain the setting of the dimensions of Tab using the Properties model:

1.   In the **Controller**, create an instance of **MobTabModel**, and pass the instance through **View Specific Data** to **View** as given below:**
