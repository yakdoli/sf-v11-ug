#### Defining the dimensions {#defining-the-dimensions style="tab-stops: 0pt"}

Generic drop-down supports customizing the dimensions.

**[]{style="FONT-FAMILY: 'Calibri','sans-serif'; FONT-SIZE: 12pt"}** 

**[Properties]{style="FONT-FAMILY: 'Calibri','sans-serif'; FONT-SIZE: 12pt"}**

+------------------+---------------------------------------------------------------------+--------------------------------------------------------------------------------------------------+-------------------------------------------+-------------+
| Name             | Description                                                         | Type of the property                                                                             | Value it accepts                          | Dependency  |
+------------------+---------------------------------------------------------------------+--------------------------------------------------------------------------------------------------+-------------------------------------------+-------------+
| Width            | Sets the width of the generic drop-down text box in pixels.         | [[struct]{style="COLOR: windowtext; TEXT-DECORATION: none; text-underline: none"}]{.UGHyperlink} | Members of [Unit]{style="COLOR: #2b91af"} | NA          |
+------------------+---------------------------------------------------------------------+--------------------------------------------------------------------------------------------------+-------------------------------------------+-------------+
| PopupPanelHeight | Sets the height of the pop-up panel holding the contents in pixels. | [[struct]{style="COLOR: windowtext; TEXT-DECORATION: none; text-underline: none"}]{.UGHyperlink} | Members of [Unit]{style="COLOR: #2b91af"} | NA          |
|                  |                                                                     |                                                                                                  |                                           |             |
|                  |                                                                     |                                                                                                  |                                           |             |
+------------------+---------------------------------------------------------------------+--------------------------------------------------------------------------------------------------+-------------------------------------------+-------------+
| PopupPanelWidth  | Sets the width of the pop-up panel holding the contents in pixels.  | [[struct]{style="COLOR: windowtext; TEXT-DECORATION: none; text-underline: none"}]{.UGHyperlink} | Members of [Unit]{style="COLOR: #2b91af"} | NA          |
