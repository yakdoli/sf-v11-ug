#### Adding Disable Grouping by Specific Fields to an Application {#adding-disable-grouping-by-specific-fields-to-an-application style="tab-stops: 0pt"}

The user needs to set the AllowRunTimeGroupByField property value to enable/disable grouping. The AllowRunTimeGroupByField property is added to PivotItem, PivotComputationInfo and FilterItemsCollection. The ShowDisabledGroupBackground property will allow the user to set a disabled look to the grouping disabled pivot items.
