##### Chart Watermark Support {#chart-watermark-support style="tab-stops: 0pt"}

Essential Chart supports watermarks which can be used to show text or images behind the chart area.

[·      ]{style="FONT-FAMILY: Symbol"}Watermarks can be rotated by specifying the rotation angle for the text or image watermark.

[·      ]{style="FONT-FAMILY: Symbol"}The opacity of a watermark can be set as a percentage.

[·      ]{style="FONT-FAMILY: Symbol"}Watermarks can be aligned horizontally or vertically with respect to the chart area.

 

**Use Case Scenarios**

[·      ]{style="FONT-FAMILY: Symbol"}Watermarking is used in copyright protection systems. They are intended to prevent or detect illegal copying of charts by unauthorized users.

[·      ]{style="FONT-FAMILY: Symbol"}Also, chart watermarks provide indisputable proof of the image origin or its author.

[·      ]{style="FONT-FAMILY: Symbol"}Owner identification like a username can be added to the chart so it is easy to find the owner.

[·      ]{style="FONT-FAMILY: Symbol"}A unique and secure digital signature can be added to each chart. This can be used to verify the origin and generation details of a chart.
