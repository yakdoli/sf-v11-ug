#### Doughnut Chart {#doughnut-chart style="tab-stops: 0pt"}

[]{style="COLOR: #15428b"} 

Doughnut chart is similar to the Pie chart. The only difference is that there is an hole at the center of the Doughnut chart. The Doughnut chart supports all Pie chart features. It can render only one series in the Chart Area. The value of the \"hole\" can be set by using the DoughnutCoefficient property of ChartDoughnutType class.

The following code example explains how to create a Doughnut Chart.
