#### Adding AjaxActionLink Button Control to a Mobile MVC Application {#adding-ajaxactionlink-button-control-to-a-mobile-mvc-application style="tab-stops: 0pt"}

In the [Getting Started]{.UGHyperlink} section, we discussed how to [create a Mobile MVC application]{.UGHyperlink} and [add the Tools package to an application]{.UGHyperlink}. This section guides you to add the AjaxActionLink Button control to an application.

1.   In the **view**, invoke the **AjaxActionLink Button** helper with the text of the button as the first argument.
