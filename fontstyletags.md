### Font Style Tags {#font-style-tags style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

The **Font Style** tags are used to format the appearance of the specified text when rendered in HTMLUI. The following are the font style tags supported in HTMLUI:

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

[·      ]{style="FONT-FAMILY: Symbol"}**\<b\>**: Bold tag renders the text in bold face

[·      ]{style="FONT-FAMILY: Symbol"}**\<i\>**: Italics tag renders italicized text

[·      ]{style="FONT-FAMILY: Symbol"}**\<u\>**: Underline tag renders underlined text

[·      ]{style="FONT-FAMILY: Symbol"}**\<em\>**: Emphasizing Text tag highlights important text in the document

[·      ]{style="FONT-FAMILY: Symbol"}**\<strong\>**: Strong tag renders specified text in bold face

[·      ]{style="FONT-FAMILY: Symbol"}**\<code\>**: Code tag renders specified text similar to computer coded text
