## Event Mechanism {#event-mechanism style="tab-stops: 0pt"}

This section describes the events of the Map control. When the events are triggered, parameters for  the events will be passed to the event handling function.

There tabulated events that exist in Map control:

 

::: {align="center"}
  ------------------ ------------------------------------------------ ----------------------------------------------------- --------------
  Event              Description                                      Arguments                                             Type
  PreviewZoomIn      Triggered before zoomin gthe Map                 Double Latitude,Double Longitude, Double ZoomFactor   Routed Event
  PreviewZoomOut     Triggered before zoomingout the Map              Double Latitude,Double Longitude, Double ZoomFactor   Routed Event
  ZoomedIn           Triggered after Map is zoomed in                 Double Latitude,Double Longitude, Double ZoomFactor   Routed Event
  ZoomedOut          Triggered after Map zoomed out                   Double Latitude,Double Longitude, Double ZoomFactor   Routed Event
  Panning            Triggered while Panning the Map                  Double Latitude,Double Longitude, PanMode panMode     Routed Event
  Panned             Triggered after Panned the Map                   Double Latitude,Double Longitude, PanMode panMode     Routed Event
  SelectionChanged   Triggered when Selection is changed in the Map   List RemovedItems, List AddedItems                    Routed Event
  ------------------ ------------------------------------------------ ----------------------------------------------------- --------------
:::
