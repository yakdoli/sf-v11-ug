### Chart Axis Header {#chart-axis-header style="tab-stops: 0pt"}

[]{#p88}[]{style="COLOR: #15428b"} 

Essential Chart for Windows Phone provides support for Chart Axis Titles with the help of the Header property. This property can hold the other elements. We can assign any other Windows Phone element to this Header property to display the data in the Chart area.
