##### Creating a PercentTextBox by using XAML {#creating-a-percenttextbox-by-using-xaml style="tab-stops: 0pt"}

The steps to create a PercentTextBox by using Visual Studio in XAML are as follows:[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"}

1.    Create a new application in Visual Studio.

2.    In the Visual Studio Toolbox, click the Syncfusion WPF Toolbox tab and select PercentTextBox.

3.    Drag and drop the PercentTextBox to Design View, to add the PercentTextBox to the application.

 

![](ImagesExt/image30_671.png){border="0"}

Figure 767: Control added to Design View[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b"}

 

4.   On the Properties window, customize the properties of the CurrencyTextBox.
