### Appointment Content {#appointment-content style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

Text can be set for an appointment using the **Subject** property.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

Text can also be set using the **Location** property, but it will be displayed inside \'()\' brackets.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

::: {align="center"}
  ---------------------- ----------------------------------------------------------------------
  Appointment Property   Description
  Subject                Specifies the text to be displayed in an appointment.
  Location               Specifies the text to be displayed in an appointment, inside \'()\'.
  ---------------------- ----------------------------------------------------------------------
:::
