#### Scrollable and Resizable {#scrollable-and-resizable style="LINE-HEIGHT: 12pt; TEXT-INDENT: -43.2pt; MARGIN: 10pt 0pt 0pt 43.2pt; tab-stops: 43.2pt"}

These properties allow you to set thebrowser default scroll bar and resize the splitter panes.

**Properties**

+-------------+----------------------------------------------------------------------------------------+-------------+--------------------------------------------------+--------------+
| Name        | Description                                                                            | Type        | Value Accepted                                   | Dependencies |
+-------------+----------------------------------------------------------------------------------------+-------------+--------------------------------------------------+--------------+
| Resizable   | [Specifies the splitter bar to be draggable.]{style="BACKGROUND: white; COLOR: black"} | Boolean     | True/False                                       | NA           |
|             |                                                                                        |             |                                                  |              |
|             |                                                                                        |             | Default value is true.[]{style="COLOR: #2b91af"} |              |
+-------------+----------------------------------------------------------------------------------------+-------------+--------------------------------------------------+--------------+
| Scrollable  | [Allows you to enable scrollable pane.]{style="BACKGROUND: white; COLOR: black"}       | Boolean     | True/False                                       | NA           |
|             |                                                                                        |             |                                                  |              |
|             |                                                                                        |             | Default value is true.                           |              |
+=============+========================================================================================+=============+==================================================+==============+

 

You can set the **Scrollable** and **Resizable** properties in both ASPX and Razor engine modes.
