#### AllowSplitterBar and CollapseMode {#allowsplitterbar-and-collapsemode style="LINE-HEIGHT: 12pt; TEXT-INDENT: -43.2pt; MARGIN: 10pt 0pt 0pt 43.2pt; tab-stops: 43.2pt"}

These properties allow you to enable the splitter bar and to set the collapse mode of the splitter bar.

[·      ]{style="FONT-FAMILY: Symbol; COLOR: black"}**[None]{style="COLOR: black"}**[: Will not display collapse button and the adjacent panes will not be collapsible]{style="COLOR: black"}

[·      ]{style="FONT-FAMILY: Symbol; COLOR: black"}**[Forward:]{style="COLOR: black"}**[ The splitter bar will display either left-to-right or top-to-bottom collapse button]{style="COLOR: black"}

[When the user clicks the button, the left (top) adjacent pane will increase its size with the width/height of the right (bottom) adjacent pane.]{style="COLOR: black"}

[·      ]{style="FONT-FAMILY: Symbol; COLOR: black"}**[Backward]{style="COLOR: black"}**[: The splitter bar will display either right-to-left or bottom-to-top collapse button]{style="COLOR: black"}

[When the user clicks the button, the right (bottom) adjacent pane will increase its size with the width/height of the left (top) adjacent pane.]{style="COLOR: black"}

[·      ]{style="FONT-FAMILY: Symbol; COLOR: black"}**[Both]{style="COLOR: black"}**[: Displays both buttons for forward and backward collapsing.]{style="COLOR: black"}

|                  |                                                                                                     |             |                                                                           |              |
|                  |                                                                                                     |             | [CollapseType]{style="COLOR: #4bacc6"}.Forward[]{style="COLOR: #4bacc6"}  |              |
|                  |                                                                                                     |             |                                                                           |              |
|                  |                                                                                                     |             | [CollapseType]{style="COLOR: #4bacc6"}.Backward[]{style="COLOR: #4bacc6"} |              |
|                  |                                                                                                     |             |                                                                           |              |
|                  |                                                                                                     |             | [CollapseType]{style="COLOR: #4bacc6"}.None                               |              |
+------------------+-----------------------------------------------------------------------------------------------------+-------------+---------------------------------------------------------------------------+--------------+
| AllowSplitterBar | [Enables the splitter bar.]{style="BACKGROUND: white; COLOR: black"}                                | Boolean     | True/False                                                                | NA           |
|                  |                                                                                                     |             |                                                                           |              |
|                  |                                                                                                     |             | Default value is true.                                                    |              |
