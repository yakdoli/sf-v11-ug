#### Add Custom Style to Ribbon Control

The Ribbon control template can be easily customizable using Expression Blend. Right-click the Ribbon control and select the option "Edit a Copy..." in Edit Template.

 

 

![Description: C:\\Users\\Jawahar\\Desktop\\sshot-2.png](../ImagesExt/image261_570.jpg){border="0"}

Figure 689: Edit Style Window
