##### ContextMenuOpened

ContextMenuOpened -- This event occurs after context menu opens.

ContextMenuEventArgs

ContextMenuEventArgs contains the following members:

Node -- TreeViewItemAdv represents the TreeViewItemAdv when right-clicked by the user.

Handled -- You can handle the Opening of ContextMenu using this property.
