#### Brush Selection

 

BrushSelector provides support to set the selected brush value. Brushes can be of two types:

 

[·      ]{style="FONT-FAMILY: Symbol"}Gradient - A gradient brush uses two colors. These colors merge to create a transition or fading effect.

[·      ]{style="FONT-FAMILY: Symbol"}Solid - This type of brush is equipped with only one color.
