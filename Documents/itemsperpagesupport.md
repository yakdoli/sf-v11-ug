#### ItemsPerPage Support

Using **ItemsPerPage** property, you can get or set the number of items to be displayed in the Carousel control.

 

Using ItemsPerPage in an Application

In the below sample, ItemsPerPage is set as 4. Hence while running the sample; only **4** items will be displayed in the path.
