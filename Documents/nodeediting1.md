##### []{#_NodeEditing}NodeEditing

The Event is triggered when TreeViewItemAdv is in Edit mode.

NodeCancellableEditEventArgs

 

[·      ]{style="FONT-FAMILY: Symbol"}You can cancel editing using the Cancel property of NodeCancellableEditEventArgs.

[·      ]{style="FONT-FAMILY: Symbol"}You can identify the node that is currently in edit mode using the Node property of NodeCancellableEditEventArgs.

[·      ]{style="FONT-FAMILY: Symbol"}Text property of NodeCancellableEditEventArgs represents the string the user will edit.
