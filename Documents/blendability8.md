#### Blendability

You can Customize ContextMenu in Blend. Add ContextMenuAdv to the design view without setting the ContextMenuAdvService attached property. Now you can see ContextMenuAdv in Object and Timelines Tap in the design view.

![](../ImagesExt/image261_795.png){border="0"}

[]{style="FONT-FAMILY: 'Calibri','sans-serif'; FONT-SIZE: 11pt"} 

Figure 893: Object and Timeline

![](../ImagesExt/image261_796.png){border="0"}

Figure 894: Design View

[]{style="FONT-FAMILY: 'Calibri','sans-serif'; FONT-SIZE: 11pt"} 

1.   Right-click the ContextMenuItemAdv and click the *Edit Template* option.

![](../ImagesExt/image261_797.png){border="0"}

Figure 895: Object and Timeline

2.   On the window, provide the Style name and click *OK*.

![](../ImagesExt/image261_798.png){border="0"}

Figure 896: Create Style Resource

 

 

Now you can customize each part in the ContextMenuItemAdv Template.
