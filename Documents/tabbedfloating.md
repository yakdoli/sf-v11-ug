##### [Tabbed Floating]{#_Ref261394066}

[]{style="FONT-FAMILY: 'Calibri','sans-serif'"} 

A Window can be tabbed inside another floating window. This can be achieved through XAML, c# or through mouse.

 

**SideInFloatMode** property describes the docking position of the element in a WindowContainer. If this is set to **Tabbed**, window will be tabbed.

 

**TargetNameInFloatMode** property must be set to specify where the window must be tabbed in floating mode.
