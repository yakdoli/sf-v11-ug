#### Inserting Images

 

In order to insert an image in the document, **ParagraphAdv** provides an inline called **ImageContainerAdv**.

 

Properties

Table 76: Property/Properties Table

::: {align="center"}
+--------------+---------------------------------------------------------------------+---------------------+--------------+-----------------+
| Property     | Description                                                         | Type                | Data Type    | Reference links |
+==============+=====================================================================+=====================+==============+=================+
| Height       | It specifies the height of the image.                               | Dependency Property | Double       | NA              |
+--------------+---------------------------------------------------------------------+---------------------+--------------+-----------------+
| Width        | It specifies the width of the image.                                | Dependency Property | Double       | NA              |
+--------------+---------------------------------------------------------------------+---------------------+--------------+-----------------+
| ImageSource  | Source of the image. It can be bytes or URL.                        | Dependency Property | ImageSource. | NA              |
+--------------+---------------------------------------------------------------------+---------------------+--------------+-----------------+
| ImageBytes   | Image source can be represented as byte array.                      | Dependency Property | Byte\[\]     | NA              |
+--------------+---------------------------------------------------------------------+---------------------+--------------+-----------------+
| FitToContent | It fits the image inside  the layout when it exceeds the size limit | Dependency Property | Boolean      | NA              |
|              |                                                                     |                     |              |                 |
|              |                                                                     |                     |              |                 |
+--------------+---------------------------------------------------------------------+---------------------+--------------+-----------------+

Limitations

Inserting an image has the following limitations:

3.   It does not have a separate context menu.

4.   It does not deal with Brightness, Reflection, Glow effects of image.
