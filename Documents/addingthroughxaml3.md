#### Adding through XAML

Following are the steps to add the PropertyGrid control by using Visual Studio in XAML.

 

5.   Create a new application in Visual Studio.

6.   In the Visual Studio Toolbox, click Syncfusion Silverlight Toolbox tab and select PropertyGrid.

Drag-and-drop the PropertyGrid to Design View, to add PropertyGrid to your application.

In the properties window, customize the properties of the PropertyGrid.
