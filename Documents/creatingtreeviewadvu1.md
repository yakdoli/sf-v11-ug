##### Creating TreeViewAdv using XAML

To create *TreeViewAdv* using VisualStudio in XAML:     

1.   Create a new Silverlight application in *Visual Studio*.

2.   In Visual Studio Toolbox, click syncfusion Silverlight Toolbox tab and select TreeViewAdv.

![](../ImagesExt/image261_642.png){border="0"}

Figure 733: Visual Studio Toolbox

 

3.   Drag and Drop the TreeViewAdv to the Design View, to add TreeViewAdv to the application.

 

 

![](../ImagesExt/image261_643.jpg){border="0"}

Figure 734: New Silverlight application in Visual Studio

 

4.   Now you can customize the properties of TreeViewAdv in the Properties Window.
