##### []{#_Setting_Size_Mode_5}Setting Size Mode

The Size Mode of the DropDownButtonAdv can be easily changed by using the property **SizeMode.** The **SizeMode** contains the following values:

[·      ]{style="FONT-FAMILY: Symbol"}Small

[·      ]{style="FONT-FAMILY: Symbol"}Normal

[·      ]{style="FONT-FAMILY: Symbol"}Large
