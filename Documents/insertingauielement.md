#### Inserting a UI Element

 

In order to insert a UI element in the document, **ParagraphAdv** provides an inline called **UIContainerAdv**.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

Properties

Table 77: Property/Properties Table

::: {align="center"}
  -------------- ---------------------------------------------------- --------------------- ----------- -----------------
  Property       Description                                          Type                  Data Type   Reference links
  Height         It specifies the height of the image.                Dependency Property   Double      NA
  Width          It specifies the width of the image.                 Dependency Property   Double      NA
  UIElement      It specifies the UI element to be added.             Dependency Property   UIElement   NA
  FitToContent   It fits the UIElement based on the size of a page.   Dependency Property   Boolean     NA
  -------------- ---------------------------------------------------- --------------------- ----------- -----------------
:::

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"}
