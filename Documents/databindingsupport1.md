#### Data Binding Support

Data Binding is the process of establishing a connection between the application UI and business logic. To bind the business object collection, **ItemSource** and **ItemTemplate** properties are used. You can bind the data object to any control and can show them in the TileViewItem.

 

Adding Data Binding to an Application

Data Binding can be added to an application by using either XAML or C# code.

 

The following code example illustrates how to add the Data Binding to an application through XAML and C#.
