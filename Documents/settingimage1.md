##### Setting Image

The Image displayed in the control can be set using two properties namely:

[·      ]{style="FONT-FAMILY: Symbol"}SmallIcon -- The property used to set Image, when SizeMode is Normal or Small.

[·      ]{style="FONT-FAMILY: Symbol"}LargeIcon -- the property used to set image, when SizeMode is Large.

The *SmallIcon* property can be set as follows.
