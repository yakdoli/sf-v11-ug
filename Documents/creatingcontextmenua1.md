##### Creating ContextMenuAdv using XAML

The steps to create ContextMenuAdv using Visual Studio in XAMLare:

1.   Open Visual Studio.

2.   On the File menu click New -\> Project. This opens the New Project Dialog box.

 

![](../ImagesExt/image261_787.png){border="0"}

Figure 883: New Project Dialog box

3.   In the Project Dialog window, select Silverlight application and in the name field type the name of the project and click OK.

![](../ImagesExt/image261_46.png){border="0"}

Figure 884: New Project

4.   Add the following reference with the sample project:

[·      ]{style="FONT-FAMILY: Symbol"}Syncfusion.Shared.Silverlight.dll

5.   Click and Open XAML file and add the following code to create ContextMenuAdv.

You can set ContextMenu to any control by setting the ContextMenuAdvService attached property. Here is the code to create ContextMenuAdv control using XAML.
