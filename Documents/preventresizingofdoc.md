#### []{#_Ref261334478}[]{#_Prevent_resizing_of}Prevent resizing of docking windows

[]{style="FONT-SIZE: 14pt"} 

Resizing of docking windows can be controlled in two ways:

[·      ]{style="FONT-FAMILY: Symbol"}Using CanResize property

[·      ]{style="FONT-FAMILY: Symbol"}Using FreezeLayout property

 

CanResize

By default, this property will be set to **True**. This allows the user to resize a window. If it is set to false, window cannot be resized. This property is used to control the resizing of a particular control.

FreezeLayout

When this property is set to false, we will not be able to resize any docked window. By default this will be set to **True**.
