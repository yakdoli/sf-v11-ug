##### []{#_Ref261309442}[]{#_Nested_Docking}Nested Docking

[]{style="FONT-FAMILY: 'Calibri','sans-serif'"} 

DockingManager framework provides full-fledged support for nested docking and floating. Nested Docking Layouts can be created using XAML or through C# code. This nested docking layouts can also be created using mouse while runtime.

 

**TargetNameInDockedMode** property is used to dock a window to the left, right, bottom or top of target window. **TargetNameInDockedMode** specifies the target window's WindowName.

 

**DesiredWidthInDockedMode** property sets the width of the window in docked mode.

 

**DesiredHeightInDockedMode** property sets the height of the window in docked mode.
