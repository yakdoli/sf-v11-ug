#### Checking Items by a Single Mouse-click

CheckedListBox control provides support to check items by a single mouse-click. The user can enable or disable this feature by using the CheckOnClick property. The default value for this property is set to false.

[]{style="LINE-HEIGHT: 150%"} 

Use Case Scenarios

This feature allows users to select an item (check the CheckBox / Radio Button) by clicking anywhere on the item.

 

Adding Checking Items to an Application by a Single Mouse-click

The following code example describes the addition of Checking Items (a single mouse-click feature) to the application using XAML.
