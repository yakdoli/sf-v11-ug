##### []{#_Template_customization_with}Template customization with Expression Blend

The Ribbon control template can be easily customizable using the Expression Blend. Right click the Ribbon control and select the option 'Edit a Copy..." in the Edit template.

 

 

![Description: C:\\Users\\Jawahar\\Desktop\\sshot-2.png](../ImagesExt/image261_570.jpg){border="0"}

Figure 652: Edit Style Window
