#### []{#_Culture_and_Number}Binding Support

Data binding is the process of establishing connection between application UI and business logic. Databinding can be unidirectional (source to target), or bidirectional (source to target and target to source). The user can bind the data to integer textbox through the Value property.

 

The following example shows a simple binding between the value of Integer Textbox and another Integer textbox value that reflects the typed value:

The DataContext property inherits its value to child elements. So you can set the DataContext on a superior layout container and its value is inherited to all child elements. This is very useful if you want to build a form that is bound to multiple properties of the same data object. If you don\'t explicitly define a source of a binding, it takes the data context by default.
