![](../ImagesExt/image261_59.png){border="0"}

Figure 55: Culture property set to Latin culture (bs-Latn)

In first sample culture set to "en-US" (US Culture), in the second one culture set to "bs-Latn" (Latin Culture). US culture uses "," as NumberGroupSeparator and Latin culture uses "." as NumberGroupSeparator. When you change the Culture property then the Value is formatted based on the Culture.
