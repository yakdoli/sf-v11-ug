#### []{#_Binding_Support_1}Binding Support

Data binding is the process of establishing connection between application UI and business logic. Data binding can be unidirectional (source to target), or bidirectional (source to target and target to source). You can bind the data to PercentTextBox through **Value** property.

 

The following example shows a simple binding between the PercentValue of PercentTextBox and another PercentTextBox PercentValue that reflects the typed value:

The Data Context property inherits its value to child elements. So you can set the Data Context on a superior layout container and its value is inherited to all child elements. This is very useful if you want to build a form that is bound to multiple properties of the same data object. If you don\'t explicitly define a source of a binding, it takes the data context by default.
