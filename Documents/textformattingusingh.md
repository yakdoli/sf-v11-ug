#### Text Formatting using HyperlinkAdv

 

**HyperlinkAdv** can be kept only inside the "ParagraphAdv" since the content property of ParagraphAdv is Inline. It allows you to display the formatted text using advanced features like NavigationUrl, and TargetType. It simply differs by these properties when we compare to "SpanAdv".

[]{style="COLOR: #15428b"} 

Properties

Table 75: Property/Properties Table

::: {align="center"}
  Property         Description                                                                    Type                   Data Type             Reference links
  ---------------- ------------------------------------------------------------------------------ ---------------------- --------------------- -----------------
  Text             It contains the text information.                                              Dependency Property    String                 
  Baseline         It indicates whether the inline text is subscript or superscript.              Dependency Property    Baseline              NA
  NavigateURL      Holds the URL to which the page navigates to.                                  Dependency Property    String                NA
  TargetType       It indicates whether to open the link in the same window or in a new window.   Dependency Property    HyperlinkTargetType   NA
  FontFamily       It specifies the FontFamily of the text.                                       Dependency Property    FontFamily            NA
  FontSize         It specifies the size of the font.                                             Dependency Property.   Double                NA
  Foreground       It designates the font color.                                                  Dependency Property    Color                 NA
  HighlightColor   It designates the highlight color of the text.                                 Dependency Property    Color                 NA
  Strikethrough    It shows whether the text has SingleStrikeThrough or DoubleStrikeThrough.      Dependency Property    StrikeThrough         NA
  FontWeight       It indicates the font weight of the text.                                      Dependency Property    FontWeight            NA
  Underline        It indicates whether the text should be underlined or not.                     Dependency Property    Underline             NA
:::
