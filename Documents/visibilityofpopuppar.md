##### Visibility of Popup Parts

[·      ]{style="FONT-FAMILY: Symbol"}Calendar Button visibility can be enabled by setting **IsCalendarEnabled** property to True.

[·      ]{style="FONT-FAMILY: Symbol"}None Button visibility can be enabled by setting **IsEmptyDateEnabled** property to True.

###### Working with DropDown Popup

By clicking None Button you can set Null Value to **DateTime** property of DateTimeEdit control. You can use **Calendar** for changing the Date and Time of the DateTimeEdit control.
