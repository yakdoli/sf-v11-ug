##### []{#_Expand_and_Collapse}Expand and Collapse Image support

ExpandImageSource and CollapseImageSource properties are used to set the expanded and collapsed image for the TreeViewItem that is used to differ the state of the treeviewitem. When expanding, the TreeViewItemAdv Expand image will display and when collapsing,the  TreeViewItemAdv Collapsed image will display.

ExpandImageHeight, CollapseImageHeight, ExpandImageWidth, ExpandImageHeight Properties are used to customize the height and width of the images.
