#### Dock Ability

 

Indicates where you can dock a window inside another window.  It is flagged enumeration. By default, DockAbility will be set as **DockAbility.All** so that all the drag indicators appears for all windows.  It contains the following values:

 

  ---------- ------------------------------------
  Position   Description
  Left       Left drag indicator will appear.
  Right      Right drag indicator will appear.
  Tabbed     Center drag indicator will appear.
  Top        Top drag indicator will appear.
  Bottom     Bottom drag indicator will appear.
  None       No drag indicator will appear.
  ---------- ------------------------------------

OuterDockAbility:

Controls the visibility of the outer drag providers. It is a flagged enumeration. By default, OuterDockAbility will be set as **OuterDockAbility.All** so that all the outer drag indicators appear.  It contains the following tabulated values:

  ---------- ------------------------------------
  Position   Description
  Left       Left drag indicator will appear.
  Right      Right drag indicator will appear.
  Top        Top drag indicator will appear.
  Bottom     Bottom drag indicator will appear.
  None       No drag indicator will appear.
  ---------- ------------------------------------
