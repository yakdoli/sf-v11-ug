#### Customize ContextMenuAdv using Blend

[]{style="FONT-FAMILY: 'Calibri','sans-serif'; FONT-SIZE: 11pt"} 

You can customize ContextMenuAdv in Blend to create ContextMenuItemAdv lsuch as color palettes.  Directly add ContextMenuAdv to design the view without setting the ContextMenuAdvService attached property. Now you can see ContextMenuAdv in Object and the Timelines tab in Design view.

![](../ImagesExt/image261_795.png){border="0"}

Figure 900: Object and Timeline

![](../ImagesExt/image261_796.png){border="0"}

Figure 901: Design View

[]{style="FONT-FAMILY: 'Calibri','sans-serif'; FONT-SIZE: 11pt"} 

1.   Right-click the ContextMenuItemAdv and click the Edit Template option.

![](../ImagesExt/image261_797.png){border="0"}

Figure 902: Object and Timeline

2.   On the window, give the Style name and click OK.

![](../ImagesExt/image261_798.png){border="0"}

Figure 903: Create Style Resource

 

 

3.   Now you can customize each part in the ContextMenuItemAdv Template.

4.   In the following example, the MouseOver effects for ContextMenuItemAdv are removed. Give a custom look to the ContextMenuItemAdv.
