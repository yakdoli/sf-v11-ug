##### []{#_Zero_Color}Zero Color

 

The Foreground of the IntegerTextBox can be customized based on the Value property. When 'Zero' is assigned as a value to Value property, then automatically the ZeroColor is set to Foreground property.

::: {style="BORDER-BOTTOM: windowtext 1pt solid; BORDER-LEFT: medium none; PADDING-BOTTOM: 1pt; MARGIN-TOP: 9pt; PADDING-LEFT: 0pt; PADDING-RIGHT: 0pt; MARGIN-BOTTOM: 9pt; BORDER-TOP: windowtext 1pt solid; BORDER-RIGHT: medium none; PADDING-TOP: 1pt"}
![](../ImagesExt/image261_3.jpg){border="0"}Note: The ZeroColor in integer textbox could be enabled by setting ApplyZeroColor property to true.
