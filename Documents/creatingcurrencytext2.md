##### []{#_Creating_CurrencyTextBox_using}Creating CurrencyTextBox using XAML

Steps to create CurrencyTextBox by using Visual Studio in XAML as follows:

1.   Create a new application in Visual Studio.

2.   In the Visual Studio Toolbox, click Syncfusion Silverlight Toolbox tab and select CurrencyTextBox.

3.   Drag and Drop the CurrencyTextBox to design view in order  to add CurrencyTextBox to your application.

 

![](../ImagesExt/image261_202.png){border="0"}

Figure 259: Control added to Design View

4.   In the properties window, customize the properties of the CurrencyTextBox.
