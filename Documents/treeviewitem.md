#### []{#_TreeViewItem}TreeViewItem

TreeViewItemAdv is a HeaderedItemsControl that contains multiple items and a header. To expand or collapse a TreeViewItemAdv, use the IsExpanded property.

SelectedItem (Object): Gets or sets the selected item in TreeViewAdv.

SelectedNode (TreeViewItemAdv): Gets or sets the Tree Node that is currently selected in TreeViewAdv.

SelectedItems (List\<Object\>): Gets or sets the Tree Nodes that are currently selected in TreeViewAdv.

SelectedNodes (List\<TreeViewItemAdv\>): Gets or sets the selected items in TreeViewAdv.
