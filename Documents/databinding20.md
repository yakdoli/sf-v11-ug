#### Data Binding

The CheckedListBox control provides a rich DataBinding support that allows users to fill the CheckedListBox items using Collections.

 

Use Case Scenarios

This Feature will be useful when you want to bind the data from the collections or from the database by setting the ItemsSource property of the CheckeddListBox as Collections or DataSet.Not applicable.

 

Adding Data Binding Feature to an Application
