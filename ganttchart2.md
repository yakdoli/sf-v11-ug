#### Gantt Chart {#gantt-chart style="tab-stops: 0pt"}

[]{#p29}[]{style="COLOR: #15428b"} 

A Gantt chart is a graphical representation of the duration of tasks against the progression of time. In a Gantt chart, each task takes up one row. The expected time for each task is represented by a horizontal bar, the left end of which marks the expected beginning of the task and right end the expected completion of the task. Tasks may run sequentially, in parallel or overlapping. The Gantt Chart requires minimum 2 Y point values.

 

The following code example illustrates how to create a Gantt Chart.
