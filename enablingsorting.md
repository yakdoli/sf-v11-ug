#### Enabling Sorting {#enabling-sorting style="tab-stops: 0pt"}

 

The tab control allows content to be sorted; this sorting can be enabled or disabled.

Property

 

  ---------- ---------------------------------------------------------- ---------------------- ---------------------- ----------------
  **Name**   **Description**                                            **Type of property**   **Value it accepts**   **Dependency**
  Sortable   When set, this property allows tab content to be sorted.   bool                   True/False             NA
  ---------- ---------------------------------------------------------- ---------------------- ---------------------- ----------------

*[[]{style="TEXT-DECORATION: none"}]{.underline}* 

Using Builder

 

The following steps explain how to enable sorting in tabs using Builder.**[]{style="FONT-SIZE: 12pt"}**

1.   In **View**, create the contents of the tabs with *ul* and *li* (for headers) and *div* tags (for content), and invoke the tab helper with the control ID as the first argument, followed by the **Sortable** method with the ID of the tab content as an argument.

Using Properties Model

 

The following steps explain how to enable sorting in tabs through the properties model.

1.   In the controller, create an instance of **TabModel**.**

2.   Define the **TargetControlId** property and pass the instance through the view-specific data to the view.**
