#### Setting Minimum Width for the Banner Image on the Exterior Wizard Page {#setting-minimum-width-for-the-banner-image-on-the-exterior-wizard-page style="TEXT-JUSTIFY: inter-ideograph; TEXT-ALIGN: justify; tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

You can set the minimum width of the banner image on the \'Exterior\' wizard page by using the **ExteriorPageBannerImageMinWidth** property.
