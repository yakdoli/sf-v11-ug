## Editors Package {#editors-package style="tab-stops: 0pt"}

 

The Editors package is a comprehensive collection of controls that are most commonly used in creating data entry forms. All the editor controls have been designed in order to provide a simple yet powerful means for creating a rich client interface and client applications.
