#### Resize Mode {#resize-mode style="tab-stops: 0pt"}

 

When the Gallery is resized, the size of the gallery items remains unchanged by default. You can enable the resizing ability for the gallery items also, by using the **AllowedItemResizeMode** property. This property supports the following modes for resizing the items.

 

[·      ]{style="FONT-FAMILY: Symbol"}Resize

[·      ]{style="FONT-FAMILY: Symbol"}Space

[·      ]{style="FONT-FAMILY: Symbol"}None

 

Use the following code to set item resize mode to Space.
