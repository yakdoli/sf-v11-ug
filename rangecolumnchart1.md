#### Range Column Chart {#range-column-chart style="tab-stops: 0pt"}

[]{#p35} 

The Range Column chart is rendered based on the start and end value of the range. The Range Column chart is similar to the Range Area chart. It has a minimum value and a maximum value.

The following code example illustrates how to create a Range Column Chart.
