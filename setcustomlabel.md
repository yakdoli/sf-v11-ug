#### Set Custom Label {#set-custom-label style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

In addition to the Numeric labels, Range Slider supports custom labels. This can be achieved using the **SetCustomLabel** property. The label and its corresponding position must be mentioned.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"}
