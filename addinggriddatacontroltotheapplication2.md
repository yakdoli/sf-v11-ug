## Adding GridData Control to the Application {#adding-griddata-control-to-the-application style="tab-stops: 0pt"}

[]{style="COLOR: #15428b"} 

Now that we have [[Created a WPF application]{.UGHyperlink}](ms-xhelp:///?Id=dbe471fc-cf73-44ca-8808-2e8e561fac94) and we have seen the basic steps to be followed before adding our Grid controls.

**[]{style="COLOR: #15428b"}** 

In this section, we will see how to add a GridData control to the WPF application and load data from the database.

Step-by-step Procedure to Create a GridData control and Load it with Sample Data

The following steps will add a GridData control and load it with Northwind Customers table.
