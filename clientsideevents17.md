##### Client-Side Events {#client-side-events style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

1.   In the HTML view of the project, define the various client functions to be invoked on event trigger. Here, for the **BeforePopup**, **OnPopup** and **TimeOut** events, the names are displayed. When **OnCloseUp** event is triggered, it displays a set of given links.
