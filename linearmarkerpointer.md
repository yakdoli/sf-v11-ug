##### Linear Marker Pointer {#linear-marker-pointer style="tab-stops: 0pt"}

Linear Marker Pointer is used to represent the values in the same manner like a bar pointer. But we can specify the placement of the marker pointer and marker style. The following are few important properties of Linear Marker Pointers.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

Table 20: Property Table

::: {align="center"}
+-----------------------------------+--------------------------------------------------------------------------------------------------+
| Property                          | Description                                                                                      |
+-----------------------------------+--------------------------------------------------------------------------------------------------+
| BackgroundBrush                   | To set the Background brush for the pointer.                                                     |
+-----------------------------------+--------------------------------------------------------------------------------------------------+
| BorderBrush                       | To set the Border brush for the pointer.                                                         |
+-----------------------------------+--------------------------------------------------------------------------------------------------+
| BorderWidth                       | To set the Border width of the pointer cap. Default value is 0.                                  |
+-----------------------------------+--------------------------------------------------------------------------------------------------+
| MarkerStyle                       | To specify the Marker style of the pointer. Default value is Rectangle.                          |
+-----------------------------------+--------------------------------------------------------------------------------------------------+
| PointerWidth                      | To set width of the pointer. Default value is 0.                                                 |
+-----------------------------------+--------------------------------------------------------------------------------------------------+
| PointerLength                     | To set the length of the pointer. Default value is 0.                                            |
+-----------------------------------+--------------------------------------------------------------------------------------------------+
| PointerPlacement                  | To set the placement where the pointer will show on the scale.                                   |
|                                   |                                                                                                  |
|                                   |                                                                                                  |
|                                   |                                                                                                  |
|                                   | *Cross*-draw on middle of the scale.                                                             |
|                                   |                                                                                                  |
|                                   | *Inside*-draw inside the scale.                                                                  |
|                                   |                                                                                                  |
|                                   | *Outside*-draw outside the scale.                                                                |
|                                   |                                                                                                  |
|                                   |                                                                                                  |
|                                   |                                                                                                  |
|                                   | Default value is Cross.                                                                          |
+-----------------------------------+--------------------------------------------------------------------------------------------------+
| Value                             | To set the value of the pointer. Default value is 0.                                             |
+-----------------------------------+--------------------------------------------------------------------------------------------------+
| MarkerCustomGeometry              | To specify a custom geometry for the marker pointer.                                             |
+-----------------------------------+--------------------------------------------------------------------------------------------------+
| EnablePointerInteraction          | Using this property, the Linear Pointer can be  dynamically changed using the keyboard or mouse. |
+-----------------------------------+--------------------------------------------------------------------------------------------------+
:::

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

Marker can be customized using the **MarkerStyle** property. Following are the in-built styles available for the MarkerStyle property.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

[·      ]{style="FONT-FAMILY: Symbol"}**Ellipse**---marker displays in elliptical form.

[·      ]{style="FONT-FAMILY: Symbol"}**Diamond---**marker displays in diamond form.

[·      ]{style="FONT-FAMILY: Symbol"}**Trapezoid**---marker displays in trapezoidal form.

[·      ]{style="FONT-FAMILY: Symbol"}**Pentagon**---marker displays in pentagonal form.

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"}
