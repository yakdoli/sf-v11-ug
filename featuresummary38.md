## Feature Summary {#feature-summary style="tab-stops: 0pt"}

Drag and Drop -- You can drag and drop the cube elements to create a report.

Load and Save Report -- You can save the current session in an XML file for future use and reload it whenever required.

Measure Editor Support -- You can rearrange, delete, include or exclude the measure in the current report.

Slice and Dice -- You can slice and dice (When you cut the data with one member it is called slicing. When you cut data with sets of members from two or more dimensions, it's called dicing.) The Cube Data Base, by filtering the members of dimensions using Member Editor.

Pivot Toggling -- You can easily toggle the pivot if required, by the click of a button.

Binding custom report -- You can load the report created using API in OLAP Client.

WPF report support -- You can load the report created in WPF OLAP Client.

Adding custom report -- You can add the list of custom reports to view all the reports.
