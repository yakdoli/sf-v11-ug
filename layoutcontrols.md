### Layout Controls {#layout-controls style="tab-stops: 0pt"}

 

Layout controls are used to design various layouts in web pages. They are used to create form layouts, row layouts, cell layouts or table layouts. Layout controls provide enhanced design-time support, autoformat support, various content alignment options, and much more.
