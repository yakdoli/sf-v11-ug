#### Radius {#radius style="tab-stops: 0pt"}

[]{style="FONT-FAMILY: 'Trebuchet MS','sans-serif'; COLOR: #15428b; FONT-SIZE: 9pt"} 

You can get or set the radius of the Magnifier frame when the FrameType is Circle. This is achieved by using the **FrameRadius** property.
