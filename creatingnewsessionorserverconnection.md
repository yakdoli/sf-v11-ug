### Creating New Session or Server Connection {#creating-new-session-or-server-connection style="tab-stops: 0pt"}

To create a new server connection:

1.   Click the Create New Session button on the OlapClient Toolbar as shown in the below screen shot:

 

![](ImagesExt/image50_73.png){border="0"}

Figure 40 OlapClientToolbar

 

8.   The Connection Option dialog opens:

 

![](ImagesExt/image50_74.png){border="0"}

Figure 41 Connection Option Dialog

 

9.   Do one of the following to provide the connection string:

[·      ]{style="FONT-FAMILY: Symbol"}Give the **Server Name** and the **Database Name**

[·      ]{style="FONT-FAMILY: Symbol"}Provide the connection string in the **Connection String** textbox

::: {style="BORDER-BOTTOM: windowtext 1pt solid; BORDER-LEFT: medium none; PADDING-BOTTOM: 1pt; MARGIN-TOP: 9pt; PADDING-LEFT: 0pt; PADDING-RIGHT: 0pt; MARGIN-BOTTOM: 9pt; BORDER-TOP: windowtext 1pt solid; BORDER-RIGHT: medium none; PADDING-TOP: 1pt"}
Note: If the server type is Non-Windows Authenticated (tightly coupled) then the 'Credentials' checkbox, available in the Server option has be checked  and the 'User ID and Password' details should be provided.
:::
